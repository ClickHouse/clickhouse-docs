:::tip SQL控制台
如果您需要 SQL 客户端连接，您的 ClickHouse Cloud 服务有一个关联的基于 Web 的 SQL 控制台； 展开下面的 **连接到 SQL 控制台** 以了解详细信息。
:::

<details><summary>连接到 SQL 控制台</summary>

从 ClickHouse Cloud 服务列表中，选择您将使用的服务，然后单击 **连接**。 从这里您可以**打开 SQL 控制台**：
