- [Managed ClickStack](/use-cases/observability/clickstack/getting-started/managed) - The **easiest and simplest** way to get started, deploying **Managed ClickStack** on **ClickHouse Cloud**.
- [Open-Source ClickStack](/use-cases/observability/clickstack/getting-started/oss) - For those of you who wish to run a fully open source architecture yourself.
