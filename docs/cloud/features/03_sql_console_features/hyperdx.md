Managed ClickStack is the managed [**ClickStack**](/use-cases/observability/clickstack) experience in ClickHouse Cloud. It unifies logs, traces, metrics, and session replay in one observability workflow built on ClickHouse and OpenTelemetry (OTel).

Managed ClickStack gives you a purpose-built interface for exploring and visualizing observability data with Lucene-style search and SQL, interactive dashboards, alerting, and trace exploration.

You launch Managed ClickStack with one click, connect it to your data, and use your existing ClickHouse Cloud authentication setup.

## Deployment {#main-concepts}

Managed ClickStack is generally available in ClickHouse Cloud. Open it from the left navigation menu when you select any service.

<Image img={hyperdx_cloud} alt="ClickHouse Cloud Managed ClickStack" size="lg"/>

To get started with Managed ClickStack in ClickHouse Cloud, use the dedicated [getting started guide](/use-cases/observability/clickstack/deployment/clickstack-clickhouse-cloud).

For further details on ClickStack, see the [full documentation](/use-cases/observability/clickstack).
