# Opik

## What is Opik? {#what-is-opik}

[Opik](https://www.comet.com/docs/opik/) is an open-source platform for LLM observability, evaluation, and prompt experimentation. Teams use Opik to trace agent and LLM workflows, evaluate quality, and monitor production behavior.

## Why Opik? {#why-opik}

- **Open source:** Self-hosted and cloud options
- **Tracing:** End-to-end traces for LLM and agent workflows
- **Evaluation:** Built-in evaluation workflows and score tracking
- **Prompt and experiment workflows:** Compare prompts and model behavior over time
- **Ecosystem support:** Integrates with OpenTelemetry-compatible instrumentation

## ClickHouse compatibility {#clickhouse-compatibility}

Opik is part of the ClickHouse ecosystem and supports high-volume observability workloads. For architecture and deployment details, see the official Opik documentation.
