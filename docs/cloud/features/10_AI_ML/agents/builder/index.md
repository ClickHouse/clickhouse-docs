The Agent Builder is where users create and configure agents. This page covers the core builder UI: name, description, instructions, and model selection. Subpages cover each capability that can be attached to an agent — tools, actions, MCP servers, skills, and subagents.

## Built-in tools

- [File search](/cloud/features/ai-ml/agents/builder/file-search) — RAG over uploaded documents.
- [Code interpreter](/cloud/features/ai-ml/agents/builder/code-interpreter) — Sandboxed code execution.
- [Web search](/cloud/features/ai-ml/agents/builder/web-search) — External web search.
- [Image generation](/cloud/features/ai-ml/agents/builder/image-generation) — Generate images from prompts.
- [Vision](/cloud/features/ai-ml/agents/builder/vision) — Accept image inputs.
- [Voice](/cloud/features/ai-ml/agents/builder/voice) — Speech-to-text and text-to-speech.

## Extensibility

- [Actions](/cloud/features/ai-ml/agents/builder/actions) — Call external APIs via OpenAPI specs.
- [MCP servers](/cloud/features/ai-ml/agents/builder/mcp-servers) — Attach third-party MCP servers to an agent.
- [Skills](/cloud/features/ai-ml/agents/builder/skills) — Reusable skill packages.
- [Subagents](/cloud/features/ai-ml/agents/builder/subagents) — Delegate work to child agents.
