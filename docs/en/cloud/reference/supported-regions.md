# Supported Cloud regions

## AWS regions

- ap-south-1
- ap-southeast-1
- eu-central-1
- eu-west-1
- us-east-1
- us-east-2
- us-west-2

:::note
Development services are currently not supported in ap-southeast-1 and ap-south-1 regions.
:::

## Google Cloud regions

- asia-southeast1
- europe-west4
- us-central1
