## Введение \{#introduction\}

Существует несколько способов задать настройку для пользователя в ClickHouse в зависимости от сценария использования и того, на какой срок вы хотите её задать. Рассмотрим несколько сценариев...

## Настройка параметров для отдельного запроса

Запрос `SELECT` может содержать секцию `SETTINGS`, в которой можно задать произвольное количество настроек. Эти настройки применяются только к этому запросу. Например:

```sql
SELECT *
FROM my_table
SETTINGS max_threads = 8;
```

Для этого запроса максимальное число потоков составит 8.


## Настройка параметра для сессии

Вы можете задать параметр на время жизни клиентской сессии, используя оператор `SET`. Это удобно для разового (ad-hoc) тестирования или когда вы хотите, чтобы параметр действовал только в течение выполнения нескольких запросов — но не дольше.

```sql
SET max_threads = 8;

SELECT *
FROM my_table;
```


## Настройка параметра для отдельного пользователя

Используйте `ALTER USER`, чтобы задать настройку только для одного пользователя. Например:

```sql
ALTER USER имя_пользователя SETTINGS max_threads = 8;
```

Вы можете убедиться, что всё сработало, выйдя из клиента, снова войдя в него, а затем вызвав функцию `getSetting`:

```sql
SELECT getSetting('max_threads');
```
