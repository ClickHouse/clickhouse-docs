## Что делать

Если вы используете Oracle в качестве источника внешних словарей ClickHouse через драйвер Oracle ODBC, вам необходимо задать правильное значение для переменной окружения `NLS_LANG` в `/etc/default/clickhouse`.

Дополнительную информацию см. в [Oracle NLS&#95;LANG FAQ](https://www.oracle.com/technetwork/products/globalization/nls-lang-099431.html).

**Пример**

```sql
NLS_LANG=RUSSIAN_RUSSIA.UTF8
```
