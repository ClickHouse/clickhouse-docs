## Игнорирование некорректных настроек в ClickHouse

Когда пользовательский параметр задан в неправильном месте, сервер не запускается, а в журнал записывается сообщение об исключении. Однако вы можете настроить ClickHouse на игнорирование некорректного параметра с помощью настройки `skip_check_for_incorrect_settings`.

Добавьте следующее в `config.xml`:

```xml
<skip_check_for_incorrect_settings>1</skip_check_for_incorrect_settings>
```

:::note
Настройки на уровне пользователя должны указываться в `users.xml` внутри раздела `<profile>` для конкретного профиля пользователя (или в `<default>` для настроек по умолчанию).
:::
