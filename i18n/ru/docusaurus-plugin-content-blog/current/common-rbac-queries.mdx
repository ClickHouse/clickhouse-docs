## Введение \{#introduction\}

Ниже приведены типовые запросы, которые помогут выдать пользователям конкретные права доступа.

## Как предоставить другому пользователю те же права доступа, что и у текущего пользователя?

```sql
GRANT CURRENT GRANTS ON *.* TO another_user;
```


## Как выдать пользователю конкретное разрешение на основе привилегий текущего пользователя?

В примере ниже `another_user` сможет выполнять операции `SELECT` во всех базах данных и таблицах текущего пользователя.

```sql
GRANT CURRENT GRANTS(SELECT ON *.*) TO another_user;
```


## Как выдать пользователю конкретное право доступа к определённой базе данных на основе прав текущего пользователя?

В приведённом ниже примере `another_user` сможет выполнять команды `INSERT` для всех таблиц в `my_database`.

```sql
GRANT INSERT ON my_database.* TO another_user;
```


## Как предоставить пользователю доступ ко всем привилегиям на основе пользователя `default`?

```sql
GRANT default_role TO another_user;
```
