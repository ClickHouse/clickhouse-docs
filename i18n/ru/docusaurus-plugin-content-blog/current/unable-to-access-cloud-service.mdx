## Я не могу получить доступ к сервису ClickHouse Cloud \{#i-am-unable-to-access-a-clickhouse-cloud-service\}

Если вы видите одно из следующих сообщений об ошибке, возможно, ваш список доступа по IP (IP Access List) блокирует подключение:

```response
curl: (35) error:02FFF036:системная библиотека:func(4095):Соединение сброшено удалённым узлом
```

или

```response
curl: (35) LibreSSL SSL_connect: SSL_ERROR_SYSCALL при соединении с HOSTNAME.clickhouse.cloud:8443
```

или

```response
Код: 210. DB::NetException: SSL-соединение неожиданно разорвано (e46453teek.us-east-2.aws.clickhouse-staging.com:9440). (NETWORK_ERROR)
```

Проверьте [список доступа по IP](/cloud/security/setting-ip-filters): если вы пытаетесь подключиться с адреса, не входящего в разрешённый список, соединение не будет установлено.
