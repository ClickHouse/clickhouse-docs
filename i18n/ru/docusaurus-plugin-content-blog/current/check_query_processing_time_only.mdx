## Question \{#question\}

Мой запрос возвращает много строк, но меня интересует только время выполнения запроса. Как отключить вывод результатов и посмотреть время его выполнения?

## Ответ

Добавьте `FORMAT Null` к запросу, чтобы установить формат вывода в `Null`. Это предотвратит передачу данных клиенту.

Например:

```sql
SELECT
    customer_id,
    count() AS total,
    any(review_headline)
FROM amazon_reviews
GROUP BY customer_id
ORDER BY total DESC
FORMAT Null
```

В ответе будет указано количество обработанных строк и затраченное время, но сами данные (0 строк) возвращены не будут:

```response
0 строк в наборе. Прошло: 25,288 сек. Обработано 222,04 млн строк, 13,50 ГБ (8,78 млн строк/с., 533,77 МБ/с.)
```
