Каждый файл конфигурации будет содержать следующую уникальную конфигурацию (представленную ниже).  
`server_id`, используемый для этого конкретного узла ClickHouse Keeper в кластере, должен быть уникальным и соответствовать значению `<id>`, определенному в разделе `<raft_configuration>`.  
`tcp_port` — это порт, используемый _клиентами_ ClickHouse Keeper.

```xml
<tcp_port>9181</tcp_port>
<server_id>{id}</server_id>
```

Следующий раздел используется для настройки серверов, которые участвуют в квorum для [алгоритма Raft](https://en.wikipedia.org/wiki/Raft_(algorithm)):

```xml
<raft_configuration>
    <server>
        <id>1</id>
        <hostname>clickhouse-keeper-01</hostname>
        <!-- TCP port used for communication between ClickHouse Keeper nodes -->
        <!--highlight-next-line-->
        <port>9234</port>
    </server>
    <server>
        <id>2</id>
        <hostname>clickhouse-keeper-02</hostname>
        <port>9234</port>
    </server>
    <server>
        <id>3</id>
        <hostname>clickhouse-keeper-03</hostname>
        <port>9234</port>
    </server>
</raft_configuration>
```
