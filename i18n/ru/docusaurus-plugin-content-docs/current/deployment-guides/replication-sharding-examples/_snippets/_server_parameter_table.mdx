Для каждого сервера указываются следующие параметры:

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | Адрес удалённого сервера. Можно использовать доменное имя или IPv4-/IPv6-адрес. Если указано доменное имя, сервер при запуске выполняет DNS-запрос, и полученный результат используется всё время работы сервера. Если DNS-запрос завершился с ошибкой, сервер не запускается. При изменении DNS-записи необходимо перезапустить сервер. | -            |
| `port`        | TCP-порт для работы протокола взаимодействия (`tcp_port` в конфигурации, обычно имеет значение 9000). Не путать с `http_port`.                                                                                                                                                                                                                      | -            |
