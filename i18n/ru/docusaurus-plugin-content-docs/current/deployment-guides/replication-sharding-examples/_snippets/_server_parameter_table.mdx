Для каждого сервера задаются следующие параметры:

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | Адрес удалённого сервера. Можно использовать доменное имя или IPv4- либо IPv6-адрес. Если указано доменное имя, сервер при запуске выполняет DNS-запрос, и его результат используется на протяжении всей работы сервера. Если DNS-запрос не удался, сервер не запустится. При изменении DNS-записи необходимо перезапустить сервер. | -            |
| `port`        | TCP-порт для работы сервера (`tcp_port` в конфигурации, обычно равен 9000). Не путать с `http_port`.                                                                                                                                                                                                                                                 | -            |
