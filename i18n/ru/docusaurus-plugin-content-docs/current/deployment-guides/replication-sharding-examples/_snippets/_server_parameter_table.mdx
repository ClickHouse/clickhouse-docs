Для каждого сервера задаются следующие параметры:

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | Адрес удалённого сервера. Можно использовать доменное имя или IPv4/IPv6‑адрес. Если указано доменное имя, сервер выполняет DNS‑запрос при запуске и сохраняет результат до завершения работы. Если DNS‑запрос завершается с ошибкой, сервер не запускается. При изменении DNS‑записи необходимо перезапустить сервер. | -            |
| `port`        | TCP‑порт для работы сервера (`tcp_port` в конфигурации, обычно 9000). Не путать с `http_port`.                                                                                                                                                                                                                                                       | -            |
