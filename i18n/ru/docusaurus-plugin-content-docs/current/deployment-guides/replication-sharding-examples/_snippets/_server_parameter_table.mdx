Для каждого сервера задаются следующие параметры:

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | Адрес удалённого сервера. Можно использовать доменное имя или IPv4/IPv6-адрес. При указании доменного имени сервер выполняет DNS-запрос при запуске и сохраняет результат на всё время работы. Если DNS-запрос завершился неудачно, сервер не запускается. При изменении DNS-записи необходимо перезапустить сервер. | -            |
| `port`        | TCP-порт для работы мессенджера (`tcp_port` в конфигурации, обычно 9000). Не путать с `http_port`.                                                                                                                                                                                                                                                   | -            |
