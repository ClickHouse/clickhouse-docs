* Каталог `config.d` содержит файл конфигурации сервера ClickHouse `config.xml`,
  в котором задаётся индивидуальная конфигурация для каждого узла ClickHouse. Эта
  конфигурация объединяется с файлом конфигурации ClickHouse по умолчанию
  `config.xml`, который устанавливается вместе с ClickHouse.
* Каталог `users.d` содержит файл конфигурации пользователей `users.xml`, в котором
  задаётся индивидуальная конфигурация для пользователей. Эта конфигурация объединяется
  с файлом конфигурации ClickHouse `users.xml` по умолчанию, который устанавливается
  вместе с ClickHouse.

:::tip Custom configuration directories
Рекомендуется использовать каталоги `config.d` и `users.d` для вашей собственной
конфигурации, а не вносить изменения напрямую в конфигурацию по умолчанию
в `/etc/clickhouse-server/config.xml` и `etc/clickhouse-server/users.xml`.

Строка

```xml
<clickhouse replace="true">
```

Гарантирует, что разделы конфигурации, определённые в каталогах `config.d` и `users.d`,
переопределяют разделы конфигурации по умолчанию, определённые в стандартных файлах
`config.xml` и `users.xml`.
:::
