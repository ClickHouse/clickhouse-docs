* Каталог `config.d` содержит файл конфигурации сервера ClickHouse `config.xml`,
  в котором задаётся пользовательская конфигурация для каждого узла ClickHouse. Эта
  конфигурация объединяется с файлом конфигурации ClickHouse по умолчанию `config.xml`,
  который устанавливается вместе с ClickHouse.
* Каталог `users.d` содержит файл конфигурации пользователей `users.xml`, в котором
  задаётся пользовательская конфигурация для пользователей. Эта конфигурация объединяется
  с файлом конфигурации ClickHouse по умолчанию `users.xml`, который устанавливается вместе
  с ClickHouse.

:::tip Custom configuration directories
Рекомендуется использовать каталоги `config.d` и `users.d` при
написании собственной конфигурации, а не изменять напрямую конфигурацию
по умолчанию в `/etc/clickhouse-server/config.xml` и `etc/clickhouse-server/users.xml`.

Строка

```xml
<clickhouse replace="true">
```

Гарантирует, что секции конфигурации, определённые в каталогах `config.d` и `users.d`,
переопределяют секции конфигурации по умолчанию, заданные в стандартных файлах
`config.xml` и `users.xml`.
:::
