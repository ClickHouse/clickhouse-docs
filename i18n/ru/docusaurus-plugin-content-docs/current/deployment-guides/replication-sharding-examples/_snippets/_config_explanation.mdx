* Каталог `config.d` содержит файл конфигурации сервера ClickHouse `config.xml`,
  в котором задаётся индивидуальная конфигурация для каждого узла ClickHouse. Эта
  конфигурация объединяется с файлом конфигурации ClickHouse по умолчанию `config.xml`,
  который устанавливается вместе с каждым экземпляром ClickHouse.
* Каталог `users.d` содержит файл конфигурации пользователей `users.xml`, в котором
  задаётся индивидуальная конфигурация пользователей. Эта конфигурация объединяется
  с файлом конфигурации ClickHouse по умолчанию `users.xml`, который устанавливается вместе
  с каждым экземпляром ClickHouse.

:::tip Каталоги пользовательской конфигурации
Рекомендуется использовать каталоги `config.d` и `users.d` для собственной конфигурации,
а не изменять напрямую конфигурацию по умолчанию в
`/etc/clickhouse-server/config.xml` и `etc/clickhouse-server/users.xml`.

Строка

```xml
<clickhouse replace="true">
```

Обеспечивает, что разделы конфигурации, определённые в каталогах `config.d` и `users.d`,
имеют приоритет над разделами конфигурации по умолчанию, заданными в файлах `config.xml` и `users.xml`.
:::
