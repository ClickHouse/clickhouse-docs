* Каталог `config.d` содержит файл конфигурации сервера ClickHouse `config.xml`,
  в котором задаётся пользовательская конфигурация для каждого узла ClickHouse. Эта
  конфигурация объединяется с файлом конфигурации ClickHouse по умолчанию `config.xml`,
  который устанавливается вместе с ClickHouse.
* Каталог `users.d` содержит файл конфигурации пользователей `users.xml`, в котором
  задаётся пользовательская конфигурация для пользователей. Эта конфигурация объединяется
  с файлом конфигурации ClickHouse по умолчанию `users.xml`, который устанавливается
  вместе с ClickHouse.

:::tip Пользовательские каталоги конфигурации
Рекомендуется использовать каталоги `config.d` и `users.d` при написании собственной
конфигурации, вместо прямого изменения конфигурации по умолчанию в
`/etc/clickhouse-server/config.xml` и `etc/clickhouse-server/users.xml`.

Строка

```xml
<clickhouse replace="true">
```

Гарантирует, что разделы конфигурации, определённые в директориях `config.d` и `users.d`,
переопределяют разделы конфигурации по умолчанию, заданные в файлах
`config.xml` и `users.xml` по умолчанию.
:::
