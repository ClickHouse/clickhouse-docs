:::tip ClickHouse Cloud упрощает управление
[ClickHouse Cloud](/cloud/overview)
снимает операционное бремя, связанное с управлением шардами и репликами.
Платформа автоматически обеспечивает высокую доступность, репликацию
и масштабирование. Вычислительные ресурсы и хранилище разделены и масштабируются
по мере необходимости, без ручной настройки и постоянного обслуживания.
