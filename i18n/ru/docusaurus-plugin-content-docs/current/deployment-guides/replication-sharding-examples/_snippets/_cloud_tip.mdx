:::tip ClickHouse Cloud упрощает управление
[ClickHouse Cloud](/cloud/overview)
снимает операционные сложности, связанные с управлением шардами и репликами.
Платформа автоматически обеспечивает высокую доступность, репликацию и принимает решения о масштабировании.
Вычислительные ресурсы и хранилище разделены и масштабируются по мере необходимости без ручной
конфигурации и постоянного обслуживания.

[Подробнее](/manage/scaling)
:::
