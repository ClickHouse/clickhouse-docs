Внешнее взаимодействие через сетевой интерфейс включается с помощью настройки
listen&#95;host. Это гарантирует, что хост сервера ClickHouse доступен с других
хостов:

```xml
<listen_host>0.0.0.0</listen_host>
```

Порт для HTTP API — `8123`:

```xml
<http_port>8123</http_port>
```

TCP-порт для взаимодействия по нативному протоколу ClickHouse между `clickhouse-client`
и другими нативными инструментами ClickHouse, а также между `clickhouse-server` и другими серверами ClickHouse
по умолчанию равен `9000`:

```xml
<tcp_port>9000</tcp_port>
```
