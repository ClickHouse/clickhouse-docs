Внешнее взаимодействие через сетевой интерфейс включается с помощью настройки listen&#95;host. Это гарантирует, что хост сервера ClickHouse будет доступен с других хостов:

```xml
<listen_host>0.0.0.0</listen_host>
```

Для HTTP API используется порт `8123`:

```xml
<http_port>8123</http_port>
```

TCP-порт для взаимодействия по собственному протоколу ClickHouse между `clickhouse-client`
и другими нативными инструментами ClickHouse, а также между `clickhouse-server` и другими серверами ClickHouse
имеет значение `9000`:

```xml
<tcp_port>9000</tcp_port>
```
