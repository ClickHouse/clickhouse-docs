Внешние подключения к сетевому интерфейсу разрешаются за счёт включения параметра
listen&#95;host. Это обеспечивает доступность хоста сервера ClickHouse с других
хостов:

```xml
<listen_host>0.0.0.0</listen_host>
```

Порт HTTP API задан как `8123`:

```xml
<http_port>8123</http_port>
```

TCP-порт для взаимодействия по нативному протоколу ClickHouse между `clickhouse-client`
и другими нативными инструментами ClickHouse, а также `clickhouse-server` и другими серверами ClickHouse
имеет значение `9000`:

```xml
<tcp_port>9000</tcp_port>
```
