Внешние подключения к сетевому интерфейсу включаются путем настройки параметра
listen&#95;host. Это обеспечивает доступность хоста сервера ClickHouse с других
хостов:

```xml
<listen_host>0.0.0.0</listen_host>
```

Порт HTTP API установлен в значение `8123`:

```xml
<http_port>8123</http_port>
```

TCP-порт для взаимодействия по нативному протоколу ClickHouse между clickhouse-client
и другими нативными инструментами ClickHouse, а также между clickhouse-server и другими серверами clickhouse-server
равен `9000`:

```xml
<tcp_port>9000</tcp_port>
```
