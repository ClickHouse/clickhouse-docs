:::note
Хотя можно запускать ClickHouse Server и ClickHouse Keeper вместе на одном сервере,
мы настоятельно рекомендуем использовать *выделенные* хосты для ClickHouse Keeper в производственных средах —
именно такой подход мы продемонстрируем в этом примере.

Серверы Keeper могут быть менее производительными, и в большинстве случаев 4 ГБ ОЗУ достаточно для каждого сервера Keeper,
пока ваши серверы ClickHouse существенно не вырастут.
:::
