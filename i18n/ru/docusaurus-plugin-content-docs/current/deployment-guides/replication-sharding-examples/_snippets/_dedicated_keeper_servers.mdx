:::note
Хотя можно запускать ClickHouse Server и ClickHouse Keeper вместе на одном сервере,
мы настоятельно рекомендуем в продуктивных средах использовать *выделенные* хосты для ClickHouse Keeper —
именно такой подход мы и продемонстрируем в этом примере.

Серверы Keeper могут быть менее мощными, и 4 ГБ RAM обычно достаточно для каждого сервера Keeper,
пока ваши серверы ClickHouse не вырастут до крупных масштабов.
:::
