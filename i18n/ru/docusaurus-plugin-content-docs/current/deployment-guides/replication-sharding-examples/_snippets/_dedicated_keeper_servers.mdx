:::note
Хотя можно запускать ClickHouse Server и ClickHouse Keeper совместно на одном сервере,
мы настоятельно рекомендуем в продуктивных средах использовать *выделенные* хосты для ClickHouse Keeper —
именно этот подход мы продемонстрируем в данном примере.

Серверы Keeper могут быть менее ресурсоёмкими, и, как правило, каждому серверу Keeper достаточно 4 ГБ ОЗУ,
пока ваши серверы ClickHouse существенно не вырастут.
:::
