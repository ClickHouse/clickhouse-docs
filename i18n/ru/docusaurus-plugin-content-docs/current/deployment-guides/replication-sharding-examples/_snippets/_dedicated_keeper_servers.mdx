:::note
Хотя возможно запускать ClickHouse Server и ClickHouse Keeper на одном и том же сервере, 
мы настоятельно рекомендуем использовать *выделенные* хосты для ClickHouse Keeper в производственных средах, 
что является подходом, который мы продемонстрируем в этом примере.

Серверы Keeper могут быть меньшими, и обычно 4 ГБ ОП достаточно для каждого сервера Keeper, 
пока ваши серверы ClickHouse не вырастут в размерах.
:::
