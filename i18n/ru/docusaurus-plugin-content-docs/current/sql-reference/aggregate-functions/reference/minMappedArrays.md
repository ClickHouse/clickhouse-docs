## minMappedArrays \{#minMappedArrays\}

Введена в версии: v20.5

Вычисляет минимум по массиву `value` в соответствии с ключами, заданными в массиве `key`.

:::note

* Передача кортежа массивов ключей и значений идентична передаче массива ключей и массива значений.
* Число элементов в `key` и `value` должно быть одинаковым для каждой строки, участвующей в агрегации.
  :::

**Синтаксис**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**Аргументы**

* `key` — массив ключей. [`Array(T)`](/sql-reference/data-types/array)
* `value` — массив значений. [`Array(T)`](/sql-reference/data-types/array)

**Возвращаемое значение**

Возвращает кортеж из двух массивов: ключи в отсортированном порядке и значения, вычисленные для соответствующих ключей. [`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**Примеры**

**Пример использования**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
