## boundingRatio \{#boundingRatio\}

Впервые добавлена в версии: v20.1

Вычисляет наклон между крайней левой и крайней правой точками в группе значений.

**Синтаксис**

```sql
boundingRatio(x, y)
```

**Аргументы**

* `x` — значения координаты X. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)
* `y` — значения координаты Y. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Возвращаемое значение**

Возвращает наклон прямой между самой левой и самой правой точками, иначе возвращает `NaN`, если данные отсутствуют. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Пример данных**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
┌─number─┬─multiply(number, 1.5)─┐
│      0 │                     0 │
│      1 │                   1.5 │
│      2 │                     3 │
│      3 │                   4.5 │
│      4 │                     6 │
│      5 │                   7.5 │
│      6 │                     9 │
│      7 │                  10.5 │
│      8 │                    12 │
│      9 │                  13.5 │
└────────┴───────────────────────┘
```

**Пример использования**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```

```response title=Response
┌─boundingRatio(number, multiply(number, 1.5))─┐
│                                          1.5 │
└──────────────────────────────────────────────┘
```
