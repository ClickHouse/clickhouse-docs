## entropy \{#entropy\}

Появилась в версии v20.1.

Вычисляет [энтропию Шеннона](https://en.wikipedia.org/wiki/Entropy_\(information_theory\)) для столбца значений.

**Синтаксис**

```sql
entropy(val)
```

**Аргументы**

* `val` — столбец значений любого типа. [`Any`](/sql-reference/data-types)

**Возвращаемое значение**

Возвращает значение энтропии Шеннона. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Базовый пример вычисления энтропии**

```sql title=Query
CREATE TABLE entropy (`vals` UInt32,`strings` String)
ENGINE = Memory;

INSERT INTO entropy VALUES (1, 'A'), (1, 'A'), (1,'A'), (1,'A'), (2,'B'), (2,'B'), (2,'C'), (2,'D');

SELECT entropy(vals), entropy(strings) FROM entropy
```

```response title=Response
┌─entropy(vals)─┬─entropy(strings)─┐
│             1 │             1.75 │
└───────────────┴──────────────────┘
```
