## distinctDynamicTypes \{#distinctDynamicTypes\}

Появилась в версии v24.9.

Вычисляет список уникальных типов данных, содержащихся в столбце [Dynamic](https://clickhouse.com/docs/sql-reference/data-types/dynamic).

**Синтаксис**

```sql
distinctDynamicTypes(dynamic)
```

**Аргументы**

* `dynamic` — столбец типа [`Dynamic`](/sql-reference/data-types/dynamic)

**Возвращаемое значение**

Возвращает отсортированный список имён типов данных в виде [`Array(String)`](/sql-reference/data-types/array)

**Примеры**

**Базовое использование со смешанными типами**

```sql title=Query
DROP TABLE IF EXISTS test_dynamic;
CREATE TABLE test_dynamic(d Dynamic) ENGINE = Memory;
INSERT INTO test_dynamic VALUES (42), (NULL), ('Hello'), ([1, 2, 3]), ('2020-01-01'), (map(1, 2)), (43), ([4, 5]), (NULL), ('World'), (map(3, 4));

SELECT distinctDynamicTypes(d) FROM test_dynamic;
```

```response title=Response
┌─distinctDynamicTypes(d)──────────────────────────────────────────┐
│ ['Array(Int64)', 'Date', 'Int64', 'Map(UInt8, UInt8)', 'String'] │
└──────────────────────────────────────────────────────────────────┘
```
