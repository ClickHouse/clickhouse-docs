# kurtSamp \{#kurtsamp\}

Вычисляет [выборочный эксцесс](https://en.wikipedia.org/wiki/Kurtosis) по последовательности.

Даёт несмещённую оценку эксцесса случайной величины, если переданные значения образуют её выборку.

```sql
kurtSamp(expr)
```

**Аргументы**

`expr` — [выражение](/sql-reference/syntax#expressions), возвращающее число.

**Возвращаемое значение**

Эксцесс заданного распределения. Тип — [Float64](../../../sql-reference/data-types/float.md). Если `n <= 1` (`n` — размер выборки), то функция возвращает `nan`.

**Пример**

```sql
SELECT kurtSamp(value) FROM series_with_value_column;
```
