## maxIntersectionsPosition \{#maxIntersectionsPosition\}

Добавлена в версии: v1.1

Агрегатная функция, вычисляющая позиции вхождений функции [`maxIntersections`](/sql-reference/aggregate-functions/reference/maxintersections).

**Синтаксис**

```sql
maxIntersectionsPosition(start_column, end_column)
```

**Аргументы**

* `start_column` — числовой столбец, задающий начало каждого интервала. Если `start_column` имеет значение `NULL` или 0, интервал будет пропущен. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float)
* `end_column` — числовой столбец, задающий конец каждого интервала. Если `end_column` имеет значение `NULL` или 0, интервал будет пропущен. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float)

**Возвращаемое значение**

Возвращает начальные позиции максимального числа пересекающихся интервалов. [`Any`](/sql-reference/data-types)

**Примеры**

**Поиск позиции максимального пересечения**

```sql title=Query
CREATE TABLE my_events (
    start UInt32,
    end UInt32
)
ENGINE = MergeTree
ORDER BY tuple();

INSERT INTO my_events VALUES
(1, 3),
(1, 6),
(2, 5),
(3, 7);

SELECT maxIntersectionsPosition(start, end) FROM my_events;
```

```response title=Response
┌─maxIntersectionsPosition(start, end)─┐
│                                    2 │
└──────────────────────────────────────┘
```
