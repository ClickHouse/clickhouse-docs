# anyHeavy {#anyheavy}

Выбирает часто встречающееся значение с использованием алгоритма [heavy hitters](https://doi.org/10.1145/762471.762473). Если существует значение, которое встречается более чем в половине случаев в каждом потоке выполнения запроса, возвращается именно оно. Обычно результат недетерминирован.

```sql
anyHeavy(столбец)
```

**Аргументы**

* `column` – имя столбца.

**Пример**

Возьмите набор данных [OnTime](../../../getting-started/example-datasets/ontime.md) и выберите любое часто встречающееся значение в столбце `AirlineID`.

```sql
SELECT anyHeavy(AirlineID) AS res
FROM ontime
```

```text
┌───res─┐
│ 19690 │
└───────┘
```
