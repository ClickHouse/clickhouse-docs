## maxMappedArrays \{#maxMappedArrays\}

Добавлена в версии: v20.5

Вычисляет максимум из массива `value` в соответствии с ключами, указанными в массиве `key`.

:::note

* Передача кортежа массивов ключей и значений идентична передаче массива ключей и массива значений.
* Количество элементов в `key` и `value` должно быть одинаковым для каждой агрегируемой строки.
  :::

**Синтаксис**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**Аргументы**

* `key` — массив ключей. [`Array(T)`](/sql-reference/data-types/array)
* `value` — массив значений. [`Array(T)`](/sql-reference/data-types/array)

**Возвращаемое значение**

Возвращает кортеж из двух массивов: ключи в отсортированном порядке, а также значения, вычисленные для соответствующих ключей. [`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**Примеры**

**Пример использования**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)────────────────┐
│ [['x', 'y', 'z'], [2, 3, 1]].        │
└──────────────────────────────────────┘
```
