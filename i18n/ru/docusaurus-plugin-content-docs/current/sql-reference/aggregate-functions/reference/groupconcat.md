Вычисляет строку, полученную объединением группы строк, опционально с заданным разделителем и/или ограничением на максимальное количество элементов.

**Синтаксис**

```sql
groupConcat[(delimiter [, limit])](expression);
```

Псевдоним: `group_concat`

**Аргументы**

* `expression` — выражение или имя столбца, результатом которого являются строки, подлежащие объединению.
* `delimiter` — [строка](../../../sql-reference/data-types/string.md), которая будет использоваться для разделения объединяемых значений. Этот параметр является необязательным и по умолчанию принимает пустую строку или разделитель из параметров, если он не указан.

**Параметры**

* `delimiter` — [строка](../../../sql-reference/data-types/string.md), которая будет использоваться для разделения объединяемых значений. Этот параметр является необязательным и по умолчанию принимает пустую строку, если он не указан.
* `limit` — положительное [целое число](../../../sql-reference/data-types/int-uint.md), задающее максимальное количество элементов для объединения. Если элементов больше, лишние элементы игнорируются. Этот параметр является необязательным.

:::note
Если `delimiter` указан без `limit`, он должен быть первым параметром. Если указаны и `delimiter`, и `limit`, `delimiter` должен предшествовать `limit`.

Кроме того, если разные разделители указаны как в параметрах, так и в аргументах, будет использоваться только разделитель из аргументов.
:::

**Возвращаемое значение**

* Возвращает [строку](../../../sql-reference/data-types/string.md), состоящую из объединённых значений столбца или выражения. Если в группе нет элементов или есть только значения `NULL`, и функция не задаёт обработку случая только значений `NULL`, результатом будет тип `Nullable(String)` со значением `NULL`.

**Примеры**

Входная таблица:

```text
┌─id─┬─name─┐
│  1 │ John │
│  2 │ Jane │
│  3 │ Bob  │
└────┴──────┘
```

1. Простейшее использование без разделителя:

Запрос:

```sql
SELECT groupConcat(Name) FROM Employees;
```

Результат:

```text
JohnJaneBob
```

Это объединяет все имена в одну непрерывную строку без разделителя.

2. Использование запятой в качестве разделителя:

Запрос:

```sql
SELECT groupConcat(', ')(Name)  FROM Employees;
```

или

```sql
SELECT groupConcat(Name, ', ')  FROM Employees;
```

Результат:

```text
John, Jane, Bob
```

Результат показывает имена, разделённые запятой и пробелом.

3. Ограничение количества объединяемых элементов

Запрос:

```sql
SELECT groupConcat(', ', 2)(Name) FROM Employees;
```

Результат:

```text
John, Jane
```

Этот запрос ограничивает вывод первыми двумя именами, хотя в таблице есть и другие имена.
