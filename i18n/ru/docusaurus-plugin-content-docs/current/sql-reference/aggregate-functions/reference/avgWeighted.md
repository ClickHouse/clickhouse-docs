## avgWeighted \{#avgWeighted\}

Введена в версии v20.1

Вычисляет [взвешенное среднее арифметическое](https://en.wikipedia.org/wiki/Weighted_arithmetic_mean).

**Синтаксис**

```sql
avgWeighted(x, weight)
```

**Аргументы**

* `x` — значения. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float)
* `weight` — веса значений. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float)

**Возвращаемое значение**

Возвращает `NaN`, если все веса равны 0 или переданный параметр `weight` пуст, в противном случае — взвешенное среднее. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Пример использования**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**Смешанные целочисленные и дробные веса**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Float64', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**Если все веса равны нулю, возвращается NaN**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (0, 0), (1, 0), (10, 0))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│               nan │
└───────────────────┘
```

**Пустая таблица возвращает NaN**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;
SELECT avgWeighted(t, t) FROM test
```

```response title=Response
┌─avgWeighted(t, t)─┐
│               nan │
└───────────────────┘
```
