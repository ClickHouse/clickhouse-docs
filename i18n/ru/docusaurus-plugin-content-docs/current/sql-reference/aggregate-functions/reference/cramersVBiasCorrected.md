## cramersVBiasCorrected \{#cramersVBiasCorrected\}

Введено в: v22.1

V Крамера — это мера взаимосвязи между двумя столбцами в таблице.
Результат функции [`cramersV`](/sql-reference/aggregate-functions/reference/cramersv) лежит в диапазоне от 0 (соответствует отсутствию взаимосвязи между переменными) до 1 и может принимать значение 1 только тогда, когда каждое значение полностью определяется другим.
Функция может давать существенно смещённую оценку, поэтому эта версия V Крамера использует [коррекцию смещения](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V#Bias_correction).

**Синтаксис**

```sql
cramersVBiasCorrected(column1, column2)
```

**Аргументы**

* `column1` — первый столбец для сравнения. [`Any`](/sql-reference/data-types)
* `column2` — второй столбец для сравнения. [`Any`](/sql-reference/data-types)

**Возвращаемое значение**

Возвращает значение от 0 (соответствует отсутствию связи между значениями столбцов) до 1 (полная взаимосвязь). [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Сравнение с обычной версией cramersV**

```sql title=Query
SELECT
    cramersV(a, b),
    cramersVBiasCorrected(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            number % 4 AS b
        FROM
            numbers(150)
    )
```

```response title=Response
┌──────cramersV(a, b)─┬─cramersVBiasCorrected(a, b)─┐
│ 0.41171788506213564 │         0.33369281784141364 │
└─────────────────────┴─────────────────────────────┘
```
