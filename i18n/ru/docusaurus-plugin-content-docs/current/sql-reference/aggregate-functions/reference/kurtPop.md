# kurtPop {#kurtpop}

Вычисляет [эксцесс](https://en.wikipedia.org/wiki/Kurtosis) для последовательности значений.

```sql
kurtPop(expr)
```

**Аргументы**

`expr` — [выражение](/sql-reference/syntax#expressions), возвращающее числовое значение.

**Возвращаемое значение**

Эксцесс данного распределения. Тип — [Float64](../../../sql-reference/data-types/float.md)

**Пример**

```sql
SELECT kurtPop(value) FROM series_with_value_column;
```
