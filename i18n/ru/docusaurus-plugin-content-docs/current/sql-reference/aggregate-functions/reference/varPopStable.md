## varPopStable \{#varPopStable\}

Добавлена в: v1.1

Возвращает дисперсию генеральной совокупности.
В отличие от [`varPop`](/sql-reference/aggregate-functions/reference/varPop), эта функция использует [численно устойчивый](https://en.wikipedia.org/wiki/Numerical_stability) алгоритм.
Она работает медленнее, но обеспечивает меньшую вычислительную погрешность.

**Синтаксис**

```sql
varPopStable(x)
```

**Аргументы**

* `x` — совокупность значений, для которой вычисляется дисперсия генеральной совокупности. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal*`](/sql-reference/data-types/decimal)

**Возвращаемое значение**

Возвращает дисперсию генеральной совокупности для `x`. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Вычисление численно устойчивой дисперсии генеральной совокупности**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    varPopStable(x) AS var_pop_stable
FROM test_data;
```

```response title=Response
┌─var_pop_stable─┐
│           14.4 │
└────────────────┘
```
