## exponentialTimeDecayedMax \{#exponentialTimeDecayedMax\}

Введено в: v21.12

Возвращает максимальное значение вычисленного экспоненциально сглаженного скользящего среднего в момент времени `t` по сравнению со значением в момент времени `t-1`.

**Синтаксис**

```sql
exponentialTimeDecayedMax(x)(value, timeunit)
```

**Параметры**

* `x` — Период полураспада. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Аргументы**

* `value` — Значение. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)
* `timeunit` — Единица измерения времени. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal) или [`DateTime`](/sql-reference/data-types/datetime) или [`DateTime64`](/sql-reference/data-types/datetime64)

**Возвращаемое значение**

Возвращает максимум экспоненциально сглаженного взвешенного скользящего среднего в моменты `t` и `t-1`. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Использование оконной функции с визуальным представлением**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 5, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedMax(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
