# singleValueOrNull \{#singlevalueornull\}

Агрегатная функция `singleValueOrNull` используется для реализации операторов с подзапросами, таких как `x = ALL (SELECT ...)`. Она проверяет, есть ли в данных ровно одно уникальное значение, не равное NULL.
Если уникальное значение только одно, она возвращает его. Если значений нет или как минимум два различных значения, она возвращает NULL.

**Синтаксис**

```sql
singleValueOrNull(x)
```

**Параметры**

* `x` — столбец любого [типа данных](../../data-types/index.md) (кроме [Map](../../data-types/map.md), [Array](../../data-types/array.md) или [Tuple](../../data-types/tuple), которые не могут быть типа [Nullable](../../data-types/nullable.md)).

**Возвращаемые значения**

* Уникальное значение, если в `x` есть ровно одно уникальное значение, отличное от `NULL`.
* `NULL`, если нет ни одного значения или есть как минимум два различных значения.

**Примеры**

Запрос:

```sql
CREATE TABLE test (x UInt8 NULL) ENGINE=Log;
INSERT INTO test (x) VALUES (NULL), (NULL), (5), (NULL), (NULL);
SELECT singleValueOrNull(x) FROM test;
```

Результат:

```response
┌─singleValueOrNull(x)─┐
│                    5 │
└──────────────────────┘
```

Запрос:

```sql
INSERT INTO test (x) VALUES (10);
SELECT singleValueOrNull(x) FROM test;
```

Результат:

```response
┌─singleValueOrNull(x)─┐
│                 ᴺᵁᴸᴸ │
└──────────────────────┘
```
