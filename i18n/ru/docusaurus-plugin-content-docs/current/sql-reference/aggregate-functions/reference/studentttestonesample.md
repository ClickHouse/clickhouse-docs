# studentTTestOneSample

Применяет одновыборочный t‑критерий Стьюдента для проверки, отличается ли среднее значение выборки от известного среднего значения генеральной совокупности.

Предполагается нормальное распределение. Нулевая гипотеза заключается в том, что среднее значение выборки равно среднему значению генеральной совокупности.

**Синтаксис**

```sql
studentTTestOneSample([уровень_доверия])(данные_выборки, среднее_генеральной_совокупности)
```

Необязательный параметр `confidence_level` позволяет вычислять доверительный интервал.

**Аргументы**

* `sample_data` — Выборочные данные. Integer, Float или Decimal.
* `population_mean` — Известное среднее генеральной совокупности, с которым проводится проверка. Integer, Float или Decimal (обычно константа).

**Параметры**

* `confidence_level` — Уровень доверия для доверительных интервалов. Float в диапазоне (0, 1).

Примечания:

* Требуются как минимум два наблюдения; в противном случае результат — `(nan, nan)` (а интервалы, если запрошены, — `nan`).
* Постоянные или почти постоянные входные данные также вернут `nan` из-за нулевой (или практически нулевой) стандартной ошибки.

**Возвращаемые значения**

[Tuple](../../../sql-reference/data-types/tuple.md) с двумя или четырьмя элементами (если указан `confidence_level`):

* вычисленная t-статистика. Float64.
* вычисленное p-значение (двустороннее). Float64.
* вычисленная нижняя граница доверительного интервала. Float64. (необязательно)
* вычисленная верхняя граница доверительного интервала. Float64. (необязательно)

Доверительные интервалы рассчитываются для выборочного среднего при заданном уровне доверия.

**Примеры**

Входная таблица:

```text
┌─value─┐
│  20.3 │
│  21.1 │
│  21.7 │
│  19.9 │
│  21.8 │
└───────┘
```

Без доверительного интервала:

```sql
SELECT studentTTestOneSample()(value, 20.0) FROM t;
-- или просто
SELECT studentTTestOneSample(value, 20.0) FROM t;
```

Доверительный интервал (95 %):

```sql
SELECT studentTTestOneSample(0.95)(value, 20.0) FROM t;
```
