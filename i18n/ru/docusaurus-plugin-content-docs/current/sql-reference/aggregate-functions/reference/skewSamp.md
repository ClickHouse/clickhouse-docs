# skewSamp \{#skewsamp\}

Вычисляет [выборочную асимметрию](https://en.wikipedia.org/wiki/Skewness) последовательности.

Является несмещённой оценкой асимметрии случайной величины, если переданные значения являются её выборкой.

```sql
skewSamp(expr)
```

**Аргументы**

`expr` — [выражение](/sql-reference/syntax#expressions), возвращающее число.

**Возвращаемое значение**

Коэффициент асимметрии заданного распределения. Тип — [Float64](../../../sql-reference/data-types/float.md). Если `n <= 1` (`n` — размер выборки), функция возвращает `nan`.

**Пример**

```sql
SELECT skewSamp(value) FROM series_with_value_column;
```
