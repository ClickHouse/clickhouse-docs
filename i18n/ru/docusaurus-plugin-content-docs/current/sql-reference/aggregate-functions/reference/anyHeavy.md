## anyHeavy \{#anyHeavy\}

Впервые представлена в версии v1.1

Выбирает часто встречающееся значение с использованием алгоритма «heavy hitters» [heavy hitters](https://doi.org/10.1145/762471.762473).
Если существует значение, которое встречается более чем в половине случаев в каждом из потоков выполнения запроса, возвращается именно оно.
Как правило, результат недетерминированный.

**Синтаксис**

```sql
anyHeavy(column)
```

**Аргументы**

* `column` — имя столбца. [`String`](/sql-reference/data-types/string)

**Возвращаемое значение**

Возвращает часто встречающееся значение. Результат недетерминированный. [`Any`](/sql-reference/data-types)

**Примеры**

**Пример использования**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
