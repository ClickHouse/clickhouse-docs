## groupUniqArray \{#groupUniqArray\}

Введена в версии: v1.1

Создаёт массив из различных значений аргумента.
Потребление памяти этой функции такое же, как у функции [`uniqExact`](/sql-reference/aggregate-functions/reference/uniqexact).

**Синтаксис**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**Параметры**

* `max_size` — Ограничивает размер результирующего массива до `max_size` элементов. `groupUniqArray(1)(x)` эквивалентно `[any(x)]`. [`UInt64`](/sql-reference/data-types/int-uint)

**Аргументы**

* `x` — выражение. [`Any`](/sql-reference/data-types)

**Возвращаемое значение**

Возвращает массив уникальных значений. [`Array`](/sql-reference/data-types/array)

**Примеры**

**Пример использования**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**С параметром max&#95;size**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
