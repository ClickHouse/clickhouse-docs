## groupBitXor \{#groupBitXor\}

Впервые появилось в версии v1.1

Применяет побитовую операцию XOR к последовательности чисел.

**Синтаксис**

```sql
groupBitXor(expr)
```

**Синонимы**: `BIT_XOR`

**Аргументы**

* `expr` — выражение типа `(U)Int*`. [`(U)Int*`](/sql-reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает значение типа `(U)Int*`. [`(U)Int*`](/sql-reference/data-types/int-uint)

**Примеры**

**Пример поразрядной операции XOR**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitXor(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01101000 = 104

┌─groupBitXor(num)─┐
│              104 │
└──────────────────┘
```
