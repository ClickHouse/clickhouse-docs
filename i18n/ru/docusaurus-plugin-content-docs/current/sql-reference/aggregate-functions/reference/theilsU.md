# theilsU \{#theilsu\}

Функция `theilsU` вычисляет [коэффициент неопределённости U Тейла](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient) — величину, которая измеряет степень связи между двумя столбцами в таблице. Её значения лежат в диапазоне от 0.0 (нет связи) до 1.0 (полное соответствие).

**Синтаксис**

```sql
theilsU(column1, column2)
```

**Аргументы**

* `column1` и `column2` — сравниваемые столбцы

**Возвращаемое значение**

* значение в диапазоне от 0 до 1

**Тип возвращаемого значения** всегда [Float64](../../../sql-reference/data-types/float.md).

**Пример**

Следующие два столбца, сравниваемые ниже, слабо взаимосвязаны, поэтому значение `theilsU` небольшое и положительное:

```sql
SELECT
    theilsU(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            number % 4 AS b
        FROM
            numbers(150)
    );
```

Результат:

```response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
