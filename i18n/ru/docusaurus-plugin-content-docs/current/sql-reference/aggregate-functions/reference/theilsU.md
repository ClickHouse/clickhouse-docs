## theilsU \{#theilsU\}

Введена в версии: v22.1.0

Функция `theilsU` вычисляет [коэффициент неопределённости Тейлса U](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient) — величину, которая измеряет степень связи между двумя столбцами в таблице.
Её значения лежат в диапазоне от 0.0 (отсутствие связи) до 1.0 (полное соответствие).

**Синтаксис**

```sql
theilsU(column1, column2)
```

**Аргументы**

* `column1` — первый столбец для сравнения. [`Any`](/sql-reference/data-types)
* `column2` — второй столбец для сравнения. [`Any`](/sql-reference/data-types)

**Возвращаемое значение**

Возвращает значение в диапазоне от 0 до 1. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Пример использования**

```sql title=Query
SELECT theilsU(a, b)
FROM (
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
