Вычисляет значение `(P(tag = 1) - P(tag = 0))(log(P(tag = 1)) - log(P(tag = 0)))` для каждой категории.

```sql
categoricalInformationValue(category1, category2, ..., tag)
```

Результат показывает, как дискретный (категориальный) признак `[category1, category2, ...]` вносит вклад в обучающую модель, предсказывающую значение `tag`.
