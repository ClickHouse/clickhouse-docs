## distinctJSONPaths \{#distinctJSONPaths\}

Добавлена в версии: v24.9

Вычисляет список уникальных путей, сохранённых в столбце [JSON](https://clickhouse.com/docs/sql-reference/data-types/newjson).

**Синтаксис**

```sql
distinctJSONPaths(json)
```

**Аргументы**

* `json` — столбец типа JSON. [`JSON`](/sql-reference/data-types/newjson)

**Возвращаемое значение**

Возвращает отсортированный список путей. [`Array(String)`](/sql-reference/data-types/array)

**Примеры**

**Базовое использование с вложенным JSON**

```sql title=Query
DROP TABLE IF EXISTS test_json;
CREATE TABLE test_json(json JSON) ENGINE = Memory;
INSERT INTO test_json VALUES ('{"a" : 42, "b" : "Hello"}'), ('{"b" : [1, 2, 3], "c" : {"d" : {"e" : "2020-01-01"}}}'), ('{"a" : 43, "c" : {"d" : {"f" : [{"g" : 42}]}}}');

SELECT distinctJSONPaths(json) FROM test_json;
```

```response title=Response
┌─distinctJSONPaths(json)───┐
│ ['a','b','c.d.e','c.d.f'] │
└───────────────────────────┘
```

**С явно заданными JSON-путями**

```sql title=Query
DROP TABLE IF EXISTS test_json;
CREATE TABLE test_json(json JSON) ENGINE = Memory;
INSERT INTO test_json VALUES ('{"a" : 42, "b" : "Hello"}'), ('{"b" : [1, 2, 3], "c" : {"d" : {"e" : "2020-01-01"}}}'), ('{"a" : 43, "c" : {"d" : {"f" : [{"g" : 42}]}}}')

SELECT distinctJSONPaths(json) FROM test_json;
```

```response title=Response
┌─distinctJSONPaths(json)─┐
│ ['a','b','c']           │
└─────────────────────────┘
```
