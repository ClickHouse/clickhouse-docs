# theilsU

Функция `theilsU` вычисляет [коэффициент неопределённости Тейла (Theil’s U)](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient) — величину, измеряющую степень связи между двумя столбцами в таблице. Значения лежат в диапазоне от −1.0 (100% отрицательная связь, или идеальная инверсия) до +1.0 (100% положительная связь, или идеальное соответствие). Значение 0.0 указывает на отсутствие связи.

**Синтаксис**

```sql
theilsU(column1, column2)
```

**Аргументы**

* `column1` и `column2` — сравниваемые столбцы

**Возвращаемое значение**

* значение в диапазоне от -1 до 1

**Тип возвращаемого значения** — всегда [Float64](../../../sql-reference/data-types/float.md).

**Пример**

Следующие два столбца, сравниваемые ниже, имеют слабую связь между собой, поэтому значение `theilsU` отрицательное:

```sql
SELECT
    theilsU(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            number % 4 AS b
        FROM
            numbers(150)
    );
```

Результат:

```response
┌────────theilsU(a, b)─┐
│ -0.30195720557678846 │
└──────────────────────┘
```
