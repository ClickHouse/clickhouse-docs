## studentTTestOneSample \{#studentTTestOneSample\}

Добавлена в версии: v25.10

Применяет одновыборочный критерий Стьюдента для определения, отличается ли среднее значение выборки от известного среднего значения генеральной совокупности.

Предполагается нормальность распределения. Нулевая гипотеза состоит в том, что среднее выборки равно среднему генеральной совокупности.

Необязательный параметр `confidence_level` позволяет рассчитывать доверительный интервал.

**Примечания:**

* Требуется как минимум 2 наблюдения; в противном случае результат — `(nan, nan)` (а интервалы, если запрошены, — `nan`).
* Постоянные или почти постоянные входные данные также вернут `nan` из-за нулевой (или практически нулевой) стандартной ошибки.

**См. также**

* [t-тест Стьюдента](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [функция studentTTest](/sql-reference/aggregate-functions/reference/studentttest)

**Синтаксис**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

**Параметры**

* `confidence_level` — Необязательный параметр. Уровень доверия для доверительных интервалов. Число с плавающей запятой в диапазоне (0, 1). [`Float*`](/sql-reference/data-types/float)

**Аргументы**

* `sample_data` — Выборка. [`Integer`](/sql-reference/data-types/int-uint) или [`Float`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)
* `population_mean` — Известное среднее генеральной совокупности, с которым выполняется проверка (обычно константа). [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Возвращаемое значение**

Возвращает кортеж из двух или четырёх элементов (если указан `confidence_level`): вычисленная t-статистика, вычисленное p-значение (двустороннее), [вычисленная нижняя граница доверительного интервала], [вычисленная верхняя граница доверительного интервала]. Доверительные интервалы относятся к выборочному среднему при заданном уровне доверия. [`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple) или [`Tuple(Float64, Float64, Float64, Float64)`](/sql-reference/data-types/tuple)

**Примеры**

**Без доверительного интервала**

```sql title=Query
SELECT studentTTestOneSample()(value, 20.0) FROM t;
```
