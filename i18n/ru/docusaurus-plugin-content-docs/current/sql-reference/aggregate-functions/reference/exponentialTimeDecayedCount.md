## exponentialTimeDecayedCount \{#exponentialTimeDecayedCount\}

Добавлена в версии: v21.12

Возвращает кумулятивное значение с экспоненциальным затуханием для временного ряда в момент времени с индексом `t`.

**Синтаксис**

```sql
exponentialTimeDecayedCount(x)(t)
```

**Параметры**

* `x` — Период полураспада. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Аргументы**

* `t` — Время. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal) или [`DateTime`](/sql-reference/data-types/datetime) или [`DateTime64`](/sql-reference/data-types/datetime64)

**Возвращаемое значение**

Возвращает значение накопленного экспоненциального затухания в заданный момент времени. [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Использование оконной функции с визуализацией**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 20, 50) AS bar
FROM
(
    SELECT
        (number % 5) = 0 AS value,
        number AS time,
        exponentialTimeDecayedCount(10)(time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
)
```
