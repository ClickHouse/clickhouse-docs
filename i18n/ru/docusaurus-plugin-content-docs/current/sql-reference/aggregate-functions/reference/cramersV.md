## cramersV \{#cramersV\}

Введено в версии v22.1

[Cramer&#39;s V](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V) (иногда также называемая Cramer&#39;s phi) — это мера взаимосвязи между двумя столбцами в таблице.
Результат функции `cramersV` лежит в диапазоне от 0 (что соответствует отсутствию связи между переменными) до 1 и может достигать 1 только тогда, когда каждое значение полностью определяется другим.
Эту меру можно рассматривать как взаимосвязь между двумя переменными, выраженную в процентах от их максимально возможной вариации.

:::note
Вариант Cramer&#39;s V с коррекцией смещения см.: [cramersVBiasCorrected](/sql-reference/aggregate-functions/reference/cramersvbiascorrected)
:::

**Синтаксис**

```sql
cramersV(column1, column2)
```

**Аргументы**

* `column1` — Первый столбец для сравнения. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)
* `column2` — Второй столбец для сравнения. [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Возвращаемое значение**

Возвращает значение от 0 (что соответствует отсутствию зависимости между значениями столбцов) до 1 (полная зависимость). [`Float64`](/sql-reference/data-types/float)

**Примеры**

**Отсутствие взаимосвязи между столбцами**

```sql title=Query
SELECT
    cramersV(a, b)
FROM
    (
        SELECT
            number % 3 AS a,
            number % 5 AS b
        FROM
            numbers(150)
    )
```

```response title=Response
┌─cramersV(a, b)─┐
│              0 │
└────────────────┘
```

**Высокая взаимосвязь между столбцами**

```sql title=Query
SELECT
    cramersV(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            number % 5 AS b
        FROM
            numbers(150)
    )
```

```response title=Response
┌─────cramersV(a, b)─┐
│ 0.8944271909999159 │
└────────────────────┘
```
