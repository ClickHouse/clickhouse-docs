## sumKahan \{#sumKahan\}

Впервые появилась в: v1.1

Вычисляет сумму чисел с использованием [алгоритма компенсированного суммирования Кэхэна](https://en.wikipedia.org/wiki/Kahan_summation_algorithm).
Работает медленнее, чем функция [`sum`](/sql-reference/aggregate-functions/reference/sum).
Компенсация применяется только к типам данных [Float](/sql-reference/data-types/float).

**Синтаксис**

```sql
sumKahan(x)
```

**Аргументы**

* `x` — входное значение. [`Integer`](/sql-reference/data-types/int-uint) или [`Float`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Возвращаемое значение**

Возвращает сумму чисел: [`(U)Int*`](/sql-reference/data-types/int-uint) или [`Float*`](/sql-reference/data-types/float) или [`Decimal`](/sql-reference/data-types/decimal)

**Примеры**

**Демонстрация повышения точности с помощью алгоритма суммирования Кэхэна**

```sql title=Query
SELECT sum(0.1), sumKahan(0.1) FROM numbers(10);
```

```response title=Response
┌───────────sum(0.1)─┬─sumKahan(0.1)─┐
│ 0.9999999999999999 │             1 │
└────────────────────┴───────────────┘
```
