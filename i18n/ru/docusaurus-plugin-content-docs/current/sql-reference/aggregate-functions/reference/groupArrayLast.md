# groupArrayLast {#grouparraylast}

Синтаксис: `groupArrayLast(max_size)(x)`

Создаёт массив, содержащий последние значения аргумента.
Например, `groupArrayLast(1)(x)` эквивалентен `[anyLast (x)]`.

В некоторых случаях вы всё ещё можете полагаться на порядок выполнения. Это относится к случаям, когда `SELECT` выполняется по подзапросу, использующему `ORDER BY`, если результат подзапроса достаточно небольшой по размеру.

**Пример**

Запрос:

```sql
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10)
```

Результат:

```text
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

По сравнению с `groupArray`:

```sql
SELECT groupArray(2)(number+1) numbers FROM numbers(10)
```

```text
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
