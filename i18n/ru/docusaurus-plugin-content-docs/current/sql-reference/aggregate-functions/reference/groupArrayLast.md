## groupArrayLast \{#groupArrayLast\}

Впервые появилась в: v23.1

Создает массив из последних значений аргумента.
Например, `groupArrayLast(1)(x)` эквивалентно `[anyLast(x)]`.
В некоторых случаях вы по‑прежнему можете полагаться на порядок вычислений.
Это относится к случаям, когда результат SELECT получается из подзапроса, использующего ORDER BY, если результат подзапроса достаточно мал.

**Синтаксис**

```sql
groupArrayLast(max_size)(x)
```

**Параметры**

* `max_size` — Максимальный размер результирующего массива. [`UInt64`](/sql-reference/data-types/int-uint)

**Аргументы**

* `max_size` — Максимальный размер результирующего массива. [`UInt64`](/sql-reference/data-types/int-uint)
* `x` — Аргумент (имя столбца или выражение). [`Any`](/sql-reference/data-types)

**Возвращаемое значение**

Возвращает массив последних значений аргумента. [`Array(T)`](/sql-reference/data-types/array)

**Примеры**

**Пример использования**

```sql title=Query
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

**Сравнение с groupArray**

```sql title=Query
-- Compare with groupArray (first values)
SELECT groupArray(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
