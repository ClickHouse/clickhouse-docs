# Домены

Домены — это специализированные типы данных, которые добавляют дополнительные возможности поверх существующих базовых типов, сохраняя при этом формат представления базового типа данных при передаче и хранении без изменений. В настоящее время ClickHouse не поддерживает пользовательские домены.

Домены можно использовать везде, где может использоваться соответствующий базовый тип, например:

- Создание столбца доменного типа
- Чтение и запись значений из столбца домена и в столбец домена
- Использование в качестве индекса, если базовый тип может использоваться как индекс
- Вызов функций со значениями столбца домена

### Дополнительные возможности доменов {#extra-features-of-domains}

- Явное имя типа столбца в `SHOW CREATE TABLE` или `DESCRIBE TABLE`
- Ввод данных в удобочитаемом формате с помощью `INSERT INTO domain_table(domain_column) VALUES(...)`
- Вывод данных в удобочитаемом формате для `SELECT domain_column FROM domain_table`
- Загрузка данных из внешнего источника в удобочитаемом формате: `INSERT INTO domain_table FORMAT CSV ...`

### Ограничения {#limitations}

- Невозможно преобразовать индексный столбец базового типа в доменный тип с помощью `ALTER TABLE`.
- Невозможно неявно преобразовать строковые значения в значения домена при вставке данных из другого столбца или таблицы.
- Домен не накладывает ограничений на сохраняемые значения.
