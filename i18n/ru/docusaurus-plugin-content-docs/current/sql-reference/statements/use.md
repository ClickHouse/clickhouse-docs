# Инструкция USE

```sql
USE [DATABASE] db
```

Позволяет установить текущую базу данных для сеанса.

Текущая база данных используется для поиска таблиц, если база данных для таблицы явно не указана в запросе (через точку перед именем таблицы).

Этот запрос нельзя выполнять при использовании протокола HTTP, так как в нём отсутствует понятие сеанса.
