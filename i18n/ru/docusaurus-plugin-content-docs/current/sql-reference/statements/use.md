# Оператор USE \{#use-statement\}

```sql
USE [DATABASE] db
```

Позволяет задать текущую базу данных для сессии.

Текущая база данных используется для поиска таблиц в тех случаях, когда база данных явно не указана в запросе с помощью записи с точкой перед именем таблицы.

Этот запрос невозможно выполнить при использовании протокола HTTP, поскольку в нём отсутствует понятие сессии.
