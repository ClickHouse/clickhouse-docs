# Табличная функция merge

Создаёт временную таблицу [Merge](../../engines/table-engines/special/merge.md).
Схема таблицы формируется на основе исходных таблиц путём объединения их столбцов и выведения общих типов.
Доступны те же виртуальные столбцы, что и для движка таблиц [Merge](../../engines/table-engines/special/merge.md).



## Синтаксис {#syntax}


```sql
merge(['db_name',] 'tables_regexp')
```

## Аргументы {#arguments}

| Аргумент        | Описание                                                                                                                                                                                                                                                                               |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `db_name`       | Возможные значения (необязательный параметр, по умолчанию `currentDatabase()`):<br/> - имя базы данных,<br/> - константное выражение, возвращающее строку с именем базы данных, например `currentDatabase()`,<br/> - `REGEXP(expression)`, где `expression` — регулярное выражение для сопоставления имён баз данных. |
| `tables_regexp` | Регулярное выражение для сопоставления имён таблиц в указанной базе данных или базах данных.                                                                                                                                                                                                 |
