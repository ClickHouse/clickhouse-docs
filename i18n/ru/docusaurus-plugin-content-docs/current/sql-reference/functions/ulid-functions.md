## ULIDStringToDateTime {#ULIDStringToDateTime}

Добавлена в версии: v23.3

Функция извлекает временную метку из [ULID]((https://github.com/ulid/spec).

**Синтаксис**

```sql
ULIDStringToDateTime(ulid[, timezone])
```

**Аргументы**

- `ulid` — входной ULID. [`String`](/sql-reference/data-types/string) или [`FixedString(26)`](/sql-reference/data-types/fixedstring)
- `timezone` — необязательный параметр. Название часового пояса для возвращаемого значения. [`String`](/sql-reference/data-types/string)

**Возвращаемое значение**

Временная метка с точностью до миллисекунд. [`DateTime64(3)`](/sql-reference/data-types/datetime64)

**Примеры**

**Пример использования**

```sql title=Запрос
SELECT ULIDStringToDateTime('01GNB2S2FGN2P93QPXDNB4EN2R')
```

```response title=Результат
┌─ULIDStringToDateTime('01GNB2S2FGN2P93QPXDNB4EN2R')─┐
│                            2022-12-28 00:40:37.616 │
└────────────────────────────────────────────────────┘
```


## generateULID {#generateULID}

Введена в версии: v23.2

Генерирует [универсальный уникальный лексикографически сортируемый идентификатор (ULID)](https://github.com/ulid/spec).

**Синтаксис**

```sql
generateULID([x])
```

**Аргументы**

- `x` — Необязательный параметр. Выражение, возвращающее значение любого из поддерживаемых типов данных. Возвращаемое значение игнорируется, но само выражение используется для обхода [устранения общих подвыражений](/sql-reference/functions/overview#common-subexpression-elimination) при многократном вызове функции в одном запросе. [`Any`](/sql-reference/data-types)

**Возвращаемое значение**

Возвращает ULID. [`FixedString(26)`](/sql-reference/data-types/fixedstring)

**Примеры**

**Пример использования**

```sql title=Запрос
SELECT generateULID()
```

```response title=Результат
┌─generateULID()─────────────┐
│ 01GNB2S2FGN2P93QPXDNB4EN2R │
└────────────────────────────┘
```

**Пример использования для генерации нескольких значений в одной строке**

```sql title=Запрос
SELECT generateULID(1), generateULID(2)
```

```response title=Результат
┌─generateULID(1)────────────┬─generateULID(2)────────────┐
│ 01GNB2SGG4RHKVNT9ZGA4FFMNP │ 01GNB2SGG4V0HMQVH4VBVPSSRB │
└────────────────────────────┴────────────────────────────┘
```
