# Функции для работы с битовыми картами

Битовые карты можно создать двумя способами. Первый способ — с помощью агрегатной функции groupBitmap с суффиксом `-State`, второй способ — построение битовой карты из объекта Array.

## bitmapAnd {#bitmapAnd}

Введена в версии: v20.1

Вычисляет логическое пересечение (AND) двух битовых карт.

**Синтаксис**

```sql
bitmapAnd(bitmap1, bitmap2)
```

**Аргументы**

- `bitmap1` — первая битовая карта. [`AggregateFunction(groupBitmap, T)`](/sql-reference/data-types/aggregatefunction).
- `bitmap2` — вторая битовая карта. [`AggregateFunction(groupBitmap, T)`](/sql-reference/data-types/aggregatefunction).

**Возвращаемое значение**

Возвращает битовую карту, содержащую биты, присутствующие в обеих входных битовых картах. [`AggregateFunction(groupBitmap, T)`](/sql-reference/data-types/aggregatefunction)

**Примеры**

**Пример использования**

```sql title=Запрос
SELECT bitmapToArray(bitmapAnd(bitmapBuild([1, 2, 3]), bitmapBuild([3, 4, 5]))) AS res;
```

```response title=Ответ
┌─res─┐
│ [3] │
└─────┘
```

```response title=Response
└────────┘
```

## bitmapSubsetInRange {#bitmapSubsetInRange}

**Возвращаемое значение**

Возвращает битовую карту, содержащую только установленные биты в заданном диапазоне. [`AggregateFunction(groupBitmap, T)`](/sql-reference/data-types/aggregatefunction)

**Примеры**

**Пример использования**

```sql title=Запрос
SELECT bitmapToArray(bitmapSubsetInRange(bitmapBuild([1, 2, 3, 4, 5]), 2, 5)) AS res;
```

```response title=Результат
┌─res───────┐
│ [2, 3, 4] │
└───────────┘
```

Возвращает битовую карту, содержащую не более `cardinality_limit` установленных битов, начиная с позиции `range_start`. [`AggregateFunction(groupBitmap, T)`](/sql-reference/data-types/aggregatefunction)

- `cardinality_limit` — максимальное количество установленных битов для включения в подмножество. [`UInt32`](/sql-reference/data-types/int-uint).

**Возвращаемое значение**

Возвращает битовую карту, содержащую не более `cardinality_limit` установленных битов, начиная после пропуска `offset` установленных битов в порядке возрастания. [`AggregateFunction(groupBitmap, T)`](/sql-reference/data-types/aggregatefunction).
