Набор данных содержит все события на GitHub с 2011 года по 6 декабря 2020 года, его размер — 3,1 миллиарда записей. Объём загружаемых данных — 75 ГБ, и потребуется до 200 ГБ дискового пространства при хранении в таблице с компрессией lz4.

Полное описание набора данных, аналитические выводы, инструкции по скачиванию и интерактивные запросы опубликованы [здесь](https://ghe.clickhouse.tech/).
