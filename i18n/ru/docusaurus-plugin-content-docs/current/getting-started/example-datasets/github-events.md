Набор данных содержит все события на GitHub с 2011 года по 6 декабря 2020 года, его объем — 3,1 миллиарда записей. Размер скачиваемого архива — 75 ГБ, а для хранения в таблице со сжатием lz4 потребуется до 200 ГБ дискового пространства.

Полное описание набора данных, аналитика, инструкции по загрузке и интерактивные запросы доступны [здесь](https://ghe.clickhouse.tech/).
