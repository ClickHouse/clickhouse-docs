Набор данных содержит все события на GitHub с 2011 года по 6 декабря 2020 года, его объём составляет 3,1 миллиарда записей. Объём загружаемого архива — 75 ГБ, и потребуется до 200 ГБ места на диске, если хранить данные в таблице сжатой с помощью LZ4.

Полное описание набора данных, аналитические материалы, инструкции по загрузке и интерактивные запросы опубликованы [здесь](https://ghe.clickhouse.tech/).
