Набор данных содержит все события на GitHub с 2011 года по 6 декабря 2020 года; его объем составляет 3,1 миллиарда записей. Размер загрузки — 75 ГБ, и потребуется до 200 ГБ дискового пространства при хранении в таблице со сжатием LZ4.

Полное описание набора данных, аналитические материалы, инструкции по загрузке и интерактивные запросы размещены [здесь](https://ghe.clickhouse.tech/).
