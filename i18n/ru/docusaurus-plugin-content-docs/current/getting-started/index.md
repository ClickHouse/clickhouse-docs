# Руководства и примеры наборов данных

У нас есть множество ресурсов, которые помогут вам начать работу и понять, как работает ClickHouse:

- Если вам нужно установить и запустить ClickHouse, ознакомьтесь с нашим [Быстрым стартом](/get-started/quick-start)
- [Руководство по ClickHouse](../tutorial.md) анализирует набор данных о поездках такси в Нью-Йорке

Кроме того, примеры наборов данных предоставляют отличную возможность для работы с ClickHouse,
изучения важных техник и приёмов, а также демонстрируют, как воспользоваться многочисленными мощными
функциями ClickHouse. Примеры наборов данных включают:
