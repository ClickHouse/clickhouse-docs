# Машинное обучение и GenAI

ClickHouse идеально подходит в качестве базы данных реального времени для машинного обучения.
С ClickHouse стало как никогда просто использовать GenAI для работы с аналитическими данными.
В этом разделе вы найдете несколько руководств о том, как ClickHouse используется для 
машинного обучения и GenAI.

| [AI chat](/use-cases/AI_ML/AIChat)                                   | В этом руководстве объясняется, как включить и использовать функцию AI Chat в ClickHouse Cloud Console.                                         |
| [MCP](/use-cases/AI/MCP)                                             | Набор руководств, помогающих вам настроить использование Model Context Protocol (MCP) с ClickHouse                                               |
| [AI-powered SQL generation](/use-cases/AI/ai-powered-sql-generation) | Эта функция позволяет пользователям описывать свои требования к данным на естественном языке, после чего система переводит их в соответствующие SQL-запросы. |
