Если вы хотите поэкспериментировать с тестовым набором данных, скачайте следующий пример для nginx.

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

:::note
Эти данные были собраны с экземпляра Nginx, настроенного на вывод логов в формате JSON для упрощения их разбора. Конфигурацию Nginx для этих логов см. в разделе [&quot;Monitoring Nginx Logs with ClickStack&quot;](/use-cases/observability/clickstack/integrations/nginx#configure-nginx).
:::
