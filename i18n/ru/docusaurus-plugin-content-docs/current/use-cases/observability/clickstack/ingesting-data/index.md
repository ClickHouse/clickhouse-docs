ClickStack предоставляет несколько способов приёма данных наблюдаемости в ваш экземпляр ClickHouse. Независимо от того, собираете ли вы логи, метрики, трассировки или данные о сессиях, вы можете использовать коллектор OpenTelemetry (OTel) как единую точку приёма данных или задействовать платформенные интеграции для специализированных сценариев использования.
