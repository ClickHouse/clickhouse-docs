ClickStack предоставляет несколько способов приёма данных наблюдаемости в ваш экземпляр ClickHouse. Независимо от того, собираете ли вы логи, метрики, трейсы или данные сессий, вы можете использовать коллектор OpenTelemetry (OTel) как единую точку ингестии или задействовать платформенные интеграции для специфических сценариев.
