ClickStack можно развернуть несколькими способами в зависимости от ваших операционных предпочтений и требований:

- [Managed ClickStack](/use-cases/observability/clickstack/getting-started/managed) — **наиболее простой и быстрый** способ начать работу: развертывание **Managed ClickStack** в **ClickHouse Cloud**.
- [Open-Source ClickStack](/use-cases/observability/clickstack/getting-started/oss) — для тех, кто хочет самостоятельно развернуть и эксплуатировать полностью open source архитектуру.
