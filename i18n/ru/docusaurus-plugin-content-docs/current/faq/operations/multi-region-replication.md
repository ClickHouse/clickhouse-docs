# Поддерживает ли ClickHouse мультирегиональную репликацию? {#does-clickhouse-support-multi-region-replication}

Краткий ответ — «да». Однако мы рекомендуем поддерживать задержку между всеми регионами/дата-центрами в пределах двузначных значений (в миллисекундах), иначе производительность записи снизится из-за прохождения через протокол распределённого консенсуса. Например, репликация между побережьями США, скорее всего, будет работать нормально, а между США и Европой — нет.

С точки зрения конфигурации нет никакой разницы по сравнению с однорегиональной репликацией — просто используйте хосты, расположенные в разных местах, для реплик.

Для получения дополнительной информации см. [полную статью о репликации данных](../../engines/table-engines/mergetree-family/replication.md).
