# Поддерживает ли ClickHouse мультирегиональную репликацию? {#does-clickhouse-support-multi-region-replication}

Короткий ответ — да. Однако мы рекомендуем поддерживать задержку между всеми регионами/дата-центрами в пределах двузначных значений (миллисекунд), иначе производительность записи снизится из-за прохождения через протокол распределённого консенсуса. Например, репликация между побережьями США, скорее всего, будет работать нормально, а между США и Европой — нет.

С точки зрения конфигурации нет никакой разницы по сравнению с однорегиональной репликацией — просто используйте для реплик хосты, расположенные в разных местах.

Для получения дополнительной информации см. [полную статью о репликации данных](../../engines/table-engines/mergetree-family/replication.md).
