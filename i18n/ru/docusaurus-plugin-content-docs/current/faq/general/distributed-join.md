# Поддерживает ли ClickHouse распределённый JOIN?

ClickHouse поддерживает распределённый JOIN в кластере.

Когда данные соразмещены в кластере (например, JOIN выполняется по идентификатору пользователя, который также является ключом шардинга), ClickHouse позволяет выполнять JOIN без передачи данных по сети.

Когда данные не соразмещены, ClickHouse позволяет выполнять broadcast JOIN, при котором части присоединяемых данных распространяются по узлам кластера.

По состоянию на 2025 год ClickHouse не выполняет алгоритм shuffle-join, который подразумевает перераспределение обеих сторон соединения по сети по всему кластеру в соответствии с ключами соединения.
