# Поддерживает ли ClickHouse распределённый JOIN?

ClickHouse поддерживает распределённый JOIN в кластере.

Когда данные расположены совместно на одних и тех же узлах кластера (например, JOIN выполняется по идентификатору пользователя, который также является ключом шардирования), ClickHouse позволяет выполнять JOIN без передачи данных по сети.

Когда данные не расположены совместно, ClickHouse позволяет выполнять широковещательный JOIN, при котором части присоединяемых данных распределяются по узлам кластера.

По состоянию на 2025 год ClickHouse не выполняет алгоритм shuffle-join, который предполагает перераспределение по сети обеих сторон соединения по всему кластеру в соответствии с ключами соединения.
