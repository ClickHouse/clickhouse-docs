# Что означает «ClickHouse»? \\{#what-does-clickhouse-mean\\}

Это сочетание «**Click**stream» и «Data ware**House**». Название происходит от исходного сценария использования в Yandex.Metrica, где ClickHouse должен был хранить записи обо всех кликах пользователей со всего интернета и до сих пор успешно с этим справляется. Подробнее об этом сценарии можно прочитать на странице [история ClickHouse](../../about-us/history.md).

У этого двойного смысла есть два следствия:

- Единственно правильный способ написания Click**H**ouse — с заглавной буквой H.
- Если вам нужно сократить название, используйте **CH**. По историческим причинам сокращение CK также популярно в Китае, в основном потому, что один из первых докладов о ClickHouse на китайском языке использовал именно эту форму.

:::info    
Спустя многие годы после того, как ClickHouse получил своё имя, такой подход к объединению двух самостоятельных по смыслу слов был отмечен как лучший способ назвать базу данных в [исследовании Энди Павло](https://www.cs.cmu.edu/~pavlo/blog/2020/03/on-naming-a-database-management-system.html), доцента по базам данных в Университете Карнеги-Меллона. ClickHouse разделил с Postgres его награду «лучшее название базы данных всех времён».
:::
