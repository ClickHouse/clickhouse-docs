# ClickHouse Cloud {#clickhouse-cloud}

ClickHouse Cloud — это облачный сервис, созданный создателями популярной OLAP‑базы данных с открытым исходным кодом ClickHouse. 
Вы можете опробовать ClickHouse Cloud, [оформив бесплатный пробный период](https://console.clickhouse.cloud/signUp).

## Преимущества ClickHouse Cloud {#clickhouse-cloud-benefits}

Ниже перечислены некоторые преимущества использования ClickHouse Cloud:

* **Быстрое получение результата**: Начните работу мгновенно, без необходимости подбирать размер и масштабировать кластер.
* **Бесшовное масштабирование**: Автоматическое масштабирование подстраивается под переменные нагрузки, чтобы вам не приходилось избыточно выделять ресурсы под пиковое потребление.
* **Serverless-операции**: Мы берём на себя подбор размера, масштабирование, безопасность, надёжность и обновления.
* **Прозрачное ценообразование**: Платите только за фактическое потребление с возможностью резервирования ресурсов и управления масштабированием.
* **Полная стоимость владения**: Лучшее соотношение цены и производительности и низкие административные накладные расходы.
* **Широкая экосистема**: Используйте ваши любимые коннекторы данных, инструменты визуализации, SQL- и языковые клиенты.

## Какую версию ClickHouse использует ClickHouse Cloud? {#what-version-of-clickhouse-does-clickhouse-cloud-use}

ClickHouse Cloud периодически обновляет ваш сервис до более новых версий, включающих исправления, новые функции и улучшения производительности. После публикации основной версии базы данных в open source мы выполняем дополнительное тестирование в нашей облачной staging-среде, что обычно занимает 6–8 недель до вывода версии в продакшен-среду. Внедрение обновлений выполняется поэтапно в зависимости от облачного провайдера, типа сервиса и региона.

Вы можете задать график обновлений для своего сервиса ClickHouse Cloud, подписавшись на определённый канал релизов. Например, мы предлагаем канал обновлений ["Fast" Release Channel](/manage/updates#fast-release-channel-early-upgrades) для получения обновлений раньше стандартного графика релизов, а также канал обновлений ["Slow" Release Channel](/manage/updates#slow-release-channel-deferred-upgrades) и другие, более гибкие варианты планирования отложенных обновлений.

Общий обзор процесса обновления ClickHouse Cloud, включая гарантии обратной совместимости, см. в справочной документации по разделу [Upgrades](/manage/updates).
