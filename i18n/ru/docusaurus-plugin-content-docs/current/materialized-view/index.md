| [Incremental materialized view](/materialized-view/incremental-materialized-view) | Позволяют перенести вычислительные затраты с момента выполнения запроса на момент вставки данных, что делает запросы `SELECT` быстрее.                                                          |
| [Refreshable materialized view](/materialized-view/refreshable-materialized-view) | Концептуально похожи на инкрементальные материализованные представления, но требуют периодического выполнения запроса по всему набору данных; результаты этого запроса сохраняются в целевой таблице для дальнейшего использования при выборках. |
