# Моделирование данных \{#data-modeling\}

В этом разделе рассматривается моделирование данных в ClickHouse и следующие темы:

| Страница                                                        | Описание                                                                                                                                                                                      |
|-----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schema Design](/data-modeling/schema-design)                   | Рассматривается проектирование схемы ClickHouse для достижения оптимальной производительности с учётом таких факторов, как запросы, обновления данных, задержки и объёмы.                   |
| [Dictionary](/dictionary)                                       | Объясняется, как определять и использовать словари для улучшения производительности запросов и обогащения данных.                                                                            |
| [Materialized Views](/materialized-views)                       | Информация о материализованных представлениях и обновляемых материализованных представлениях в ClickHouse.                                                                                   |
| [Projections](/data-modeling/projections)                       | Информация о проекциях в ClickHouse.                                                                                                                                                          |
| [Data Compression](/data-compression/compression-in-clickhouse) | Рассматриваются различные методы сжатия в ClickHouse и способы оптимизации хранения данных и производительности запросов за счёт выбора подходящего метода сжатия под ваши типы данных и рабочие нагрузки. |
| [Denormalizing Data](/data-modeling/denormalization)            | Рассматривается подход к денормализации, используемый в ClickHouse, который направлен на повышение производительности запросов за счёт хранения связанных данных в одной таблице.           |
