# system.settings&#95;changes {#systemsettings&#95;changes}

Содержит информацию об изменениях настроек в предыдущих версиях ClickHouse.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `type` ([Enum8(&#39;Session&#39; = 0, &#39;MergeTree&#39; = 1)](../../sql-reference/data-types/)) — Группа настроек (Session, MergeTree...)
* `version` ([String](../../sql-reference/data-types/)) — Версия сервера ClickHouse.
* `changes` ([Array(Tuple(name String, previous&#95;value String, new&#95;value String, reason String))](../../sql-reference/data-types/)) — Список изменений в настройках, которые привели к изменению поведения ClickHouse.

{/*AUTOGENERATED_END*/ }

**Пример**

```sql
SELECT *
FROM system.settings_changes
WHERE version = '23.5'
FORMAT Vertical
```

```text
Row 1:
──────
type:    Core
version: 23.5
changes: [('input_format_parquet_preserve_order','1','0','Позволяет читателю Parquet изменять порядок строк для улучшения параллелизма.'),('parallelize_output_from_storages','0','1','Позволяет использовать параллелизм при выполнении запросов, читающих данные из file/url/s3/и т. д. Может изменить порядок строк.'),('use_with_fill_by_sorting_prefix','0','1','Столбцы, предшествующие столбцам WITH FILL в предложении ORDER BY, формируют префикс сортировки. Строки с разными значениями в префиксе сортировки заполняются независимо'),('output_format_parquet_compliant_nested_types','0','1','Изменяет имя внутреннего поля в схеме выходного файла Parquet.')]
```
