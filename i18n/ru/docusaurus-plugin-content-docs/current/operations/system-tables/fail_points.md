## Описание \{#description\}

Содержит список всех доступных failpoint&#39;ов, зарегистрированных на сервере, с указанием их типа и того, включены ли они в данный момент.

Failpoint&#39;ы могут быть включены и отключены во время работы с помощью команд `SYSTEM ENABLE FAILPOINT` и `SYSTEM DISABLE FAILPOINT`.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя failpoint.
* `type` ([Enum8(&#39;once&#39; = 0, &#39;regular&#39; = 1, &#39;pauseable&#95;once&#39; = 2, &#39;pauseable&#39; = 3)](../../sql-reference/data-types/)) — Тип failpoint: &#39;once&#39; срабатывает один раз и затем автоматически отключается, &#39;regular&#39; срабатывает каждый раз, &#39;pauseable&#95;once&#39; один раз блокирует выполнение, &#39;pauseable&#39; блокирует выполнение каждый раз, пока оно не будет возобновлено.
* `enabled` ([UInt8](../../sql-reference/data-types/)) — Указывает, включён ли failpoint в данный момент (1) или отключён (0).

{/*AUTOGENERATED_END*/ }

## Пример \{#example\}

```sql title="Query"
SYSTEM ENABLE FAILPOINT replicated_merge_tree_insert_retry_pause;
SELECT * FROM system.fail_points WHERE enabled = 1
```
