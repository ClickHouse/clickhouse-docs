## Описание \{#description\}

Содержит активные роли текущего пользователя. Команда `SET ROLE` изменяет содержимое этой таблицы.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — Имя роли.
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — 1, если у роли есть привилегия ADMIN OPTION.
* `is_default` ([UInt8](../../sql-reference/data-types/)) — 1, если роль задана по умолчанию.

{/*AUTOGENERATED_END*/ }
