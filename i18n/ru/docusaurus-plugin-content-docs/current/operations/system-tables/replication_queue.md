## Описание \{#description\}

Содержит информацию о задачах из очередей репликации, хранящихся в ClickHouse Keeper или ZooKeeper, для таблиц семейства `ReplicatedMergeTree`.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Имя базы данных.
* `table` ([String](../../sql-reference/data-types/)) — Имя таблицы.
* `replica_name` ([String](../../sql-reference/data-types/)) — Имя реплики в ClickHouse Keeper. У разных реплик одной и той же таблицы разные имена.
* `position` ([UInt32](../../sql-reference/data-types/)) — Позиция задачи в очереди.
* `node_name` ([String](../../sql-reference/data-types/)) — Имя узла в ClickHouse Keeper.
* `type` ([String](../../sql-reference/data-types/)) — Тип задачи в очереди, одно из следующих значений: • GET&#95;PART — Получить часть с другой реплики, • ATTACH&#95;PART — Присоединить часть, возможно, из этой же реплики (если она найдена в папке detached). Это можно рассматривать как GET&#95;PART с некоторыми оптимизациями, поскольку эти операции почти идентичны, • MERGE&#95;PARTS — Выполнить слияние частей, • DROP&#95;RANGE — Удалить части в указанной партиции в заданном диапазоне номеров. • CLEAR&#95;COLUMN — Примечание: Устарело. Удалить конкретный столбец из указанной партиции. • CLEAR&#95;INDEX — Примечание: Устарело. Удалить конкретный индекс из указанной партиции. • REPLACE&#95;RANGE — Удалить определённый диапазон частей и заменить их новыми. • MUTATE&#95;PART — Применить одну или несколько мутаций к части. • ALTER&#95;METADATA — Применить изменение ALTER в соответствии с глобальными путями /metadata и /columns.
* `create_time` ([DateTime](../../sql-reference/data-types/)) — Дата и время, когда задача была отправлена на выполнение.
* `required_quorum` ([UInt32](../../sql-reference/data-types/)) — Количество реплик, ожидающих подтверждения завершения задачи. Этот столбец актуален только для задачи GET&#95;PARTS.
* `source_replica` ([String](../../sql-reference/data-types/)) — Имя исходной реплики.
* `new_part_name` ([String](../../sql-reference/data-types/)) — Имя новой части.
* `parts_to_merge` ([Array(String)](../../sql-reference/data-types/)) — Имена частей для слияния или обновления.
* `is_detach` ([UInt8](../../sql-reference/data-types/)) — Флаг указывает, находится ли задача DETACH&#95;PARTS в очереди.
* `is_currently_executing` ([UInt8](../../sql-reference/data-types/)) — Флаг указывает, выполняется ли конкретная задача в данный момент.
* `num_tries` ([UInt32](../../sql-reference/data-types/)) — Количество неудачных попыток выполнить задачу.
* `last_exception` ([String](../../sql-reference/data-types/)) — Текст сообщения о последней возникшей ошибке (если была).
* `last_exception_time` ([DateTime](../../sql-reference/data-types/)) — Дата и время, когда произошла последняя ошибка.
* `last_attempt_time` ([DateTime](../../sql-reference/data-types/)) — Дата и время последней попытки выполнить задачу.
* `num_postponed` ([UInt32](../../sql-reference/data-types/)) — Количество случаев, когда действие откладывалось.
* `postpone_reason` ([String](../../sql-reference/data-types/)) — Причина, по которой задача была отложена.
* `last_postpone_time` ([DateTime](../../sql-reference/data-types/)) — Дата и время, когда задача была отложена в последний раз.
* `merge_type` ([String](../../sql-reference/data-types/)) — Тип текущего слияния. Пусто, если это mutation.
