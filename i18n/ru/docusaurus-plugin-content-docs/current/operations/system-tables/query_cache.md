## Описание \{#description\}

Показывает содержимое [кэша запросов](../query-cache.md).

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `query` ([String](../../sql-reference/data-types/)) — Строка запроса.
* `query_id` ([String](../../sql-reference/data-types/)) — Идентификатор запроса.
* `result_size` ([UInt64](../../sql-reference/data-types/)) — Размер записи кэша запросов.
* `tag` ([LowCardinality(String)](../../sql-reference/data-types/)) — Тег записи кэша запросов.
* `stale` ([UInt8](../../sql-reference/data-types/)) — Указывает, устарела ли запись кэша запросов.
* `shared` ([UInt8](../../sql-reference/data-types/)) — Указывает, используется ли запись кэша запросов совместно несколькими пользователями.
* `compressed` ([UInt8](../../sql-reference/data-types/)) — Указывает, сжата ли запись кэша запросов.
* `expires_at` ([DateTime](../../sql-reference/data-types/)) — Время, когда запись кэша запросов становится устаревшей.
* `key_hash` ([UInt64](../../sql-reference/data-types/)) — Хэш строки запроса, используемый в качестве ключа для поиска записей кэша запросов.

{/*AUTOGENERATED_END*/ }
