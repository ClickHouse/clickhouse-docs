## Описание \{#description\}

Содержит информацию о кэшированных DNS-записях.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([String](../../sql-reference/data-types/)) — Имя хоста.
* `ip_address` ([String](../../sql-reference/data-types/)) — IP-адрес.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](../../sql-reference/data-types/)) — Семейство IP-адреса.
* `cached_at` ([DateTime](../../sql-reference/data-types/)) — Таймстамп помещения записи в кэш.

{/*AUTOGENERATED_END*/ }

## Пример \{#example\}

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
