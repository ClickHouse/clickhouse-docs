Содержит историю всех событий, произошедших с файловым кэшем объектов в удалённой файловой системе.

Эту таблицу можно безопасно очищать или удалять в любое время.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата события
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время события
* `query_id` ([String](/sql-reference/data-types/string)) — Идентификатор запроса
* `source_file_path` ([String](/sql-reference/data-types/string)) — Путь к сегменту файла в файловой системе
* `file_segment_range` ([Tuple(UInt64, UInt64)](/sql-reference/data-types/tuple)) — Диапазон сегмента файла
* `total_requested_range` ([Tuple(UInt64, UInt64)](/sql-reference/data-types/tuple)) — Полный диапазон чтения
* `key` ([String](/sql-reference/data-types/string)) — Ключ сегмента файла
* `offset` ([UInt64](/sql-reference/data-types/int-uint)) — Смещение сегмента файла
* `size` ([UInt64](/sql-reference/data-types/int-uint)) — Размер считываемых данных
* `read_type` ([String](/sql-reference/data-types/string)) — Тип чтения: READ&#95;FROM&#95;CACHE, READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE, READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/sql-reference/data-types/int-uint)) — Выполнялась ли попытка чтения из кэша
* `read_buffer_id` ([String](/sql-reference/data-types/string)) — Идентификатор буфера чтения во внутренней реализации
* `user_id` ([String](/sql-reference/data-types/string)) — Идентификатор пользователя, создавшего сегмент файла

{/*AUTOGENERATED_END*/ }
