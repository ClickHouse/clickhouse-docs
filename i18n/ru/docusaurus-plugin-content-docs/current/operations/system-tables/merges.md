## Описание \{#description\}

Содержит информацию о слияниях и мутациях частей, которые в данный момент выполняются для таблиц семейства MergeTree.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Имя базы данных, в которой находится таблица.
* `table` ([String](../../sql-reference/data-types/)) — Имя таблицы.
* `elapsed` ([Float64](../../sql-reference/data-types/)) — Время, прошедшее с начала слияния (в секундах).
* `progress` ([Float64](../../sql-reference/data-types/)) — Доля выполненной работы от 0 до 1.
* `num_parts` ([UInt64](../../sql-reference/data-types/)) — Количество частей, подлежащих слиянию.
* `source_part_names` ([Array(String)](../../sql-reference/data-types/)) — Список имён исходных частей.
* `result_part_name` ([String](../../sql-reference/data-types/)) — Имя части, которая будет сформирована в результате слияния.
* `source_part_paths` ([Array(String)](../../sql-reference/data-types/)) — Список путей для каждой исходной части.
* `result_part_path` ([String](../../sql-reference/data-types/)) — Путь к части, которая будет сформирована в результате слияния.
* `partition_id` ([String](../../sql-reference/data-types/)) — Идентификатор партиции, в которой происходит слияние.
* `partition` ([String](../../sql-reference/data-types/)) — Имя партиции
* `is_mutation` ([UInt8](../../sql-reference/data-types/)) — 1, если этот процесс является mutation части.
* `total_size_bytes_compressed` ([UInt64](../../sql-reference/data-types/)) — Общий размер сжатых данных в слитых фрагментах.
* `total_size_bytes_uncompressed` ([UInt64](../../sql-reference/data-types/)) — Общий размер несжатых данных в слитых фрагментах.
* `total_size_marks` ([UInt64](../../sql-reference/data-types/)) — Общее количество марок в слитых частях.
* `bytes_read_uncompressed` ([UInt64](../../sql-reference/data-types/)) — Количество прочитанных байтов в несжатом виде.
* `rows_read` ([UInt64](../../sql-reference/data-types/)) — Количество прочитанных строк.
* `bytes_written_uncompressed` ([UInt64](../../sql-reference/data-types/)) — Количество записанных байтов в несжатом виде.
* `rows_written` ([UInt64](../../sql-reference/data-types/)) — Количество записанных строк.
* `columns_written` ([UInt64](../../sql-reference/data-types/)) — Количество записанных столбцов (для алгоритма Вертикального слияния).
* `memory_usage` ([UInt64](../../sql-reference/data-types/)) — Объём памяти, используемой процессом слияния.
* `thread_id` ([UInt64](../../sql-reference/data-types/)) — Идентификатор потока процесса слияния.
* `merge_type` ([String](../../sql-reference/data-types/)) — Тип текущего слияния. Пусто, если это mutation.
* `merge_algorithm` ([String](../../sql-reference/data-types/)) — Алгоритм, используемый в текущем слиянии. Пусто, если это mutation.
