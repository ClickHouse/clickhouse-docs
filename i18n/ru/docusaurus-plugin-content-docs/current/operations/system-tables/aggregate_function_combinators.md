## Описание \{#description\}

Содержит список всех доступных комбинаторов агрегатных функций, которые можно применять к агрегатным функциям, чтобы изменять их поведение.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Название комбинатора.
* `is_internal` ([UInt8](../../sql-reference/data-types/)) — Указывает, предназначен ли этот комбинатор исключительно для внутреннего использования.

{/*AUTOGENERATED_END*/ }
