# system.time&#95;zones {#systemtime&#95;zones}

Содержит список часовых поясов, поддерживаемых сервером ClickHouse. Этот список часовых поясов может различаться в зависимости от версии ClickHouse.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `time_zone` ([String](../../sql-reference/data-types/)) — список поддерживаемых часовых поясов.

{/*AUTOGENERATED_END*/ }

**Пример**

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
