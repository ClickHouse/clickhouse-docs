# system.metrics

<SystemTableCloud />

Содержит метрики, которые могут быть вычислены мгновенно или имеют текущее значение. Например, количество одновременно обрабатываемых запросов или текущая задержка реплики. Эта таблица всегда актуальна.

Столбцы:

<!--AUTOGENERATED_START-->

- `metric` ([String](../../sql-reference/data-types/)) — Название метрики.
- `value` ([Int64](../../sql-reference/data-types/)) — Значение метрики.
- `description` ([String](../../sql-reference/data-types/)) — Описание метрики.
<!--AUTOGENERATED_END-->

Все поддерживаемые метрики можно найти в исходном файле [src/Common/CurrentMetrics.cpp](https://github.com/ClickHouse/ClickHouse/blob/master/src/Common/CurrentMetrics.cpp).

**Пример**

```sql
SELECT * FROM system.metrics LIMIT 10
```

```text
┌─metric───────────────────────────────┬─value─┬─description────────────────────────────────────────────────────────────┐
│ Query                                │     1 │ Количество выполняемых запросов                                        │
│ Merge                                │     0 │ Количество выполняемых фоновых слияний                                 │
│ PartMutation                         │     0 │ Количество мутаций (ALTER DELETE/UPDATE)                               │
│ ReplicatedFetch                      │     0 │ Количество частей данных, загружаемых из реплик                        │
│ ReplicatedSend                       │     0 │ Количество частей данных, отправляемых на реплики                      │
│ ReplicatedChecks                     │     0 │ Количество частей данных, проверяемых на согласованность               │
│ BackgroundMergesAndMutationsPoolTask │     0 │ Количество активных слияний и мутаций в соответствующем фоновом пуле   │
│ BackgroundFetchesPoolTask            │     0 │ Количество активных загрузок в соответствующем фоновом пуле            │
│ BackgroundCommonPoolTask             │     0 │ Количество активных задач в соответствующем фоновом пуле               │
│ BackgroundMovePoolTask               │     0 │ Количество активных задач в BackgroundProcessingPool для перемещений   │
└──────────────────────────────────────┴───────┴────────────────────────────────────────────────────────────────────────┘
```
