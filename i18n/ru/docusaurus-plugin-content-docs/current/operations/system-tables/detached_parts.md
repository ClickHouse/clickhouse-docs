## Описание \{#purpose\}

Содержит информацию об отсоединённых частях таблиц [MergeTree](../../engines/table-engines/mergetree-family/mergetree.md). Столбец `reason` указывает, по какой причине часть была отсоединена.

Для частей, отсоединённых пользователем, причина не указана. Такие части можно снова присоединить командой [ALTER TABLE ATTACH PARTITION|PART](/sql-reference/statements/alter/partition#attach-partitionpart).

Описание остальных столбцов см. в [system.parts](../../operations/system-tables/parts.md).

Если имя части некорректно, значения некоторых столбцов могут быть `NULL`. Такие части можно удалить командой [ALTER TABLE DROP DETACHED PART](/sql-reference/statements/alter/partition#drop-detached-partitionpart).

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Имя базы данных, которой принадлежит эта часть.
* `table` ([String](../../sql-reference/data-types/)) — Имя таблицы, которой принадлежит эта часть.
* `partition_id` ([Nullable(String)](../../sql-reference/data-types/)) — Идентификатор партиции, к которой относится эта часть.
* `name` ([String](../../sql-reference/data-types/)) — Имя части.
* `bytes_on_disk` ([UInt64](../../sql-reference/data-types/)) — Общий размер всех файлов этой части данных в байтах.
* `modification_time` ([DateTime](../../sql-reference/data-types/)) — Время изменения каталога с этой частью данных. Обычно соответствует времени, когда произошло отсоединение.
* `disk` ([String](../../sql-reference/data-types/)) — Имя диска, на котором хранится эта часть данных.
* `path` ([String](../../sql-reference/data-types/)) — Путь на диске к файлу этой части данных.
* `reason` ([Nullable(String)](../../sql-reference/data-types/)) — Причина, по которой эта часть была отсоединена.
* `min_block_number` ([Nullable(Int64)](../../sql-reference/data-types/)) — Минимальный номер части данных, входящей в состав текущей части после слияния.
* `max_block_number` ([Nullable(Int64)](../../sql-reference/data-types/)) — Максимальный номер части данных, входящей в состав текущей части после слияния.
* `level` ([Nullable(UInt32)](../../sql-reference/data-types/)) — Глубина дерева слияния. Ноль означает, что текущая часть была создана операцией вставки, а не слиянием других частей.
