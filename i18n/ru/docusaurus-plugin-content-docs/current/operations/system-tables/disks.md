## Описание \{#description\}

Содержит информацию о дисках, заданных в [конфигурации сервера](../../engines/table-engines/mergetree-family/mergetree.md#table_engine-mergetree-multiple-volumes_configure).

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя диска в конфигурации сервера.
* `path` ([String](../../sql-reference/data-types/)) — Путь к точке монтирования в файловой системе.
* `free_space` ([UInt64](../../sql-reference/data-types/)) — Свободное место на диске в байтах.
* `total_space` ([UInt64](../../sql-reference/data-types/)) — Общий объем диска в байтах.
* `unreserved_space` ([UInt64](../../sql-reference/data-types/)) — Свободное пространство, не занятое резервированиями (`free&#95;space` минус объем резервирований, используемых слияниями, вставками и другими выполняющимися в данный момент операциями записи на диск).
* `keep_free_space` ([UInt64](../../sql-reference/data-types/)) — Объем дискового пространства в байтах, который должен оставаться свободным на диске. Задается параметром `keep&#95;free&#95;space&#95;bytes` в конфигурации диска.
* `type` ([String](../../sql-reference/data-types/)) — Тип диска, указывающий, где хранятся данные: оперативная память, локальный диск или удаленное хранилище.
* `object_storage_type` ([String](../../sql-reference/data-types/)) — Тип Объектного хранилища, если тип диска — `object&#95;storage`
* `metadata_type` ([String](../../sql-reference/data-types/)) — Тип хранилища метаданных, если тип диска — `object&#95;storage`
* `is_encrypted` ([UInt8](../../sql-reference/data-types/)) — Флаг, показывающий, шифрует ли этот диск хранящиеся на нем данные.
* `is_read_only` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, что с этим диском можно выполнять только операции чтения.
* `is_write_once` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, является ли диск однократно записываемым. Это означает, что `BACKUP` на этот диск поддерживается, но `INSERT` в таблицу MergeTree на этом диске не поддерживается.
* `is_remote` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, для каких операций с этим диском требуется сетевое взаимодействие.
* `is_broken` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, неисправен ли диск. Неисправные диски имеют 0 свободного места и не могут использоваться.
* `cache_path` ([String](../../sql-reference/data-types/)) — Путь к каталогу кэша на локальном диске, если диск поддерживает кэширование.

{/*AUTOGENERATED_END*/ }
