## Описание \{#description\}

Содержит информацию обо всех успешных и неуспешных событиях входа в систему и выхода из неё.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/sql-reference/data-types/enum)) — Результат входа/выхода. Возможные значения: LoginFailure — ошибка входа. LoginSuccess — успешный вход. Logout — выход из системы.
* `auth_id` ([UUID](/sql-reference/data-types/uuid)) — Идентификатор аутентификации — UUID, автоматически генерируемый при каждом входе пользователя в систему.
* `session_id` ([String](/sql-reference/data-types/string)) — Идентификатор сеанса, передаваемый клиентом через HTTP-интерфейс.
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата входа/выхода.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время входа/выхода.
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — Время начала входа/выхода с точностью до микросекунд.
* `user` ([Nullable(String)](/sql-reference/data-types/nullable)) — Имя пользователя.
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/sql-reference/data-types/nullable)) — Тип аутентификации.
* `profiles` ([Array(LowCardinality(String))](/sql-reference/data-types/array)) — Список профилей, заданных для всех ролей и/или пользователей.
* `roles` ([Array(LowCardinality(String))](/sql-reference/data-types/array)) — Список ролей, к которым применяется профиль.
* `settings` ([Array(Tuple(LowCardinality(String), String))](/sql-reference/data-types/array)) — Settings, измененные при входе/выходе клиента.
* `client_address` ([IPv6](/sql-reference/data-types/ipv6)) — IP-адрес, использовавшийся для входа/выхода.
* `client_port` ([UInt16](/sql-reference/data-types/int-uint)) — Порт клиента, использовавшийся для входа/выхода.
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/sql-reference/data-types/enum)) — Интерфейс, через который был выполнен вход.
* `client_hostname` ([String](/sql-reference/data-types/string)) — Имя хоста клиентской машины, на которой запущен clickhouse-client или другой TCP-клиент.
* `client_name` ([String](/sql-reference/data-types/string)) — Имя clickhouse-client или другого TCP-клиента.
* `client_revision` ([UInt32](/sql-reference/data-types/int-uint)) — Ревизия clickhouse-client или другого TCP-клиента.
* `client_version_major` ([UInt32](/sql-reference/data-types/int-uint)) — Основная версия clickhouse-client или другого TCP-клиента.
* `client_version_minor` ([UInt32](/sql-reference/data-types/int-uint)) — Дополнительная версия clickhouse-client или другого TCP-клиента.
* `client_version_patch` ([UInt32](/sql-reference/data-types/int-uint)) — Патч-версия clickhouse-client или другого TCP-клиента.
* `failure_reason` ([String](/sql-reference/data-types/string)) — Сообщение об исключении, содержащее причину сбоя входа/выхода.

{/*AUTOGENERATED_END*/ }
