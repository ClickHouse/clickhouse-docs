# system.iceberg&#95;history \\{#systemiceberg&#95;history\\}

Эта системная таблица содержит историю снимков таблиц Iceberg, существующих в ClickHouse. Она будет пустой, если в ClickHouse нет ни одной таблицы Iceberg.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Имя базы данных.
* `table` ([String](../../sql-reference/data-types/)) — Имя таблицы.
* `made_current_at` ([Nullable(DateTime64(3))](../../sql-reference/data-types/)) — Дата и время, когда этот снимок был сделан текущим.
* `snapshot_id` ([UInt64](../../sql-reference/data-types/)) — Идентификатор снимка, используемый для его однозначной идентификации.
* `parent_id` ([UInt64](../../sql-reference/data-types/)) — Идентификатор родительского снимка.
* `is_current_ancestor` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, является ли этот снимок предком текущего снимка.
