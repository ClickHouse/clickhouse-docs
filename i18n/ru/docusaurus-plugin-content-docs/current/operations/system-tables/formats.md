## Описание \{#description\}

Содержит список всех форматов с указанием, подходит ли каждый формат для ввода/вывода и поддерживает ли он распараллеливание.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя формата.
* `is_input` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, подходит ли формат для ввода данных.
* `is_output` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, подходит ли формат для вывода данных.
* `supports_parallel_parsing` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, поддерживает ли формат параллельный разбор.
* `supports_parallel_formatting` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, поддерживает ли формат параллельное форматирование.
* `is_tty_friendly` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, обычно ли формат корректно отображается в терминале. Для других форматов CLI запросит подтверждение перед выводом.
* `content_type` ([String](../../sql-reference/data-types/)) — HTTP Content-Type, соответствующий выходному формату. Может зависеть от текущих настроек формата.
* `supports_random_access` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, поддерживает ли формат произвольный доступ при вводе.
* `has_schema_inference` ([UInt8](../../sql-reference/data-types/)) — Формат может динамически определять schema по данным (либо из встроенного заголовка или метаданных, либо по самому фрагменту данных).
* `has_external_schema` ([UInt8](../../sql-reference/data-types/)) — Формат либо имеет фиксированную schema, либо принимает заранее заданную schema в собственном формате.
* `prefers_large_blocks` ([UInt8](../../sql-reference/data-types/)) — Формат записывает более крупные блоки при выводе и генерирует более крупные блоки при вводе.
* `supports_append` ([UInt8](../../sql-reference/data-types/)) — С этим форматом можно дописывать данные в один файл.
* `supports_subsets_of_columns` ([UInt8](../../sql-reference/data-types/)) — Входной формат может распознавать отсутствие некоторых столбцов.
