## Описание \{#description\}

Содержит сведения о настройках таблиц [S3Queue](../../engines/table-engines/integrations/s3queue.md). Эта системная таблица доступна начиная с версии сервера `24.10`.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — База данных таблицы с движком S3Queue.
* `table` ([String](../../sql-reference/data-types/)) — Имя таблицы с движком S3Queue.
* `name` ([String](../../sql-reference/data-types/)) — Имя настройки.
* `value` ([String](../../sql-reference/data-types/)) — Значение настройки.
* `type` ([String](../../sql-reference/data-types/)) — Тип настройки (строковое значение, зависящее от реализации).
* `changed` ([UInt8](../../sql-reference/data-types/)) — 1, если настройка явно задана в конфигурации или была явно изменена.
* `description` ([String](../../sql-reference/data-types/)) — Описание настройки.
* `alterable` ([UInt8](../../sql-reference/data-types/)) — Указывает, может ли текущий пользователь изменить настройку с помощью ALTER TABLE MODIFY SETTING: 0 — текущий пользователь может изменить настройку, 1 — текущий пользователь не может изменить настройку.

{/*AUTOGENERATED_END*/ }
