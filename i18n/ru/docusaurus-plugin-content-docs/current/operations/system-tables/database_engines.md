Содержит список движков баз данных, поддерживаемых сервером.

Таблица содержит следующие столбцы (тип столбца указан в скобках):

{/*AUTOGENERATED_START*/ }

{/*AUTOGENERATED_END*/ }

Пример:

```sql
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

│ Ordinary │
│ Atomic   │
