## Описание \{#description\}

Содержит список движков баз данных, поддерживаемых сервером.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Название движка базы данных.

{/*AUTOGENERATED_END*/ }

## Пример \{#example\}

```sql title="Query"
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text title="Response"
┌─name─────┐
│ Ordinary │
│ Atomic   │
│ Lazy     │
└──────────┘
```
