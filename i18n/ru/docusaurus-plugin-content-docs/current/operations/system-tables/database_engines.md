Содержит список движков баз данных, поддерживаемых сервером.

В этой таблице представлены следующие столбцы (тип столбца указан в скобках):

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя движка базы данных.

{/*AUTOGENERATED_END*/ }

Пример:

```sql
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text
┌─name─────┐
│ Обычный  │
│ Атомный  │
│ Ленивый  │
└──────────┘
```
