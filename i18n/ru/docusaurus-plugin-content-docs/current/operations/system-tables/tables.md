## Описание \{#description\}

Содержит метаданные для каждой таблицы, известной серверу.

[Отсоединённые](../../sql-reference/statements/detach.md) таблицы не отображаются в `system.tables`.

[Временные таблицы](../../sql-reference/statements/create/table.md#temporary-tables) видны в `system.tables` только в тех сессиях, в которых они были созданы. Они отображаются с пустым полем `database` и с включённым флагом `is_temporary`.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Имя базы данных, в которой находится эта таблица.
* `name` ([String](../../sql-reference/data-types/)) — имя таблицы.
* `uuid` ([UUID](../../sql-reference/data-types/)) — UUID таблицы (база данных Atomic).
* `engine` ([String](../../sql-reference/data-types/)) — имя движка таблицы (без параметров).
* `is_temporary` ([UInt8](../../sql-reference/data-types/)) — флаг, указывающий, является ли таблица временной.
* `data_paths` ([Array(String)](../../sql-reference/data-types/)) — Пути к данным таблицы в файловых системах.
* `metadata_path` ([String](../../sql-reference/data-types/)) — Путь к метаданным таблицы в файловой системе.
* `metadata_modification_time` ([DateTime](../../sql-reference/data-types/)) — Время последнего изменения метаданных таблицы.
* `metadata_version` ([Int32](../../sql-reference/data-types/)) — Версия метаданных для таблицы ReplicatedMergeTree; для таблицы, отличной от ReplicatedMergeTree, — 0.
* `dependencies_database` ([Array(String)](../../sql-reference/data-types/)) — Зависимости базы данных.
* `dependencies_table` ([Array(String)](../../sql-reference/data-types/)) — Зависимости таблицы (materialized views, зависящие от текущей таблицы).
* `create_table_query` ([String](../../sql-reference/data-types/)) — запрос, использованный для создания таблицы.
* `engine_full` ([String](../../sql-reference/data-types/)) — Параметры движка таблицы.
* `as_select` ([String](../../sql-reference/data-types/)) — SELECT-запрос для представления.
* `parameterized_view_parameters` ([Array(Tuple(name String, type String))](../../sql-reference/data-types/)) — Параметры параметризованного представления.
* `partition_key` ([String](../../sql-reference/data-types/)) — Выражение ключа партиции, заданное в таблице.
* `sorting_key` ([String](../../sql-reference/data-types/)) — Выражение сортировочного ключа, заданное в таблице.
* `primary_key` ([String](../../sql-reference/data-types/)) — Выражение первичного ключа, заданное в таблице.
* `sampling_key` ([String](../../sql-reference/data-types/)) — Выражение ключа семплирования, указанное в таблице.
* `storage_policy` ([String](../../sql-reference/data-types/)) — Политика хранения. Применимо к таблицам, использующим движки MergeTree и Distributed.
* `total_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее количество строк, если точное число строк в таблице можно быстро определить; в противном случае — NULL (включая базовую таблицу Buffer).
* `total_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее количество байтов, если можно быстро определить точный объём данных таблицы в хранилище; в противном случае — NULL (без учёта какого-либо нижележащего хранилища). Если таблица хранит данные на диске, возвращает объём занятого места на диске (то есть в сжатом виде). Если таблица хранит данные в памяти, возвращает приблизительное количество занятых байтов в памяти.
* `total_bytes_uncompressed` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее количество несжатых байтов, если точное число байтов можно быстро определить по контрольным суммам части таблицы в хранилище, иначе NULL (не учитывает нижележащее хранилище, если оно есть).
* `parts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее количество частей в таблице.
* `active_parts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Количество активных частей в этой таблице.
* `total_marks` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее количество марок во всех частях таблицы.
* `active_on_fly_data_mutations` ([UInt64](../../sql-reference/data-types/)) — Общее количество активных мутаций данных (UPDATE и DELETE), которые могут применяться на лету.
* `active_on_fly_alter_mutations` ([UInt64](../../sql-reference/data-types/)) — Общее число активных ALTER-мутаций (MODIFY COLUMN), пригодных для применения на лету.
* `active_on_fly_metadata_mutations` ([UInt64](../../sql-reference/data-types/)) — Общее количество активных мутаций метаданных (RENAME), которые можно применять на лету.
* `columns_descriptions_cache_size` ([UInt64](../../sql-reference/data-types/)) — Размер кэша описаний столбцов для таблиц семейства *MergeTree
* `lifetime_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее количество строк, вставленных с момента запуска сервера (только для таблиц Buffer).
* `lifetime_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее число байтов, вставленных с момента запуска сервера (только для таблиц Buffer).
* `comment` ([String](../../sql-reference/data-types/)) — Комментарий к таблице.
* `has_own_data` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, хранит ли таблица собственные данные на диске или только обращается к другому источнику.
* `loading_dependencies_database` ([Array(String)](../../sql-reference/data-types/)) — Зависимости базы данных при загрузке (список объектов, которые должны быть загружены перед текущим объектом).
* `loading_dependencies_table` ([Array(String)](../../sql-reference/data-types/)) — зависимости загрузки таблицы (список объектов, которые должны быть загружены до текущего объекта).
* `loading_dependent_database` ([Array(String)](../../sql-reference/data-types/)) — База данных, от которой зависит загрузка.
* `loading_dependent_table` ([Array(String)](../../sql-reference/data-types/)) — Таблица, от которой зависит загрузка.
* `definer` ([String](../../sql-reference/data-types/)) — имя определяющего пользователя SQL security, используемое для таблицы.

{/*AUTOGENERATED_END*/ }
