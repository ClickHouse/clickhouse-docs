## Описание \{#description\}

Содержит информацию о кодеках сжатия и шифрования.

Эту таблицу можно использовать для получения информации о доступных кодеках сжатия и шифрования.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя кодека.
* `method_byte` ([UInt8](../../sql-reference/data-types/)) — Байт, указывающий кодек в сжатом файле.
* `is_compression` ([UInt8](../../sql-reference/data-types/)) — Истинно, если этот кодек что-то сжимает. В противном случае это может быть просто преобразование, помогающее сжатию.
* `is_generic_compression` ([UInt8](../../sql-reference/data-types/)) — Кодек представляет собой универсальный алгоритм сжатия, например lz4 или zstd.
* `is_encryption` ([UInt8](../../sql-reference/data-types/)) — Кодек выполняет шифрование.
* `is_timeseries_codec` ([UInt8](../../sql-reference/data-types/)) — Кодек предназначен для временных рядов с числами с плавающей точкой.
* `is_experimental` ([UInt8](../../sql-reference/data-types/)) — Кодек является экспериментальным.
* `description` ([String](../../sql-reference/data-types/)) — Краткое описание кодека.

{/*AUTOGENERATED_END*/ }

## Пример \{#example\}

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```
