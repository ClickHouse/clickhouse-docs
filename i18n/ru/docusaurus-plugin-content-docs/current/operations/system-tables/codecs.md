Содержит информацию о кодеках сжатия и шифрования.

Вы можете использовать эту таблицу, чтобы получить информацию о доступных кодеках сжатия и шифрования.

Таблица `system.codecs` содержит следующие столбцы (тип столбца указан в скобках):

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя кодека.
* `method_byte` ([UInt8](../../sql-reference/data-types/)) — Байт, обозначающий кодек в сжатом файле.
* `is_compression` ([UInt8](../../sql-reference/data-types/)) — Равно 1, если этот кодек что‑то сжимает. В противном случае это может быть просто преобразование, помогающее сжатию.
* `is_generic_compression` ([UInt8](../../sql-reference/data-types/)) — Кодек является универсальным алгоритмом сжатия, таким как LZ4, ZSTD.
* `is_encryption` ([UInt8](../../sql-reference/data-types/)) — Кодек выполняет шифрование.
* `is_timeseries_codec` ([UInt8](../../sql-reference/data-types/)) — Кодек предназначен для временных рядов с плавающей запятой.
* `is_experimental` ([UInt8](../../sql-reference/data-types/)) — Кодек является экспериментальным.
* `description` ([String](../../sql-reference/data-types/)) — Краткое высокоуровневое описание кодека.

{/*AUTOGENERATED_END*/ }

**Пример**

Запрос:

```sql
SELECT * FROM system.codecs WHERE name='LZ4'
```

Результат:

```text
Row 1:
──────
name:                   LZ4
method_byte:            130
is_compression:         1
is_generic_compression: 1
is_encryption:          0
is_timeseries_codec:    0
is_experimental:        0
description:            Extremely fast; good compression; balanced speed and efficiency.
```
