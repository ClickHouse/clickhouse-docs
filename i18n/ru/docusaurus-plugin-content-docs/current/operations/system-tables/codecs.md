Содержит информацию о кодеках сжатия и шифрования.

Эту таблицу можно использовать для получения информации о доступных кодеках сжатия и шифрования.

Таблица `system.codecs` содержит следующие столбцы (тип столбца указан в скобках):

{/*AUTOGENERATED_START*/ }

{/*AUTOGENERATED_END*/ }

**Пример**

Запрос:

```sql
SELECT * FROM system.codecs WHERE name='LZ4'
```

Результат:

```text
Row 1:
──────
name:                   LZ4
method_byte:            130
is_compression:         1
is_generic_compression: 1
is_encryption:          0
is_timeseries_codec:    0
is_experimental:        0
description:            Очень быстрый; хорошее сжатие; сбалансированное соотношение скорости и эффективности.
```
