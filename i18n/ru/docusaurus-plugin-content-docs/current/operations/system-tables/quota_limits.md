## Описание \{#description\}

Содержит информацию о максимальных значениях для всех интервалов всех квот. Одной квоте может соответствовать любое количество строк или ни одной.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `quota_name` ([String](../../sql-reference/data-types/)) — Имя квоты.
* `duration` ([UInt32](../../sql-reference/data-types/)) — Длительность интервала времени для расчета потребления ресурсов, в секундах.
* `is_randomized_interval` ([UInt8](../../sql-reference/data-types/)) — Логическое значение. Показывает, является ли интервал случайным. Если интервал не случайный, он всегда начинается в одно и то же время. Например, интервал в 1 минуту всегда начинается с целого числа минут (то есть он может начаться в 11:20:00, но никогда не начинается в 11:20:01), а интервал в один день всегда начинается в полночь UTC. Если интервал случайный, самый первый интервал начинается в случайный момент времени, а последующие интервалы идут один за другим. Значения: 0 — интервал не случайный, 1 — интервал случайный.
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов всех типов, которые можно выполнить.
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов SELECT, которые можно выполнить.
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов INSERT, которые можно выполнить.
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов, завершившихся ошибкой, в течение указанного периода времени.
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое общее количество строк в результирующих наборах всех запросов в течение указанного периода времени.
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое общее количество байтов в результирующих наборах всех запросов в течение указанного периода времени.
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество строк, которые можно прочитать при выполнении всех запросов в течение указанного периода времени.
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество байтов, которые можно прочитать при выполнении всех запросов в течение указанного периода времени.
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — Максимально допустимое время выполнения всех запросов (в наносекундах) в течение указанного периода времени
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество байтов, которые можно записать при выполнении всех запросов в течение указанного периода времени.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество последовательных неудачных попыток аутентификации в течение указанного периода времени.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество выполнений любого отдельного нормализованного запроса в течение указанного периода времени.
