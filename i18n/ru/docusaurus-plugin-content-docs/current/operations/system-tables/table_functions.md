## Описание \{#description\}

Содержит список всех доступных табличных функций и их описания.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя табличной функции.
* `description` ([String](../../sql-reference/data-types/)) — Краткое описание табличной функции.
* `allow_readonly` ([UInt8](../../sql-reference/data-types/)) — Флаг, указывающий, может ли пользователь с доступом только для чтения использовать эту функцию.
