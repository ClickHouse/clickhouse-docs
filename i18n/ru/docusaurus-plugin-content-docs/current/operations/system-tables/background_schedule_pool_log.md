## Описание \{#description\}

Таблица `system.background_schedule_pool_log` создаётся только если указана серверная настройка [background&#95;schedule&#95;pool&#95;log](/operations/server-configuration-parameters/settings#background_schedule_pool_log).

Эта таблица содержит историю выполнения задач пула фонового планировщика. Пулы фонового планировщика используются для выполнения периодических задач, таких как распределённые отправки, сброс буферов и операции с брокером сообщений.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата события.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время события.
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — Время события с точностью до микросекунд.
* `query_id` ([String](/sql-reference/data-types/string)) — Идентификатор запроса, связанного с фоновой задачей.
* `database` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя базы данных.
* `table` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя таблицы.
* `table_uuid` ([UUID](/sql-reference/data-types/uuid)) — UUID таблицы, к которой относится фоновая задача.
* `log_name` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя фоновой задачи.
* `duration_ms` ([UInt64](/sql-reference/data-types/int-uint)) — Длительность выполнения задачи в миллисекундах.
* `error` ([UInt16](/sql-reference/data-types/int-uint)) — Код ошибки, вызвавшей исключение.
* `exception` ([String](/sql-reference/data-types/string)) — Текст сообщения об ошибке.

{/*AUTOGENERATED_END*/ }

Таблица `system.background_schedule_pool_log` создается после первого выполнения фоновой задачи.
