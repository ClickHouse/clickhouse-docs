## Описание \{#description\}

Содержит информацию о файлах данных, хранящихся на удалённых дисках (например, S3, Azure Blob Storage), включая соответствие между путями локальных метаданных и путями удалённых blob-объектов.

Каждая строка представляет один удалённый blob-объект, связанный с файлом данных.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `disk_name` ([String](../../sql-reference/data-types/)) — Имя диска.
* `path` ([String](../../sql-reference/data-types/)) — Путь к диску.
* `cache_base_path` ([String](../../sql-reference/data-types/)) — Базовый каталог файлов кэша.
* `local_path` ([String](../../sql-reference/data-types/)) — Путь к файлу ClickHouse; также используется как путь к метаданным.
* `remote_path` ([String](../../sql-reference/data-types/)) — Путь к blob-объекту в объектном хранилище, с которым связан файл ClickHouse.
* `size` ([UInt64](../../sql-reference/data-types/)) — Размер файла (в сжатом виде).
* `common_prefix_for_blobs` ([String](../../sql-reference/data-types/)) — Общий префикс для blob-объектов в объектном хранилище.
* `cache_paths` ([Array(String)](../../sql-reference/data-types/)) — Файлы кэша для соответствующего blob-объекта.

{/*AUTOGENERATED_END*/ }

## Пример \{#example\}

```sql
SELECT * FROM system.remote_data_paths LIMIT 1 FORMAT Vertical;
```

```text
Row 1:
──────
disk_name:              s3
path:                   /var/lib/clickhouse/disks/s3/
cache_base_path:        /var/lib/clickhouse/disks/s3_cache/
local_path:             store/123/1234abcd-1234-1234-1234-1234abcd1234/all_0_0_0/data.bin
remote_path:            abc123/all_0_0_0/data.bin
size:                   1048576
common_prefix_for_blobs:
cache_paths:            ['/var/lib/clickhouse/disks/s3_cache/a1/b2/c3d4e5f6']
```
