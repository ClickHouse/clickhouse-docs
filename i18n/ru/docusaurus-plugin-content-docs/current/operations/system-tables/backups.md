## Описание \{#description\}

Содержит список всех операций `BACKUP` или `RESTORE` с их текущими состояниями и другими свойствами. Обратите внимание, что эта таблица не является персистентной и отображает только операции, выполненные после последнего перезапуска сервера.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `id` ([String](../../sql-reference/data-types/)) — Идентификатор операции; может быть передан через SETTINGS id=... или случайным образом сгенерирован как UUID.
* `name` ([String](../../sql-reference/data-types/)) — Имя операции, строка вида `Disk('backups', 'my_backup')`
* `base_backup_name` ([String](../../sql-reference/data-types/)) — Имя операции базовой резервной копии, строка вида `Disk('backups', 'my_base_backup')`
* `query_id` ([String](../../sql-reference/data-types/)) — Идентификатор запроса, который запустил резервное копирование.
* `status` ([Enum8(&#39;CREATING&#95;BACKUP&#39; = 0, &#39;BACKUP&#95;CREATED&#39; = 1, &#39;BACKUP&#95;FAILED&#39; = 2, &#39;RESTORING&#39; = 3, &#39;RESTORED&#39; = 4, &#39;RESTORE&#95;FAILED&#39; = 5, &#39;BACKUP&#95;CANCELLED&#39; = 6, &#39;RESTORE&#95;CANCELLED&#39; = 7)](../../sql-reference/data-types/)) — Статус операции резервного копирования или восстановления.
* `error` ([String](../../sql-reference/data-types/)) — Сообщение об ошибке, если оно есть.
* `start_time` ([DateTime64(6)](../../sql-reference/data-types/)) — Время начала операции.
* `end_time` ([DateTime64(6)](../../sql-reference/data-types/)) — Время завершения операции.
* `num_files` ([UInt64](../../sql-reference/data-types/)) — Количество файлов, сохранённых в резервной копии.
* `total_size` ([UInt64](../../sql-reference/data-types/)) — Общий размер файлов, сохранённых в резервной копии.
* `num_entries` ([UInt64](../../sql-reference/data-types/)) — Количество записей в резервной копии, то есть количество файлов в папке, если резервная копия хранится в виде папки.
* `uncompressed_size` ([UInt64](../../sql-reference/data-types/)) — Размер резервной копии в несжатом виде.
* `compressed_size` ([UInt64](../../sql-reference/data-types/)) — Размер резервной копии в сжатом виде.
* `files_read` ([UInt64](../../sql-reference/data-types/)) — Возвращает количество файлов, прочитанных при RESTORE из этой резервной копии.
* `bytes_read` ([UInt64](../../sql-reference/data-types/)) — Возвращает общий размер файлов, прочитанных при RESTORE из этой резервной копии.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](../../sql-reference/data-types/)) — Все profile events, собранные во время этой операции.
