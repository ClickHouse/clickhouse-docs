## Описание \{#description\}

Возвращает статистику jemalloc одной строкой с одним столбцом. Эквивалентно команде SYSTEM JEMALLOC STATS.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `stats` ([String](../../sql-reference/data-types/)) — вывод статистики JEMalloc, полученный с помощью malloc&#95;stats&#95;print.
