## Описание \{#description\}

Содержит сведения о доступных сертификатах и источниках их получения.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `version` ([Int32](../../sql-reference/data-types/)) — Версия сертификата. Значения: 0 для v1, 1 для v2, 2 для v3.
* `serial_number` ([Nullable(String)](../../sql-reference/data-types/)) — Серийный номер сертификата, присвоенный центром сертификации.
* `signature_algo` ([Nullable(String)](../../sql-reference/data-types/)) — Алгоритм подписи — алгоритм, который центр сертификации использовал для подписи этого сертификата.
* `issuer` ([Nullable(String)](../../sql-reference/data-types/)) — Издатель — уникальный идентификатор центра сертификации, выдавшего этот сертификат.
* `not_before` ([Nullable(String)](../../sql-reference/data-types/)) — Начало периода, в течение которого сертификат действителен.
* `not_after` ([Nullable(String)](../../sql-reference/data-types/)) — Конец периода, в течение которого сертификат действителен.
* `subject` ([Nullable(String)](../../sql-reference/data-types/)) — Субъект — идентификатор владельца открытого ключа.
* `pkey_algo` ([Nullable(String)](../../sql-reference/data-types/)) — Алгоритм открытого ключа определяет, с каким алгоритмом может использоваться открытый ключ.
* `path` ([String](../../sql-reference/data-types/)) — Путь к файлу или каталогу, содержащему этот сертификат.
* `default` ([UInt8](../../sql-reference/data-types/)) — Сертификат находится в стандартном расположении сертификатов.
* `protocol` ([String](../../sql-reference/data-types/)) — Имя протокола, если сертификат взят из конфигурации TLS для конкретного протокола; в противном случае пусто.

{/*AUTOGENERATED_END*/ }
