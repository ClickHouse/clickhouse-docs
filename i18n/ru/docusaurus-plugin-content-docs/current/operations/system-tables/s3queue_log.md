Содержит записи журнала с информацией о файлах, обработанных движком S3Queue.

Эту таблицу можно в любой момент безопасно очистить с помощью TRUNCATE или удалить.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата записи этой строки лога
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время записи этой строки лога
* `database` ([String](/sql-reference/data-types/string)) — Имя базы данных, в которой находится таблица очереди (`S3Queue` или `AzureQueue`).
* `table` ([String](/sql-reference/data-types/string)) — Имя таблицы очереди (`S3Queue` или `AzureQueue`).
* `uuid` ([String](/sql-reference/data-types/string)) — UUID таблицы очереди (`S3Queue` или `AzureQueue`).
* `file_name` ([String](/sql-reference/data-types/string)) — Имя обрабатываемого файла.
* `rows_processed` ([UInt64](/sql-reference/data-types/int-uint)) — Количество обработанных строк.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/sql-reference/data-types/enum)) — Статус обрабатываемого файла.
* `processing_start_time` ([Nullable(DateTime)](/sql-reference/data-types/nullable)) — Время начала обработки файла.
* `processing_end_time` ([Nullable(DateTime)](/sql-reference/data-types/nullable)) — Время завершения обработки файла.
* `exception` ([String](/sql-reference/data-types/string)) — Сообщение об исключении, если оно возникло.
* `commit_id` ([UInt64](/sql-reference/data-types/int-uint)) — Идентификатор транзакции, в которой был зафиксирован этот файл.
* `commit_time` ([DateTime](/sql-reference/data-types/datetime)) — Время фиксации файла в Keeper (как обработанного, так и завершившегося с ошибкой).
* `transaction_start_time` ([DateTime](/sql-reference/data-types/datetime)) — Время начала всей транзакции обработки.
* `get_object_time_ms` ([UInt64](/sql-reference/data-types/int-uint)) — Время, затраченное на поиск объекта в объектном хранилище.
