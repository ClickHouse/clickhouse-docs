Использование квот по всем пользователям.

* `quota_name` ([String](../../sql-reference/data-types/)) — Название QUOTA.
* `quota_key` ([String](../../sql-reference/data-types/)) — Значение ключа.
* `is_current` ([UInt8](../../sql-reference/data-types/)) — Использование QUOTA текущим пользователем.
* `start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — Время начала расчёта потребления ресурсов.
* `end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — Время окончания расчёта потребления ресурсов.
* `duration` ([Nullable(UInt32)](../../sql-reference/data-types/)) — Длительность временного интервала для расчета потребления ресурсов, в секундах.
* `queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее число выполненных запросов.
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов всех типов, разрешённых к выполнению.
* `query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее количество выполненных SELECT-запросов.
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество SELECT-запросов, которые можно выполнить.
* `query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее число выполненных INSERT-запросов.
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов INSERT, которые можно выполнить.
* `errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее число запросов, завершившихся ошибкой.
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество запросов, завершившихся ошибкой, в течение указанного периода времени.
* `result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее общее количество строк в результирующих наборах всех запросов за текущий период времени.
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимальное суммарное количество строк в результатах всех запросов, разрешённое в течение указанного периода времени.
* `result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее суммарное количество байтов во всех наборах результатов запросов за текущий период времени.
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимальное общее количество байтов во всех результирующих наборах запросов, разрешённых в течение указанного периода времени.
* `read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее число строк, прочитанных к текущему моменту при выполнении всех запросов за текущий период времени.
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимальное количество строк, которые можно прочитать при выполнении всех запросов за указанный период времени.
* `read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее суммарное количество байтов, прочитанных при выполнении всех запросов за текущий период времени.
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимальное количество байт, которое можно прочитать при выполнении всех запросов в течение указанного периода времени.
* `execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — Общее время (в наносекундах), затраченное на выполнение запросов за текущий период
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — Максимальное время (в наносекундах), отведённое на выполнение всех запросов за указанный период времени
* `written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Общее текущее количество байтов, записанных при выполнении всех запросов за текущий период времени.
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимальное количество байтов, которое разрешено записать в течение указанного периода времени при выполнении всех запросов.
* `failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее количество последовательных ошибок аутентификации за текущий период времени.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое количество последовательных неудачных попыток аутентификации в течение указанного периода времени.
* `queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Текущее максимальное число выполнений любого отдельного нормализованного запроса за текущий период времени.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — Максимально допустимое число выполнений одного нормализованного запроса за указанный период времени.
