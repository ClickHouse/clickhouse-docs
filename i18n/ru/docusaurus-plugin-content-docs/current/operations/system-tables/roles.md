# system.roles \\{#systemroles\\}

Содержит информацию о настроенных [ролях](../../guides/sre/user-management/index.md#role-management).

Столбцы:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя роли.
* `id` ([UUID](../../sql-reference/data-types/)) — Идентификатор роли.
* `storage` ([String](../../sql-reference/data-types/)) — Путь к хранилищу ролей. Настраивается параметром `access_control_path`.

{/*AUTOGENERATED_END*/ }
