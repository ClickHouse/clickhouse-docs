Содержит информацию об участниках. Порядок строк при выполнении запроса является случайным.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — имя участника (автора) из журнала Git.

{/*AUTOGENERATED_END*/ }

**Пример**

```sql
SELECT * FROM system.contributors LIMIT 10
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
│ Max Vetrov       │
│ LiuYangkuan      │
│ svladykin        │
│ zamulla          │
│ Šimon Podlipský  │
│ BayoNet          │
│ Ilya Khomutov    │
│ Amy Krishnevsky  │
│ Loud_Scream      │
└──────────────────┘
```

Чтобы найти свою запись в таблице, выполните запрос:

```sql
SELECT * FROM system.contributors WHERE name = 'Olga Khvostikova'
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
└──────────────────┘
```
