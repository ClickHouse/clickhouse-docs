## Описание \{#description\}

Содержит записи журнала. Уровень логирования, записи с которым попадают в эту таблицу, можно ограничить с помощью серверной настройки `text_log.level`.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата записи.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время записи.
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — Время записи с точностью до микросекунд.
* `thread_name` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя потока, из которого была выполнена запись в журнал.
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — Идентификатор потока OS.
* `level` ([Enum8(&#39;Fatal&#39; = 1, &#39;Critical&#39; = 2, &#39;Error&#39; = 3, &#39;Warning&#39; = 4, &#39;Notice&#39; = 5, &#39;Information&#39; = 6, &#39;Debug&#39; = 7, &#39;Trace&#39; = 8, &#39;Test&#39; = 9)](/sql-reference/data-types/enum)) — Уровень записи.
