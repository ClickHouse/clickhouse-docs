# system.projections {#systemprojections}

Содержит информацию о существующих проекциях во всех таблицах.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — имя базы данных.
* `table` ([String](../../sql-reference/data-types/)) — имя таблицы.
* `name` ([String](../../sql-reference/data-types/)) — имя проекции.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](../../sql-reference/data-types/)) — тип проекции.
* `sorting_key` ([Array(String)](../../sql-reference/data-types/)) — ключ сортировки проекции.
* `query` ([String](../../sql-reference/data-types/)) — запрос проекции.

{/*AUTOGENERATED_END*/ }

**Пример**

```sql
SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       landing
name:        improved_sorting_key
type:        Normal
sorting_key: ['user_id','date']
query:       SELECT * ORDER BY user_id, date
settings:     {}

Row 2:
──────
database:    default
table:       landing
name:        agg_no_key
type:        Aggregate
sorting_key: []
query:       SELECT count()
settings:     {}
```
