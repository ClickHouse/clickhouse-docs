## Описание \{#description\}

Содержит все активные роли на текущий момент, включая текущую роль текущего пользователя и предоставленные роли для текущей роли.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — Имя роли.
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — 1, если у роли есть привилегия `ADMIN OPTION`.
* `is_current` ([UInt8](../../sql-reference/data-types/)) — Флаг, показывающий, что `enabled_role` является текущей ролью текущего пользователя.
* `is_default` ([UInt8](../../sql-reference/data-types/)) — Флаг, показывающий, что `enabled_role` является ролью по умолчанию.
