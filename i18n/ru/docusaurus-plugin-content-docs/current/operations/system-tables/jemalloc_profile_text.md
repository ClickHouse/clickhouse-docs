## Описание \{#description\}

Отображает символизированный профиль кучи jemalloc. Сначала выполните команду &#39;SYSTEM JEMALLOC FLUSH PROFILE&#39;, чтобы сгенерировать профиль.

{/*AUTOGENERATED_START*/ }

* `line` ([String](../../sql-reference/data-types/)) — Строка из символизированного профиля кучи jemalloc.
