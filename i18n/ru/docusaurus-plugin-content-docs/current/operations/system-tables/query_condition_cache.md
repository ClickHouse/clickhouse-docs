# system.query_condition_cache \{#systemquery_condition_cache\}

<SystemTableCloud />

Отображает содержимое [кэша условий запросов](../query-condition-cache.md).

Столбцы:

{/*AUTOGENERATED_START*/ }

* `key_hash` ([UInt128](../../sql-reference/data-types/)) — Хеш (table&#95;uuid, part&#95;name, condition&#95;hash).
* `entry_size` ([UInt64](../../sql-reference/data-types/)) — Размер записи в байтах.
* `matching_marks` ([String](../../sql-reference/data-types/)) — Совпадающие метки.

{/*AUTOGENERATED_END*/ }
