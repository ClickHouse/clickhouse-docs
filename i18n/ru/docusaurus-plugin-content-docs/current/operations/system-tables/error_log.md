## Описание \{#description\}

Содержит историю значений ошибок из таблицы `system.errors`, периодически сбрасываемую на диск.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата события.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время события.
* `code` ([Int32](/sql-reference/data-types/int-uint)) — Код ошибки.
* `error` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя ошибки.
* `value` ([UInt64](/sql-reference/data-types/int-uint)) — Количество ошибок, произошедших за интервал времени.
* `remote` ([UInt8](/sql-reference/data-types/int-uint)) — Удалённое исключение (то есть полученное при выполнении одного из распределённых запросов).
* `last_error_time` ([DateTime](/sql-reference/data-types/datetime)) — Время возникновения последней ошибки.
* `last_error_message` ([String](/sql-reference/data-types/string)) — Сообщение о последней ошибке.
* `last_error_query_id` ([String](/sql-reference/data-types/string)) — Идентификатор запроса, вызвавшего последнюю ошибку (если доступен).
* `last_error_trace` ([Array(UInt64)](/sql-reference/data-types/array)) — Стек вызовов в виде списка физических адресов, по которым расположены вызванные методы.

{/*AUTOGENERATED_END*/ }
