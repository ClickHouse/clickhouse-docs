Содержит информацию о параметрах [graphite_rollup](../../operations/server-configuration-parameters/settings.md#graphite), которые используются в таблицах с движками [\*GraphiteMergeTree](../../engines/table-engines/mergetree-family/graphitemergetree.md).

Столбцы:

<!--AUTOGENERATED_START-->

- `config_name` ([String](../../sql-reference/data-types/)) — Имя параметра graphite_rollup.
- `rule_type` ([String](../../sql-reference/data-types/)) — Тип правила. Возможные значения: RuleTypeAll = 0 — по умолчанию, с регулярным выражением, совместим со старой схемой; RuleTypePlain = 1 — простые метрики, с регулярным выражением, совместим со старой схемой; RuleTypeTagged = 2 — метрики с тегами, с регулярным выражением, совместим со старой схемой; RuleTypeTagList = 3 — метрики с тегами, с регулярным выражением (преобразуется в RuleTypeTagged из строки вида 'retention=10min ; env=(staging|prod)')
- `regexp` ([String](../../sql-reference/data-types/)) — Шаблон для имени метрики.
- `function` ([String](../../sql-reference/data-types/)) — Имя агрегатной функции.
- `age` ([UInt64](../../sql-reference/data-types/)) — Минимальный возраст данных в секундах.
- `precision` ([UInt64](../../sql-reference/data-types/)) — Точность определения возраста данных в секундах.
- `priority` ([UInt16](../../sql-reference/data-types/)) — Приоритет шаблона.
- `is_default` ([UInt8](../../sql-reference/data-types/)) — Является ли шаблон шаблоном по умолчанию.
- `Tables.database` ([Array(String)](../../sql-reference/data-types/)) — Массив имен баз данных таблиц, использующих параметр `config_name`.
- `Tables.table` ([Array(String)](../../sql-reference/data-types/)) — Массив имен таблиц, использующих параметр `config_name`.
<!--AUTOGENERATED_END-->
