## Описание \{#description\}

Содержит информацию о параметрах [graphite&#95;rollup](../../operations/server-configuration-parameters/settings.md#graphite), которые используются в таблицах с движками типа [*GraphiteMergeTree](../../engines/table-engines/mergetree-family/graphitemergetree.md).

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `config_name` ([String](../../sql-reference/data-types/)) — имя параметра graphite&#95;rollup.
* `rule_type` ([String](../../sql-reference/data-types/)) — Тип правила. Возможные значения: RuleTypeAll = 0 — значение по умолчанию, с Regex, совместимо со старой схемой; RuleTypePlain = 1 — обычные метрики, с Regex, совместимо со старой схемой; RuleTypeTagged = 2 — метрики с тегами, с Regex, совместимо со старой схемой; RuleTypeTagList = 3 — метрики с тегами, с Regex (преобразуется в RuleTypeTagged из строки вида &#39;retention=10min ; env=(staging|prod)&#39;)
* `regexp` ([String](../../sql-reference/data-types/)) — Шаблон для имени метрики.
* `function` ([String](../../sql-reference/data-types/)) — Имя агрегирующей функции.
* `age` ([UInt64](../../sql-reference/data-types/)) — Минимальный возраст данных в секундах.
* `precision` ([UInt64](../../sql-reference/data-types/)) — Точность определения возраста данных в секундах.
* `priority` ([UInt16](../../sql-reference/data-types/)) — Приоритет шаблона.
* `is_default` ([UInt8](../../sql-reference/data-types/)) — Является ли шаблон шаблоном по умолчанию.
* `Tables.database` ([Array(String)](../../sql-reference/data-types/)) — Массив имен таблиц базы данных, использующих параметр `config_name`.
* `Tables.table` ([Array(String)](../../sql-reference/data-types/)) — Массив имен таблиц, использующих параметр `config_name`.
