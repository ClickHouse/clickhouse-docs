## Описание \{#description\}

Содержит сведения о параметрах сборки сервера ClickHouse.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя параметра сборки.
* `value` ([String](../../sql-reference/data-types/)) — Значение параметра сборки.

{/*AUTOGENERATED_END*/ }

## Пример \{#example\}

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
