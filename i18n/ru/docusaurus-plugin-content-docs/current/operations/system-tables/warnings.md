## Описание \{#description\}

Содержит предупреждения о конфигурации сервера, которые отображаются в clickhouse-client сразу после подключения к серверу.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `message` ([String](../../sql-reference/data-types/)) — Предупреждение, выданное сервером ClickHouse.
* `message_format_string` ([LowCardinality(String)](../../sql-reference/data-types/)) — Форматная строка, использованная для форматирования сообщения.

{/*AUTOGENERATED_END*/ }
