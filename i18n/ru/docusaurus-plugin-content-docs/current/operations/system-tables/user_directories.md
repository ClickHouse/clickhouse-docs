## Описание \{#description\}

Содержит сведения о настроенных пользовательских каталогах — каталогах файловой системы, из которых серверу ClickHouse разрешено читать данные, предоставленные пользователем.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя каталога.
* `type` ([String](../../sql-reference/data-types/)) — Тип хранилища доступа, например `users.xml`, `replicated` или `memory`.
* `params` ([String](../../sql-reference/data-types/)) — JSON с параметрами хранилища доступа.
* `precedence` ([UInt64](../../sql-reference/data-types/)) — Порядок, в котором этот каталог задан в конфигурации. Этот же порядок ClickHouse использует при поиске пользователя или роли.

{/*AUTOGENERATED_END*/ }
