## Описание \{#description\}

Содержит точки инструментирования, использующие функциональность XRay в LLVM.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `id` ([UInt32](../../sql-reference/data-types/)) — Идентификатор точки инструментирования
* `function_id` ([Int32](../../sql-reference/data-types/)) — Идентификатор, назначенный функции в разделе xray&#95;instr&#95;map ELF-бинарного файла.
* `function_name` ([LowCardinality(String)](../../sql-reference/data-types/)) — Имя, используемое для инструментирования функции.
* `handler` ([LowCardinality(String)](../../sql-reference/data-types/)) — Handler, встроенный в точки инструментирования функции.
* `entry_type` ([Enum8(&#39;Entry&#39; = 0, &#39;Exit&#39; = 1, &#39;EntryAndExit&#39; = 2)](../../sql-reference/data-types/)) — Тип точки входа для патча.
* `symbol` ([LowCardinality(String)](../../sql-reference/data-types/)) — Полное деманглированное имя символа.
* `parameters` ([Array(Dynamic)](../../sql-reference/data-types/)) — Параметры вызова handler.

{/*AUTOGENERATED_END*/ }
