Содержит информацию о выделениях памяти, выполненных через аллокатор jemalloc в различных классах размеров (бинах), агрегированных со всех арен. Эти статистические данные могут быть не абсолютно точными из-за кэширования, специфичного для потоков, в jemalloc.

Столбцы:

- `index` (UInt64) — Индекс бина, упорядоченный по размеру
- `large` (Bool) — Истина для больших выделений и Ложь для мелких
- `size` (UInt64) — Размер выделений в этом бине
- `allocations` (UInt64) — Количество выделений
- `deallocations` (UInt64) — Количество освобождений

**Пример**

Найдите размеры выделений, которые больше всего способствовали текущему общему использованию памяти.

```sql
SELECT
    *,
    allocations - deallocations AS active_allocations,
    size * active_allocations AS allocated_bytes
FROM system.jemalloc_bins
WHERE allocated_bytes > 0
ORDER BY allocated_bytes DESC
LIMIT 10
```
