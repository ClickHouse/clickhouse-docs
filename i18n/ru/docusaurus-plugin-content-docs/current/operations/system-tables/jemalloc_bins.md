Содержит информацию о выделениях памяти, выполненных аллокатором jemalloc в различных классах размеров (bins), агрегированную из всех арен.
Эти статистические данные могут быть неточными из-за локального кэширования потоков в jemalloc.

Столбцы:

<!--AUTOGENERATED_START-->

- `index` ([UInt16](../../sql-reference/data-types/)) — Индекс bin, упорядоченный по размеру.
- `large` ([UInt8](../../sql-reference/data-types/)) — True для больших выделений и False для малых.
- `size` ([UInt64](../../sql-reference/data-types/)) — Размер выделений в данном bin.
- `allocations` ([Int64](../../sql-reference/data-types/)) — Количество выделений.
- `deallocations` ([Int64](../../sql-reference/data-types/)) — Количество освобождений.
<!--AUTOGENERATED_END-->

**Пример**

Найти размеры выделений, которые внесли наибольший вклад в текущее общее использование памяти.

```sql
SELECT
    *,
    allocations - deallocations AS active_allocations,
    size * active_allocations AS allocated_bytes
FROM system.jemalloc_bins
WHERE allocated_bytes > 0
ORDER BY allocated_bytes DESC
LIMIT 10
```
