# system.background_schedule_pool \{#systembackground_schedule_pool\}

<SystemTableCloud />

## Описание \{#description\}

Содержит информацию о заданиях в пулах фонового планировщика. Пулы фонового планировщика используются для выполнения периодических задач, таких как отправка данных в distributed таблицы, сброс буферов и операции с брокером сообщений.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `pool` ([String](../../sql-reference/data-types/)) — Имя пула. Возможные значения: `schedule` — пул планирования общего назначения, `buffer_flush` — пул для сброса данных таблицы Buffer, `distributed` — пул для операций distributed таблицы, `message_broker` — пул для операций брокера сообщений.
* `database` ([String](../../sql-reference/data-types/)) — Имя базы данных.
* `table` ([String](../../sql-reference/data-types/)) — Имя таблицы.
* `table_uuid` ([UUID](../../sql-reference/data-types/)) — UUID таблицы.
* `query_id` ([String](../../sql-reference/data-types/)) — ID запроса (если сейчас выполняется). Примечание: это не настоящий запрос, а случайно сгенерированный идентификатор для сопоставления записей журнала в `system.text_log`.
* `elapsed_ms` ([UInt64](../../sql-reference/data-types/)) — Время выполнения задачи (если сейчас выполняется).
* `log_name` ([String](../../sql-reference/data-types/)) — Имя журнала для задачи.
* `deactivated` ([UInt8](../../sql-reference/data-types/)) — Указывает, деактивирована ли задача (всегда false, так как деактивированные задачи удаляются из пула).
* `scheduled` ([UInt8](../../sql-reference/data-types/)) — Указывает, запланирована ли задача к выполнению.
* `delayed` ([UInt8](../../sql-reference/data-types/)) — Указывает, запланирована ли задача с задержкой.
* `executing` ([UInt8](../../sql-reference/data-types/)) — Указывает, выполняется ли задача в данный момент.

{/*AUTOGENERATED_END*/ }
