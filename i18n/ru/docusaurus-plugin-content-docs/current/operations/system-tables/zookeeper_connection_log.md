## Описание \{#description\}

Таблица &#39;system.zookeeper&#95;connection&#95;log&#39; показывает историю подключений к ZooKeeper (включая вспомогательные экземпляры ZooKeeper). Каждая строка содержит информацию об одном событии, связанном с подключениями.

:::note
Таблица не содержит событий для отключений, вызванных остановкой сервера.
:::

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста сервера, который подключается к ZooKeeper или отключается от него.
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/sql-reference/data-types/enum)) — Тип события. Возможные значения: Connected, Disconnected.
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата записи.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — Время записи
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — Время записи с точностью до микросекунд.
* `name` ([String](/sql-reference/data-types/string)) — Имя кластера ZooKeeper.
* `host` ([String](/sql-reference/data-types/string)) — Имя хоста или IP-адрес узла ZooKeeper, к которому ClickHouse подключился или от которого отключился.
* `port` ([UInt16](/sql-reference/data-types/int-uint)) — Порт узла ZooKeeper, к которому ClickHouse подключился или от которого отключился.
* `index` ([UInt8](/sql-reference/data-types/int-uint)) — Индекс узла ZooKeeper, к которому ClickHouse подключился или от которого отключился. Индекс берётся из конфигурации ZooKeeper.
* `client_id` ([Int64](/sql-reference/data-types/int-uint)) — Идентификатор сеанса соединения.
* `keeper_api_version` ([UInt8](/sql-reference/data-types/int-uint)) — Версия API Keeper.
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/sql-reference/data-types/array)) — Включённые флаги возможностей. Применяется только к ClickHouse Keeper.
* `availability_zone` ([String](/sql-reference/data-types/string)) — Зона доступности
* `reason` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Причина подключения или отключения.

{/*AUTOGENERATED_END*/ }
