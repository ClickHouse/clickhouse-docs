Содержит информацию обо всех транзакциях, выполненных на данном сервере.

Эту таблицу можно в любой момент безопасно очистить с помощью TRUNCATE или удалить.

Столбцы:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — Имя хоста, на котором была выполнена транзакция.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/sql-reference/data-types/enum)) — Тип транзакции. Возможные значения: Begin, Commit, Rollback, AddPart, LockPart, UnlockPart.
* `event_date` ([Date](/sql-reference/data-types/date)) — Дата записи.
* `event_time` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — Время записи
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — Идентификатор потока.
* `query_id` ([String](/sql-reference/data-types/string)) — Идентификатор запроса, выполненного в рамках транзакции.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — Идентификатор транзакции.
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — Хеш идентификатора.
* `csn` ([UInt64](/sql-reference/data-types/int-uint)) — Номер последовательности фиксации
* `database` ([String](/sql-reference/data-types/string)) — Имя базы данных, в которой была выполнена транзакция.
* `table` ([String](/sql-reference/data-types/string)) — Имя таблицы, в которой была выполнена транзакция.
* `uuid` ([UUID](/sql-reference/data-types/uuid)) — UUID таблицы, в которой была выполнена транзакция.
* `part` ([String](/sql-reference/data-types/string)) — Имя части, участвовавшей в транзакции.

{/*AUTOGENERATED_END*/ }
