## Описание \{#description\}

Содержит список всех именованных коллекций, созданных с помощью SQL-запроса или считанных из файла конфигурации.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — Имя коллекции.
* `collection` ([Map(String, String)](../../sql-reference/data-types/)) — Содержимое коллекции.
* `source` ([String](../../sql-reference/data-types/)) — Источник именованной коллекции.
* `create_query` ([String](../../sql-reference/data-types/)) — Запрос CREATE для именованной коллекции.
