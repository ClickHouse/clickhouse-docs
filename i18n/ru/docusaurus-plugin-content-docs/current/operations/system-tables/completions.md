## Описание \{#description\}

Содержит список токенов автодополнения.

## Столбцы \{#columns\}

{/*AUTOGENERATED_START*/ }

* `word` ([String](../../sql-reference/data-types/)) — Токен автодополнения.
* `context` ([String](../../sql-reference/data-types/)) — Тип сущности токена (например, таблица).
* `belongs` ([Nullable(String)](../../sql-reference/data-types/)) — Токен сущности, к которой относится этот токен (например, имя родительской базы данных).

{/*AUTOGENERATED_END*/ }
