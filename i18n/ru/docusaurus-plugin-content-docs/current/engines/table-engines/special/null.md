# Движок таблицы Null \\{#null-table-engine\\}

При записи данных в таблицу `Null` они игнорируются.
При чтении из таблицы `Null` результат будет пустым.

Движок таблицы `Null` полезен для преобразования данных, когда исходные данные больше не нужны после преобразования.
Для этого вы можете создать материализованное представление поверх таблицы `Null`.
Данные, записываемые в таблицу, будут считываться представлением, но исходные сырые данные будут отбрасываться.
