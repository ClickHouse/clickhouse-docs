# Движок таблицы Null 

При записи данных в таблицу `Null` данные игнорируются.
При чтении из таблицы `Null` результат будет пустым.

Движок таблицы `Null` полезен для преобразования данных, когда исходные данные больше не нужны после их преобразования.
Для этого можно создать материализованное представление на основе таблицы `Null`.
Данные, записанные в таблицу, будут считываться представлением, а исходные сырые данные будут отброшены.
