# Движок таблицы View {#view-table-engine}

Используется для реализации представлений (подробнее см. запрос `CREATE VIEW`). Он не хранит данные, а только сохраняет указанный запрос `SELECT`. При чтении из таблицы выполняется этот запрос (при этом из него удаляются все ненужные столбцы).
