# Движок таблицы View

Используется для реализации представлений (подробности см. в описании запроса `CREATE VIEW`). Он не хранит данные, а только указанный запрос `SELECT`. При чтении из таблицы этот запрос выполняется (и все ненужные столбцы удаляются из запроса).
