# Периоды обслуживания ClickPipes для Postgres

Ближайшее окно обслуживания ClickPipes для Postgres запланировано на:

- **Дата:** 17 апреля 2025 года
- **Время:** 07:00–08:00 UTC

В это время ваши Postgres Pipes будут испытывать кратковременный простой.
После завершения окна обслуживания ClickPipes снова будут доступны и продолжат работу в штатном режиме.
