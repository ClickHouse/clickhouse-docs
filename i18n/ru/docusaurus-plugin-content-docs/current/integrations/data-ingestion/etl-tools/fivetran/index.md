# Fivetran и ClickHouse Cloud

<ClickHouseSupportedBadge/>



## Обзор {#overview}

[Fivetran](https://www.fivetran.com) — это автоматизированная платформа для перемещения данных между облачными платформами данных.

[ClickHouse Cloud](https://clickhouse.com/cloud) поддерживается в качестве [целевой системы Fivetran](https://fivetran.com/docs/destinations/clickhouse), что позволяет пользователям загружать данные из различных источников в ClickHouse.

:::note
[Целевая система ClickHouse Cloud](https://fivetran.com/docs/destinations/clickhouse) в настоящее время находится в закрытом предварительном просмотре. В случае возникновения проблем обращайтесь в службу поддержки ClickHouse.
:::
