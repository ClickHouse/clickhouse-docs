# Загрузка данных

ClickHouse интегрируется с рядом решений для интеграции и трансформации данных.
Для получения дополнительной информации ознакомьтесь со страницами ниже:

| Инструмент загрузки данных                                       | Описание                                                                                                                                                                                                                              |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Airbyte](/integrations/airbyte)                                 | Платформа интеграции данных с открытым исходным кодом. Позволяет создавать конвейеры ELT и поставляется с более чем 140 готовыми коннекторами.                                                                                       |
| [Apache Spark](/integrations/apache-spark)                       | Многопрофильный движок для выполнения задач инженерии данных (data engineering), науки о данных (data science) и машинного обучения на одном сервере или в кластере.                                                                |
| [Apache Flink](https://github.com/ClickHouse/flink-connector-clickhouse) | Потоковая загрузка и обработка данных в ClickHouse через DataStream API Flink с поддержкой пакетной записи.                                                                                                                          |
| [Amazon Glue](/integrations/glue)                                | Полностью управляемый бессерверный сервис интеграции данных от Amazon Web Services (AWS), упрощающий процесс обнаружения, подготовки и трансформации данных для аналитики, машинного обучения и разработки приложений.              |
| [Azure Synapse](/integrations/azure-synapse)                     | Полностью управляемый облачный аналитический сервис от Microsoft Azure, объединяющий big data и хранилище данных для упрощения интеграции, трансформации и аналитики данных в масштабе с использованием SQL, Apache Spark и конвейеров данных. |
| [Azure Data Factory](/integrations/azure-data-factory)           | Облачный сервис интеграции данных, позволяющий создавать, планировать и оркестрировать рабочие процессы обработки данных в масштабе.                                                                                                |
| [Apache Beam](/integrations/apache-beam)                         | Унифицированная модель программирования с открытым исходным кодом, позволяющая разработчикам определять и выполнять как пакетные, так и потоковые (непрерывные) конвейеры обработки данных.                                          |
| [BladePipe](/integrations/bladepipe)                             | Сквозной инструмент интеграции данных в режиме реального времени с задержкой менее одной секунды, обеспечивающий бесшовный поток данных между платформами.                                                                          |
| [dbt](/integrations/dbt)                                         | Позволяет аналитическим инженерам преобразовывать данные в их хранилищах, просто используя операторы SELECT.                                                                                                                        |
| [dlt](/integrations/data-ingestion/etl-tools/dlt-and-clickhouse) | Библиотека с открытым исходным кодом, которую можно добавить в ваши Python-скрипты для загрузки данных из различных, часто неструктурированных источников в хорошо структурированные, постоянно актуальные наборы данных.          |
| [Fivetran](/integrations/fivetran)                               | Автоматизированная платформа перемещения данных, синхронизирующая данные из, в и между вашими облачными платформами данных.                                                                                                         |
| [NiFi](/integrations/nifi)                                       | Программное обеспечение управления рабочими процессами с открытым исходным кодом, предназначенное для автоматизации потоков данных между программными системами.                                                                     |
| [Vector](/integrations/vector)                                   | Высокопроизводительный конвейер данных для observability, дающий организациям полный контроль над их данными наблюдаемости.                                                                                                         |
