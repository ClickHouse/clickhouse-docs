# Интеграции с SQL‑клиентами

В этом разделе описано, как интегрировать ClickHouse с различными распространёнными инструментами для управления базами данных, анализа и визуализации.

| Tool                                                | Description                                                 |
|-----------------------------------------------------|-------------------------------------------------------------|
| [DataGrip](/integrations/datagrip)                  | Мощная IDE для работы с базами данных                      |
| [DBeaver](/integrations/dbeaver)                    | Инструмент для администрирования и разработки баз данных   |
| [DbVisualizer](/integrations/dbvisualizer)          | Инструмент управления базами данных для разработчиков, администраторов БД и аналитиков |
| [Jupyter Notebooks](/integrations/jupysql)          | Интерактивные ноутбуки для кода, визуализаций и текста     |
| [QStudio](/integrations/qstudio)                    | Бесплатный SQL‑клиент с графическим интерфейсом и открытым исходным кодом |
| [TABLUM.IO](/integrations/tablumio)                 | Облачная платформа для визуализации данных                 |
| [marimo](/integrations/marimo)                      | Open source реактивный ноутбук для Python со встроенной поддержкой SQL  |
