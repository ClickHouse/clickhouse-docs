# Интеграции

Подключайте ClickHouse к вашим любимым инструментам и сервисам, чтобы создавать мощные конвейеры обработки данных и аналитические процессы.

Интеграции ClickHouse классифицированы по уровню поддержки:

| Tier | Description |
|------|-------------|
| Core integrations | Создаются или поддерживаются ClickHouse, официально поддерживаются ClickHouse и размещаются в организации ClickHouse на GitHub |
| Partner integrations | Создаются или поддерживаются и сопровождаются сторонними поставщиками программного обеспечения |
| Community integrations | Создаются или поддерживаются и сопровождаются участниками сообщества. Прямая поддержка не предоставляется, за исключением публичных репозиториев GitHub и каналов сообщества в Slack |
