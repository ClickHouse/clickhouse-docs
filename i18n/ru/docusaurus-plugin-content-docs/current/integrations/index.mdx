Интеграции ClickHouse организованы по уровню их поддержки:

- **Основные интеграции:** разработанные или поддерживаемые ClickHouse, они поддерживаются ClickHouse и находятся в организации ClickHouse на GitHub
- **Интеграции партнеров:** разработанные или поддерживаемые сторонними поставщиками программного обеспечения и поддерживаемые ими
- **Сообщество интеграций:** разработанные или поддерживаемые и поддерживаемые членами сообщества. Прямой поддержки, кроме общественных репозиториев GitHub и каналов Slack сообщества, нет

Каждая интеграция дополнительно классифицируется по категориям: **Клиент языка**, **Прием данных**, **Визуализация данных** и **SQL клиент**.

:::note
Мы активно составляем этот список интеграций ClickHouse ниже, поэтому он не является исчерпывающим. Не стесняйтесь 
[вносить вклад](https://github.com/ClickHouse/clickhouse-docs#contributing) в список любых относящихся к делу интеграций ClickHouse.
:::
