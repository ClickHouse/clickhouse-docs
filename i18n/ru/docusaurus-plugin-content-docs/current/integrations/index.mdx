# Интеграции

Подключайте ClickHouse к вашим любимым инструментам и сервисам, чтобы создавать мощные конвейеры данных и аналитические рабочие процессы.

Интеграции ClickHouse сгруппированы по уровню поддержки:

| Уровень | Описание |
|--------|----------|
| Core integrations | Созданы или поддерживаются ClickHouse, официально поддерживаются ClickHouse и размещаются в организации ClickHouse на GitHub |
| Partner integrations | Созданы или поддерживаются и сопровождаются сторонними поставщиками программного обеспечения |
| Community integrations | Созданы или поддерживаются и сопровождаются участниками сообщества. Прямая поддержка недоступна, кроме общедоступных репозиториев GitHub и каналов сообщества в Slack |
