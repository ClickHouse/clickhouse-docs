# Интеграции

Подключайте ClickHouse к вашим любимым инструментам и сервисам, чтобы создавать мощные конвейеры обработки данных и аналитические рабочие процессы.

Интеграции ClickHouse сгруппированы по уровню поддержки:

| Tier | Description |
|------|-------------|
| Core integrations | Разработаны или поддерживаются ClickHouse, поддерживаются командой ClickHouse и размещаются в организации ClickHouse на GitHub |
| Partner integrations | Разработаны или поддерживаются и находятся на сопровождении у сторонних поставщиков программного обеспечения |
| Community integrations | Разработаны или поддерживаются и сопровождаются участниками сообщества. Прямая поддержка недоступна, за исключением публичных репозиториев GitHub и каналов сообщества в Slack |
