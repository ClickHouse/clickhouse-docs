# Интеграции \{#integrations\}

Подключайте ClickHouse к вашим любимым инструментам и сервисам, чтобы создавать мощные конвейеры данных и аналитические процессы.

Интеграции с ClickHouse организованы по уровням поддержки:

| Tier | Description |
|------|-------------|
| Core integrations | Разработаны или поддерживаются компанией ClickHouse, официально поддерживаются ею и размещаются в организации ClickHouse на GitHub |
| Partner integrations | Разрабатываются или поддерживаются и сопровождаются сторонними поставщиками программного обеспечения |
| Community integrations | Разрабатываются или поддерживаются и сопровождаются участниками сообщества. Прямая поддержка отсутствует; доступны только публичные репозитории GitHub и каналы сообщества в Slack |
