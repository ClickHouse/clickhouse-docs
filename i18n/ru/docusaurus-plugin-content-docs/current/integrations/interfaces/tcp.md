# Нативный интерфейс (TCP) \{#native-interface-tcp\}

Нативный протокол используется в [клиенте командной строки](/interfaces/cli), для межсерверного взаимодействия при обработке распределённых запросов, а также в других программах на C++. К сожалению, нативный протокол ClickHouse пока не имеет формальной спецификации, но его можно восстановить методом обратного проектирования по исходному коду ClickHouse (начиная [примерно отсюда](https://github.com/ClickHouse/ClickHouse/tree/master/src/Client)) и/или с помощью перехвата и анализа TCP-трафика.
