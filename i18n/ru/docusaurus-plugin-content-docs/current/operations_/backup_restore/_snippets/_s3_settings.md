| Параметр                                            | Описание                                                                                                                                                                                                                                                                                                             | Значение по умолчанию |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------|
| `use_same_s3_credentials_for_base_backup`           | Определяет, будет ли базовая резервная копия в S3 наследовать учетные данные запроса. | `s3_storage_class`                                  | Класс хранилища, используемый для резервного копирования в S3, например `STANDARD`.                                                                                                                                                                                                                                  |                        |
