| Setting                                             | Description                                                                                                                                                                                                                                                                                                           | Default value |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------|
| `use_same_s3_credentials_for_base_backup`           | Определяет, должен ли базовый бэкап в S3 наследовать учётные данные из запроса. Работает только с `S3`.                                                                                                                                                                                                              |               |
| `s3_storage_class`                                  | Класс хранения, используемый для резервной копии в S3. Например: `STANDARD`                                                                                                                                                                                                                                          |               |
