| Setting                                             | Description                                                                                                                                                                                                                                                                                                           | Default value |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------|
| `use_same_s3_credentials_for_base_backup`           | Должен ли базовый бэкап в S3 использовать те же учетные данные, что и запрос. | `s3_storage_class`                                  | Класс хранилища, используемый для бэкапа в S3.
