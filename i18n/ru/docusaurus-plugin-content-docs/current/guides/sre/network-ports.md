# Сетевые порты {#network-ports}

:::note
Порты, обозначенные как **по умолчанию**, означают, что номер порта задан в `/etc/clickhouse-server/config.xml`. Чтобы переопределить эти настройки, добавьте файл в `/etc/clickhouse-server/config.d/`. См. документацию по [конфигурационному файлу](/operations/configuration-files).
:::

|8123|HTTP-порт по умолчанию||✓|
|8443|Порт HTTP SSL/TLS по умолчанию|✓|✓|
|9000|Порт нативного протокола ClickHouse (также упоминается как протокол ClickHouse TCP). Используется приложениями и процессами ClickHouse, такими как `clickhouse-server`, `clickhouse-client` и нативные инструменты ClickHouse. Используется для межсерверного взаимодействия при распределённых запросах.||✓|
|9004|Порт эмуляции MySQL||✓|
