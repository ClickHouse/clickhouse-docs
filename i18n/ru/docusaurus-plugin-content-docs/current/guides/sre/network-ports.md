# Сетевые порты

:::note
Порты, помеченные как **по умолчанию**, означают, что номер порта задан в `/etc/clickhouse-server/config.xml`. Чтобы изменить настройки, добавьте файл в `/etc/clickhouse-server/config.d/`. См. документацию по [файлам конфигурации](/operations/configuration-files).
:::

|Port|Description|Cloud|OSS|
|----|-----------|-----|---|
|2181|Порт службы ZooKeeper по умолчанию. **Примечание: см. `9181` для ClickHouse Keeper**||✓|
|8123|HTTP-порт по умолчанию||✓|
|8443|HTTP-порт SSL/TLS по умолчанию|✓|✓|
|9000|Порт Native Protocol (также называется протокол ClickHouse TCP). Используется приложениями и процессами ClickHouse, такими как `clickhouse-server`, `clickhouse-client` и нативные инструменты ClickHouse. Используется для межсерверного взаимодействия при распределённых запросах.||✓|
|9004|Порт эмуляции MySQL||✓|
|9005|Порт эмуляции PostgreSQL (также используется для защищённого взаимодействия, если для ClickHouse включён SSL).||✓|
|9009|Порт межсерверного взаимодействия для низкоуровневого доступа к данным. Используется для обмена данными, репликации и межсерверного взаимодействия.||✓|
|9010|SSL/TLS для межсерверного взаимодействия||✓|
|9011|Порт протокола PROXYv1 для Native Protocol||✓|
|9019|JDBC Bridge||✓|
|9100|Порт gRPC||✓|
|9181|Рекомендуемый порт ClickHouse Keeper||✓|
|9234|Рекомендуемый порт ClickHouse Keeper Raft (также используется для защищённого взаимодействия, если включён `<secure>1</secure>`)||✓|
|9363|Порт метрик Prometheus по умолчанию||✓|
|9281|Рекомендуемый защищённый SSL-порт ClickHouse Keeper||✓|
|9440|Порт Native Protocol с SSL/TLS|✓|✓|
|42000|Порт Graphite по умолчанию||✓|
