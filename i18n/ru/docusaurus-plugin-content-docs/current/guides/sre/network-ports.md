# Сетевые порты

:::note
Порты, обозначенные как **стандартные**, означают, что номер порта задан в `/etc/clickhouse-server/config.xml`. Чтобы переопределить настройки, добавьте файл в `/etc/clickhouse-server/config.d/`. См. документацию по [конфигурационным файлам](/operations/configuration-files).
:::

|Port|Description|Cloud|OSS|
|----|-----------|-----|---|
|2181|Стандартный сервисный порт ZooKeeper. **Примечание: см. `9181` для ClickHouse Keeper**||✓|
|8123|Стандартный HTTP-порт||✓|
|8443|Стандартный порт HTTP SSL/TLS|✓|✓|
|9000|Порт нативного протокола (также называется протоколом ClickHouse TCP). Используется приложениями и процессами ClickHouse, такими как `clickhouse-server`, `clickhouse-client`, а также встроенными утилитами ClickHouse. Используется для межсерверного взаимодействия при распределённых запросах.||✓|
|9004|Порт эмуляции MySQL||✓|
|9005|Порт эмуляции PostgreSQL (также используется для защищённого соединения, если для ClickHouse включён SSL).||✓|
|9009|Порт межсерверного взаимодействия для низкоуровневого доступа к данным. Используется для обмена данными, репликации и межсерверного взаимодействия.||✓|
|9010|Порт SSL/TLS для межсерверных соединений||✓|
|9011|Порт протокола PROXYv1 для нативного протокола||✓|
|9019|JDBC bridge||✓|
|9100|Порт gRPC||✓|
|9181|Рекомендуемый порт ClickHouse Keeper||✓|
|9234|Рекомендуемый порт Raft ClickHouse Keeper (также используется для защищённого соединения, если включено `<secure>1</secure>`)||✓|
|9363|Стандартный порт метрик Prometheus||✓|
|9281|Рекомендуемый защищённый SSL-порт ClickHouse Keeper||✓|
