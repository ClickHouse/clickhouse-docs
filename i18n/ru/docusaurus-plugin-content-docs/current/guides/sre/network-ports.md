# Сетевые порты

:::note
Порты, обозначенные как **по умолчанию**, означают, что номер порта задан в `/etc/clickhouse-server/config.xml`. Чтобы изменить настройки, добавьте файл в `/etc/clickhouse-server/config.d/`. См. документацию по [конфигурационным файлам](/operations/configuration-files).
:::

|Port|Описание|Cloud|OSS|
|----|-----------|-----|---|
|2181|Стандартный порт службы ZooKeeper. **Примечание: см. `9181` для ClickHouse Keeper**||✓|
|8123|Стандартный HTTP-порт||✓|
|8443|Стандартный порт HTTP SSL/TLS|✓|✓|
|9000|Порт протокола Native (также называется протоколом ClickHouse TCP). Используется приложениями и процессами ClickHouse, такими как `clickhouse-server`, `clickhouse-client` и нативные инструменты ClickHouse. Применяется для межсерверного взаимодействия при распределённых запросах.||✓|
|9004|Порт эмуляции MySQL||✓|
|9005|Порт эмуляции PostgreSQL (также используется для защищённого взаимодействия, если для ClickHouse включён SSL).||✓|
|9009|Порт межсерверного взаимодействия для низкоуровневого доступа к данным. Используется для обмена данными, репликации и межсерверного взаимодействия.||✓|
|9010|SSL/TLS для межсерверного взаимодействия||✓|
|9011|Порт протокола Native PROXYv1||✓|
|9019|JDBC bridge||✓|
|9100|gRPC-порт||✓|
|9181|Рекомендуемый порт ClickHouse Keeper||✓|
|9234|Рекомендуемый порт Raft для ClickHouse Keeper (также используется для защищённого взаимодействия, если включено `<secure>1</secure>`)||✓|
|9363|Стандартный порт метрик Prometheus||✓|
|9281|Рекомендуемый защищённый SSL-порт для ClickHouse Keeper||✓|
|9440|Порт протокола Native SSL/TLS|✓|✓|
|42000|Стандартный порт Graphite||✓|
