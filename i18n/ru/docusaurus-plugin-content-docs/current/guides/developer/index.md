# Расширенные руководства

В этом разделе представлены следующие расширенные руководства:

| Guide                                                                                                                  | Description                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alternative Query Languages](../developer/alternative-query-languages)                                         | Руководство по альтернативным поддерживаемым диалектам и работе с ними. Приводит примеры запросов для каждого диалекта.                                                                                                                                                                                                                       |
| [Cascading Materialized Views](../developer/cascading-materialized-views)                                       | Руководство по созданию каскадных материализованных представлений (`Materialized Views`), позволяющих объединять несколько исходных таблиц в одну целевую таблицу. Содержит пример использования каскадных материализованных представлений для агрегирования данных по месяцам и годам для группы доменных имён.                          |
| [Debugging memory issues](../developer/debugging-memory-issues)                                                 | Руководство по отладке проблем с использованием памяти в ClickHouse.                                                                                                                                                                                                                                                                          |
| [Deduplicating Inserts on Retries](../developer/deduplicating-inserts-on-retries)                               | Руководство по обработке ситуаций, когда требуется повторно выполнить неуспешные операции вставки.                                                                                                                                                                                                                                            |
| [Deduplication strategies](../developer/deduplication)                                                          | Руководство, подробно рассматривающее дедупликацию данных — технику удаления дублирующихся строк из базы данных. Объясняет отличия от дедупликации на основе первичных ключей в OLTP‑системах, подход ClickHouse к дедупликации и способы обработки сценариев с дублирующимися данными в запросах ClickHouse.                              |
| [Filling gaps in time-series data](../developer/time-series-filling-gaps)                                       | Руководство, раскрывающее возможности ClickHouse по работе с временными рядами, включая техники заполнения пропусков в данных для формирования более полной и непрерывной картины временных рядов.                                                                                                                                          |
| [Manage Data with TTL (Time-to-live)](../developer/ttl)                                                         | Руководство, в котором обсуждается использование секции `WITH FILL` для заполнения пропусков во временных рядах. Описывает, как заполнять пропуски нулевыми значениями, как задавать начальную точку заполнения, как заполнять пропуски до определённой конечной точки и как интерполировать значения для кумулятивных вычислений.        |
| [Stored procedures & query parameters](../developer/stored-procedures-and-prepared-statements)                  | Руководство, объясняющее, что ClickHouse не поддерживает традиционные хранимые процедуры, и предлагающее рекомендуемые альтернативы, включая пользовательские функции (UDF), параметризованные представления, материализованные представления и внешнюю оркестрацию. Также рассматриваются параметры запросов для безопасных параметризованных запросов (аналогов prepared statements). |
| [Understanding query execution with the Analyzer](../developer/understanding-query-execution-with-the-analyzer) | Руководство, поясняющее выполнение запросов в ClickHouse с помощью инструмента `analyzer`. Описывает, как `analyzer` разбивает запрос на последовательность шагов, позволяя визуализировать и отлаживать весь процесс выполнения для достижения оптимальной производительности.                                                               |
| [Using JOINs in ClickHouse](../joining-tables)                                                                  | Руководство по объединению таблиц (`JOIN`) в ClickHouse. Охватывает различные типы объединений (`INNER`, `LEFT`, `RIGHT` и т. д.), рассматривает передовые практики эффективных объединений (например, размещение меньших таблиц справа) и даёт представление о внутренних алгоритмах `JOIN` в ClickHouse, чтобы помочь оптимизировать запросы для сложных связей в данных. |
