# Расширенные руководства

В этом разделе представлены следующие расширенные руководства:

| Guide                                                                                                                  | Description                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alternative Query Languages](../developer/alternative-query-languages)                                         | Руководство по альтернативным поддерживаемым диалектам и их использованию. Приведены примеры запросов для каждого диалекта.                                                                                                                                                                                                                   |
| [Cascading Materialized Views](../developer/cascading-materialized-views)                                       | Руководство по созданию и каскадированию материализованных представлений, объединению нескольких исходных таблиц в одну целевую таблицу. Содержит пример использования каскадных материализованных представлений для агрегирования данных по месяцам и годам для группы доменных имён.                                                     |
| [Debugging memory issues](../developer/debugging-memory-issues)                                                 | Руководство по диагностике проблем с использованием памяти в ClickHouse.                                                                                                                                                                                                                                                                       |
| [Deduplicating Inserts on Retries](../developer/deduplicating-inserts-on-retries)                               | Руководство по обработке ситуаций, когда требуется повторно выполнять неуспешные вставки.                                                                                                                                                                                                                                                      |
| [Deduplication strategies](../developer/deduplication)                                                          | Руководство, подробно разбирающее дедупликацию данных — технику удаления дублирующихся строк из базы данных. Объясняет отличия от дедупликации на основе первичных ключей в OLTP‑системах, подход ClickHouse к дедупликации и способы обработки сценариев с дублирующимися данными в запросах ClickHouse.                                   |
| [Filling gaps in time-series data](../developer/time-series-filling-gaps)                                       | Руководство, которое раскрывает возможности ClickHouse по работе с временными рядами, включая техники заполнения пропусков в данных для создания более полной и непрерывной картины временных рядов.                                                                                                                                        |
| [Manage Data with TTL (Time-to-live)](../developer/ttl)                                                         | Руководство по управлению данными с помощью TTL (time‑to‑live). Объясняет, как задать время жизни данных, управлять автоматическим удалением устаревших записей и использовать TTL для оптимизации хранения и производительности.                                                                                                             |
| [Stored procedures & query parameters](../developer/stored-procedures-and-prepared-statements)                  | Руководство, объясняющее, что ClickHouse не поддерживает традиционные хранимые процедуры, и описывающее рекомендуемые альтернативы, включая пользовательские функции (User‑Defined Functions, UDF), параметризованные представления, материализованные представления и внешнюю оркестрацию. Также рассматриваются параметры запросов для безопасного выполнения параметризованных запросов (аналогично prepared statements). |
| [Understanding query execution with the Analyzer](../developer/understanding-query-execution-with-the-analyzer) | Руководство, которое проясняет механизм выполнения запросов в ClickHouse с помощью инструмента Analyzer. Объясняет, как Analyzer разбивает запрос на последовательность шагов, что позволяет визуализировать и отлаживать весь процесс выполнения для достижения оптимальной производительности.                                             |
| [Using JOINs in ClickHouse](../joining-tables)                                                                  | Руководство, упрощающее работу с объединением таблиц в ClickHouse. Охватывает различные типы объединений (`INNER`, `LEFT`, `RIGHT` и т. д.), рассматривает лучшие практики эффективных объединений (например, размещение меньших таблиц справа) и даёт представление о внутренних алгоритмах JOIN в ClickHouse, чтобы помочь оптимизировать запросы для сложных связей в данных. |
