# Расширенные руководства

В этом разделе содержатся следующие расширенные руководства:

| Guide                                                                                                                  | Description                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alternative Query Languages](../developer/alternative-query-languages)                                               | Руководство по альтернативным поддерживаемым диалектам и их использованию. Содержит примеры запросов на каждом диалекте.                                                                                                                                                                                                                       |
| [Cascading Materialized Views](../developer/cascading-materialized-views)                                             | Руководство по созданию материализованных представлений и организации их каскадного использования, а также по объединению нескольких исходных таблиц в одну целевую таблицу. Содержит пример использования каскадных материализованных представлений для агрегации данных по месяцам и годам для группы доменных имён.                        |
| [Debugging memory issues](../developer/debugging-memory-issues)                                                       | Руководство по отладке проблем с памятью в ClickHouse.                                                                                                                                                                                                                                                                                         |
| [Deduplicating Inserts on Retries](../developer/deduplicating-inserts-on-retries)                                     | Руководство по обработке ситуаций, когда требуется повторно выполнить неудачные операции вставки.                                                                                                                                                                                                                                              |
| [Deduplication strategies](../developer/deduplication)                                                                | Руководство, подробно рассматривающее дедупликацию данных — метод удаления дублирующихся строк из базы данных. Объясняет отличия от дедупликации на основе первичного ключа в OLTP-системах, подход ClickHouse к дедупликации и способы обработки сценариев с дублирующимися данными в ваших запросах ClickHouse.                             |
| [Filling gaps in time-series data](../developer/time-series-filling-gaps)                                             | Руководство, в котором рассматриваются возможности ClickHouse по работе с временными рядами, включая методы заполнения пропусков в данных для построения более полной и непрерывной картины временных рядов.                                                                                                                                 |
| [Manage Data with TTL (Time-to-live)](../developer/ttl)                                                               | Руководство, в котором рассматривается использование конструкции `WITH FILL` для заполнения пропусков во временных рядах. Описывает, как заполнять пропуски нулевыми значениями, как задавать начальную точку заполнения, как заполнять пропуски до заданной конечной точки и как интерполировать значения для кумулятивных вычислений. |
| [Stored procedures & query parameters](../developer/stored-procedures-and-prepared-statements)                        | Руководство, объясняющее, что ClickHouse не поддерживает традиционные хранимые процедуры, и предлагающее рекомендуемые альтернативы, включая пользовательские функции (UDF), параметризованные представления, материализованные представления и внешнюю оркестрацию. Также рассматриваются параметры запросов для безопасных параметризованных запросов (аналогично prepared statements). |
| [Understanding query execution with the Analyzer](../developer/understanding-query-execution-with-the-analyzer)       | Руководство, проясняющее выполнение запросов в ClickHouse посредством инструмента Analyzer. Объясняет, как Analyzer разбивает запрос на последовательность шагов, что позволяет визуализировать и диагностировать весь процесс выполнения для достижения оптимальной производительности.                                                       |
| [Using JOINs in ClickHouse](../joining-tables)                                                                        | Руководство, упрощающее работу с объединением таблиц в ClickHouse. Охватывает различные типы соединений (`INNER`, `LEFT`, `RIGHT` и др.), описывает лучшие практики для эффективных соединений (например, размещение меньших таблиц справа) и раскрывает внутренние алгоритмы соединений в ClickHouse, чтобы помочь оптимизировать запросы для сложных связей в данных.                  |
