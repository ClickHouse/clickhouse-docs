ClickHouse — это база данных SQL, и вы запрашиваете данные, используя тот же тип запросов `SELECT`, с которыми уже знакомы. Например:

```sql
SELECT *
FROM helloworld.my_first_table
ORDER BY timestamp
```

:::note
Дополнительные сведения о синтаксисе, доступных разделах и параметрах см. в разделе [SQL Reference](/sql-reference/statements/select).
:::

Обратите внимание, что результат возвращается в удобном табличном формате:

```response
┌─user_id─┬─message────────────────────────────────────────────┬───────────timestamp─┬──metric─┐
│     102 │ Вставляйте много строк за один батч                │ 2022-03-21 00:00:00 │ 1.41421 │
│     102 │ Сортируйте данные под часто используемые запросы   │ 2022-03-22 00:00:00 │   2.718 │
│     101 │ Привет, ClickHouse!                                │ 2022-03-22 14:04:09 │      -1 │
│     101 │ Гранулы — это наименьшие читаемые фрагменты данных │ 2022-03-22 14:04:14 │ 3.14159 │
└─────────┴────────────────────────────────────────────────────┴─────────────────────┴─────────┘

4 строки в наборе. Время выполнения: 0,008 с.
```

Добавьте оператор `FORMAT`, чтобы указать один из [многочисленных поддерживаемых форматов вывода в ClickHouse](/interfaces/formats#formats-overview):

```sql
SELECT *
FROM helloworld.my_first_table
ORDER BY timestamp
FORMAT TabSeparated
```

В приведённом выше запросе результат возвращается в виде таблички с разделителями табуляции:

```response
ID запроса: 3604df1c-acfd-4117-9c56-f86c69721121

102 Вставляйте много строк одним батчем      2022-03-21 00:00:00     1.41421
102 Сортируйте данные с учётом наиболее часто выполняемых запросов  2022-03-22 00:00:00     2.718
101 Привет, ClickHouse!  2022-03-22 14:04:09     -1
101 Гранулы — это наименьшие порции данных для чтения       2022-03-22 14:04:14     3.14159

4 строки в выборке. Время выполнения: 0.005 сек.
```

:::note
ClickHouse поддерживает более 70 форматов ввода и вывода, поэтому, благодаря тысячам функций и всем этим форматам данных, вы можете использовать ClickHouse для выполнения впечатляющих и быстрых ETL‑преобразований данных. При этом вам даже не нужен запущенный сервер ClickHouse для преобразования данных — вы можете использовать утилиту `clickhouse-local`. Подробности см. на [странице документации по `clickhouse-local`](/interfaces/cli).
:::
