Управляемый Postgres включает отобранный набор расширений, позволяющих расширить возможности вашей базы данных. Ниже приведен список доступных расширений.


## Установка расширений \{#installing-extensions\}

Чтобы установить расширение, подключитесь к своей базе данных и выполните:

```sql
CREATE EXTENSION extension_name;
```

Чтобы увидеть, какие расширения установлены сейчас:

```sql
SELECT * FROM pg_extension;
```

Чтобы просмотреть список всех доступных расширений и их версий:

```sql
SELECT * FROM pg_available_extensions;
```

## Расширение pg_clickhouse \{#pg-clickhouse\}

Расширение `pg_clickhouse` предустановлено на каждом управляемом экземпляре Postgres. Оно позволяет выполнять запросы к базам данных ClickHouse непосредственно из PostgreSQL, обеспечивая единый слой запросов как для транзакций, так и для аналитики.

Подробные инструкции по настройке и использованию см. в [документации по pg_clickhouse](/integrations/pg_clickhouse).

## Расширение pg_stat_ch \{#pg-stat-ch\}

Расширение `pg_stat_ch` обеспечивает обсервабилити на уровне запросов для вашего экземпляра Управляемого Postgres. Оно собирает телеметрию по каждому запросу (время выполнения, строки, буферы, WAL, ошибки) и передает ее в интерфейс ClickHouse Cloud, где она отображается в представлениях медленных запросов, ошибок и рабочих нагрузок. Настройка не требуется.

## Расширение pg_re2 \{#pg-re2\}

Расширение `pg_re2` добавляет в Управляемый Postgres совместимые с ClickHouse функции регулярных выражений на базе движка Google RE2. Оно полезно при переносе запросов между ClickHouse и Управляемым Postgres, а также при выполнении гибридных рабочих нагрузок, где требуется одинаковая семантика сопоставления с шаблоном в обеих системах.
