## Оповещения о доступности {#uptime-alerts}

Пользователи теперь могут просматривать данные о доступности по регионам на [странице статуса](https://status.clickhouse.com/) и подписываться на оповещения при сбоях в работе сервиса.



## SLA {#sla}

Мы также предлагаем соглашения об уровне обслуживания (SLA) для отдельных контрактов с обязательством по расходам. Пожалуйста, свяжитесь с нами по адресу [sales@clickhouse.com](mailto:sales@clickhouse.com), чтобы получить дополнительную информацию о наших условиях SLA.
