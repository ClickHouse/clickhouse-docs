## Оповещения о доступности {#uptime-alerts}

Пользователи теперь могут просматривать показатели доступности по регионам на [странице статуса](https://status.clickhouse.com/) и подписываться на уведомления о сбоях в работе сервиса.


## SLA {#sla}

Мы также предлагаем соглашения об уровне обслуживания для отдельных контрактов с фиксированными обязательствами по расходам. Свяжитесь с нами по адресу [sales@clickhouse.com](mailto:sales@clickhouse.com), чтобы узнать больше о нашей политике SLA.
