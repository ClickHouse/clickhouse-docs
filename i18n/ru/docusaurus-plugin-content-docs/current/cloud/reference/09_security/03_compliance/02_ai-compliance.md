ClickHouse использует ИИ, чтобы помогать клиентам и расширять возможности платформы. В этом документе описано, где применяется ИИ, как его можно включить или отключить, а также приведены ссылки на другую документацию с инструкциями по его использованию.

:::info Хранение данных и обучение моделей ИИ
По умолчанию ни одна из наших функций ИИ не сохраняет данные клиентов и не использует их для обучения моделей.
:::

## Поддержка ИИ \{#ai-support\}

Пользователь с разрешениями `organization:manage` может просматривать и изменять настройку, включающую поддержку ИИ. Она доступна на странице настроек организации.

ClickHouse предоставляет возможности ИИ, позволяющие клиентам взаимодействовать с чат-ассистентом для получения помощи по настройке или устранению неполадок. Ассистент использует информацию из нашей документации и систем управления обращениями, чтобы быстро отвечать на запросы клиентов.

## Передача данных для улучшения продукта \{#sharing-data-for-product-improvement\}

Пользователь с разрешением `organization:manage` может просматривать и изменять параметр, включающий передачу данных для улучшения продукта. Этот параметр доступен на странице настроек организации.

Если для вашей организации включен административный параметр, разрешающий передачу данных ClickHouse для улучшения продукта, ClickHouse может использовать данные, созданные или собранные в результате использования вашими пользователями AI-функции, включая входные и выходные данные, журналы и метаданные (за исключением прямых результатов запросов к ClickHouse Cloud), в таких целях, как оценка моделей (тестирование точности, производительности и безопасности/соответствия требованиям), повышение надежности сервиса и безопасности, улучшение пользовательского опыта и проведение аналитики. Мы не будем использовать эти данные для обучения, переобучения или тонкой настройки моделей.

## ClickHouse Assistant \{#clickhouse-assistant\}

Параметр для включения [ClickHouse Assistant](/cloud/features/ai-ml/ask-ai) может просматривать и изменять пользователь с разрешением `service:manage-generative-ai`. Он доступен на странице настроек сервиса. Эта возможность включает две разные функции:

| Feature              | Description                                                                      | Data used                                                                      | User validation                                                                    |
| :------------------- | :------------------------------------------------------------------------------- | :----------------------------------------------------------------------------- | :--------------------------------------------------------------------------------- |
| SQL AI               | Создает, анализирует и рекомендует улучшения для запросов                        | Метаданные сервиса, текст запроса и запрос пользователя                        | Результат предоставляется пользователю для проверки перед выполнением запроса      |
| ClickHouse Assistant | Позволяет пользователям задавать вопросы на естественном языке и получать ответы | Метаданные и содержимое сервиса, документация ClickHouse и запрос пользователя | Результат предоставляется для проверки; перед использованием его следует проверить |

## Протокол контекста модели \{#model-comtext-protocol\}

[ClickHouse MCP](/cloud/features/ai-ml/remote-mcp) позволяет клиентам проще создавать собственных агентов, используя данные из своих сервисов ClickHouse. Это требует отдельной настройки, а для обработки данных используется выбранная клиентом модель ИИ. ClickHouse не управляет моделями ИИ клиента и не имеет к ним доступа.
