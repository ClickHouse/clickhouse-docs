## События журнала аудита консоли {#console-audit-log-events}

Различные типы событий, фиксируемых для организации, сгруппированы в 3 категории: **Organization** (Организация), **Service** (Сервис) и **User** (Пользователь). Дополнительную информацию о журнале аудита, а также о том, как экспортировать данные или добавить интеграцию API, см. в документации по [журналу аудита консоли](/cloud/security/audit-logging/console-audit-log) в разделе руководств выше.

В журнал аудита записываются следующие события.

### Организация {#organization}

- Организация создана
- Организация удалена
- Название организации изменено

### Сервис {#service}

- Сервис создан
- Сервис удален
- Сервис остановлен
- Сервис запущен
- Название сервиса изменено
- Список IP-адресов для доступа к сервису изменен
- Пароль сервиса сброшен

### Пользователь {#user}

- Роль пользователя изменена
- Пользователь удален из организации
- Пользователь приглашен в организацию
- Пользователь присоединился к организации
- Приглашение пользователя удалено
- Пользователь покинул организацию
