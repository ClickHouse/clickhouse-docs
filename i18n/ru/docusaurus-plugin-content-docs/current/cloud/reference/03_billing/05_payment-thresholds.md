# Платёжные пороги

Когда сумма к оплате за расчётный период в ClickHouse Cloud достигает 10 000 долларов США или эквивалентной суммы, с вашего платёжного метода автоматически списываются средства. Неудачное списание приведёт к приостановке или прекращению предоставления услуг по истечении льготного периода. 

:::note
Этот платёжный порог не распространяется на клиентов, у которых есть контракт с обязательством по расходам или иное согласованное договорное соглашение с ClickHouse.
:::

Если ваша организация достигнет 90% платёжного порога и, согласно прогнозу, превысит его в середине расчётного периода, на платёжный адрес электронной почты, связанный с организацией, будет отправлено уведомление. Вы также получите уведомление по электронной почте и счёт, когда превысите платёжный порог.

Эти платёжные пороги могут быть уменьшены ниже 10 000 долларов США по запросу клиентов или по инициативе финансовой команды ClickHouse. Если у вас есть вопросы, свяжитесь с support@clickhouse.com для получения дополнительной информации.
