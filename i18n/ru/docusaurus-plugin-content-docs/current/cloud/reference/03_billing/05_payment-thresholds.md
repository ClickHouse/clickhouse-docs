# Платёжные пороги

Когда сумма к оплате за расчётный период в ClickHouse Cloud достигает 10 000 долларов США или эквивалентной суммы, с вашего платёжного средства автоматически списывается оплата. В случае неудачного списания по истечении льготного периода ваши услуги будут приостановлены или прекращены. 

:::note
Этот платёжный порог не применяется к клиентам, у которых есть контракт с фиксированным объёмом расходов или иное согласованное договорное соглашение с ClickHouse.
:::

Если ваша организация достигнет 90% платёжного порога и, согласно текущему прогнозу, превысит платёжный порог в середине расчётного периода, на адрес электронной почты для выставления счетов, связанный с организацией, будет отправлено уведомление. Вы также получите уведомление по электронной почте и счёт, когда превысите платёжный порог.

Эти платёжные пороги могут быть уменьшены ниже 10 000 долларов США по запросу клиентов или по инициативе финансовой команды ClickHouse. Если у вас есть вопросы, свяжитесь с support@clickhouse.com для получения дополнительной информации.
