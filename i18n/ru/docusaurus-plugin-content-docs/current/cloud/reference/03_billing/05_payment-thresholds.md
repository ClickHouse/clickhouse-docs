# Платёжные пороги

Когда ваша сумма к оплате за расчётный период в ClickHouse Cloud достигает 10 000 долл. США или эквивалентной суммы, с выбранного способа оплаты автоматически списываются средства. Неудачная попытка списания приведёт к приостановке или прекращению предоставления услуг после льготного периода. 

:::note
Этот платёжный порог не применяется к клиентам, у которых есть договор на гарантированный объём расходов или иное согласованное договорное соглашение с ClickHouse.
:::

Если ваша организация достигнет 90% платёжного порога и при текущем уровне потребления, как ожидается, превысит порог в середине расчётного периода, на адрес для выставления счетов, связанный с организацией, будет отправлено уведомление по электронной почте. Вы также получите уведомление по электронной почте, а при превышении порога — счёт на оплату.

Эти платёжные пороги могут быть установлены ниже 10 000 долл. США по запросу клиентов или по инициативе финансовой команды ClickHouse. Если у вас есть вопросы, свяжитесь с support@clickhouse.com для получения дополнительной информации.
