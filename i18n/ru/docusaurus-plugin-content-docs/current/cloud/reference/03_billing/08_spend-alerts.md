Оповещения о расходах сообщают, когда использование ресурсов вашей организации достигает настроенных пороговых значений в течение расчётного периода, чтобы вы могли избежать непредвиденных расходов. Они только отправляют уведомления; сведения о пороговых значениях, при достижении которых счета выставляются автоматически, см. в разделе [Платёжные пороги](/cloud/billing/payment-thresholds).

## Предварительные требования \{#prerequisites\}

* Чтобы настраивать и просматривать оповещения о расходах, у вас должна быть роль **Organization Admin** или **Billing Admin**.

## Как работают оповещения о расходах \{#how-spend-alerts-work\}

* Оповещения основаны на совокупном использовании в рамках расчетного периода и сбрасываются в начале каждого нового цикла.
* Оповещения срабатывают каждый час на основе совокупного использования, рассчитанного на этот момент, то есть оповещение срабатывает после превышения порога.

:::note Оповещения о расходах не ограничивают использование
Достижение 100% по вашему оповещению о расходах не ограничивает использование — расходы продолжают накапливаться и после достижения порога в 100%.
:::

## Настройте оповещение о расходах \{#set-up-spend-alert\}

<VerticalStepper headerLevel="h3">
  ### Откройте диалоговое окно оповещений о расходах \{#open-spend-alerts-dialog\}

  1. Перейдите в **Organization &gt; Billing** в консоли ClickHouse Cloud.
  2. Нажмите **Actions &gt; Add spend alerts**.

  <Image img={spend_alerts_actions} alt="Страница Billing с открытым меню Actions и выделенным пунктом Add spend alerts" size="lg" />

  ### Настройте и сохраните оповещение \{#configure-and-save\}

  1. Укажите сумму оповещения о расходах в долларах или кредитах за расчетный период.
  2. Выберите каналы уведомлений: **Email**, **UI** и/или **Slack**.
  3. Нажмите **Save**.

  <Image img={spend_alerts_config} alt="Диалоговое окно оповещений о расходах организации с лимитом расходов, автоматическими пороговыми оповещениями и каналами уведомлений" size="md" />
</VerticalStepper>

## Автоматические пороговые оповещения \{#threshold-alerts\}

На основе настроенной суммы оповещения о расходах автоматически создаются и срабатывают три оповещения:

| Порог    | Условие срабатывания            | Пример (оповещение на $10) |
| -------- | ------------------------------- | -------------------------- |
| **50%**  | Достигнута половина расходов    | $5                         |
| **75%**  | Достигнуто 75% расходов         | $7.50                      |
| **100%** | Достигнут полный объём расходов | $10                        |

## Каналы уведомлений \{#notification-channels\}

| Канал     | Описание                                                                                                              |
| --------- | --------------------------------------------------------------------------------------------------------------------- |
| **Email** | Отправляет оповещения на указанный адрес электронной почты. Нажмите значок редактирования, чтобы изменить получателя. |
| **UI**    | Отображает оповещения в консоли ClickHouse Cloud.                                                                     |
| **Slack** | Отправляет оповещения в канал Slack. Нажмите значок редактирования, чтобы выполнить настройку.                        |
