# ClickPipes для потоковой передачи и объектного хранилища {#clickpipes-for-streaming-object-storage}

В этом разделе описывается модель ценообразования ClickPipes для потоковой передачи и объектного хранилища.


## Как выглядит структура ценообразования ClickPipes? {#what-does-the-clickpipes-pricing-structure-look-like}

Она состоит из двух компонентов:

- **Вычислительные ресурсы**: Цена **за единицу в час**.
  Вычислительные ресурсы — это стоимость работы подов-реплик ClickPipes независимо от того, активно ли они принимают данные или нет.
  Применяется ко всем типам ClickPipes.
- **Принятые данные**: Цена **за ГБ**.
  Тариф на принятые данные применяется ко всем потоковым ClickPipes
  (Kafka, Confluent, Amazon MSK, Amazon Kinesis, Redpanda, WarpStream, Azure Event Hubs)
  для данных, передаваемых через поды-реплики. Объем принятых данных (ГБ) тарифицируется на основе байтов, полученных от источника (несжатых или сжатых).


## Что такое реплики ClickPipes? {#what-are-clickpipes-replicas}

ClickPipes загружает данные из удалённых источников через выделенную инфраструктуру,
которая работает и масштабируется независимо от сервиса ClickHouse Cloud.
Поэтому для этого используются выделенные вычислительные реплики.


## Каково количество реплик по умолчанию и их размер? {#what-is-the-default-number-of-replicas-and-their-size}

Каждый ClickPipe по умолчанию имеет 1 реплику с 512 МиБ ОЗУ и 0,125 vCPU (XS).
Это соответствует **0,0625** вычислительных единиц ClickHouse (1 единица = 8 ГиБ ОЗУ, 2 vCPU).


## Какова стоимость ClickPipes? {#what-are-the-clickpipes-public-prices}

- Вычислительные ресурсы: $0.20 за единицу в час ($0.0125 за реплику в час для размера реплики по умолчанию)
- Загруженные данные: $0.04 за ГБ

Стоимость вычислительных ресурсов зависит от **количества** и **размера** реплик в ClickPipe. Размер реплики по умолчанию можно изменить с помощью вертикального масштабирования. Стоимость каждого размера реплики указана ниже:

| Размер реплики             | Вычислительные единицы | RAM     | vCPU   | Цена за час |
| -------------------------- | ------------- | ------- | ------ | -------------- |
| Очень малый (XS) (по умолчанию) | 0.0625        | 512 МиБ | 0.125  | $0.0125        |
| Малый (S)                  | 0.125         | 1 ГиБ   | 0.25   | $0.025         |
| Средний (M)                | 0.25          | 2 ГиБ   | 0.5    | $0.05          |
| Большой (L)                | 0.5           | 4 ГиБ   | 1.0    | $0.10          |
| Очень большой (XL)         | 1.0           | 8 ГиБ   | 2.0    | $0.20          |
