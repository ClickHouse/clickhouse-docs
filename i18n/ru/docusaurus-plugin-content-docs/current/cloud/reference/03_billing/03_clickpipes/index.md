:::note
Использование MySQL и MongoDB CDC ClickPipes является бесплатным до момента достижения статуса General Availability (GA). Клиенты будут уведомлены заблаговременно перед запуском GA, чтобы они могли пересмотреть и оптимизировать использование ClickPipes.
:::
