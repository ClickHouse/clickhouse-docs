# Переход с биллинга по модели оплаты по мере использования (PAYG) на контракт с обязательствами по расходам в облачном маркетплейсе \\{#migrate-payg-to-committed\\}

Если ваша организация ClickHouse в данный момент использует биллинг через активную подписку (или заказ) в облачном маркетплейсе по модели оплаты по мере использования (PAYG) и вы хотите перейти на биллинг по контракту с обязательствами по расходам через тот же облачный маркетплейс, примите новое предложение, а затем выполните приведённые ниже шаги в зависимости от вашего облачного провайдера.



## Важные замечания \\{#important-notes\\}

Обратите внимание, что отмена вашей PAYG-подписки в маркетплейсе не удаляет вашу учетную запись ClickHouse Cloud — прекращаются только платежные отношения через маркетплейс. После отмены наша система прекратит выставление счетов за услуги ClickHouse Cloud через маркетплейс. (Примечание: этот процесс не является мгновенным и может занять несколько минут.)

После отмены вашей подписки в маркетплейсе, если в вашей организации ClickHouse зарегистрирована банковская карта, мы спишем оплату с этой карты в конце вашего расчетного периода — если только до этого момента не будет подключена новая подписка через маркетплейс.

Если после отмены не настроена банковская карта, у вас будет 14 дней, чтобы добавить либо действительную банковскую карту, либо новую подписку через облачный маркетплейс в вашу организацию. Если в указанный период не будет настроен ни один способ оплаты, ваши сервисы будут приостановлены, а ваша организация будет считаться не соответствующей требованиям по [платежной дисциплине](/manage/clickhouse-cloud-billing-compliance).

Любое использование услуг, начисленное после отмены подписки, будет выставлено к оплате на следующий настроенный действительный способ оплаты — предоплаченный кредит, подписку через маркетплейс или банковскую карту, в таком порядке.

По любым вопросам или при возникновении проблем с привязкой вашей организации к новой подписке через маркетплейс, пожалуйста, обратитесь в [службу поддержки](https://clickhouse.com/support/program) ClickHouse за помощью.



## AWS Marketplace \\{#aws-marketplace\\}

Если вы хотите использовать тот же AWS Account ID для перевода подписки PAYG на контракт с фиксированным объемом расходов, мы рекомендуем [связаться с отделом продаж](https://clickhouse.com/company/contact), чтобы внести это изменение. В этом случае не потребуется выполнять дополнительные действия, и работа вашей организации ClickHouse и связанных сервисов не будет нарушена.

Если вы хотите использовать другой AWS Account ID для перевода вашей организации ClickHouse с подписки PAYG на контракт с фиксированным объемом расходов, выполните следующие шаги:

### Шаги по отмене подписки AWS PAYG \\{#cancel-aws-payg\\}

1. **Перейдите в [AWS Marketplace](https://us-east-1.console.aws.amazon.com/marketplace)**
2. **Нажмите кнопку «Manage Subscriptions»**
3. **Перейдите в «Your Subscriptions»:**
    - Нажмите «Manage Subscriptions»
4. **Найдите ClickHouse Cloud в списке:**
    - Найдите и нажмите ClickHouse Cloud в разделе «Your Subscriptions»
5. **Отмените подписку:**
    - В разделе «Agreement» нажмите выпадающий список или кнопку «Actions» рядом с ClickHouse Cloud
    - Выберите «Cancel subscription»

> **Примечание:** Если вам нужна помощь с отменой подписки (например, если кнопка отмены подписки недоступна), обратитесь в [службу поддержки AWS](https://support.console.aws.amazon.com/support/home#/).

Затем выполните следующие [шаги](/cloud/billing/marketplace/aws-marketplace-committed-contract), чтобы перевести вашу организацию ClickHouse на новый контракт AWS с фиксированным объемом расходов, который вы приняли.



## GCP Marketplace \\{#gcp-marketplace\\}

### Шаги по отмене заказа GCP PAYG \\{#cancel-gcp-payg\\}

1. **Перейдите в консоль [Google Cloud Marketplace](https://console.cloud.google.com/marketplace):**
    - Убедитесь, что вы вошли в нужный аккаунт GCP и выбрали соответствующий проект
2. **Найдите свой заказ ClickHouse:**
    - В левом меню нажмите «Your Orders»
    - Найдите нужный заказ ClickHouse в списке активных заказов
3. **Отмените заказ:**
    - Найдите меню с тремя точками справа от вашего заказа и следуйте инструкциям, чтобы отменить заказ ClickHouse

> **Примечание.** Для помощи с отменой этого заказа свяжитесь со [службой поддержки GCP](https://cloud.google.com/support/docs/get-billing-support).

Далее выполните эти [шаги](/cloud/billing/marketplace/gcp-marketplace-committed-contract), чтобы настроить вашу организацию ClickHouse для нового контракта GCP с фиксированным обязательством по расходам.



## Azure Marketplace \\{#azure-marketplace\\}

### Шаги по отмене подписки Azure PAYG \\{#cancel-azure-payg\\}

1. **Перейдите в [портал Microsoft Azure](http://portal.azure.com)**
2. **Откройте раздел «Subscriptions» («Подписки»)**
3. **Найдите активную подписку ClickHouse, которую требуется отменить**
4. **Отмените подписку:**
    - Нажмите на подписку ClickHouse Cloud, чтобы открыть сведения о подписке
    - Нажмите кнопку «Cancel subscription» («Отменить подписку»)

> **Примечание:** Чтобы получить помощь с отменой этого заказа, откройте запрос в службу поддержки через портал Azure.

Затем выполните следующие [шаги](/cloud/billing/marketplace/azure-marketplace-committed-contract), чтобы настроить вашу организацию ClickHouse для работы с новым контрактом Azure на фиксированный объем расходов.



## Требования для привязки контракта с фиксированным объемом расходов \\{#linking-requirements\\}

> **Примечание:** чтобы привязать вашу организацию к контракту маркетплейса с фиксированным объемом расходов:
> - пользователь, выполняющий описанные ниже шаги, должен быть администратором организации ClickHouse, к которой вы привязываете подписку;
> - все неоплаченные счета организации должны быть погашены (по любым вопросам обратитесь в [службу поддержки](https://clickhouse.com/support/program) ClickHouse).
