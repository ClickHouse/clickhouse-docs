# Миграция биллинга с модели pay-as-you-go (PAYG) на контракт с фиксированными расходами в облачном маркетплейсе {#migrate-payg-to-committed}

Если ваша организация ClickHouse в настоящее время использует биллинг через активную подписку pay-as-you-go (PAYG) в облачном маркетплейсе (или заказ) и вы хотите перейти на биллинг через контракт с фиксированными расходами в том же облачном маркетплейсе, примите ваше новое предложение, а затем следуйте приведенным ниже инструкциям в зависимости от вашего облачного провайдера.


## Важные замечания {#important-notes}

Обратите внимание, что отмена подписки PAYG в маркетплейсе не удаляет вашу учетную запись ClickHouse Cloud — отменяются только платежные отношения через маркетплейс. После отмены наша система прекратит выставление счетов за сервисы ClickHouse Cloud через маркетплейс. (Примечание: этот процесс происходит не мгновенно и может занять несколько минут).

После отмены подписки в маркетплейсе, если в вашей организации ClickHouse сохранена кредитная карта, мы спишем средства с этой карты в конце расчетного периода — если только до этого момента не будет подключена новая подписка через маркетплейс.

Если после отмены кредитная карта не настроена, у вас будет 14 дней, чтобы добавить действующую кредитную карту или новую подписку через облачный маркетплейс в вашу организацию. Если в течение этого периода способ оплаты не будет настроен, ваши сервисы будут приостановлены, а ваша организация будет считаться не соответствующей [требованиям биллинга](/manage/clickhouse-cloud-billing-compliance).

Любое использование, накопленное после отмены подписки, будет оплачено следующим настроенным действующим способом оплаты — в порядке приоритета: предоплаченный кредит, подписка через маркетплейс или кредитная карта.

По любым вопросам или для получения поддержки при настройке вашей организации с новой подпиской через маркетплейс обращайтесь в [службу поддержки](https://clickhouse.com/support/program) ClickHouse.


## AWS Marketplace {#aws-marketplace}

Если вы хотите использовать тот же идентификатор учетной записи AWS (AWS Account ID) для перехода с подписки с оплатой по факту использования (PAYG) на контракт с фиксированными расходами, рекомендуем [связаться с отделом продаж](https://clickhouse.com/company/contact) для внесения этого изменения. В этом случае не потребуется никаких дополнительных действий, и работа вашей организации ClickHouse и сервисов не будет прервана.

Если вы хотите использовать другой идентификатор учетной записи AWS для перехода вашей организации ClickHouse с подписки PAYG на контракт с фиксированными расходами, выполните следующие шаги:

### Шаги для отмены подписки AWS PAYG {#cancel-aws-payg}

1. **Перейдите на [AWS Marketplace](https://us-east-1.console.aws.amazon.com/marketplace)**
2. **Нажмите на кнопку "Manage Subscriptions"**
3. **Перейдите в раздел "Your Subscriptions":**
   - Нажмите на "Manage Subscriptions"
4. **Найдите ClickHouse Cloud в списке:**
   - Найдите и нажмите на ClickHouse Cloud в разделе "Your Subscriptions"
5. **Отмените подписку:**
   - В разделе "Agreement" нажмите на выпадающий список или кнопку "Actions" рядом с записью ClickHouse Cloud
   - Выберите "Cancel subscription"

> **Примечание:** Для получения помощи по отмене подписки (например, если кнопка отмены подписки недоступна) обратитесь в [службу поддержки AWS](https://support.console.aws.amazon.com/support/home#/).

Затем выполните эти [шаги](/cloud/billing/marketplace/aws-marketplace-committed-contract), чтобы настроить вашу организацию ClickHouse для работы с новым контрактом с фиксированными расходами AWS, который вы приняли.


## GCP Marketplace {#gcp-marketplace}

### Шаги по отмене заказа GCP PAYG {#cancel-gcp-payg}

1. **Перейдите в [консоль Google Cloud Marketplace](https://console.cloud.google.com/marketplace):**
   - Убедитесь, что вы вошли в нужную учетную запись GCP и выбрали соответствующий проект
2. **Найдите ваш заказ ClickHouse:**
   - В меню слева нажмите «Your Orders»
   - Найдите нужный заказ ClickHouse в списке активных заказов
3. **Отмените заказ:**
   - Найдите меню с тремя точками справа от вашего заказа и следуйте инструкциям по отмене заказа ClickHouse

> **Примечание:** Для получения помощи по отмене заказа обратитесь в [службу поддержки GCP](https://cloud.google.com/support/docs/get-billing-support).

Далее выполните следующие [шаги](/cloud/billing/marketplace/gcp-marketplace-committed-contract), чтобы настроить вашу организацию ClickHouse для работы с новым контрактом с фиксированными расходами GCP.


## Azure Marketplace {#azure-marketplace}

### Шаги по отмене подписки Azure PAYG {#cancel-azure-payg}

1. **Перейдите на [портал Microsoft Azure](http://portal.azure.com)**
2. **Откройте раздел «Подписки»**
3. **Найдите активную подписку ClickHouse, которую необходимо отменить**
4. **Отмените подписку:**
   - Нажмите на подписку ClickHouse Cloud, чтобы открыть сведения о подписке
   - Нажмите кнопку «Отменить подписку»

> **Примечание:** Для получения помощи по отмене заказа откройте запрос в службу поддержки на портале Azure.

Далее выполните следующие [шаги](/cloud/billing/marketplace/azure-marketplace-committed-contract), чтобы настроить вашу организацию ClickHouse для работы с новым контрактом с фиксированными расходами Azure.


## Требования для привязки к контракту с гарантированными расходами {#linking-requirements}

> **Примечание:** Чтобы привязать вашу организацию к контракту с гарантированными расходами в маркетплейсе:
>
> - Пользователь, выполняющий эти действия, должен иметь права администратора организации ClickHouse, к которой привязывается подписка
> - Все неоплаченные счета организации должны быть оплачены (при возникновении вопросов обращайтесь в [службу поддержки](https://clickhouse.com/support/program) ClickHouse)
