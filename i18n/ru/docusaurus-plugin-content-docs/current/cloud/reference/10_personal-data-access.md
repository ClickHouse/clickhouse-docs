## Введение {#intro}

Как зарегистрированный пользователь ClickHouse, вы можете просматривать и управлять данными своей учетной записи, включая контактную информацию. В зависимости от вашей роли вам также может быть доступна контактная информация других пользователей вашей организации, сведения об API-ключах и другая релевантная информация. Вы можете управлять этими данными напрямую через консоль ClickHouse в режиме самообслуживания.

**Что такое запрос на доступ субъекта данных (DSAR)**

В зависимости от вашего местоположения применимое законодательство может предоставлять вам дополнительные права в отношении персональных данных, которые ClickHouse хранит о вас (права субъекта данных), как описано в Политике конфиденциальности ClickHouse. Процесс реализации прав субъекта данных называется запросом на доступ субъекта данных (DSAR).

**Объем персональных данных**

Ознакомьтесь с Политикой конфиденциальности ClickHouse для получения подробной информации о персональных данных, которые собирает ClickHouse, и о том, как они могут использоваться.


## Самообслуживание {#self-service}

По умолчанию ClickHouse позволяет пользователям просматривать свои персональные данные непосредственно в консоли ClickHouse.

Ниже приведена сводка данных, которые ClickHouse собирает при регистрации учетной записи и использовании сервиса, а также информация о том, где можно просмотреть конкретные персональные данные в консоли ClickHouse.

| Расположение/URL                                             | Описание                                          | Персональные данные    |
| ------------------------------------------------------------ | ------------------------------------------------- | ---------------------- |
| https://auth.clickhouse.cloud/u/signup/                      | Регистрация учетной записи                        | email, password        |
| https://console.clickhouse.cloud/profile                     | Общие сведения профиля пользователя               | name, email            |
| https://console.clickhouse.cloud/organizations/OrgID/members | Список пользователей в организации                | name, email            |
| https://console.clickhouse.cloud/organizations/OrgID/keys    | Список API-ключей и информация об их создателях   | email                  |
| https://console.clickhouse.cloud/organizations/OrgID/audit   | Журнал активности с перечнем действий пользователей | email                  |
| https://console.clickhouse.cloud/organizations/OrgID/billing | Платежная информация и счета                      | billing address, email |
| https://console.clickhouse.cloud/support                     | Взаимодействие со службой поддержки ClickHouse    | name, email            |

Примечание: в URL-адресах с `OrgID` необходимо указать `OrgID` вашей учетной записи.

### Текущие клиенты {#current-customers}

Если у вас есть учетная запись и функция самообслуживания не решила вашу проблему с персональными данными, вы можете подать запрос субъекта данных на доступ в соответствии с Политикой конфиденциальности. Для этого войдите в свою учетную запись ClickHouse и создайте [обращение в службу поддержки](https://console.clickhouse.cloud/support). Это поможет нам подтвердить вашу личность и ускорить обработку вашего запроса.

Обязательно укажите следующие сведения в обращении:

| Поле        | Текст для включения в запрос                                                                        |
| ----------- | --------------------------------------------------------------------------------------------------- |
| Тема        | Запрос субъекта данных на доступ (DSAR)                                                             |
| Описание    | Подробное описание информации, которую вы хотите, чтобы ClickHouse нашел, собрал и/или предоставил. |

### Лица без учетной записи {#individuals-without-an-account}

Если у вас нет учетной записи и указанная выше функция самообслуживания не решила вашу проблему с персональными данными, и вы хотите подать запрос субъекта данных на доступ в соответствии с Политикой конфиденциальности, вы можете отправить такие запросы по электронной почте на адрес [privacy@clickhouse.com](mailto:privacy@clickhouse.com).


## Проверка личности {#identity-verification}

Если вы отправите запрос на доступ к персональным данным по электронной почте, мы можем запросить у вас определённую информацию для подтверждения вашей личности и обработки запроса. Действующее законодательство может требовать или разрешать нам отклонить ваш запрос. Если мы отклоним ваш запрос, мы сообщим вам причину в рамках законодательных ограничений.

Для получения дополнительной информации ознакомьтесь с [Политикой конфиденциальности ClickHouse](https://clickhouse.com/legal/privacy-policy)
