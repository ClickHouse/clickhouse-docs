## Что такое ClickHouse Cloud? {#what-is-clickhouse-cloud}

ClickHouse Cloud — это полностью управляемый облачный сервис, созданный оригинальными разработчиками
ClickHouse, самой быстрой и популярной колоночной СУБД с открытым исходным кодом для онлайн-аналитической
обработки данных (OLAP).

С ClickHouse Cloud инфраструктура, обслуживание, масштабирование и эксплуатация выполняются
за вас, что позволяет сосредоточиться на главном — на более быстром создании ценности
для вашей организации и клиентов.


## Преимущества ClickHouse Cloud {#benefits-of-clickhouse-cloud}

ClickHouse Cloud предлагает ряд важных преимуществ по сравнению с версией с открытым исходным кодом:

- **Быстрый выход на результат**: Начните разработку мгновенно, без необходимости планировать размер и масштабирование кластера.
- **Бесшовное масштабирование**: Автоматическое масштабирование адаптируется к изменяющимся нагрузкам, поэтому вам не нужно избыточно выделять ресурсы для пиковых нагрузок.
- **Бессерверная модель работы**: Мы позаботимся о размерах, масштабировании, безопасности, надежности и обновлениях.
- **Прозрачное ценообразование**: Платите только за фактическое использование с возможностью резервирования ресурсов и контроля масштабирования.
- **Совокупная стоимость владения**: Оптимальное соотношение цены и производительности при низких административных издержках.
- **Широкая экосистема**: Используйте привычные коннекторы данных, инструменты визуализации, SQL и языковые клиенты.
