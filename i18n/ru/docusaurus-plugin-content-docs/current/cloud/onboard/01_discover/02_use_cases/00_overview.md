ClickHouse Cloud подходит как для использования в качестве **основного хранилища данных**, так и в качестве **аналитического слоя**.

Колонночная архитектура ClickHouse, векторизованная обработка и облачно-ориентированный дизайн делают его уникально подходящим для аналитических нагрузок, которым необходимы одновременно высокая скорость и масштабируемость. В целом, наиболее распространённые варианты использования ClickHouse Cloud:

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud превосходно подходит для аналитики в реальном времени, обеспечивая отклик на запросы менее чем за секунду по миллиардам строк благодаря колонночному хранилищу и векторизованному движку выполнения. Платформа обрабатывает высокопроизводительный поток загрузки данных — миллионы событий в секунду — при этом позволяя выполнять прямые запросы к необработанным данным без необходимости предварительной агрегации. Материализованные представления обеспечивают агрегации в реальном времени и предвычисленные результаты, а приблизительные функции для квантилей и подсчётов предоставляют мгновенные инсайты, идеально подходящие для интерактивных дашбордов и принятия решений в реальном времени.                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud хорошо подходит для задач наблюдаемости, предлагая специализированные движки и функции, оптимизированные для временных рядов, которые позволяют с лёгкостью загружать и анализировать терабайты логов, метрик и трейсов. С помощью ClickStack, комплексного решения наблюдаемости на базе ClickHouse, организации могут преодолеть традиционное разделение на три изолированных домена — логи, метрики и трейсы — объединив все данные наблюдаемости на единой платформе, что позволяет выполнять коррелированный анализ и избавляет от сложности управления раздельными системами. Такой унифицированный подход делает решение идеальным для мониторинга производительности приложений, мониторинга инфраструктуры и анализа событий безопасности на уровне предприятия, при этом ClickStack предоставляет инструменты и интеграции, необходимые для полноценной реализации процессов наблюдаемости без изолированных хранилищ данных. |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | Экосистема ClickHouse для хранилищ данных обеспечивает подключение всего в несколько кликов и простую загрузку данных в ClickHouse. Благодаря отличной поддержке аналитики по историческим данным, озёр данных (data lakes), федеративных запросов и JSON как нативного типа данных пользователи могут хранить данные с высокой экономической эффективностью в масштабах всего предприятия.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud можно использовать на всех этапах ML-цепочки создания ценности — от исследования и подготовки до обучения, тестирования и инференса. Инструменты, такие как `clickhouse-local`, `clickhouse-server` и `chdb`, можно использовать для исследования, обнаружения и преобразования данных, в то время как ClickHouse может выступать в роли `feature store`, `vector store` или хранилища наблюдаемости для MLOps. Кроме того, он обеспечивает агентную аналитику с помощью встроенных средств, таких как полностью управляемый удалённый MCP-сервер, встроенное дополнение текста запросов, настройка графиков на основе ИИ и функция Ask AI в продукте.                                                                                                                                                                                                                                                         |
