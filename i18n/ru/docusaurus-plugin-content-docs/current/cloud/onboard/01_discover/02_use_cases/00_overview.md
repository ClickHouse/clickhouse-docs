ClickHouse Cloud подходит как для использования в качестве **основного хранилища данных**, так и как **аналитического слоя**.

Колоночная архитектура ClickHouse, векторизованная обработка и облачно-родной дизайн 
делают его особенно подходящим для аналитических нагрузок, которым одновременно нужны высокая скорость и масштабируемость.
В целом, наиболее распространённые варианты использования ClickHouse Cloud:

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud превосходно подходит для аналитики в реальном времени, обеспечивая ответы на запросы за доли секунды по миллиардам строк благодаря колоночной архитектуре хранения и векторизованному движку выполнения. Платформа обрабатывает высокопроизводительный приём данных — миллионы событий в секунду — и при этом позволяет выполнять прямые запросы к «сырым» данным без необходимости предварительной агрегации. Материализованные представления обеспечивают агрегирование и предвычисленные результаты в реальном времени, а аппроксимационные функции для квантилей и подсчётов дают мгновенные инсайты, идеально подходящие для интерактивных дашбордов и принятия решений в реальном времени.                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud хорошо подходит для задач наблюдаемости, предоставляя специализированные движки и функции, оптимизированные для временных рядов, которые позволяют с лёгкостью принимать и обрабатывать терабайты логов, метрик и трейсов. С помощью ClickStack — комплексного решения ClickHouse для наблюдаемости — организации могут преодолеть традиционное разделение на три изолированных хранилища логов, метрик и трейсов, объединив все данные наблюдаемости на одной платформе, что обеспечивает коррелированный анализ и устраняет сложность управления разрозненными системами. Такой единый подход делает платформу идеальной для мониторинга производительности приложений, мониторинга инфраструктуры и анализа событий безопасности в масштабах предприятия, при этом ClickStack предоставляет инструменты и интеграции, необходимые для полноценного цикла наблюдаемости без разрозненных хранилищ данных. |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | Экосистема ClickHouse для data warehousing и её коннекторы позволяют пользователям настроить систему в несколько кликов и легко загрузить свои данные в ClickHouse. Благодаря отличной поддержке анализа исторических данных, data lakes, федеративных запросов и JSON как нативного типа данных ClickHouse позволяет хранить данные масштабно и экономично.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud можно использовать на всех этапах ML-цепочки создания ценности — от исследования и подготовки данных до обучения, тестирования и инференса. Инструменты, такие как Clickhouse-local, Clickhouse-server и chdb, подходят для исследования, обнаружения и преобразования данных, тогда как ClickHouse может использоваться как feature store, vector store или хранилище наблюдаемости для MLOps. Кроме того, он обеспечивает agentic-аналитику благодаря встроенным инструментам, таким как полностью управляемый удалённый MCP server, inline-дополнение текста для запросов, настройка графиков на основе AI и Ask AI прямо в продукте.                                                                                                                                                                                                                                                         |
