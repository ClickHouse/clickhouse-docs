ClickHouse Cloud подходит как для использования в качестве **основного хранилища данных**, так и в качестве **аналитического слоя**.

Колоночная архитектура ClickHouse, векторизованная обработка и облачно-родная архитектура 
делают его особенно подходящим для аналитических нагрузок, которым необходимы и скорость, и масштабируемость.
В целом, наиболее распространенные варианты использования ClickHouse Cloud следующие:

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud превосходно подходит для аналитики в реальном времени, обеспечивая ответы на запросы за доли секунды по миллиардам строк благодаря колоночной архитектуре хранения и векторизованному движку выполнения. Платформа обрабатывает высокопроизводительный приём данных — миллионы событий в секунду — и при этом позволяет выполнять прямые запросы к сырым данным без необходимости предварительной агрегации. Materialized Views обеспечивают агрегирование в реальном времени и предварительно вычисленные результаты, а приближённые функции для квантилей и подсчётов предоставляют мгновенные аналитические выводы, идеально подходящие для интерактивных дашбордов и принятия решений в реальном времени.                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud хорошо подходит для задач наблюдаемости, предоставляя специализированные движки и функции, оптимизированные для временных рядов, которые позволяют с лёгкостью принимать и запрашивать терабайты логов, метрик и трейсов. С помощью ClickStack, комплексного решения наблюдаемости на базе ClickHouse, организации могут преодолеть традиционное разделение на три изолированных хранилища логов, метрик и трейсов, объединив все данные наблюдаемости на одной платформе, что обеспечивает коррелированный анализ и устраняет сложность управления разрозненными системами. Такой единый подход делает решение идеальным для мониторинга производительности приложений, мониторинга инфраструктуры и анализа событий безопасности в масштабах предприятия, при этом ClickStack предоставляет инструменты и интеграции, необходимые для полноценных процессов наблюдаемости без изолированных хранилищ данных. |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | Экосистема ClickHouse для хранилищ данных и подключения к ним позволяет пользователям настроить систему в несколько кликов и легко загрузить свои данные в ClickHouse. Благодаря отличной поддержке анализа исторических данных, озёр данных (data lakes), федеративных запросов и JSON в качестве нативного типа данных платформа позволяет экономично и масштабируемо хранить данные.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud может использоваться на всём протяжении жизненного цикла ML-решений — от исследования и подготовки данных до обучения, тестирования и инференса. Инструменты вроде Clickhouse-local, Clickhouse-server и chdb могут применяться для исследования, обнаружения и трансформации данных, а ClickHouse может использоваться как feature store, vector store или хранилище наблюдаемости для MLOps. Кроме того, платформа обеспечивает агентную аналитику (agentic analytics) благодаря встроенным инструментам, таким как полностью управляемый удалённый MCP-сервер, встроенное дополнение текста запросов, настройка графиков на базе ИИ и встроенная в продукт функция Ask AI.                                                                                                                                                                                                                                                         |
