ClickHouse Cloud подходит как для использования в качестве **основного хранилища данных**, так и в качестве **аналитического слоя**.

Колоночная архитектура ClickHouse, векторизованная обработка и облачно-ориентированный дизайн
делают его уникально подходящим для аналитических нагрузок, которым одновременно нужны скорость и масштабируемость.
В целом, наиболее распространённые варианты использования ClickHouse Cloud включают:

| Вариант использования                                                                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)           | ClickHouse Cloud превосходно подходит для аналитики в реальном времени, обеспечивая отклик на запросы менее секунды по миллиардам строк благодаря колоночной архитектуре хранения и векторизованному движку выполнения. Платформа обрабатывает высокопроизводительную ингестию данных — миллионы событий в секунду — при этом позволяя выполнять прямые запросы к «сырым» данным без необходимости предварительной агрегации. Материализованные представления (Materialized Views) обеспечивают агрегирование в реальном времени и предвычисленные результаты, а аппроксимирующие функции для квантилей и подсчётов дают мгновенные инсайты, идеально подходящие для интерактивных дашбордов и принятия решений в реальном времени.                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                       | ClickHouse Cloud отлично подходит для нагрузок, связанных с наблюдаемостью, предлагая специализированные движки и функции, оптимизированные для временных рядов и способные с лёгкостью выполнять ингестию и обработку запросов к терабайтам логов, метрик и трассировок. С помощью ClickStack, комплексного решения ClickHouse для наблюдаемости, организации могут преодолеть традиционное разделение на три изолированных хранилища логов, метрик и трассировок, объединив все данные наблюдаемости на единой платформе, что позволяет выполнять коррелированный анализ и устраняет сложность управления разрозненными системами. Такой унифицированный подход делает платформу идеальной для мониторинга производительности приложений, мониторинга инфраструктуры и анализа событий безопасности в масштабах предприятия, при этом ClickStack предоставляет инструменты и интеграции, необходимые для полноценных процессов наблюдаемости без изолированных хранилищ данных. |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)          | Экосистема ClickHouse для хранилищ данных и интеграций позволяет пользователям настроить решение в несколько кликов и легко загрузить свои данные в ClickHouse. Благодаря отличной поддержке анализа исторических данных, озёр данных, федеративных запросов и JSON как родного типа данных платформа позволяет пользователям хранить свои данные масштабируемо и экономично.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud может использоваться на всём протяжении жизненного цикла ML — от исследования и подготовки данных до обучения, тестирования и инференса. Инструменты вроде Clickhouse-local, Clickhouse-server и chdb можно использовать для исследования, поиска и трансформации данных, в то время как ClickHouse может выступать в роли feature store, векторного хранилища или хранилища данных наблюдаемости для MLOps. Кроме того, платформа обеспечивает аналитику на основе AI-агентов (agentic analytics) с помощью встроенных средств, таких как полностью управляемый удалённый MCP-сервер, автодополнение текста запросов, настройка графиков на основе ИИ и функция Ask AI в интерфейсе продукта.                                                                                                                                                                                                                                                         |
