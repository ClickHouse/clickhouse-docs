# Уведомления \{#notifications\}

ClickHouse Cloud отправляет уведомления о критических событиях, связанных с вашим сервисом или организацией. Чтобы понять, как отправляются и настраиваются уведомления, нужно учитывать несколько понятий:

1. **Категория уведомлений**: Обозначает группы уведомлений, такие как уведомления о биллинге, уведомления, связанные с сервисом, и т. д. В каждой категории есть несколько уведомлений, для которых можно настроить способ доставки.
2. **Уровень важности уведомления**: Уровень важности уведомления может быть `info`, `warning` или `critical` в зависимости от того, насколько важно уведомление. Это не настраивается.
3. **Канал уведомлений**: Канал — это способ получения уведомления, например UI, электронная почта, Slack и т. д. Для большинства уведомлений это настраивается.

## Получение уведомлений \{#receiving-notifications\}

Уведомления можно получать по разным каналам. ClickHouse Cloud поддерживает получение уведомлений по электронной почте, через UI ClickHouse Cloud и через Slack. Чтобы просмотреть текущие уведомления, нажмите значок колокольчика в меню в левом верхнем углу — откроется выезжающая панель. Нажав кнопку **View All** в нижней части выезжающей панели, вы перейдете на страницу, где отображается журнал активности всех уведомлений.

## Настройка уведомлений \{#customizing-notifications\}

Для каждого уведомления можно настроить способ его получения. Экран настроек можно открыть из выезжающей панели уведомлений или на второй вкладке в журнале активности уведомлений.

Вы можете настраивать уведомления, доставляемые через UI ClickHouse Cloud, и эти настройки применяются отдельно для каждого пользователя. Вы также можете настраивать уведомления, доставляемые на ваши собственные адреса электронной почты, но только пользователи с правами администратора могут настраивать уведомления, доставляемые на произвольные адреса электронной почты, и уведомления, доставляемые в каналы Slack.

Чтобы настроить доставку для конкретного уведомления, нажмите на значок карандаша, чтобы изменить каналы доставки уведомления.

## Уведомления ClickPipes \{#clickpipes-notifications\}

ClickHouse отправляет уведомления ClickPipes, когда в вашем ClickPipe возникают сбои или другие проблемы.

## Уведомления о биллинге \{#billing-notifications\}

ClickHouse отправляет уведомления о биллинге при проблемах с оплатой, а также когда использование по предоплаченным обязательствам достигает определенных пороговых значений.

## Поддерживаемые уведомления \{#supported-notifications\}

В настоящее время мы отправляем уведомления, связанные с биллингом (сбой платежа, превышение определенного порога использования и т. д.), а также уведомления о событиях масштабирования (масштабирование завершено, масштабирование заблокировано и т. д.).
