В этом документе перечислены модели в ClickHouse Cloud, на которые распространяются отдельные условия сторонних разработчиков моделей.
Все используемые модели в настоящее время размещаются в AWS Bedrock, а не у соответствующих разработчиков моделей.
Эти ссылки приведены для вашего удобства. Вы несете ответственность за соблюдение применимых условий для моделей и всех применимых политик использования.
