Мутации обновления — это запросы `ALTER`, которые изменяют данные в таблице посредством операций обновления. Прежде всего это такие запросы, как `ALTER TABLE UPDATE` и т.п. Выполнение таких запросов приводит к созданию новых версий частей данных с применёнными мутациями. Это означает, что такие запросы инициируют перезапись целых частей данных для всех строк, вставленных до мутации, что приводит к большому количеству операций записи.

:::info
Для операций обновления можно избежать большого количества операций записи, используя специализированные табличные движки, такие как [ReplacingMergeTree](/guides/replacing-merge-tree) или [CollapsingMergeTree](/engines/table-engines/mergetree-family/collapsingmergetree), вместо стандартного табличного движка MergeTree.
:::
