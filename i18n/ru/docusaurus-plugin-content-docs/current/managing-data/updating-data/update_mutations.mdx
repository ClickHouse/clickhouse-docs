Под мутациями обновления понимаются запросы `ALTER`, которые изменяют данные в таблице посредством обновлений. Прежде всего это такие запросы, как `ALTER TABLE UPDATE` и т.п. Выполнение таких запросов создаёт новые мутировавшие версии кусков данных. Это означает, что такие операции приводят к перезаписи целых кусков данных для всех данных, вставленных до мутации, что выражается в большом количестве операций записи.

:::info
Для обновлений вы можете избежать большого количества операций записи, используя специализированные движки таблиц, такие как [ReplacingMergeTree](/guides/replacing-merge-tree) или [CollapsingMergeTree](/engines/table-engines/mergetree-family/collapsingmergetree) вместо стандартного движка таблиц MergeTree.
:::
