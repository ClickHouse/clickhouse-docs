Под мутациями удаления понимаются запросы `ALTER`, которые изменяют данные таблицы посредством удаления. В частности, это такие запросы, как `ALTER TABLE DELETE` и т. п. Выполнение таких запросов приводит к созданию новых мутировавших версий кусков данных. Это означает, что такие операторы инициируют полную перезапись кусков данных для всех данных, вставленных до мутации, что приводит к большому количеству операций записи.

:::info
При удалении данных большого количества операций записи можно избежать, используя специализированные движки таблиц, такие как [ReplacingMergeTree](/guides/replacing-merge-tree) или [CollapsingMergeTree](/engines/table-engines/mergetree-family/collapsingmergetree), вместо стандартного движка таблиц MergeTree.
:::
