Чтобы подключиться к ClickHouse по HTTP(S), вам потребуется следующая информация:

| Parameter(s)              | Description                                                                                              |
| ------------------------- | -------------------------------------------------------------------------------------------------------- |
| `HOST` and `PORT`         | Обычно используется порт 8443 при использовании TLS или 8123 при работе без TLS.                         |
| `DATABASE NAME`           | По умолчанию доступна база данных `default`. Укажите имя базы данных, к которой вы хотите подключиться.  |
| `USERNAME` and `PASSWORD` | По умолчанию имя пользователя — `default`. Используйте имя пользователя, подходящее для вашего сценария. |

Сведения о вашем сервисе ClickHouse Cloud доступны в консоли ClickHouse Cloud.
Выберите сервис и нажмите **Connect**:

<Image img={cloud_connect_button} size="md" alt="Кнопка подключения сервиса ClickHouse Cloud" border />

Выберите **HTTPS**. Параметры подключения будут показаны в примере команды `curl`.

<Image img={connection_details_https} size="md" alt="Сведения о подключении к ClickHouse Cloud по HTTPS" border />

Если вы используете самостоятельный (self-managed) ClickHouse, параметры подключения задаются вашим администратором ClickHouse.
