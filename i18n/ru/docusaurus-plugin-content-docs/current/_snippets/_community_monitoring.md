## Решения для мониторинга от сообщества {#community-monitoring}

Сообщество ClickHouse разработало комплексные решения для мониторинга, которые интегрируются с популярными стеками observability. [ClickHouse Monitoring](https://github.com/duyet/clickhouse-monitoring) предоставляет полноценную систему мониторинга с готовыми дашбордами. Этот проект с открытым исходным кодом предлагает быстрый способ развертывания для команд, желающих внедрить мониторинг ClickHouse с использованием проверенных практик и готовых конфигураций дашбордов.

:::note
Как и другие подходы к прямому мониторингу базы данных, это решение выполняет запросы непосредственно к системным таблицам ClickHouse, что препятствует переходу инстансов в режим ожидания и влияет на оптимизацию затрат.
:::
