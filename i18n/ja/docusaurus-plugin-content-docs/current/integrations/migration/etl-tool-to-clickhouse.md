# サードパーティETLツールの使用

<img src={require('./images/third-party-01.png').default} class="image" alt="セルフマネージドClickHouseの移行" style={{width: '40%', padding: '30px'}}/>

外部データソースからClickHouseにデータを移動するための優れた選択肢は、多くの人気のあるETLおよびELTツールのいずれかを使用することです。以下のドキュメントを参照してください：

- [Airbyte](/integrations/data-ingestion/etl-tools/airbyte-and-clickhouse.md)
- [dbt](/integrations/data-ingestion/etl-tools/dbt/index.md)
- [Vector](/integrations/data-ingestion/etl-tools/vector-to-clickhouse.md)

しかし、ClickHouseと統合できるETL/ELTツールは他にもたくさんあるので、お気に入りのツールのドキュメントを確認してください。
