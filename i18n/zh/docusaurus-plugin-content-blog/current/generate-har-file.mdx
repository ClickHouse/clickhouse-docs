## 在 Google Chrome 中 \{#from-google-chrome\}

1. 按下 <kbd>F12</kbd> 或 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd>（Windows/Linux）/ <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>I</kbd>（Mac）打开开发者工具。
2. 点击“Network”（网络）选项卡。
3. 重新加载页面并重现问题。
4. 在开发者工具中点击下载按钮。

## 在 Mozilla Firefox 中 \{#from-mozilla-firefox\}

1. 按下 <kbd>F12</kbd> 或 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>（Windows/Linux）/ <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>E</kbd>（Mac）打开开发者工具。
2. 点击“Network”选项卡。
3. 重新加载页面并重现问题。
4. 在开发者工具的请求列表中右键单击，选择“Save All As HAR”。

## 在 Microsoft Edge 中 \{#from-microsoft-edge\}

1. 按下 <kbd>F12</kbd> 或 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd>（Windows/Linux）/ <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>I</kbd>（Mac）打开开发者工具。
2. 单击“Network”选项卡。
3. 重新加载页面并重现问题。
4. 在开发者工具的网络列表中单击右键，然后选择“Save all as HAR with content”（将全部另存为包含内容的 HAR）。

## 在 Safari 中 \{#from-safari\}

1. 启用开发者工具（如果尚未启用）：
   * 前往 Safari &gt; 设置 &gt; 高级。
   * 在底部勾选“在菜单栏中显示‘开发’菜单”。
2. 点击“开发” &gt; “显示网页检查器”。
3. 点击“网络”标签页。
4. 重新加载页面并重现问题。
5. 在开发者工具中点击“导出”按钮。

## 发送 .HAR 文件 \{#sending-har-file\}

1. 将文件重命名为简短且易于理解的名称（例如：login-issue.har）。
2. 压缩该文件（可选，但推荐）。
3. 将其附加到您的支持工单中，或通过电子邮件发送给分配给您的支持联系人。

:::note
HAR 文件可能包含会话 cookie 或其他敏感数据。
请仅与已授权的支持人员共享这些文件。
:::
