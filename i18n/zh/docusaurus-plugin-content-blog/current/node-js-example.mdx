## 在 Node.js 中使用 @clickhouse/client \{#use-nodejs-with-clickhouseclient\}

下面是一个基本示例代码文件 `main.ts`。

`package.json` (将其放在项目根目录 `./` 下) ：

```json
{
  "name": "一个简单的 ClickHouse 客户端示例",
  "version": "1.0.0",
  "main": "main.js",
  "license": "MIT",
  "devDependencies": {
    "typescript": "^5.3.2"
  },
  "dependencies": {
    "@clickhouse/client": "^0.2.6"
  }
}
```

`Main.ts` (将其放在 `./src` 目录下) ：

```ts
import { ClickHouseClient, createClient } from '@clickhouse/client'; // or '@clickhouse/client-web'

interface ClickHouseResultSet<T> {
  meta: Meta[];
  data: T[];
  rows: number;
  statistics: Statistics;
}

interface Statistics {
  elapsed: number;
  rows_read: number;
  bytes_read: number;
}

interface Meta {
  name: string;
  type: string;
}

interface Count {
  c: number;
}

//请根据需要替换客户端连接参数,如 `host`、
//`username`、`password`、`database`。

const initClickHouseClient = async (): Promise<ClickHouseClient> => {
  const client = createClient({
    host: 'https://FQDN.aws.clickhouse.cloud',
    username: 'default',
    password: 'password',
    database: 'default',
    application: `pingpong`,
  });

  console.log('ClickHouse ping');
  if (!(await client.ping())) {
    throw new Error('ClickHouse ping 失败!');
  }
  console.log('ClickHouse pong!');
  return client;
};

const main = async () => {
  console.log('正在初始化 ClickHouse 客户端');
  const client = await initClickHouseClient();

  const row = await client.query({
    query: `SELECT count() AS c FROM system.tables WHERE database='system'`,
  });

  const jsonRow: ClickHouseResultSet<Count> = await row.json();

  console.log(`找到 ${jsonRow.data[0].c} 个系统表!`);

  await client.close();
  console.log(`👋`);
};

main();
```

要安装软件包，请在 `./` 目录下运行 `yarn`：

```
$ yarn
yarn install v1.22.19

[4/4] 🔨  正在构建软件包...

```

在 `./` 目录下使用以下命令执行 `main.ts` 中的代码：

```
$ npx ts-node src/main.ts
```

将输出：

```
正在初始化 ClickHouse 客户端
ClickHouse ping
ClickHouse pong!
已找到 120 个系统表！

```
