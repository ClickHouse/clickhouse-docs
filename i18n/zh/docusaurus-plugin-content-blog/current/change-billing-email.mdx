## 问题 \{#question\}

如何在 ClickHouse Cloud 中更改计费联系人？

## 回答 \{#answer\}

若要以管理员身份更改账单联系人，请按照以下步骤操作：

1. 邀请一位新用户以管理员身份加入 Cloud 组织。
2. 在对方接受邀请后，进入 ClickHouse Cloud 控制台的账单页面（Admin-&gt;Billing），找到 “Billing contacts” 部分。
3. 使用 Edit 按钮将这位新的管理员用户设置为账单联系人。
