## 简介 \{#introduction\}

以下是一些常用查询语句，用于为用户授予特定权限。

## 如何为其他用户授予与当前用户相同的权限？

```sql
GRANT CURRENT GRANTS ON *.* TO another_user;
```


## 如何基于当前用户的授权为某个用户授予特定权限？

在下面的示例中，`another_user` 将能够对当前用户的所有数据库和表执行 `SELECT` 查询。

```sql
GRANT CURRENT GRANTS(SELECT ON *.*) TO another_user;
```


## 如何基于当前用户的权限，为某个用户在某个数据库上授予特定权限？

在下面的示例中，`another_user` 将能够对 `my_database` 中的所有表执行 `INSERT` 操作。

```sql
GRANT INSERT ON my_database.* TO another_user;
```


## 如何基于 `default` 用户为某个特定用户授予与其相同的全部权限？

```sql
GRANT default_role TO another_user;
```
