## 如何为 ClickHouse 做出贡献 \{#how-to-contribute-in-clickhouse\}

ClickHouse 是一个在 [GitHub 上开发](https://github.com/ClickHouse/ClickHouse) 的开源项目。

按照惯例，贡献指南发布在源代码仓库根目录中的 [CONTRIBUTING](https://github.com/ClickHouse/ClickHouse/blob/master/CONTRIBUTING.md) 文件里。

如果你想对 ClickHouse 提出重大变更建议，请考虑先在 GitHub 上[创建一个 issue](https://github.com/ClickHouse/ClickHouse/issues/new/choose)，说明你想做什么，以便先与维护者和社区进行讨论。[此类 RFC issue 的示例](https://github.com/ClickHouse/ClickHouse/issues?q=is%3Aissue+is%3Aopen+rfc)。

如果你的贡献与安全相关，也请查阅[我们的安全策略](https://github.com/ClickHouse/ClickHouse/security/policy/)。
