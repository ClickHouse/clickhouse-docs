该数据集包含 2011 年至 2020 年 12 月 6 日期间 GitHub 上的所有事件，共计 31 亿条记录。下载包大小约为 75 GB，如果以使用 lz4 压缩的表形式存储在磁盘上，最多需要约 200 GB 的磁盘空间。

完整的数据集说明、分析洞见、下载步骤以及交互式查询发布在[此处](https://ghe.clickhouse.tech/)。
