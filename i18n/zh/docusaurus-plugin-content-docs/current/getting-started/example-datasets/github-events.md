该数据集包含从 2011 年至 2020 年 12 月 6 日期间在 GitHub 上发生的所有事件，总计 31 亿条记录。下载大小为 75 GB，如果以 lz4 压缩方式存储在表中，最多将需要 200 GB 的磁盘空间。

完整的数据集说明、分析洞察、下载指引以及交互式查询发布在此[代码仓库](https://github.com/ClickHouse/github-explorer)中。
