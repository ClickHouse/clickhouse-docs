数据集包含2011年到2020年12月6日之间GitHub上所有事件，大小为31亿条记录。下载大小为75 GB，如果存储在带有lz4压缩的表中，将需要高达200 GB的磁盘空间。

完整的数据集描述、见解、下载说明和交互式查询已发布在 [这里](https://ghe.clickhouse.tech/)。
