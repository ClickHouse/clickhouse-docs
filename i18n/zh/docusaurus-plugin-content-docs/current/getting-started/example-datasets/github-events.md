该数据集包含从 2011 年至 2020 年 12 月 6 日 GitHub 上的所有事件，共 31 亿条记录。下载大小为 75 GB，如果以 lz4 压缩格式存储在表中，最多需要 200 GB 磁盘空间。

完整的数据集说明、分析见解、下载说明以及交互式查询发布在[此处](https://ghe.clickhouse.tech/)。
