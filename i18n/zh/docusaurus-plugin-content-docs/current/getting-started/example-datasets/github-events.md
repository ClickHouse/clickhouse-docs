该数据集包含从 2011 年到 2020 年 12 月 6 日的所有 GitHub 事件，共 31 亿条记录。下载包大小为 75 GB，如果使用 LZ4 压缩存储在表中，磁盘空间最多需要 200 GB。

完整的数据集说明、洞察、下载指南和交互式查询见[此处](https://ghe.clickhouse.tech/)。
