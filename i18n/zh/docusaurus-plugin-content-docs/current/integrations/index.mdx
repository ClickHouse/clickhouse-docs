# 集成

将 ClickHouse 与您常用的工具和服务连接起来，构建功能强大的数据管道和分析工作流程。

ClickHouse 集成按支持级别划分：

| Tier | Description |
|------|-------------|
| Core integrations | 由 ClickHouse 构建或维护，由 ClickHouse 提供支持，并位于 ClickHouse 的 GitHub 组织下 |
| Partner integrations | 由第三方软件供应商构建或维护并提供支持 |
| Community integrations | 由社区成员构建或维护并提供支持。除公共 GitHub 仓库和社区 Slack 频道之外，不提供任何直接支持 |
