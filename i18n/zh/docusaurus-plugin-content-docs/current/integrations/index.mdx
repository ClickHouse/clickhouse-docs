将 ClickHouse 与常用工具和服务连接，构建强大的数据管道和分析工作流程。

ClickHouse 集成按支持级别进行分类：

| Tier                   | Description                                                     |
| ---------------------- | --------------------------------------------------------------- |
| Core integrations      | 由 ClickHouse 构建或维护，由 ClickHouse 提供支持，并托管在 ClickHouse GitHub 组织下 |
| Partner integrations   | 由第三方软件供应商构建或维护，并由其提供支持                                          |
| Community integrations | 由社区成员构建或维护并提供支持。除公共 GitHub 仓库和社区 Slack 频道之外，不提供其他直接支持           |

:::tip[正在构建 ClickHouse 集成？]
如果您正在构建连接 ClickHouse 的产品，请先阅读[集成开发](/integrations/integration-development)指南：[构建集成](/integrations/integration-development/building-integrations)、[测试](/integrations/integration-development/testing-your-integration)以及[为您的产品编写文档](/integrations/integration-development/documenting-your-integration)。准备提交时，请在 [partner portal](https://clickhouse.com/partners) 注册您的集成。
:::
