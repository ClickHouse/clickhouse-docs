# 集成

将 ClickHouse 与常用工具和服务连接起来，以构建强大的数据管道和分析工作流。

ClickHouse 集成按支持级别进行分类：

| Tier | Description |
|------|-------------|
| Core integrations | 由 ClickHouse 构建或维护，由 ClickHouse 提供支持，并托管在 ClickHouse 的 GitHub 组织下 |
| Partner integrations | 由第三方软件供应商构建或维护并提供支持 |
| Community integrations | 由社区成员构建或维护并提供支持。除公共 GitHub 仓库和社区 Slack 频道外，不提供直接支持 |
