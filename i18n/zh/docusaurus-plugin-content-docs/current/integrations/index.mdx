ClickHouse 集成按照其支持水平组织：

- **核心集成：** 由 ClickHouse 构建或维护，受到 ClickHouse 的支持，并位于 ClickHouse GitHub 组织中
- **合作伙伴集成：** 由第三方软件供应商构建或维护，并得到支持
- **社区集成：** 由社区成员构建或维护并提供支持。除了公共 GitHub 存储库和社区 Slack 渠道外，没有直接支持可用

每个集成进一步分类为 **语言客户端**、**数据摄取**、**数据可视化** 和 **SQL 客户端** 类别。

:::note
我们正在积极编纂下面这个 ClickHouse 集成列表，因此它并不是详尽无遗的。请随时
[贡献](https://github.com/ClickHouse/clickhouse-docs#contributing) 任何相关的 ClickHouse 集成到该列表中。
:::
