要通过 HTTP(S) 连接到 ClickHouse，您需要以下信息：

| Parameter(s)              | Description                             |
| ------------------------- | --------------------------------------- |
| `HOST` and `PORT`         | 通常在使用 TLS 时端口为 8443，未使用 TLS 时端口为 8123。  |
| `DATABASE NAME`           | 默认提供一个名为 `default` 的数据库，请使用您希望连接的数据库名称。 |
| `USERNAME` and `PASSWORD` | 默认用户名为 `default`。请使用适合您使用场景的用户名。        |

您的 ClickHouse Cloud 服务的详细信息可在 ClickHouse Cloud 控制台中查看。
选择一个服务并单击 **Connect**：

<Image img={cloud_connect_button} size="md" alt="ClickHouse Cloud 服务连接按钮" border />

选择 **HTTPS**。连接详细信息会显示在示例 `curl` 命令中。

<Image img={connection_details_https} size="md" alt="ClickHouse Cloud HTTPS 连接详细信息" border />

如果您使用的是自管的 ClickHouse，则连接详细信息由您的 ClickHouse 管理员进行设置。
