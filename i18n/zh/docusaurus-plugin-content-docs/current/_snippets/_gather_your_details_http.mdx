要通过 HTTP(S) 连接到 ClickHouse，你需要以下信息：

| Parameter(s)              | Description                                |
| ------------------------- | ------------------------------------------ |
| `HOST` and `PORT`         | 通常，在使用 TLS 时端口为 8443，在不使用 TLS 时端口为 8123。   |
| `DATABASE NAME`           | 开箱即用地提供了一个名为 `default` 的数据库，请使用你要连接的数据库名称。 |
| `USERNAME` and `PASSWORD` | 开箱即用的用户名为 `default`。请使用适合你场景的用户名。          |

你的 ClickHouse Cloud 服务的详细信息可以在 ClickHouse Cloud 控制台中查看。
选择一个服务并点击 **Connect**：

<Image img={cloud_connect_button} size="md" alt="ClickHouse Cloud 服务连接按钮" border />

选择 **HTTPS**。连接详情会显示在示例 `curl` 命令中。

<Image img={connection_details_https} size="md" alt="ClickHouse Cloud HTTPS 连接详情" border />

如果你使用的是自管 ClickHouse，则连接详情由你的 ClickHouse 管理员进行配置。
