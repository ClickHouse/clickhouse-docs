Delete mutations refers to `ALTER` 查询，旨在通过删除操作来操纵表数据。最显著的例子是 `ALTER TABLE DELETE` 等查询。执行这些查询将生成数据部分的新变异版本。这意味着，诸如此类的语句会引发对所有在变异之前插入的数据的整个数据部分进行重写，这将转化为大量的写请求。

:::info
对于删除操作，您可以通过使用专用的表引擎，如 [ReplacingMergeTree](/guides/replacing-merge-tree) 或 [CollapsingMergeTree](/engines/table-engines/mergetree-family/collapsingmergetree)，来避免产生大量的写请求，而不是使用默认的 MergeTree 表引擎。
:::
