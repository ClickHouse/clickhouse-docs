本节提供多种示例数据集和样例，帮助你开始使用 ClickStack。这些示例展示了在 ClickStack 中处理可观测性数据的不同方式，涵盖从本地开发到生产环境的各种场景。

| Dataset | Description |
|---------|-------------|
| [Sample Data](sample-data.md) | 加载一个包含来自我们演示环境的日志、追踪和指标的示例数据集 |
| [Local Data](local-data.md) | 收集本地系统的指标和日志，并将其发送到 ClickStack 进行分析 |
| [Remote Demo Data](remote-demo-data.md) | 连接到我们的远程演示集群并排查一个问题 |
