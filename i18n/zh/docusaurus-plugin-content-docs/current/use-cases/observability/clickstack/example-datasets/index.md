本节提供多种示例数据集和用例，帮助你开始使用 ClickStack。这些示例展示了在 ClickStack 中处理可观测性数据的不同方式，覆盖从本地开发到生产环境的各类场景。

| Dataset | Description |
|---------|-------------|
| [Sample Data](sample-data.md) | 加载一个来自我们演示环境的示例数据集，其中包含日志、链路追踪和指标数据 |
| [Local Data](local-data.md) | 采集本地系统的指标和日志，并将其发送到 ClickStack 进行分析 |
| [Remote Demo Data](remote-demo-data.md) | 连接到我们的远程演示集群并通过真实案例进行问题排查 |
