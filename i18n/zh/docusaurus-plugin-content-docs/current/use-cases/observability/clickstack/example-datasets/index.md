| 数据集 | 描述 |
|---------|-------------|
| [示例数据](sample-data.md) | 加载一个包含来自我们演示环境的日志、跟踪和指标的示例数据集 |
| [本地数据](local-data.md) | 收集本地系统指标和日志，并将其发送到 ClickStack 进行分析 |
| [远程演示数据](remote-demo-data.md) | 连接到我们的远程演示集群并探索一个问题 |
