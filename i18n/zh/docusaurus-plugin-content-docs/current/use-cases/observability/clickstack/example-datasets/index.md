本节提供了多种示例数据集和示例，帮助你开始使用 ClickStack。这些示例展示了在 ClickStack 中处理可观测性数据的不同方式，涵盖从本地开发到生产环境的各类场景。

| 数据集 | 描述 |
|---------|-------------|
| [示例数据](sample-data.md) | 加载一个示例数据集，其中包含来自我们演示环境的日志（logs）、链路追踪（traces）和指标（metrics） |
| [本地数据](local-data.md) | 采集本地系统的指标（metrics）和日志（logs），并将其发送到 ClickStack 进行分析 |
| [远程演示数据](remote-demo-data.md) | 连接到我们的远程演示集群并排查一个问题 |
