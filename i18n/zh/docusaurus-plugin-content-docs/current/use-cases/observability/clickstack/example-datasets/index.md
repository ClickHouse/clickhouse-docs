本节提供了多种示例数据集和示例用法，帮助你入门 ClickStack。这些示例展示了在 ClickStack 中以不同方式处理可观测性数据，涵盖从本地开发到生产环境的各类场景。

| Dataset | Description |
|---------|-------------|
| [Sample Data](sample-data.md) | 加载一个包含我们演示环境日志、追踪和指标的示例数据集 |
| [Local Data](local-data.md) | 收集本地系统指标和日志，并将其发送到 ClickStack 进行分析 |
| [Remote Demo Data](remote-demo-data.md) | 连接到我们的远程演示集群并排查问题 |
