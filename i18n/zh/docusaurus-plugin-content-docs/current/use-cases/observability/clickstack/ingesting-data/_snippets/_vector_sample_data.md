如果你想使用示例数据集进行试验，请下载以下 nginx 示例。

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

:::note
这些数据是从一个已配置为输出 JSON 格式日志（便于解析）的 Nginx 实例中收集的。有关这些日志的 Nginx 配置，请参见 [《使用 ClickStack 监控 Nginx 日志》](/use-cases/observability/clickstack/integrations/nginx#configure-nginx)。
:::
