## 描述 {#description}

与 [`PrettyCompact`](./PrettyCompact.md) 格式不同的是，该格式不使用 [ANSI 转义序列](http://en.wikipedia.org/wiki/ANSI_escape_code)。 
这是在浏览器中显示该格式以及使用 'watch' 命令行工具所必需的。
