## Description {#description}

不同于 [`PrettyCompact`](./PrettyCompact.md) 格式，因为不使用 [ANSI-escape sequences](http://en.wikipedia.org/wiki/ANSI_escape_code)。这是在浏览器中显示格式以及使用 'watch' 命令行工具所必需的。
