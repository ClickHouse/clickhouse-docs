## 描述 \\{#description\\}

与 [`Pretty`](./Pretty.md) 格式的区别在于，该格式会在表格的行之间绘制网格线。
因此，输出更加紧凑。

:::note
在交互模式下，该格式是命令行客户端的默认输出格式。
:::
