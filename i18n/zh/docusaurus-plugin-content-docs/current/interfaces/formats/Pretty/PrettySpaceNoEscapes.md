## 描述 {#description}

与 [`PrettySpace`](./PrettySpace.md) 格式不同的是，不使用 [ANSI 转义序列](http://en.wikipedia.org/wiki/ANSI_escape_code)。 
这是在浏览器中显示此格式以及使用 'watch' 命令行工具所必需的。
