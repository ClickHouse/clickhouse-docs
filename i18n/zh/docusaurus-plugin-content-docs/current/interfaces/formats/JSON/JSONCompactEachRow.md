## 描述 {#description}

与 [`JSONEachRow`](./JSONEachRow.md) 不同之处仅在于数据行被输出为数组，而不是对象。

```sql

```

以 `JSONCompactEachRow` 格式读取数据：

```sql
SELECT *
FROM football
FORMAT JSONCompactEachRow
```
