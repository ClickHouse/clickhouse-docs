## 描述 {#description}

与[`TabSeparated`](/interfaces/formats/TabSeparated)格式不同，行的写入不需要转义。

:::note
使用此格式解析时，每个字段中不允许包含制表符或换行符。
:::

对于`TabSeparatedRaw`格式和`RawBlob`格式的比较，请参见：[原始格式比较](../RawBLOB.md/#raw-formats-comparison)

```sql
SELECT *
FROM football
FORMAT TabSeparatedRaw
```
