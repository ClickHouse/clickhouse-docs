## 运行状态告警 {#uptime-alerts}

用户现在可以在[状态页面](https://status.clickhouse.com/)上查看各区域的运行时间,并订阅服务中断告警通知。


## SLA {#sla}

对于特定的承诺消费合同,我们还提供服务级别协议(SLA)。如需了解更多关于我们 SLA 政策的信息,请通过 [sales@clickhouse.com](mailto:sales@clickhouse.com) 联系我们。
