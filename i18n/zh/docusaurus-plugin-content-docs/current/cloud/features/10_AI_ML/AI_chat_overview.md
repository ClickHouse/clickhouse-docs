# Cloud 中的 ClickHouse Assistant 助手 \{#clickhouse-assistant-agent-in-cloud\}

“ClickHouse Assistant” 助手是一种开箱即用的体验，使用户能够在其 ClickHouse Cloud 服务中托管的数据之上触发复杂的分析任务。
用户无需编写 SQL 或在仪表板之间导航，只需用自然语言描述他们正在寻找的内容。
该助手会返回生成的查询、可视化或摘要，还可以结合活动标签页、已保存的查询、schema 细节和仪表板等上下文来提高准确性。
它被设计为一个嵌入式助手，帮助用户快速从问题走向洞察，并从提示转化为可用的仪表板或 API。

该体验还嵌入了一个“Docs AI”子助手，可直接在控制台中就 ClickHouse 文档提出具体问题。
用户无需在数百页文档中搜索，只需直接提问，例如 &quot;How do I configure materialized views?&quot; 或 &quot;What&#39;s the difference between ReplacingMergeTree and AggregatingMergeTree?&quot;，即可获得包含相关代码示例和源文档链接的精确答案。

有关更多详细信息，请参阅 [guides](/use-cases/AI_ML/AIChat) 部分。
