ClickHouse Agents 中的聊天界面支持对话、分支、并排比较和共享。

## 对话 \{#conversations\}

在消息输入框中输入内容并点击发送，即可开始新的对话。每个对话都会保存到侧边栏的历史记录中，之后可重新打开、重命名或删除。

你可以直接编辑自己发送的任意消息；agent 会从该位置重新生成回复。你也可以仅重新生成 agent 的上一条回复，而无需重新发送消息。

## 书签 \{#bookmarks\}

你可以为某条消息或整个对话添加书签，方便日后快速找回。书签仅自己可见，并且在对话重命名后仍会保留。

## 分叉 \{#forking\}

分叉会基于某条特定消息创建一个新的对话分支。使用它可在不影响原始对话线程的情况下探索另一条路径。提供三种分叉模式：

* **仅可见消息** — 复制通向分叉消息的直接路径。
* **包含相关分支** — 复制主路径以及所有现有分支。
* **包含到此处的全部内容** — 复制直到分叉消息为止的全部内容。

分叉出的对话彼此独立——所做的更改不会同步回原始对话。

## 多对话 \{#multi-conversation\}

多对话会并排运行两个对话，并向两边发送相同的提示词。可用它比较不同模型的回复，或对不同的智能体配置进行 A/B 测试。

点击聊天顶部栏中的 **+** 按钮，即可在当前对话旁边创建一个并行对话。

## 共享聊天 \{#sharing-chats\}

为任意对话生成共享链接，以便发送给团队成员或保存备查。接收者看到的是只读视图，其中包括工件和可见分支。你可以随时在共享面板中撤销链接。

对现有消息的编辑会显示在共享视图中；在生成链接后添加的消息则不会显示。
