智能体构建器用于创建和配置智能体。

:::tip 刚开始使用 ClickHouse Agents？
请先阅读[快速入门](/cloud/features/ai-ml/agents/quickstart)。下方页面将详细介绍构建器的各个部分。
:::

本节介绍智能体构建器的各个组成部分，包含以下页面：

| Page                                                                      | Description              |
| ------------------------------------------------------------------------- | ------------------------ |
| [Code interpreter](/cloud/features/ai-ml/agents/builder/code-interpreter) | 在沙盒环境中执行代码。              |
| [Web search](/cloud/features/ai-ml/agents/builder/web-search)             | 搜索公共网页。                  |
| [Image generation](/cloud/features/ai-ml/agents/builder/image-generation) | 根据文本生成图像。                |
| [Vision](/cloud/features/ai-ml/agents/builder/vision)                     | 支持图像输入。                  |
| [MCP servers](/cloud/features/ai-ml/agents/builder/mcp-servers)           | 将第三方 MCP servers 连接到智能体。 |
| [Skills](/cloud/features/ai-ml/agents/builder/skills)                     | 可复用的指令包。                 |
| [Subagents](/cloud/features/ai-ml/agents/builder/subagents)               | 将工作委派给子智能体。              |
