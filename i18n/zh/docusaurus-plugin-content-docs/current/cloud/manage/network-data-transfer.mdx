ClickHouse Cloud 计量传入和传出数据。  
这包括任何进出 ClickHouse Cloud 的数据，以及任何区域内和跨区域的数据传输。  
此使用情况在服务级别进行跟踪。根据此使用情况，客户会产生数据传输费用，这些费用将添加到他们的月账单中。

ClickHouse Cloud 收取以下费用：  
- 从 ClickHouse Cloud 到公共互联网的数据传出，包括到其他云供应商的其他区域。  
- 在同一云供应商内发送到另一个区域的数据。

区域内的数据传输或私有链接/私有服务连接使用和数据传输不会收取费用。  
然而，我们保留在发现影响我们合理收费能力的使用模式时实施额外数据传输定价维度的权利。

数据传输费用因云服务提供商 (CSP) 和区域而异。  
公共互联网的传出定价仅基于源区域。  
跨区域（或区域间）定价取决于源区域和目标区域。

:::note  
请注意，当通过原生协议使用非内联值（例如 `INSERT INTO [TABLE] FROM INFILE [FILE] FORMAT NATIVE`）进行插入时，ClickHouse 客户端会从服务器提取元数据以打包数据。如果元数据大于插入负载，您可能会从服务器的角度看到比实际的传入数据更多的传出数据。如果这不可接受，请考虑使用 VALUES 语法内联数据或使用 HTTP 协议。
