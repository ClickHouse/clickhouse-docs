## 什么是 ClickHouse Cloud？ {#what-is-clickhouse-cloud}

ClickHouse Cloud 是由 ClickHouse 原始创建者打造的全托管云服务。ClickHouse 是最快且最受欢迎的开源列式在线分析处理（OLAP）数据库。

使用 Cloud，基础设施、维护、扩展和运维工作均由我们负责，让您可以专注于最重要的事情——更快地为您的组织和客户创造价值。


## ClickHouse Cloud 的优势 {#benefits-of-clickhouse-cloud}

相比开源版本,ClickHouse Cloud 提供了以下几项主要优势:

- **快速创造价值**: 无需配置集群规模和扩展,即可立即开始构建应用。
- **无缝扩展**: 自动扩展功能根据工作负载变化动态调整,无需为峰值使用量预留过多资源。
- **无服务器运维**: 我们负责处理容量规划、扩展、安全、可靠性和升级,您只需专注于业务。
- **透明定价**: 按实际使用量付费,并提供资源预留和扩展控制功能。
- **总拥有成本**: 最佳性价比和极低的管理开销。
- **丰富的生态系统**: 支持您常用的数据连接器、可视化工具、SQL 客户端和各种编程语言客户端。
