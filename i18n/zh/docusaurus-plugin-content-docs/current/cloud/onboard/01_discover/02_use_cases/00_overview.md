ClickHouse Cloud 既适合作为**主数据存储**，也适合作为**分析层**。

ClickHouse 的列式架构、向量化处理以及云原生设计，使其在同时要求高速度与高扩展性的分析型工作负载中具有独特优势。
总体而言，ClickHouse Cloud 最常见的使用场景包括：

| 使用场景                                                                                 | 说明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [实时分析](/cloud/get-started/cloud/use-cases/real-time-analytics)                      | ClickHouse Cloud 在实时分析场景中表现出色，依托其列式存储架构和向量化执行引擎，即使在处理数十亿行数据时，依然能够实现亚秒级查询响应。平台可以以每秒数百万事件的高吞吐量接收数据，同时支持对原始数据进行直接查询，无需预聚合。`Materialized Views` 提供实时聚合和预计算结果，而用于分位数和计数的近似函数可以即时输出洞察，非常适合交互式仪表盘和实时决策。                                                                                                                                                                                                                                                                                                                                                              |
| [可观测性](/cloud/get-started/cloud/use-cases/observability)                            | ClickHouse Cloud 非常适合可观测性工作负载，具备针对时序数据优化的专用引擎和函数，可以轻松摄取和查询 TB 级别的日志、指标与追踪数据。通过 ClickStack（ClickHouse 提供的完整可观测性解决方案），组织可以打破传统日志、指标和追踪三大孤岛，在单一平台上统一所有可观测性数据，从而实现关联分析，并消除管理多个独立系统的复杂性。这种统一的方法非常适用于企业级的应用性能监控、基础设施监控以及安全事件分析，而 ClickStack 提供了实现完整可观测性工作流所需的工具与集成能力，避免数据孤岛。 |
| [数据湖与数据仓库](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)          | ClickHouse 在数据仓库生态中的连接能力，使用户只需点击几下就能完成配置，并轻松将数据导入 ClickHouse。借助其对历史数据分析、数据湖、查询联邦，以及将 JSON 作为原生数据类型的良好支持，用户可以在大规模场景下以高性价比的方式存储数据。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [机器学习与人工智能](/cloud/get-started/cloud/use-cases/AI_ML)                          | ClickHouse Cloud 可用于整个机器学习价值链，从数据探索与准备，到训练、测试和推理。`clickhouse-local`、`clickhouse-server` 和 `chdb` 等工具可用于数据探索、发现与转换，而 ClickHouse 本身可以作为特征存储、向量存储或 MLOps 可观测性存储使用。此外，它还通过内置工具支持智能体驱动分析（agentic analytics），例如全托管远程 MCP server、查询内联文本补全、AI 驱动的图表配置，以及产品内的 Ask AI 功能。                                                                                                                                                                                                                                                         |
