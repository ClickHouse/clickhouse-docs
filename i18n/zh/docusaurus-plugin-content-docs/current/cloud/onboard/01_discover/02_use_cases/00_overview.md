ClickHouse Cloud 既适合作为**主数据存储**，也适合作为**分析层**。

ClickHouse 的列式架构、向量化处理以及云原生设计，使其在需要同时兼顾速度和规模的分析型工作负载方面具有独特优势。总体来看，ClickHouse Cloud 最常见的使用场景包括：

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud 在实时分析方面表现出色，通过其列式存储架构和向量化执行引擎，即使在处理数十亿行数据时也能提供亚秒级查询响应。该平台可以以每秒数百万事件的高吞吐量摄取数据，同时支持直接在原始数据上执行查询，无需预聚合。借助物化视图，可以实现实时聚合和预计算结果，而用于分位数和计数的近似函数则可以即时提供洞察，非常适合交互式仪表盘和实时决策。                                                                                                                                                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud 非常适合可观测性工作负载，具备针对时序数据优化的专用引擎和函数，能够轻松摄取和查询 TB 级日志、指标与追踪数据。通过 ClickStack（ClickHouse 的完整可观测性解决方案），组织可以在单一平台中统一所有可观测性数据，打破传统的日志、指标和追踪三大数据孤岛，实现关联分析，并消除管理多个独立系统的复杂性。这种统一的方法使其非常适用于企业级的应用性能监控、基础设施监控及安全事件分析，同时 ClickStack 提供了完成全栈可观测性工作流所需的工具与集成，避免产生新的数据孤岛。 |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | ClickHouse 在数据仓库生态中的连接能力，使用户只需点击几下即可完成环境配置，并轻松将数据导入 ClickHouse。凭借对历史数据分析、数据湖、查询联邦以及原生 JSON 数据类型的出色支持，它可帮助用户在大规模场景下以高性价比的方式存储数据。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud 可覆盖整个机器学习价值链，从探索与准备到训练、测试和推理。诸如 clickhouse-local、clickhouse-server 和 chdb 等工具可用于数据探索、发现与转换，而 ClickHouse 本身可以用作特征存储、向量存储或 MLOps 可观测性存储。此外，它还通过内置工具支持智能代理驱动的分析（agentic analytics），例如完全托管的远程 MCP server、查询内联文本补全、AI 驱动的图表配置以及产品内的 Ask AI 功能。                                                                                                                                                                                                                                                         |
