ClickHouse Cloud 既适合作为**主数据存储**，也适合作为**分析层**。

ClickHouse 的列式架构、向量化处理以及云原生设计，
使其在需要同时兼顾速度与规模的分析型工作负载上具有独特优势。
总体而言，ClickHouse Cloud 最常见的用例包括：

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud 在实时分析场景中表现出色，通过其列式存储架构和向量化执行引擎，即使在数十亿行数据上也能提供亚秒级查询响应。该平台能够处理每秒数百万事件的高吞吐数据写入，并支持在原始数据上直接查询，而无需预聚合。`Materialized Views` 可以提供实时聚合和预计算结果，而用于分位数和计数的近似函数能够即时产出洞察，非常适合交互式仪表盘和实时决策。                                                                                                                                                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud 非常适合可观测性工作负载，提供针对时序数据优化的专用引擎和函数，能够轻松摄取与查询海量日志、指标和链路追踪数据（TB 级）。借助 ClickHouse 的综合可观测性解决方案 ClickStack，组织可以在单一平台上统一所有可观测性数据，打破传统的日志、指标与追踪三大孤岛，实现关联分析，并消除管理多个独立系统的复杂性。这种统一的方式非常适合大规模的应用性能监控、基础设施监控以及安全事件分析，而 ClickStack 提供了完成端到端可观测性流程所需的工具和集成能力，从而避免数据孤岛。 |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | ClickHouse 在数据仓库生态方面具备良好的连接能力，用户只需点击几下即可完成环境配置，并轻松将数据导入 ClickHouse。借助对历史数据分析、数据湖、查询联邦以及将 JSON 作为原生数据类型的出色支持，它能够帮助用户在大规模下以低成本高效存储数据。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud 可用于整个机器学习价值链，从探索与准备阶段一直到训练、测试和推理阶段。`clickhouse-local`、`clickhouse-server` 与 `chdb` 等工具可用于数据探索、发现与转换，而 ClickHouse 本身可以作为特征存储、向量存储或 MLOps 可观测性存储。此外，它还通过内置能力支持智能分析，例如全托管远程 MCP 服务器、查询内联文本补全、AI 驱动的图表配置以及产品内的 Ask AI 功能。                                                                                                                                                                                                                                                         |
