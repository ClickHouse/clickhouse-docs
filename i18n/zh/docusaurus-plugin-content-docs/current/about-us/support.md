# ClickHouse Cloud 支持服务

ClickHouse 为 ClickHouse Cloud 用户和客户提供支持服务。我们的目标是打造一支能够充分体现 ClickHouse 产品特性的支持服务团队——无与伦比的性能、易用性，以及极其快速且高质量的响应与结果。详情请访问[ClickHouse 支持计划](https://clickhouse.com/support/program/)页面。

[登录 Cloud 控制台](https://console.clickhouse.cloud/support)，在菜单中选择 **Help -> Support** 来新建支持工单，并查看你已提交工单的处理状态。 

你也可以订阅我们的[状态页面](https://status.clickhouse.com)，以便及时获知任何影响我们平台的事件。

:::note
请注意，只有订阅客户在支持事件上享有服务级别协议（SLA）。如果你目前还不是 ClickHouse Cloud 用户——尽管我们会尽力回答你的问题，但仍建议你优先使用我们的社区资源：

- [ClickHouse 社区 Slack 频道](https://clickhouse.com/slack)
- [其他社区选项](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
:::
