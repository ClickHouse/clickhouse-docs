# ClickHouse Cloud 支持服务

ClickHouse 为 ClickHouse Cloud 用户和客户提供支持服务。我们的目标是打造一支能够体现 ClickHouse 产品特质的支持服务团队——卓越的性能、易用性以及快速、高质量的服务响应。详情请参阅 [ClickHouse 支持计划](https://clickhouse.com/support/program/)页面。

[登录 Cloud 控制台](https://console.clickhouse.cloud/support),从菜单中选择 **帮助 -> 支持** 即可创建新的支持工单并查看已提交工单的状态。

您还可以订阅我们的[状态页面](https://status.clickhouse.com),以便在平台发生任何事件时及时收到通知。

:::note
请注意,只有订阅客户才享有支持事件的服务级别协议(SLA)。如果您目前不是 ClickHouse Cloud 用户,虽然我们会尽力回答您的问题,但建议您优先使用以下社区资源:

- [ClickHouse 社区 Slack 频道](https://clickhouse.com/slack)
- [其他社区资源](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
:::
