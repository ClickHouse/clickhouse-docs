# ClickHouse Cloud 支持服务

ClickHouse 为 ClickHouse Cloud 用户和客户提供支持服务。我们的目标是打造一支能够体现 ClickHouse 产品特性的支持服务团队——无与伦比的性能、易用性，以及极其快速且高质量的响应。有关详细信息，请访问我们的 [ClickHouse 支持计划](https://clickhouse.com/support/program/) 页面。

[登录 Cloud 控制台](https://console.clickhouse.cloud/support)，从菜单中选择 **Help -> Support**，以创建新的支持工单并查看已提交工单的状态。 

您还可以订阅我们的 [状态页面](https://status.clickhouse.com)，以便在有任何影响我们平台的事件时及时获得通知。

:::note
请注意，只有订阅客户在支持请求方面享有服务级别协议 (SLA)。如果您目前还不是 ClickHouse Cloud 用户——虽然我们会尽力回答您的问题，但我们更鼓励您前往我们的以下社区资源之一：

- [ClickHouse 社区 Slack 频道](https://clickhouse.com/slack)
- [其他社区选项](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
:::
