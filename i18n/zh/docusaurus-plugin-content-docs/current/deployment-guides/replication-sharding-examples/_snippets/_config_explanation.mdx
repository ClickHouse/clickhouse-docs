* `config.d` 目录包含 ClickHouse 服务器配置文件 `config.xml`，
  在该文件中可以为每个 ClickHouse 节点定义自定义配置。该配置会与
  每次安装 ClickHouse 时自带的默认配置文件 `config.xml`
  合并。
* `users.d` 目录包含用户配置文件 `users.xml`，在该文件中可以为
  用户定义自定义配置。该配置会与每次安装 ClickHouse 时自带的
  默认 `users.xml` 用户配置文件合并。

:::tip Custom configuration directories
最佳实践是在编写自定义配置时，使用 `config.d` 和 `users.d` 目录，
而不是直接修改 `/etc/clickhouse-server/config.xml` 和
`etc/clickhouse-server/users.xml` 中的默认配置。

这一行

```xml
<clickhouse replace="true">
```

确保在 `config.d` 和 `users.d` 目录中定义的配置节会覆盖默认 `config.xml` 和 `users.xml` 文件中定义的配置节。
:::
