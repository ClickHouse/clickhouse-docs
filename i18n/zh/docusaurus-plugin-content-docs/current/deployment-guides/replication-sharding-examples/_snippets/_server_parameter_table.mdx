对于每个服务器，需要指定以下参数：

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | 远程服务器的地址。您可以使用域名，也可以使用 IPv4 或 IPv6 地址。如果指定的是域名，服务器在启动时会发起一次 DNS 请求，并在整个运行期间使用该解析结果。如果 DNS 请求失败，服务器将无法启动。如果您更改了 DNS 记录，则需要重启服务器。 | -            |
| `port`        | 用于消息传递的 TCP 端口（配置中的 `tcp_port`，通常设置为 9000）。不要与 `http_port` 混淆。                                                                                                                                                                                                                           | -            |
