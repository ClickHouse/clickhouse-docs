对于每个服务器，需要指定以下参数：

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | 远程服务器的地址。可以使用域名或 IPv4 / IPv6 地址。如果指定了域名，服务器在启动时会发起 DNS 查询，并在服务器运行期间缓存结果。如果 DNS 查询失败，服务器将无法启动。如果修改了该域名的 DNS 记录，则需要重启服务器。 | -            |
| `port`        | 用于消息通信的 TCP 端口（配置中的 `tcp_port`，通常设置为 9000）。不要与 `http_port` 混淆。                                                                                                                                                                                                                       | -            |
