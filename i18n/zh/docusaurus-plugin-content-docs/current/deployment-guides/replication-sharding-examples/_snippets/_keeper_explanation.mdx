每个配置文件将包含以下唯一配置（如下所示）。  
用于 ClickHouse Keeper 节点的 `server_id` 应在集群中唯一，并且与 `<raft_configuration>` 部分中定义的服务器 `<id>` 匹配。  
`tcp_port` 是 ClickHouse Keeper 的 _客户端_ 使用的端口。

```xml
<tcp_port>9181</tcp_port>
<server_id>{id}</server_id>
```

以下部分用于配置参与 [raft 共识算法](https://en.wikipedia.org/wiki/Raft_(algorithm)) 的服务器： 

```xml
<raft_configuration>
    <server>
        <id>1</id>
        <hostname>clickhouse-keeper-01</hostname>
        <!-- TCP port used for communication between ClickHouse Keeper nodes -->
        <!--highlight-next-line-->
        <port>9234</port>
    </server>
    <server>
        <id>2</id>
        <hostname>clickhouse-keeper-02</hostname>
        <port>9234</port>
    </server>
    <server>
        <id>3</id>
        <hostname>clickhouse-keeper-03</hostname>
        <port>9234</port>
    </server>
</raft_configuration>
```
