通过启用 `listen_host` 配置，可以允许通过网络接口进行外部访问。这样可以确保 ClickHouse 服务器主机可被其他主机访问：

```xml
<listen_host>0.0.0.0</listen_host>
```

HTTP API 端口已设置为 `8123`：

```xml
<http_port>8123</http_port>
```

用于 ClickHouse 原生协议在 clickhouse-client 与其他原生 ClickHouse 工具，以及 clickhouse-server 与其他 clickhouse-server 之间交互的 TCP 端口设置为 `9000`：

```xml
<tcp_port>9000</tcp_port>
```
