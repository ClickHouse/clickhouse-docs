通过启用 `listen_host` 设置，可以为网络接口打开外部通信。这确保 ClickHouse 服务器主机可以被其他主机访问：

```xml
<listen_host>0.0.0.0</listen_host>
```

HTTP API 的端口设置为 `8123`：

```xml
<http_port>8123</http_port>
```

用于 `clickhouse-client` 与其他原生 ClickHouse 工具，以及 `clickhouse-server` 与其他 `clickhouse-server` 之间通过 ClickHouse 原生协议交互的 TCP 端口被设置为 `9000`：

```xml
<tcp_port>9000</tcp_port>
```
