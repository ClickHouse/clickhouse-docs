# 用于集成的表引擎

ClickHouse 提供多种与外部系统集成的方式，其中包括表引擎。与所有其他表引擎一样，其配置是通过 `CREATE TABLE` 或 `ALTER TABLE` 语句完成的。随后，从用户的角度来看，已配置的集成与普通表无异，但对其发起的查询会被转发到外部系统。这种透明的查询特性是该方案相较于其他集成方法（如字典或表函数，需要在每次使用时采用自定义查询方式）的关键优势之一。
