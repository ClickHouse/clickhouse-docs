## 描述 \{#description\}

包含服务器所支持的数据库引擎列表。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 数据库引擎名称。

{/*AUTOGENERATED_END*/ }

## 示例 \{#example\}

```sql title="Query"
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text title="Response"
┌─name─────┐
│ Ordinary │
│ Atomic   │
│ Lazy     │
└──────────┘
```
