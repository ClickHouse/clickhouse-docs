包含服务器支持的数据库引擎列表。

该表包含以下列（列类型见括号）：

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 数据库引擎名称。

{/*AUTOGENERATED_END*/ }

示例：

```sql
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text
┌─name─────┐
│ Ordinary │
│ Atomic   │
│ Lazy     │
└──────────┘
```
