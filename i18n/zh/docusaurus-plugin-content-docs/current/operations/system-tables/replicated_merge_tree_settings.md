## 描述 \{#description\}

包含 ReplicatedMergeTree 引擎所有专有设置的列表，以及各设置的当前值、默认值和说明。您可以在 CREATE 查询的 SETTINGS 部分修改其中任意设置。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 设置名称。
* `value` ([String](../../sql-reference/data-types/)) — 设置值。
* `default` ([String](../../sql-reference/data-types/)) — 设置的默认值。
* `changed` ([UInt8](../../sql-reference/data-types/)) — 如果该设置在配置中被显式定义，或被显式修改，则为 1。
* `description` ([String](../../sql-reference/data-types/)) — 设置说明。
* `min` ([Nullable(String)](../../sql-reference/data-types/)) — 设置的最小值 (如果通过约束设置了最小值) 。如果该设置没有最小值，则为 NULL。
* `max` ([Nullable(String)](../../sql-reference/data-types/)) — 设置的最大值 (如果通过约束设置了最大值) 。如果该设置没有最大值，则为 NULL。
* `disallowed_values` ([Array(String)](../../sql-reference/data-types/)) — 不允许的值列表
* `readonly` ([UInt8](../../sql-reference/data-types/)) — 显示当前用户是否可以修改该设置：0 — 当前用户可以修改该设置，1 — 当前用户不能修改该设置。
* `type` ([String](../../sql-reference/data-types/)) — 设置类型 (实现特定的字符串值) 。
* `is_obsolete` ([UInt8](../../sql-reference/data-types/)) — 显示该设置是否已废弃。
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](../../sql-reference/data-types/)) —
  此功能的支持级别。ClickHouse 功能按不同ティア进行划分，具体取决于其当前的
  开发状态，以及使用它们时可有的预期：

- PRODUCTION：该功能稳定且可安全使用，并且与其他 PRODUCTION 功能交互时不会出现问题。
- BETA：该功能稳定且安全。与其他功能一同使用时的结果尚不明确，且不保证正确性。欢迎测试并提交反馈。
- EXPERIMENTAL：该功能仍在开发中。仅适用于开发者和 ClickHouse 爱好者。该功能可能可用，也可能不可用，并且可能随时被移除。
- OBSOLETE：不再受支持。它要么已被移除，要么将在未来版本中被移除。

{/*AUTOGENERATED_END*/ }
