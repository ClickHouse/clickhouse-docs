## 描述 \{#description\}

`system.zookeeper&#95;connection&#95;log` 表显示 ZooKeeper 连接 (包括辅助 ZooKeeper) 的历史记录。每一行对应一个与连接相关的事件。

:::note
该表不包含由于服务器关闭导致的断开连接事件。
:::

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 已连接到 ZooKeeper 或已与 ZooKeeper 断开连接的服务器的主机名。
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/sql-reference/data-types/enum)) — 事件类型。可能的值：Connected、Disconnected。
* `event_date` ([Date](/sql-reference/data-types/date)) — 条目日期。
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 条目时间
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 精确到微秒的条目时间。
* `name` ([String](/sql-reference/data-types/string)) — ZooKeeper 集群名称。
* `host` ([String](/sql-reference/data-types/string)) — ClickHouse 所连接或断开连接的 ZooKeeper 节点的主机名/IP。
* `port` ([UInt16](/sql-reference/data-types/int-uint)) — ClickHouse 所连接或断开连接的 ZooKeeper 节点的端口。
* `index` ([UInt8](/sql-reference/data-types/int-uint)) — ClickHouse 所连接或断开连接的 ZooKeeper 节点的索引。该索引来自 ZooKeeper 配置。
* `client_id` ([Int64](/sql-reference/data-types/int-uint)) — 连接的会话 ID。
* `keeper_api_version` ([UInt8](/sql-reference/data-types/int-uint)) — Keeper API 版本。
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/sql-reference/data-types/array)) — 已启用的功能标志。仅适用于 ClickHouse Keeper。
* `availability_zone` ([String](/sql-reference/data-types/string)) — 可用区
* `reason` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 连接或断开连接的原因。

{/*AUTOGENERATED_END*/ }

```sql
SELECT * FROM system.zookeeper_connection_log;
```
