## 描述 \{#description\}

包含有关 [AzureQueue](../../engines/table-engines/integrations/azure-queue.md) 表设置的信息。
从服务器版本 `24.10` 起可用。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 使用 AzureQueue 引擎的表所在的数据库。
* `table` ([String](../../sql-reference/data-types/)) — 使用 AzureQueue 引擎的表名称。
* `name` ([String](../../sql-reference/data-types/)) — 设置名称。
* `value` ([String](../../sql-reference/data-types/)) — 设置值。
* `type` ([String](../../sql-reference/data-types/)) — 设置类型 (由具体实现决定的字符串值) 。
* `changed` ([UInt8](../../sql-reference/data-types/)) — 如果该设置在配置中已显式定义或被显式修改，则为 1。
* `description` ([String](../../sql-reference/data-types/)) — 设置说明。
* `alterable` ([UInt8](../../sql-reference/data-types/)) — 显示当前用户是否可以通过 ALTER TABLE MODIFY SETTING 修改该设置：0 — 当前用户可以修改该设置，1 — 当前用户无法修改该设置。

{/*AUTOGENERATED_END*/ }
