## 描述 \{#description\}

包含关于队列中待处理异步插入的相关信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `query` ([String](../../sql-reference/data-types/)) — 查询文本。
* `database` ([String](../../sql-reference/data-types/)) — 数据库名称。
* `table` ([String](../../sql-reference/data-types/)) — 表名。
* `format` ([String](../../sql-reference/data-types/)) — 格式名称。
* `first_update` ([DateTime64(6)](../../sql-reference/data-types/)) — 首次插入时间，精度为微秒。
* `total_bytes` ([UInt64](../../sql-reference/data-types/)) — 队列中等待处理的总字节数。
* `entries.query_id` ([Array(String)](../../sql-reference/data-types/)) — 队列中等待的插入操作对应的查询 ID 数组。
* `entries.bytes` ([Array(UInt64)](../../sql-reference/data-types/)) — 队列中每个待处理插入查询的字节数数组。

{/*AUTOGENERATED_END*/ }
