包含有关系统中已发生事件数量的信息。例如，在该表中，您可以查看自 ClickHouse 服务器启动以来已处理了多少个 `SELECT` 查询。

列：

{/*AUTOGENERATED_START*/ }

* `event` ([String](../../sql-reference/data-types/)) — 事件名称。
* `value` ([UInt64](../../sql-reference/data-types/)) — 发生的事件数量。
* `description` ([String](../../sql-reference/data-types/)) — 事件描述。

{/*AUTOGENERATED_END*/ }

**示例**

```sql
SELECT * FROM system.events LIMIT 5
```
