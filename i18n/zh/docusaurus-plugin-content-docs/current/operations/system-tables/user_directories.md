## 描述 \{#description\}

包含有关已配置用户目录的信息——这些目录位于文件系统中，ClickHouse 服务端可从中读取用户提供的数据。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 目录名称。
* `type` ([String](../../sql-reference/data-types/)) — 访问控制存储的类型，例如 users.xml、replicated 或 memory 等。
* `params` ([String](../../sql-reference/data-types/)) — 包含访问控制存储参数的 JSON。
* `precedence` ([UInt64](../../sql-reference/data-types/)) — 该目录在配置中定义的顺序。ClickHouse 在查找用户或角色时也会按相同顺序进行。

{/*AUTOGENERATED_END*/ }
