## 描述 \{#description\}

包含通过 SQL 查询创建或从设置文件中解析出的所有命名集合的列表。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 集合名称。
* `collection` ([Map(String, String)](../../sql-reference/data-types/)) — 集合的内部内容。
* `source` ([String](../../sql-reference/data-types/)) — 命名集合的来源。
* `create_query` ([String](../../sql-reference/data-types/)) — 命名集合的 CREATE 查询。

{/*AUTOGENERATED_END*/ }
