## 描述 \{#description\}

包含用户和角色的角色授权关系。要向此表添加记录，请使用 `GRANT role TO user`。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `user_name` ([Nullable(String)](../../sql-reference/data-types/)) — 用户名。
* `role_name` ([Nullable(String)](../../sql-reference/data-types/)) — 角色名称。
* `granted_role_name` ([String](../../sql-reference/data-types/)) — 授予给 `role_name` 角色的角色名称。要将一个角色授予另一个角色，请使用 `GRANT role1 TO role2`。
* `granted_role_id` ([UUID](../../sql-reference/data-types/)) — 角色 ID。
* `granted_role_is_default` ([UInt8](../../sql-reference/data-types/)) — 用于指示 `granted_role` 是否为默认角色的标志。可能的值：• 1 — `granted_role` 是默认角色，• 0 — `granted_role` 不是默认角色。
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — 用于指示 `granted_role` 是否具有 `ADMIN OPTION` 特权的标志。可能的值：• 1 — 该角色具有 `ADMIN OPTION` 特权。• 0 — 该角色不具有 `ADMIN OPTION` 特权。

{/*AUTOGENERATED_END*/ }
