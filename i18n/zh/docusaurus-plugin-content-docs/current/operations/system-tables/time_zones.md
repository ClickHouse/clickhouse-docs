# system.time&#95;zones \\{#systemtime&#95;zones\\}

包含 ClickHouse 服务器支持的时区列表。该列表可能会因 ClickHouse 版本不同而有所变化。

列：

{/*AUTOGENERATED_START*/ }

* `time_zone` ([String](../../sql-reference/data-types/)) — 受支持的时区列表。

{/*AUTOGENERATED_END*/ }

**示例**

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
