## 说明 \{#description\}

包含当前用户当前启用的角色。`SET ROLE` 会更改此表的内容。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — 角色名称。
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — 如果该角色具有 ADMIN OPTION 特权，则值为 1。
* `is_default` ([UInt8](../../sql-reference/data-types/)) — 如果该角色被设为默认角色，则值为 1。

{/*AUTOGENERATED_END*/ }
