## 描述 \{#description\}

包含已配置的profile的属性。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 设置 profile 的名称。
* `id` ([UUID](../../sql-reference/data-types/)) — 设置 profile 的 ID。
* `storage` ([String](../../sql-reference/data-types/)) — 设置 profile 存储路径。在 `access_control_path` 参数中配置。
* `num_elements` ([UInt64](../../sql-reference/data-types/)) — 此 profile 在 `system.settings_profile_elements` 表中的元素数量。
* `apply_to_all` ([UInt8](../../sql-reference/data-types/)) — 表示该设置 profile 已应用于所有角色和/或用户。
* `apply_to_list` ([Array(String)](../../sql-reference/data-types/)) — 应用该设置 profile 的角色和/或用户列表。
* `apply_to_except` ([Array(String)](../../sql-reference/data-types/)) — 该设置 profile 应用于除所列角色和/或用户之外的所有角色和/或用户。

{/*AUTOGENERATED_END*/ }
