## 描述 \{#description\}

包含有关存储在远程磁盘 (例如 S3、Azure Blob 存储) 上的数据 File 的信息，包括本地元数据路径与远程 blob 路径之间的映射关系。

每行表示一个与数据 File 关联的远程 blob 对象。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `disk_name` ([String](../../sql-reference/data-types/)) — 磁盘名称。
* `path` ([String](../../sql-reference/data-types/)) — 磁盘路径。
* `cache_base_path` ([String](../../sql-reference/data-types/)) — 缓存文件的基础目录。
* `local_path` ([String](../../sql-reference/data-types/)) — ClickHouse 文件的路径，也可用作元数据路径。
* `remote_path` ([String](../../sql-reference/data-types/)) — 对象存储中的 Blob 路径，与 ClickHouse 文件相关联。
* `size` ([UInt64](../../sql-reference/data-types/)) — 文件大小 (压缩后) 。
* `common_prefix_for_blobs` ([String](../../sql-reference/data-types/)) — 对象存储中 Blob 的公共前缀。
* `cache_paths` ([Array(String)](../../sql-reference/data-types/)) — 对应 Blob 的缓存文件。

{/*AUTOGENERATED_END*/ }

## 示例 \{#example\}

```sql
SELECT * FROM system.remote_data_paths LIMIT 1 FORMAT Vertical;
```

```text
Row 1:
──────
disk_name:              s3
path:                   /var/lib/clickhouse/disks/s3/
cache_base_path:        /var/lib/clickhouse/disks/s3_cache/
local_path:             store/123/1234abcd-1234-1234-1234-1234abcd1234/all_0_0_0/data.bin
remote_path:            abc123/all_0_0_0/data.bin
size:                   1048576
common_prefix_for_blobs:
cache_paths:            ['/var/lib/clickhouse/disks/s3_cache/a1/b2/c3d4e5f6']
```
