## 描述 \{#description\}

包含已配置[角色](../../guides/sre/user-management/index.md#role-management)的信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 角色名称。
* `id` ([UUID](../../sql-reference/data-types/)) — 角色 ID。
* `storage` ([String](../../sql-reference/data-types/)) — 角色存储路径。在 `access_control_path` 参数中配置。

{/*AUTOGENERATED_END*/ }
