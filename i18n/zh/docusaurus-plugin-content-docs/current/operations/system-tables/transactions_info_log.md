包含当前服务器上已执行的所有事务的信息。

可随时安全地截断或删除此表。

列：

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 执行事务所在的主机名。
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/sql-reference/data-types/enum)) — 事务类型。可能的值：Begin、Commit、Rollback、AddPart、LockPart、UnlockPart。
* `event_date` ([Date](/sql-reference/data-types/date)) — 条目日期。
* `event_time` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 条目时间
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — 线程标识符。
* `query_id` ([String](/sql-reference/data-types/string)) — 在事务范围内执行的查询 ID。
* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — 事务标识符。
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — 该标识符的哈希值。
* `csn` ([UInt64](/sql-reference/data-types/int-uint)) — 提交序列号
* `database` ([String](/sql-reference/data-types/string)) — 执行该事务所针对的数据库名称。
* `table` ([String](/sql-reference/data-types/string)) — 执行该事务所针对的表名称。
* `uuid` ([UUID](/sql-reference/data-types/uuid)) — 执行该事务所针对的表的 UUID。
* `part` ([String](/sql-reference/data-types/string)) — 参与该事务的 part 名称。

{/*AUTOGENERATED_END*/ }
