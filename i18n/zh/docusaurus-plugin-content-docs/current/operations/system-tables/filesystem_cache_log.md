包含远程文件系统上对象的文件系统缓存中发生的所有事件历史记录。

可随时安全地截断或删除该表。

列：

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 主机名
* `event_date` ([Date](/sql-reference/data-types/date)) — 事件日期
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 事件时间
* `query_id` ([String](/sql-reference/data-types/string)) — 查询 ID
* `source_file_path` ([String](/sql-reference/data-types/string)) — 文件系统中的 File 段路径
* `file_segment_range` ([Tuple(UInt64, UInt64)](/sql-reference/data-types/tuple)) — File 段范围
* `total_requested_range` ([Tuple(UInt64, UInt64)](/sql-reference/data-types/tuple)) — 完整读取范围
* `key` ([String](/sql-reference/data-types/string)) — File 段键
* `offset` ([UInt64](/sql-reference/data-types/int-uint)) — File 段偏移量
* `size` ([UInt64](/sql-reference/data-types/int-uint)) — 读取大小
* `read_type` ([String](/sql-reference/data-types/string)) — 读取类型：READ&#95;FROM&#95;CACHE、READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE、READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/sql-reference/data-types/int-uint)) — 是否尝试从缓存中读取
* `read_buffer_id` ([String](/sql-reference/data-types/string)) — 内部实现中的读取缓冲区 ID
* `user_id` ([String](/sql-reference/data-types/string)) — 创建该 File 段的用户 ID

{/*AUTOGENERATED_END*/ }
