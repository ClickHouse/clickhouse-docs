## 描述 \{#description\}

包含 AzureQueue 元数据的内存状态，以及当前每个文件正在处理的行。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `zookeeper_path` ([String](../../sql-reference/data-types/)) — zookeeper 中元数据的路径
* `file_path` ([String](../../sql-reference/data-types/)) — 正在处理的文件的路径
* `file_name` ([String](../../sql-reference/data-types/)) — 正在处理的文件的名称
* `rows_processed` ([UInt64](../../sql-reference/data-types/)) — 当前已处理的行数
* `status` ([String](../../sql-reference/data-types/)) — 处理状态：已处理、处理中、失败
* `processing_start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 文件开始处理的时间
* `processing_end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 文件处理结束的时间
* `exception` ([String](../../sql-reference/data-types/)) — 处理过程中发生的异常

{/*AUTOGENERATED_END*/ }
