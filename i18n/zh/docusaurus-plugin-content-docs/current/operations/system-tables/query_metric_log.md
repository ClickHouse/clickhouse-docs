## 描述 \{#description\}

包含来自表 `system.events` 的各个查询的内存和指标值历史记录，并定期写入磁盘。

一旦查询开始，就会按照 `query_metric_log_interval` 毫秒的周期采集数据 (默认值为 1000 毫秒) 。如果查询耗时超过 `query_metric_log_interval`，则在查询结束时也会采集一次数据。
