## 描述 \{#description\}

包含已加载到 LibraryBridge 内存中的 CatBoost 模型列表及其加载时间。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `model_path` ([String](../../sql-reference/data-types/)) — 已训练模型的路径。
* `type` ([String](../../sql-reference/data-types/)) — 模型类型。目前仅支持 catboost。
* `loading_start_time` ([DateTime](../../sql-reference/data-types/)) — 模型开始加载的时间。
* `loading_duration` ([Float32](../../sql-reference/data-types/)) — 加载模型所需的时间。

{/*AUTOGENERATED_END*/ }
