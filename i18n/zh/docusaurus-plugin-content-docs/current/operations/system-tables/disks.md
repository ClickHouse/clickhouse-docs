## 描述 \{#description\}

包含在[服务器配置](../../engines/table-engines/mergetree-family/mergetree.md#table_engine-mergetree-multiple-volumes_configure)中定义的磁盘的相关信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 服务器配置中磁盘的名称。
* `path` ([String](../../sql-reference/data-types/)) — 文件系统中挂载点的路径。
* `free_space` ([UInt64](../../sql-reference/data-types/)) — 磁盘上的可用空间，单位为字节。
* `total_space` ([UInt64](../../sql-reference/data-types/)) — 磁盘总空间，单位为字节。
* `unreserved_space` ([UInt64](../../sql-reference/data-types/)) — 未被预留占用的可用空间 (`free_space` 减去当前正在运行的合并、插入及其他磁盘写入操作所占用的预留空间大小) 。
* `keep_free_space` ([UInt64](../../sql-reference/data-types/)) — 磁盘上应保留的空闲空间，单位为字节。在磁盘配置的 `keep_free_space_bytes` 参数中定义。
* `type` ([String](../../sql-reference/data-types/)) — 磁盘类型，用于指示该磁盘将数据存储在何处——RAM、本地磁盘或远程存储。
* `object_storage_type` ([String](../../sql-reference/data-types/)) — 如果磁盘类型为 `object_storage`，则表示对象存储类型。
* `metadata_type` ([String](../../sql-reference/data-types/)) — 如果磁盘类型为 `object_storage`，则表示元数据存储类型。
* `is_encrypted` ([UInt8](../../sql-reference/data-types/)) — 用于显示该磁盘是否对底层数据进行加密的标志。
* `is_read_only` ([UInt8](../../sql-reference/data-types/)) — 表示该磁盘只能执行读取操作的标志。
* `is_write_once` ([UInt8](../../sql-reference/data-types/)) — 表示磁盘是否为一次写入的标志。这意味着它支持将 BACKUP 写入此磁盘，但不支持对位于此磁盘上的 MergeTree 表执行 INSERT。
* `is_remote` ([UInt8](../../sql-reference/data-types/)) — 表示对此磁盘执行操作时是否会涉及网络交互的标志。
* `is_broken` ([UInt8](../../sql-reference/data-types/)) — 表示磁盘是否损坏的标志。损坏的磁盘空间将为 0，且无法使用。
* `cache_path` ([String](../../sql-reference/data-types/)) — 当磁盘支持缓存时，本地磁盘上缓存目录的路径。

{/*AUTOGENERATED_END*/ }
