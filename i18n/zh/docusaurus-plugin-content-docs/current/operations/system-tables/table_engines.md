## 说明 \{#description\}

包含服务器支持的表引擎说明及其功能支持情况。

此表包含以下各列 (列类型显示在括号中) ：

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 表引擎名称。
* `supports_settings` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持 SETTINGS 子句。
* `supports_skipping_indices` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持跳过索引。
* `supports_projections` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持 projections。
* `supports_sort_order` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持 PARTITION&#95;BY、PRIMARY&#95;KEY、ORDER&#95;BY 和 SAMPLE&#95;BY 子句。
* `supports_ttl` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持生存时间 (TTL)。
* `supports_replication` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持数据复制。
* `supports_deduplication` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持数据去重。
* `supports_parallel_insert` ([UInt8](../../sql-reference/data-types/)) — 标志位，表示表引擎是否支持并行插入 (参见 max&#95;insert&#95;threads 设置) 。

{/*AUTOGENERATED_END*/ }
