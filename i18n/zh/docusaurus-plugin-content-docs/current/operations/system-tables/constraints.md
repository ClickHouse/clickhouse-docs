## 描述 \{#description\}

包含所有表中现有约束的信息。

在[临时表](../../sql-reference/statements/create/table.md#temporary-tables)上定义的约束，仅会在创建它们的会话中显示于 `system.constraints` 中。它们的 `database` 字段显示为空。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 数据库名。
* `table` ([String](../../sql-reference/data-types/)) — 表名。
* `name` ([String](../../sql-reference/data-types/)) — 约束名称。
* `type` ([Enum8(&#39;CHECK&#39; = 0, &#39;ASSUME&#39; = 1)](../../sql-reference/data-types/)) — 约束类型。
* `expression` ([String](../../sql-reference/data-types/)) — 约束表达式。

{/*AUTOGENERATED_END*/ }

## 示例 \{#example\}

```sql
SELECT * FROM system.constraints LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       hits
name:        check_hits
type:        CHECK
expression:  CounterID > 0

Row 2:
──────
database:    default
table:       hits
name:        assume_positive
type:        ASSUME
expression:  WatchID > 0
```
