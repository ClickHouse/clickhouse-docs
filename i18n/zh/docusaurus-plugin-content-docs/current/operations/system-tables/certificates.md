## 说明 \{#description\}

包含有关可用证书及其来源的信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `version` ([Int32](../../sql-reference/data-types/)) — 证书版本。取值为 0 表示 v1，1 表示 v2，2 表示 v3。
* `serial_number` ([Nullable(String)](../../sql-reference/data-types/)) — 由颁发者分配的证书序列号。
* `signature_algo` ([Nullable(String)](../../sql-reference/data-types/)) — 签名算法——即颁发者用于签署此证书的算法。
* `issuer` ([Nullable(String)](../../sql-reference/data-types/)) — 颁发者——即签发此证书的证书颁发机构的唯一标识符。
* `not_before` ([Nullable(String)](../../sql-reference/data-types/)) — 此证书有效时间窗口的开始时间。
* `not_after` ([Nullable(String)](../../sql-reference/data-types/)) — 此证书有效时间窗口的结束时间。
* `subject` ([Nullable(String)](../../sql-reference/data-types/)) — 使用者——用于标识公钥所有者。
* `pkey_algo` ([Nullable(String)](../../sql-reference/data-types/)) — 公钥算法，定义该公钥可与哪种算法配合使用。
* `path` ([String](../../sql-reference/data-types/)) — 包含此证书的文件或目录的路径。
* `default` ([UInt8](../../sql-reference/data-types/)) — 证书位于默认证书位置。
* `protocol` ([String](../../sql-reference/data-types/)) — 如果证书来自按协议区分的 TLS 配置，则为协议名称；否则为空。

{/*AUTOGENERATED_END*/ }
