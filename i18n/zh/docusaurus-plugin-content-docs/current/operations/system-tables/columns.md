包含所有表中列的信息。

你可以使用此表获取类似 [DESCRIBE TABLE](../../sql-reference/statements/describe-table.md) 查询返回的信息，但可以一次性针对多个表进行查询。

来自[临时表](../../sql-reference/statements/create/table.md#temporary-tables)的列仅在创建它们的会话中在 `system.columns` 中可见。此时这些列的 `database` 字段为空。
