## 描述 \{#description\}

包含当前所有已启用的角色，包括当前用户的当前角色以及为当前角色授予的角色。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — 角色名称。
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — 如果该角色具有 `ADMIN OPTION` 特权，则为 1。
* `is_current` ([UInt8](../../sql-reference/data-types/)) — 标志，指示该 `enabled_role` 是否为当前用户的当前角色。
* `is_default` ([UInt8](../../sql-reference/data-types/)) — 标志，指示该 `enabled_role` 是否为默认角色。

{/*AUTOGENERATED_END*/ }
