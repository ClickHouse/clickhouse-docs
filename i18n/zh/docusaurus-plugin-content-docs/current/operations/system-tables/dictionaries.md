**示例**

配置字典：

```sql
CREATE DICTIONARY dictionary_with_comment
(
    id UInt64,
    value String
)
PRIMARY KEY id
SOURCE(CLICKHOUSE(HOST 'localhost' PORT tcpPort() TABLE 'source_table'))
LAYOUT(FLAT())
LIFETIME(MIN 0 MAX 1000)
COMMENT '临时字典';
```

确保字典已加载。

```sql
SELECT * FROM system.dictionaries LIMIT 1 FORMAT Vertical;
```
