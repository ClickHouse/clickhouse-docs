# system.iceberg&#95;history

此 system 表包含 ClickHouse 中 Iceberg 表的快照历史。如果 ClickHouse 中不存在任何 Iceberg 表，则该表为空。

列：

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 数据库名称。
* `table` ([String](../../sql-reference/data-types/)) — 表名。
* `made_current_at` ([Nullable(DateTime64(3))](../../sql-reference/data-types/)) — 此快照被设为当前快照的日期和时间。
* `snapshot_id` ([UInt64](../../sql-reference/data-types/)) — 用于标识快照的快照 ID。
* `parent_id` ([UInt64](../../sql-reference/data-types/)) — 此快照的父快照 ID。
* `is_current_ancestor` ([UInt8](../../sql-reference/data-types/)) — 标志位，指示此快照是否为当前快照的祖先。

{/*AUTOGENERATED_END*/ }
