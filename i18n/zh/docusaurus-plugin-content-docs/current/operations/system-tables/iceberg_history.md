## 描述 \{#description\}

此 system 表包含 ClickHouse 中 Iceberg 表的快照历史记录。如果 ClickHouse 中没有任何 Iceberg 表，则该表将为空。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 数据库名称。
* `table` ([String](../../sql-reference/data-types/)) — 表名称。
* `made_current_at` ([Nullable(DateTime64(3))](../../sql-reference/data-types/)) — 此快照成为当前快照的日期和时间。
* `snapshot_id` ([UInt64](../../sql-reference/data-types/)) — 用于标识快照的快照 ID。
* `parent_id` ([UInt64](../../sql-reference/data-types/)) — 此快照的父 ID。
* `is_current_ancestor` ([UInt8](../../sql-reference/data-types/)) — 指示此快照是否为当前快照祖先的标志。

{/*AUTOGENERATED_END*/ }
