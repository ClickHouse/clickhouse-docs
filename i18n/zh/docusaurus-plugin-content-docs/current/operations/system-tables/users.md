# system.users \{#systemusers\}

包含服务器上已配置的[用户账户](../../guides/sre/user-management/index.md#user-account-management)列表。

列：

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 用户名。
* `id` ([UUID](../../sql-reference/data-types/)) — 用户 ID。
* `storage` ([String](../../sql-reference/data-types/)) — 用户存储的路径。在 `access_control_path` 参数中配置。
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](../../sql-reference/data-types/)) — 显示认证类型。用户标识有多种方式：无密码、明文密码、SHA256 编码密码、双重 SHA-1 编码密码或 bcrypt 编码密码。
* `auth_params` ([Array(String)](../../sql-reference/data-types/)) — 取决于 `auth_type` 的 JSON 格式认证参数。
* `valid_until` ([Array(DateTime)](../../sql-reference/data-types/)) — 用户凭证的过期日期和时间。
* `host_ip` ([Array(String)](../../sql-reference/data-types/)) — 允许连接到 ClickHouse 服务器的主机 IP 地址。
* `host_names` ([Array(String)](../../sql-reference/data-types/)) — 允许连接到 ClickHouse 服务器的主机名。
* `host_names_regexp` ([Array(String)](../../sql-reference/data-types/)) — 允许连接到 ClickHouse 服务器的主机名的正则表达式。
* `host_names_like` ([Array(String)](../../sql-reference/data-types/)) — 使用 LIKE 谓词设置的、允许连接到 ClickHouse 服务器的主机名。
* `default_roles_all` ([UInt8](../../sql-reference/data-types/)) — 表示为用户默认设置的所有已授予角色。
* `default_roles_list` ([Array(String)](../../sql-reference/data-types/)) — 默认授予给用户的角色列表。
* `default_roles_except` ([Array(String)](../../sql-reference/data-types/)) — 除列出的角色外，全部已授予角色都被设置为默认角色。
* `grantees_any` ([UInt8](../../sql-reference/data-types/)) — 指示具有任意 GRANT 选项权限的用户是否可以将其授予任意其他用户的标志。
* `grantees_list` ([Array(String)](../../sql-reference/data-types/)) — 该用户被允许向其授予 GRANT 选项的用户或角色列表。
* `grantees_except` ([Array(String)](../../sql-reference/data-types/)) — 该用户被禁止向其授予 GRANT 选项的用户或角色列表。
* `default_database` ([String](../../sql-reference/data-types/)) — 此用户的默认数据库名称。

{/*AUTOGENERATED_END*/ }
