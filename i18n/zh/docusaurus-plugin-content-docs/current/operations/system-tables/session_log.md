## 描述 \{#description\}

包含所有成功和失败的登录和登出事件的信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 执行该查询的服务器主机名。
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/sql-reference/data-types/enum)) — 登录/登出结果。可能的值：LoginFailure — 登录错误。LoginSuccess — 登录成功。Logout — 从系统登出。
* `auth_id` ([UUID](/sql-reference/data-types/uuid)) — 身份验证 ID，即每次用户登录时自动生成的 UUID。
* `session_id` ([String](/sql-reference/data-types/string)) — 客户端通过 HTTP interface 传递的会话 ID。
* `event_date` ([Date](/sql-reference/data-types/date)) — 登录/登出日期。
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 登录/登出时间。
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 以微秒精度记录的登录/登出开始时间。
* `user` ([Nullable(String)](/sql-reference/data-types/nullable)) — 用户名。
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/sql-reference/data-types/nullable)) — 身份验证类型。
* `profiles` ([Array(LowCardinality(String))](/sql-reference/data-types/array)) — 为所有角色和/或用户设置的 profile 列表。
* `roles` ([Array(LowCardinality(String))](/sql-reference/data-types/array)) — 应用了该 profile 的角色列表。
* `settings` ([Array(Tuple(LowCardinality(String), String))](/sql-reference/data-types/array)) — 客户端登录/登出时修改的设置。
* `client_address` ([IPv6](/sql-reference/data-types/ipv6)) — 用于登录/登出的 IP 地址。
* `client_port` ([UInt16](/sql-reference/data-types/int-uint)) — 用于登录/登出的客户端端口。
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/sql-reference/data-types/enum)) — 发起登录的 interface。
* `client_hostname` ([String](/sql-reference/data-types/string)) — 运行 clickhouse-client 或其他 TCP 客户端的客户端机器主机名。
* `client_name` ([String](/sql-reference/data-types/string)) — clickhouse-client 或其他 TCP 客户端的名称。
* `client_revision` ([UInt32](/sql-reference/data-types/int-uint)) — clickhouse-client 或其他 TCP 客户端的修订版本。
* `client_version_major` ([UInt32](/sql-reference/data-types/int-uint)) — clickhouse-client 或其他 TCP 客户端的主版本号。
* `client_version_minor` ([UInt32](/sql-reference/data-types/int-uint)) — clickhouse-client 或其他 TCP 客户端的次版本号。
* `client_version_patch` ([UInt32](/sql-reference/data-types/int-uint)) — clickhouse-client 或其他 TCP 客户端版本的补丁部分。
* `failure_reason` ([String](/sql-reference/data-types/string)) — 包含登录/登出失败原因的异常消息。

{/*AUTOGENERATED_END*/ }
