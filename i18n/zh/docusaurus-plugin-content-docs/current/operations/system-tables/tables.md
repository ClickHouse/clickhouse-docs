## 说明 \{#description\}

包含服务器已知的每个表的元数据。

[已分离](../../sql-reference/statements/detach.md) 的表不会在 `system.tables` 中显示。

[临时表](../../sql-reference/statements/create/table.md#temporary-tables) 只有在创建它们的会话中才会在 `system.tables` 中可见。它们会以 `database` 字段为空且 `is_temporary` 标志已开启的形式显示。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 该表所在数据库的名称。
* `name` ([String](../../sql-reference/data-types/)) — 表名称。
* `uuid` ([UUID](../../sql-reference/data-types/)) — 表的 uuid (Atomic 数据库) 。
* `engine` ([String](../../sql-reference/data-types/)) — 表引擎名称 (不带参数) 。
* `is_temporary` ([UInt8](../../sql-reference/data-types/)) — 用于指示该表是否为临时表的标志。
* `data_paths` ([Array(String)](../../sql-reference/data-types/)) — 表数据在文件系统中的路径。
* `metadata_path` ([String](../../sql-reference/data-types/)) — 文件系统中该表元数据的路径。
* `metadata_modification_time` ([DateTime](../../sql-reference/data-types/)) — 表元数据的最后修改时间。
* `metadata_version` ([Int32](../../sql-reference/data-types/)) — ReplicatedMergeTree 表的元数据版本；对于非 ReplicatedMergeTree 表，该值为 0。
* `dependencies_database` ([Array(String)](../../sql-reference/data-types/)) — 数据库依赖项。
* `dependencies_table` ([Array(String)](../../sql-reference/data-types/)) — 表依赖项 (当前表的 materialized view) 。
* `create_table_query` ([String](../../sql-reference/data-types/)) — 用于创建该表的查询。
* `engine_full` ([String](../../sql-reference/data-types/)) — 表引擎参数。
* `as_select` ([String](../../sql-reference/data-types/)) — 视图的 SELECT 查询。
* `parameterized_view_parameters` ([Array(Tuple(name String, type String))](../../sql-reference/data-types/)) — 参数化视图的参数。
* `partition_key` ([String](../../sql-reference/data-types/)) — 表中指定的分区键表达式。
* `sorting_key` ([String](../../sql-reference/data-types/)) — 表中指定的排序键表达式。
* `primary_key` ([String](../../sql-reference/data-types/)) — 表中指定的主键表达式。
* `sampling_key` ([String](../../sql-reference/data-types/)) — 表中指定的采样键表达式。
* `storage_policy` ([String](../../sql-reference/data-types/)) — 存储策略。适用于使用 MergeTree 和 Distributed 引擎的表。
* `total_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 总行数；如果可以快速确定表中的精确行数，则为该值，否则为 NULL (包括底层 Buffer 表) 。
* `total_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 总字节数；如果能够快速确定该表在存储中的确切字节数，则返回该值，否则为 NULL (不包括任何底层存储) 。如果表将数据存储在磁盘上，则返回磁盘上已使用的空间 (即压缩后的大小) 。如果表将数据存储在内存中，则返回内存中已使用字节数的估算值。
* `total_bytes_uncompressed` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 未压缩字节总数；如果可以根据存储中该表的 часть 校验和快速确定精确的字节数，则返回该值，否则返回 NULL (不考虑底层存储 (如果有) ) 。
* `parts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 该表中的 parts 总数。
* `active_parts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 此表中处于活动状态的 parts 数量。
* `total_marks` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 该表中所有 parts 的 marks 总数。
* `active_on_fly_data_mutations` ([UInt64](../../sql-reference/data-types/)) — 可按需应用的活动数据 mutation (UPDATE 和 DELETE) 总数。
* `active_on_fly_alter_mutations` ([UInt64](../../sql-reference/data-types/)) — 可按需应用的活动 alter mutation (MODIFY COLUMN) 总数。
* `active_on_fly_metadata_mutations` ([UInt64](../../sql-reference/data-types/)) — 可按需应用的活动元数据 mutation (RENAME) 总数。
* `columns_descriptions_cache_size` ([UInt64](../../sql-reference/data-types/)) — *MergeTree 表的列描述缓存大小
* `lifetime_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 自服务器启动以来已 INSERT 的总行数 (仅适用于 Buffer 表) 。
* `lifetime_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 自服务器启动以来已 INSERT 的总字节数 (仅适用于 Buffer 表) 。
* `comment` ([String](../../sql-reference/data-types/)) — 表注释。
* `has_own_data` ([UInt8](../../sql-reference/data-types/)) — 用于指示该表本身是否会在磁盘上存储数据，还是仅访问其他数据源的标志。
* `loading_dependencies_database` ([Array(String)](../../sql-reference/data-types/)) — 数据库加载依赖项 (应在当前对象之前加载的对象列表) 。
* `loading_dependencies_table` ([Array(String)](../../sql-reference/data-types/)) — 表的加载依赖 (应在当前对象之前加载的对象列表) 。
* `loading_dependent_database` ([Array(String)](../../sql-reference/data-types/)) — 加载所依赖的数据库。
* `loading_dependent_table` ([Array(String)](../../sql-reference/data-types/)) — 加载依赖表。
* `definer` ([String](../../sql-reference/data-types/)) — 该表使用的 SQL 安全定义者名称。

{/*AUTOGENERATED_END*/ }
