## 描述 \{#description\}

包含与服务器配置相关的警告信息，`clickhouse-client` 连接到服务器后会立即显示这些警告。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `message` ([String](../../sql-reference/data-types/)) — 由 ClickHouse 服务端发出的警告消息。
* `message_format_string` ([LowCardinality(String)](../../sql-reference/data-types/)) — 用于格式化该消息的格式字符串。

{/*AUTOGENERATED_END*/ }
