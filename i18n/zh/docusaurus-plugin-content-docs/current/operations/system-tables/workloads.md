# system.workloads

包含本地服务器上各个 [workload](/operations/workload-scheduling.md#workload_entity_storage) 的信息。该表中每一行对应一个 workload。

示例：

```sql
SELECT *
FROM system.workloads
FORMAT Vertical
```

```text
第 1 行：
──────
name:         production
parent:       all
create_query: CREATE WORKLOAD production IN `all` SETTINGS weight = 9

第 2 行：
──────
name:         development
parent:       all
create_query: CREATE WORKLOAD development IN `all`

第 3 行：
──────
name:         all
parent:
create_query: CREATE WORKLOAD `all`
```
