## 描述 \{#description\}

包含当前正在分片之间移动的 parts 及其迁移进度信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 执行移动操作的数据库名称。
* `table` ([String](../../sql-reference/data-types/)) — 执行移动操作的表名称。
* `task_name` ([String](../../sql-reference/data-types/)) — 移动任务的名称。
* `task_uuid` ([UUID](../../sql-reference/data-types/)) — 移动任务的标识符。
* `create_time` ([DateTime](../../sql-reference/data-types/)) — 任务的创建时间。
* `part_name` ([String](../../sql-reference/data-types/)) — 正在移动的 part 名称。
* `part_uuid` ([UUID](../../sql-reference/data-types/)) — 正在移动的 part 的 UUID。
* `to_shard` ([String](../../sql-reference/data-types/)) — 目标分片的名称。
* `dst_part_name` ([String](../../sql-reference/data-types/)) — 结果 part 的名称。
* `update_time` ([DateTime](../../sql-reference/data-types/)) — 最后一次更新的时间。
* `state` ([String](../../sql-reference/data-types/)) — 移动操作的当前状态。
* `rollback` ([UInt8](../../sql-reference/data-types/)) — 指示该操作是否已回滚的标志。
* `num_tries` ([UInt32](../../sql-reference/data-types/)) — 完成该操作的尝试次数。
* `last_exception` ([String](../../sql-reference/data-types/)) — 最近一次异常的名称 (如果有) 。

{/*AUTOGENERATED_END*/ }
