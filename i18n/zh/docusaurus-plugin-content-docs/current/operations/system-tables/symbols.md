## 描述 \{#description\}

包含用于对 `clickhouse` 二进制文件进行自省的信息。访问它需要具有 `introspection` 权限。
此表仅对 C++ 专家和 ClickHouse 工程师有用。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `symbol` ([String](../../sql-reference/data-types/)) — 二进制文件中的符号名称。它是经过名称修饰的。您可以对 demangle(symbol) 进行处理以获得可读名称。
* `symbol_demangled` ([Nullable(String)](../../sql-reference/data-types/)) — 用于 XRay 插桩的解修饰符号。
* `function_id` ([Nullable(Int32)](../../sql-reference/data-types/)) — XRay 插桩映射中的函数 ID。
* `address_begin` ([UInt64](../../sql-reference/data-types/)) — 二进制文件中该符号的起始地址。
* `address_end` ([UInt64](../../sql-reference/data-types/)) — 二进制文件中该符号的结束地址。

{/*AUTOGENERATED_END*/ }

## 示例 \{#example\}

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
