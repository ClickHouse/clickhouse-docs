## 描述 \{#description\}

包含有关 ClickHouse 服务器构建选项的信息。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 构建选项名称。
* `value` ([String](../../sql-reference/data-types/)) — 构建选项的值。

{/*AUTOGENERATED_END*/ }

## 示例 \{#example\}

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
