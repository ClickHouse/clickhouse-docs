包含通过 [HTTP 接口](/interfaces/http.md) 访问的 `/dashboard` 页面所使用的查询。
此表可用于监控和故障排查。表中每个仪表盘图表对应一行记录。

:::note
`/dashboard` 页面不仅可以渲染来自 `system.dashboards` 的查询，还可以渲染来自任意具有相同表结构的表中的查询。
这对于创建自定义仪表盘很有用。
:::

示例：

```sql
SELECT *
FROM system.dashboards
WHERE title ILIKE '%CPU%'
```
