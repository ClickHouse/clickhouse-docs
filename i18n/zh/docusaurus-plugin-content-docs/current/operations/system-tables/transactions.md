## 描述 \{#description\}

列出事务及其状态。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — 事务的标识符。
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — 标识符的哈希值。
* `elapsed` ([Float64](/sql-reference/data-types/float)) — 处理该事务所耗费的时间。
* `is_readonly` ([UInt8](/sql-reference/data-types/int-uint)) — 指示该事务是否执行过任何写操作的标志。
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/sql-reference/data-types/enum)) — 事务的状态。可能的值：RUNNING、COMMITTING、COMMITTED、ROLLED&#95;BACK。

{/*AUTOGENERATED_END*/ }
