## 描述 \{#description\}

以单行单列的形式返回 jemalloc 统计信息。相当于执行 SYSTEM JEMALLOC STATS 命令。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `stats` ([String](../../sql-reference/data-types/)) — 由 malloc&#95;stats&#95;print 输出的 JEMalloc 统计信息。

{/*AUTOGENERATED_END*/ }
