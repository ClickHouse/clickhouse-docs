## 描述 \{#description\}

包含使用 LLVM XRay 功能的插桩点。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `id` ([UInt32](../../sql-reference/data-types/)) — 插桩点的 ID
* `function_id` ([Int32](../../sql-reference/data-types/)) — 在 ELF 二进制文件的 xray&#95;instr&#95;map 部分中分配给该函数的 ID。
* `function_name` ([LowCardinality(String)](../../sql-reference/data-types/)) — 用于为该函数插桩的名称。
* `handler` ([LowCardinality(String)](../../sql-reference/data-types/)) — 已补丁到该函数插桩点中的 handler。
* `entry_type` ([Enum8(&#39;Entry&#39; = 0, &#39;Exit&#39; = 1, &#39;EntryAndExit&#39; = 2)](../../sql-reference/data-types/)) — 该补丁的入口类型。
* `symbol` ([LowCardinality(String)](../../sql-reference/data-types/)) — 完整且已解修饰的符号名称。
* `parameters` ([Array(Dynamic)](../../sql-reference/data-types/)) — handler 调用的参数。

{/*AUTOGENERATED_END*/ }
