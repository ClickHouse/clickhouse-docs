## 描述 \{#description\}

包含所有可用的聚合函数组合器列表，这些组合器可应用于聚合函数，并修改其工作方式。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 组合器的名称。
* `is_internal` ([UInt8](../../sql-reference/data-types/)) — 该组合器是否仅用于内部。

{/*AUTOGENERATED_END*/ }
