## 描述 \{#description\}

列出服务器中已注册的所有可用 failpoint，以及它们的类型及当前是否已启用。

可以在运行时使用 `SYSTEM ENABLE FAILPOINT` 和 `SYSTEM DISABLE FAILPOINT` 语句来启用或禁用 failpoint。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — failpoint 的名称。
* `type` ([Enum8(&#39;once&#39; = 0, &#39;regular&#39; = 1, &#39;pauseable&#95;once&#39; = 2, &#39;pauseable&#39; = 3)](../../sql-reference/data-types/)) — failpoint 的类型：&#39;once&#39; 只触发一次，随后自动禁用；&#39;regular&#39; 每次都会触发；&#39;pauseable&#95;once&#39; 会阻塞执行一次；&#39;pauseable&#39; 每次都会阻塞执行，直到恢复执行。
* `enabled` ([UInt8](../../sql-reference/data-types/)) — failpoint 当前是启用 (1) 还是禁用 (0) 。

{/*AUTOGENERATED_END*/ }

## 示例 \{#example\}

```sql title="Query"
SYSTEM ENABLE FAILPOINT replicated_merge_tree_insert_retry_pause;
SELECT * FROM system.fail_points WHERE enabled = 1
```
