## 描述 \{#description\}

该表包含 [MergeTree](/engines/table-engines/mergetree-family/mergetree.md) 表中正在进行的 [数据分区片段移动](/sql-reference/statements/alter/partition#move-partitionpart) 的信息。每个数据分区片段移动对应一行。

## 列 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 数据库名称。
* `table` ([String](../../sql-reference/data-types/)) — 包含正在移动的数据分区片段的表名称。
* `elapsed` ([Float64](../../sql-reference/data-types/)) — 自数据分区片段开始移动以来经过的时间 (以秒为单位) 。
* `target_disk_name` ([String](../../sql-reference/data-types/)) — 数据分区片段正在移入的目标磁盘名称。
* `target_disk_path` ([String](../../sql-reference/data-types/)) — 文件系统中该磁盘挂载点的路径。
* `part_name` ([String](../../sql-reference/data-types/)) — 正在移动的数据分区片段名称。
* `part_size` ([UInt64](../../sql-reference/data-types/)) — 数据分区片段大小。
* `thread_id` ([UInt64](../../sql-reference/data-types/)) — 执行移动操作的线程 ID。

{/*AUTOGENERATED_END*/ }
