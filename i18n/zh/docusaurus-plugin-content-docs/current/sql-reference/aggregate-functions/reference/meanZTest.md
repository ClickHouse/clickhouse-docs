## meanZTest \{#meanZTest\}

引入于：v22.2

对来自两个总体的样本应用均值 Z 检验。

两个样本的取值都在 `sample_data` 列中。
如果 `sample_index` 等于 0，则该行中的值属于来自第一个总体的样本。
否则，该值属于来自第二个总体的样本。
原假设是两个总体的均值相等。
假定服从正态分布。
两个总体的方差可以不相等，且方差已知。

**语法**

```sql
meanZTest(population_variance_x, population_variance_y, confidence_level)(sample_data, sample_index)
```

**参数**

* `population_variance_x` — 总体 x 的方差。[`Float*`](/sql-reference/data-types/float)
* `population_variance_y` — 总体 y 的方差。[`Float*`](/sql-reference/data-types/float)
* `confidence_level` — 用于计算置信区间的置信水平。[`Float*`](/sql-reference/data-types/float)

**参数说明**

* `sample_data` — 样本数据。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)
* `sample_index` — 样本索引。[`(U)Int*`](/sql-reference/data-types/int-uint)

**返回值**

返回一个包含四个元素的 Tuple：计算得到的 z 统计值、计算得到的 p 值、计算得到的置信区间下界、计算得到的置信区间上界。[`Tuple(Float64, Float64, Float64, Float64)`](/sql-reference/data-types/tuple)

**示例**

**均值 Z 检验示例**

```sql title=Query
CREATE TABLE mean_ztest (sample_data Float64, sample_index UInt8) ENGINE = Memory;
INSERT INTO mean_ztest VALUES (20.3, 0), (21.9, 0), (22.1, 0), (18.9, 1), (19, 1), (20.3, 1);

SELECT meanZTest(0.7, 0.45, 0.95)(sample_data, sample_index) FROM mean_ztest;
```

```response title=Response
┌─meanZTest(0.7, 0.45, 0.95)(sample_data, sample_index)───────────────────────────────┐
│ (3.2841296025548123, 0.0010229786769086013, 0.8198428246768334, 3.2468238419898365) │
└─────────────────────────────────────────────────────────────────────────────────────┘
```
