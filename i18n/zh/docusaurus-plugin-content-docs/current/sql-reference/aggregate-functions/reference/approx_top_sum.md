# approx_top_sum {#approx_top_sum}

返回一个数组，包含指定列中近似出现频率最高的值及其计数。结果数组按值的近似出现频率降序排序（不是按值本身排序）。此外，还会考虑值的权重。

```sql
approx_top_sum(N)(column, weight)
approx_top_sum(N, reserved)(column, weight)
```

此函数不保证返回精确结果。在某些情况下可能会出现误差，并且可能返回的高频值并不是最频繁的值。

`N` 的最大值为 `65536`。

**参数**

* `N` — 要返回的元素数量。可选。默认值：10。
* `reserved` — 为不同值预留的单元格数量。如果 uniq(column) &gt; reserved，则 topK 函数的结果将是近似值。可选。默认值：N * 3。

**参数说明**

* `column` — 用于计算频率的列的值。
* `weight` — 权重。每个值在频率计算中被计入 `weight` 次。[UInt64](../../../sql-reference/data-types/int-uint.md)。

**示例**

查询：

```sql
SELECT approx_top_sum(2)(k, w)
FROM VALUES('k Char, w UInt64', ('y', 1), ('y', 1), ('x', 5), ('y', 1), ('z', 10))
```

结果：

```text
┌─approx_top_sum(2)(k, w)─┐
│ [('z',10,0),('x',5,0)]  │
└─────────────────────────┘
```
