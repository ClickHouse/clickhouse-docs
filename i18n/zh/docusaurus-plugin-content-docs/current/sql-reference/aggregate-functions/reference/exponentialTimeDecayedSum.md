## exponentialTimeDecayedSum \{#exponentialTimeDecayedSum\}

引入版本：v21.12

返回时间序列在时间索引为 `t` 处，经指数平滑处理的移动平均值之和。

**语法**

```sql
exponentialTimeDecayedSum(x)(v, t)
```

**参数**

* `x` — 使某个值的权重衰减到 1/e 所需的时间差。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**函数参数**

* `v` — 数值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)
* `t` — 时间。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal) 或 [`DateTime`](/sql-reference/data-types/datetime) 或 [`DateTime64`](/sql-reference/data-types/datetime64)

**返回值**

返回在给定时间点的指数平滑移动平均值之和。[`Float64`](/sql-reference/data-types/float)

**示例**

**窗口函数用法及可视化展示**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 10, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedSum(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
