## boundingRatio \{#boundingRatio\}

首次引入版本：v20.1

计算一组数值中最左侧和最右侧点之间的斜率。

**语法**

```sql
boundingRatio(x, y)
```

**参数**

* `x` — X 坐标值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)
* `y` — Y 坐标值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**返回值**

返回最左侧点和最右侧点之间连线的斜率；当数据为空时，返回 `NaN`。[`Float64`](/sql-reference/data-types/float)

**示例**

**示例数据**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
│      2 │                     3 │

```

**用法示例**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```
