## groupBitOr \{#groupBitOr\}

引入版本：v1.1

对一系列数字执行按位或运算。

**语法**

```sql
groupBitOr(expr)
```

**别名**: `BIT_OR`

**参数**

* `expr` — `(U)Int*` 类型的表达式。[`(U)Int*`](/sql-reference/data-types/int-uint)

**返回值**

返回 `(U)Int*` 类型的值。[`(U)Int*`](/sql-reference/data-types/int-uint)

**示例**

**按位或 (OR) 示例**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
