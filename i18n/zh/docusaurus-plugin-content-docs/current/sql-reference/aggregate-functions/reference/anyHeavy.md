## anyHeavy \{#anyHeavy\}

引入版本：v1.1

使用 [heavy hitters](https://doi.org/10.1145/762471.762473) 算法选择一个高频出现的值。
如果在每个查询执行线程中，都存在某个值在超过一半的记录中出现，则返回该值。
通常，该结果是非确定性的。

**语法**

```sql
anyHeavy(column)
```

**参数**

* `column` — 列名。[`String`](/sql-reference/data-types/string)

**返回值**

返回一个出现频率较高的值。结果是非确定性的。[`Any`](/sql-reference/data-types)

**示例**

**使用示例**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
