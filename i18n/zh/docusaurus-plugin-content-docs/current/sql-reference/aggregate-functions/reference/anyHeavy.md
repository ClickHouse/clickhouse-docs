# anyHeavy {#anyheavy}

使用 [heavy hitters](https://doi.org/10.1145/762471.762473) 算法选择一个高频出现的值。如果在每个查询执行线程中，都存在某个值在超过一半的记录中出现，则返回该值。通常，该结果是非确定性的。

```sql
anyHeavy(column)
```

**参数**

* `column` – 列名。

**示例**

以 [OnTime](../../../getting-started/example-datasets/ontime.md) 数据集为例，在 `AirlineID` 列中选择任意一个出现频率较高的值。

```sql
SELECT anyHeavy(AirlineID) AS res
FROM ontime
```

```text
┌───res─┐
│ 19690 │
└───────┘
```
