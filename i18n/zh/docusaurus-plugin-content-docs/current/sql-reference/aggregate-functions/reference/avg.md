## avg \{#avg\}

自 v1.1 起引入

计算算术平均值。

**语法**

```sql
avg(x)
```

**参数**

* `x` — 输入值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**返回值**

返回算术平均值；当输入参数 `x` 为空时，返回 `NaN`。返回类型为 [`Float64`](/sql-reference/data-types/float)

**示例**

**基本用法**

```sql title=Query
SELECT avg(x) FROM VALUES('x Int8', 0, 1, 2, 3, 4, 5);
```

```response title=Response
┌─avg(x)─┐
│    2.5 │
└────────┘
```

**空表时返回 NaN**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;

SELECT avg(t) FROM test;
```

```response title=Response
┌─avg(x)─┐
│    nan │
└────────┘
```
