## groupUniqArray \{#groupUniqArray\}

引入于：v1.1

根据不同的参数值创建一个数组。
此函数的内存消耗与 [`uniqExact`](/sql-reference/aggregate-functions/reference/uniqexact) 相同。

**语法**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**参数**

* `max_size` — 将结果数组的大小限制为最多 `max_size` 个元素。`groupUniqArray(1)(x)` 等价于 `[any(x)]`。[`UInt64`](/sql-reference/data-types/int-uint)

**参数说明**

* `x` — 表达式。[`Any`](/sql-reference/data-types)

**返回值**

返回一个由唯一值组成的数组。[`Array`](/sql-reference/data-types/array)

**示例**

**用法示例**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**使用 max&#95;size 参数**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
