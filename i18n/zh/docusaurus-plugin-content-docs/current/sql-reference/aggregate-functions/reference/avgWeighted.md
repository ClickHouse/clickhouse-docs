## avgWeighted \{#avgWeighted\}

自 v20.1 起引入

计算[加权算术平均值](https://en.wikipedia.org/wiki/Weighted_arithmetic_mean)。

**语法**

```sql
avgWeighted(x, weight)
```

**参数**

* `x` — 数值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float)
* `weight` — 数值的权重。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float)

**返回值**

如果所有权重都等于 0 或提供的权重参数为空，则返回 `NaN`；否则返回加权平均值。[`Float64`](/sql-reference/data-types/float)

**示例**

**用法示例**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**混合整数和浮点型的权重**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Float64', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**如果所有权重均为 0，则返回 NaN**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (0, 0), (1, 0), (10, 0))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│               nan │
└───────────────────┘
```

**空表时返回 NaN**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;
SELECT avgWeighted(t, t) FROM test
```

```response title=Response
┌─avgWeighted(t, t)─┐
│               nan │
└───────────────────┘
```
