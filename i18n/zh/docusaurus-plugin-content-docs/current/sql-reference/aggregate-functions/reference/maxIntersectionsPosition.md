## maxIntersectionsPosition \{#maxIntersectionsPosition\}

引入版本：v1.1

用于计算[`maxIntersections`](/sql-reference/aggregate-functions/reference/maxintersections)函数各次出现位置的聚合函数。

**语法**

```sql
maxIntersectionsPosition(start_column, end_column)
```

**参数**

* `start_column` — 表示每个区间起点的数值类型列。若 `start_column` 为 `NULL` 或 0，则跳过该区间。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float)
* `end_column` — 表示每个区间终点的数值类型列。若 `end_column` 为 `NULL` 或 0，则跳过该区间。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float)

**返回值**

返回使相交区间数量最大的起始位置。[`Any`](/sql-reference/data-types)

**示例**

**查找相交区间数量最大的起始位置**

```sql title=Query
CREATE TABLE my_events (
    start UInt32,
    end UInt32
)
ENGINE = MergeTree
ORDER BY tuple();

INSERT INTO my_events VALUES
(1, 3),
(1, 6),
(2, 5),
(3, 7);

SELECT maxIntersectionsPosition(start, end) FROM my_events;
```

```response title=Response
┌─maxIntersectionsPosition(start, end)─┐
│                                    2 │
└──────────────────────────────────────┘
```
