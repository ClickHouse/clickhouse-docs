## analysisOfVariance \{#analysisOfVariance\}

引入于：v22.10

提供用于单因素方差分析（ANOVA 检验）的一种统计检验方法。该检验用于多个服从正态分布的观测组，以判断这些组的均值是否全部相同。

:::note
分组从 0 开始编号，并且至少需要两个组才能进行检验。
至少要有一个组的观测值数量大于 1。
:::

**语法**

```sql
analysisOfVariance(val, group_no)
```

**别名**: `anova`

**参数**

* `val` — 数值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)
* `group_no` — `val` 所属的分组编号。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**返回值**

返回一个元组，包含 F 统计量和 p 值。[`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple)

**示例**

**基本用法**

```sql title=Query
SELECT analysisOfVariance(number, number % 2) FROM numbers(1048575);
```

```response title=Response
┌─analysisOfVariance(number, modulo(number, 2))─┐
│ (0,1)                                         │
└───────────────────────────────────────────────┘
```
