## varPopStable \{#varPopStable\}

引入版本：v1.1

返回总体方差。
与 [`varPop`](https://clickhouse.com/docs/sql-reference/aggregate-functions/reference/varpop) 不同，此函数使用一种[数值稳定](https://en.wikipedia.org/wiki/Numerical_stability)的算法。
其运行速度较慢，但数值误差更小。

**语法**

```sql
varPopStable(x)
```

**参数**

* `x` — 用于计算总体方差的一组数值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal*`](/sql-reference/data-types/decimal)

**返回值**

返回 `x` 的总体方差。[`Float64`](/sql-reference/data-types/float)

**示例**

**计算稳定的总体方差**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    varPopStable(x) AS var_pop_stable
FROM test_data;
```

```response title=Response
┌─var_pop_stable─┐
│           14.4 │
└────────────────┘
```
