## varPopStable \{#varpopstable\}

返回总体方差。与 [`varPop`](../reference/varPop.md) 不同，此函数使用一种[数值上更稳定](https://en.wikipedia.org/wiki/Numerical_stability)的算法。它的运行速度较慢，但计算误差更小。

**语法**

```sql
varPopStable(x)
```

别名：`VAR_POP_STABLE`。

**参数**

* `x`：用于计算总体方差的总体数据。[(U)Int*](../../data-types/int-uint.md)、[Float*](../../data-types/float.md)、[Decimal*](../../data-types/decimal.md)。

**返回值**

* 返回 `x` 的总体方差。[Float64](../../data-types/float.md)。

**示例**

查询：

```sql
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    varPopStable(x) AS var_pop_stable
FROM test_data;
```

结果：

```response
┌─var_pop_stable─┐
│           14.4 │
└────────────────┘
```
