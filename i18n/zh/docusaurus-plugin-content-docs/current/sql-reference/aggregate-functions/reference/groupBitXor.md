# groupBitXor \{#groupbitxor\}

对一组数字执行按位 `XOR` 运算。

```sql
groupBitXor(expr)
```

**参数**

`expr` – 一个其结果类型为 `UInt*` 或 `Int*` 的表达式。

**返回值**

返回 `UInt*` 或 `Int*` 类型的值。

**示例**

测试数据：

```text
binary     decimal
00101100 = 44
00011100 = 28
00001101 = 13
01010101 = 85
```

查询：

```sql
SELECT groupBitXor(num) FROM t
```

其中 `num` 是存放测试数据的列。

结果：

```text
binary     decimal
01101000 = 104
```
