## groupBitXor \{#groupBitXor\}

引入版本：v1.1

对一组数字执行按位异或运算。

**语法**

```sql
groupBitXor(expr)
```

**别名**: `BIT_XOR`

**参数**

* `expr` — `(U)Int*` 类型的表达式。[`(U)Int*`](/sql-reference/data-types/int-uint)

**返回值**

返回 `(U)Int*` 类型的值。[`(U)Int*`](/sql-reference/data-types/int-uint)

**示例**

**按位异或示例**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitXor(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01101000 = 104

┌─groupBitXor(num)─┐
│              104 │
└──────────────────┘
```
