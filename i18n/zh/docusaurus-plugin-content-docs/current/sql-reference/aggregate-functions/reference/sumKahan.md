## sumKahan \{#sumKahan\}

引入于：v1.1

使用 [Kahan 补偿求和算法](https://en.wikipedia.org/wiki/Kahan_summation_algorithm)计算数值的总和。
比 [`sum`](/sql-reference/aggregate-functions/reference/sum) 函数更慢。
补偿仅适用于 [Float](/sql-reference/data-types/float) 类型。

**语法**

```sql
sumKahan(x)
```

**参数**

* `x` — 输入值。[`Integer`](/sql-reference/data-types/int-uint) 或 [`Float`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**返回值**

返回数值之和。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**示例**

**展示 Kahan 求和算法带来的精度提升**

```sql title=Query
SELECT sum(0.1), sumKahan(0.1) FROM numbers(10);
```

```response title=Response
┌───────────sum(0.1)─┬─sumKahan(0.1)─┐
│ 0.9999999999999999 │             1 │
└────────────────────┴───────────────┘
```
