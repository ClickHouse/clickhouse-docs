# groupBitAnd \{#groupbitand\}

对一系列数值执行按位 `AND` 运算。

```sql
groupBitAnd(expr)
```

**参数**

`expr` – 其结果为 `UInt*` 或 `Int*` 类型的表达式。

**返回值**

返回 `UInt*` 或 `Int*` 类型的值。

**示例**

测试数据：

```text
binary     decimal
00101100 = 44
00011100 = 28
00001101 = 13
01010101 = 85
```

查询：

```sql
SELECT groupBitAnd(num) FROM t
```

其中 `num` 是测试数据所在的列。

结果：

```text
binary     decimal
00000100 = 4
```
