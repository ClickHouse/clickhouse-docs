## groupArrayIntersect \{#groupArrayIntersect\}

引入版本：v24.2

返回给定数组的交集（返回出现在所有给定数组中的所有元素）。

**语法**

```sql
groupArrayIntersect(x)
```

**参数**

* `x` — 参数（列名或表达式）。[`Any`](/sql-reference/data-types)

**返回值**

返回一个包含所有数组共有元素的数组。[`Array`](/sql-reference/data-types/array)

**示例**

**用法示例**

```sql title=Query
-- Create table with Memory engine
CREATE TABLE numbers (
    a Array(Int32)
) ENGINE = Memory;

-- Insert sample data
INSERT INTO numbers VALUES
    ([1,2,4]),
    ([1,5,2,8,-1,0]),
    ([1,5,7,5,8,2]);

SELECT groupArrayIntersect(a) AS intersection FROM numbers;
```

```response title=Response
┌─intersection──────┐
│ [1, 2]            │
└───────────────────┘
```
