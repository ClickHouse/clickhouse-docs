## groupArrayMovingSum \{#groupArrayMovingSum\}

引入版本：v20.1

计算输入值的移动求和。

该函数可以接收窗口大小作为参数。如果未指定，函数会将窗口大小设为该列中的行数。

**语法**

```sql
groupArrayMovingSum(numbers_for_summing)
groupArrayMovingSum(window_size)(numbers_for_summing)
```

**参数**

* `window_size` — 计算窗口的大小。如果未指定，函数会将窗口大小设为列中行的数量。[`UInt64`](/sql-reference/data-types/int-uint)

**参数说明**

* `numbers_for_summing` — 结果为数值数据类型的表达式。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal`](/sql-reference/data-types/decimal)

**返回值**

返回与输入数据的大小和类型相同的数组。[`Array`](/sql-reference/data-types/array)

**示例**

**使用示例**

```sql title=Query
CREATE TABLE t
(
    `int` UInt8,
    `float` Float32,
    `dec` Decimal32(2)
)
ENGINE = Memory;

INSERT INTO t VALUES (1, 1.1, 1.10), (2, 2.2, 2.20), (4, 4.4, 4.40), (7, 7.77, 7.77);

SELECT
    groupArrayMovingSum(int) AS I,
    groupArrayMovingSum(float) AS F,
    groupArrayMovingSum(dec) AS D
FROM t;
```

```response title=Response
┌─I──────────┬─F───────────────────────────────┬─D──────────────────────┐
│ [1,3,7,14] │ [1.1,3.3000002,7.7000003,15.47] │ [1.10,3.30,7.70,15.47] │
└────────────┴─────────────────────────────────┴────────────────────────┘
```

**指定窗口大小**

```sql title=Query
SELECT
    groupArrayMovingSum(2)(int) AS I,
    groupArrayMovingSum(2)(float) AS F,
    groupArrayMovingSum(2)(dec) AS D
FROM t;
```

```response title=Response
┌─I──────────┬─F───────────────────────────────┬─D──────────────────────┐
│ [1,3,6,11] │ [1.1,3.3000002,6.6000004,12.17] │ [1.10,3.30,6.60,12.17] │
└────────────┴─────────────────────────────────┴────────────────────────┘
```
