## stddevSamp \{#stddevSamp\}

引入版本：v1.1

返回一组数值数据序列的样本标准差。
结果等于 [`varSamp`](/sql-reference/aggregate-functions/reference/varSamp) 的平方根。

:::note
此函数使用在数值上不稳定的算法。
如果在计算中需要[数值稳定性](https://en.wikipedia.org/wiki/Numerical_stability)，请使用 [`stddevSampStable`](/sql-reference/aggregate-functions/reference/stddevsampstable) 函数。它的执行速度较慢，但计算误差更小。
:::

**语法**

```sql
stddevSamp(x)
```

**别名**: `STDDEV_SAMP`

**参数**

* `x` — 要计算其样本方差平方根的值。[`(U)Int*`](/sql-reference/data-types/int-uint) 或 [`Float*`](/sql-reference/data-types/float) 或 [`Decimal*`](/sql-reference/data-types/decimal)

**返回值**

返回 `x` 的样本方差的平方根值。[`Float64`](/sql-reference/data-types/float)

**示例**

**计算样本标准差**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSamp(population)
FROM test_data;
```

```response title=Response
┌─stddevSamp(population)─┐
│                      4 │
└────────────────────────┘
```
