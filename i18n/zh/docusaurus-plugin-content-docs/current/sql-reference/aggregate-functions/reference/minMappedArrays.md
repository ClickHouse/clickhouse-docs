## minMappedArrays \{#minMappedArrays\}

引入版本：v20.5

根据 `key` 数组中指定的键，从 `value` 数组中计算最小值。

:::note

* 传递一个由键数组和值数组组成的 tuple，与分别传递一个键数组和一个值数组是等价的。
* 对于每一行进行汇总时，`key` 和 `value` 中的元素数量必须相同。
  :::

**语法**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**参数**

* `key` — 键的数组。[`Array(T)`](/sql-reference/data-types/array)
* `value` — 值的数组。[`Array(T)`](/sql-reference/data-types/array)

**返回值**

返回一个由两个数组组成的元组：按排序后顺序排列的键，以及为对应键计算得到的值。[`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**示例**

**使用示例**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
