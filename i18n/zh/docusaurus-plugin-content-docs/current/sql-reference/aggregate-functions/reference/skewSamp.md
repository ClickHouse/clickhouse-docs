## skewSamp \{#skewSamp\}

引入于：v20.1

计算序列的[样本偏度](https://en.wikipedia.org/wiki/Skewness)。

如果传入的值构成某随机变量的样本，则该函数返回该随机变量偏度的无偏估计值。

**语法**

```sql
skewSamp(expr)
```

**参数**

* `expr` — 返回数值的表达式。[`Expression`](/sql-reference/data-types/special-data-types/expression)

**返回值**

返回给定分布的偏度。如果 `n <= 1`（`n` 是样本大小），则函数返回 `nan`。[`Float64`](/sql-reference/data-types/float)

**示例**

**对称分布**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**右偏分布**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
