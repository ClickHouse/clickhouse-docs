## skewPop \{#skewPop\}

引入自：v20.1

计算序列的[偏度](https://en.wikipedia.org/wiki/Skewness)。

**语法**

```sql
skewPop(expr)
```

**参数**

* `expr` — 返回数值的表达式。[`Expression`](/sql-reference/data-types/special-data-types/expression)

**返回值**

返回给定分布的偏度。[`Float64`](/sql-reference/data-types/float)

**示例**

**对称分布**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**右偏态分布**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
