# studentTTestOneSample \{#studentttestonesample\}

用于对单个样本进行 Student&#39;s t 检验，以判断样本均值是否与已知总体均值存在差异。

假设数据服从正态分布。原假设是样本均值等于总体均值。

**语法**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

可选的 `confidence_level` 参数用于启用置信区间计算。

**参数说明**

* `sample_data` — 样本数据。Integer、Float 或 Decimal。
* `population_mean` — 用于检验的已知总体均值。Integer、Float 或 Decimal（通常为常量）。

**可选参数**

* `confidence_level` — 置信区间的置信水平。取值为 (0, 1) 区间内的 Float。

注意：

* 至少需要 2 个观测值；否则结果为 `(nan, nan)`（如果请求置信区间，则区间为 `nan`）。
* 常量或近似常量的输入也会因为标准误为零（或接近零）而返回 `nan`。

**返回值**

[Tuple](../../../sql-reference/data-types/tuple.md)，包含两个或四个元素（如果指定了 `confidence_level`，则为四个）：

* 计算得到的 t-统计量。Float64。
* 计算得到的 p 值（双尾）。Float64。
* 计算得到的置信区间下界。Float64。（可选）
* 计算得到的置信区间上界。Float64。（可选）

置信区间表示在给定置信水平下的样本均值区间。

**示例**

输入表：

```text
┌─value─┐
│  20.3 │
│  21.1 │
│  21.7 │
│  19.9 │
│  21.8 │
└───────┘
```

不含置信区间：

```sql
SELECT studentTTestOneSample()(value, 20.0) FROM t;
-- or simply
SELECT studentTTestOneSample(value, 20.0) FROM t;
```

置信区间（95%）：

```sql
SELECT studentTTestOneSample(0.95)(value, 20.0) FROM t;
```
