# 加密函数 {#encryption-functions}

这些函数使用 AES（Advanced Encryption Standard，高级加密标准）算法对数据进行加解密。

密钥长度取决于所选加密模式：对于 `-128-`、`-196-` 和 `-256-` 模式，长度分别为 `16`、`24` 和 `32` 字节。

初始化向量（IV）的长度始终为 16 字节（超出 16 字节的部分会被忽略）。
