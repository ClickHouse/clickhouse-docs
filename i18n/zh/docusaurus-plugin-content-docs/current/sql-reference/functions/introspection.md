# 自省函数

可以使用本章中描述的函数，对 [ELF](https://en.wikipedia.org/wiki/Executable_and_Linkable_Format) 和 [DWARF](https://en.wikipedia.org/wiki/DWARF) 进行自省，用于查询剖析（profiling）。

:::note
这些函数较慢，并且可能带来安全方面的风险。
:::

要使自省函数正常工作：

* 安装 `clickhouse-common-static-dbg` 软件包。

* 将 [allow&#95;introspection&#95;functions](../../operations/settings/settings.md#allow_introspection_functions) 设置为 1。

  出于安全原因，自省函数默认是禁用的。

ClickHouse 将分析器报告保存到 [trace&#95;log](/operations/system-tables/trace_log) 系统表中。请确保该表和分析器已正确配置。
