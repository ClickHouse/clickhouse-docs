
# 使用 S2 索引的功能

## S2Index {#s2index}

[S2](https://s2geometry.io/) 是一个地理索引系统，其中所有地理数据都在球体上表示（类似于地球仪）。

在 S2 库中，点被表示为 S2 索引——一个特定的数字，内部编码为单位球面上的一个点，与传统的（纬度，经度）对不同。要获取指定格式（纬度，经度）给定点的 S2 点索引，请使用 [geoToS2](#geotos2) 函数。此外，您还可以使用 [s2ToGeo](#s2togeo) 函数获取与指定 S2 点索引对应的地理坐标。

## geoToS2 {#geotos2}

返回与提供的坐标 `(经度, 纬度)` 对应的 [S2](#s2index) 点索引。

**语法**

```sql
geoToS2(lon, lat)
```

**参数**

- `lon` — 经度。 [Float64](../../data-types/float.md).
- `lat` — 纬度。 [Float64](../../data-types/float.md).

**返回值**

- S2 点索引。 [UInt64](../../data-types/int-uint.md).

**示例**

查询：

```sql
SELECT geoToS2(37.79506683, 55.71290588) AS s2Index;
```

结果：

```text
┌─────────────s2Index─┐
│ 4704772434919038107 │
└─────────────────────┘
```

## s2ToGeo {#s2togeo}

返回与提供的 [S2](#s2index) 点索引对应的地理坐标 `(经度, 纬度)`。

**语法**

```sql
s2ToGeo(s2index)
```

**参数**

- `s2index` — S2 索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- 一个 [tuple](../../data-types/tuple.md)，包含两个值：
    - `lon`。[Float64](../../data-types/float.md).
    - `lat`。[Float64](../../data-types/float.md).

**示例**

查询：

```sql
SELECT s2ToGeo(4704772434919038107) AS s2Coodrinates;
```

结果：

```text
┌─s2Coodrinates────────────────────────┐
│ (37.79506681471008,55.7129059052841) │
└──────────────────────────────────────┘
```

## s2GetNeighbors {#s2getneighbors}

返回与提供的 [S2](#s2index) 相对应的 S2 邻居索引。S2 系统中的每个单元都是由四条测地线界定的四边形。因此，每个单元有 4 个邻居。

**语法**

```sql
s2GetNeighbors(s2index)
```

**参数**

- `s2index` — S2 索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- 一个包含 4 个邻居索引的数组：`array[s2index1, s2index3, s2index2, s2index4]`。[Array](../../data-types/array.md)([UInt64](../../data-types/int-uint.md)).

**示例**

查询：

```sql
SELECT s2GetNeighbors(5074766849661468672) AS s2Neighbors;
```

结果：

```text
┌─s2Neighbors───────────────────────────────────────────────────────────────────────┐
│ [5074766987100422144,5074766712222515200,5074767536856236032,5074767261978329088] │
└───────────────────────────────────────────────────────────────────────────────────┘
```

## s2CellsIntersect {#s2cellsintersect}

确定两个提供的 [S2](#s2index) 单元是否相交。

**语法**

```sql
s2CellsIntersect(s2index1, s2index2)
```

**参数**

- `siIndex1`, `s2index2` — S2 索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- `1` — 如果单元相交。 [UInt8](../../data-types/int-uint.md).
- `0` — 如果单元不相交。 [UInt8](../../data-types/int-uint.md).

**示例**

查询：

```sql
SELECT s2CellsIntersect(9926595209846587392, 9926594385212866560) AS intersect;
```

结果：

```text
┌─intersect─┐
│         1 │
└───────────┘
```

## s2CapContains {#s2capcontains}

确定一个圆盖是否包含一个 S2 点。圆盖代表被平面切割下来的球体的一部分。它由球面上的一个点和以度为单位的半径定义。

**语法**

```sql
s2CapContains(center, degrees, point)
```

**参数**

- `center` — 与圆盖对应的 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `degrees` — 圆盖的半径（以度为单位）。 [Float64](../../data-types/float.md).
- `point` — S2 点索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- `1` — 如果圆盖包含 S2 点索引。 [UInt8](../../data-types/int-uint.md).
- `0` — 如果圆盖不包含 S2 点索引。 [UInt8](../../data-types/int-uint.md).

**示例**

查询：

```sql
SELECT s2CapContains(1157339245694594829, 1.0, 1157347770437378819) AS capContains;
```

结果：

```text
┌─capContains─┐
│           1 │
└─────────────┘
```

## s2CapUnion {#s2capunion}

确定包含给定两个输入圆盖的最小圆盖。圆盖代表被平面切割下来的球体的一部分。它由球面上的一个点和以度为单位的半径定义。

**语法**

```sql
s2CapUnion(center1, radius1, center2, radius2)
```

**参数**

- `center1`, `center2` — 与两个输入圆盖对应的 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `radius1`, `radius2` — 两个输入圆盖的半径（以度为单位）。 [Float64](../../data-types/float.md).

**返回值**

- `center` — 与包含两个输入圆盖的最小圆盖中心对应的 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `radius` — 包含两个输入圆盖的最小圆盖的半径。 [Float64](../../data-types/float.md).

**示例**

查询：

```sql
SELECT s2CapUnion(3814912406305146967, 1.0, 1157347770437378819, 1.0) AS capUnion;
```

结果：

```text
┌─capUnion───────────────────────────────┐
│ (4534655147792050737,60.2088283994957) │
└────────────────────────────────────────┘
```

## s2RectAdd {#s2rectadd}

增加边界矩形的大小，以包括给定的 S2 点。在 S2 系统中，矩形由一种称为 `S2LatLngRect` 的 S2Region 类型表示，表示在纬度-经度空间中的矩形。

**语法**

```sql
s2RectAdd(s2pointLow, s2pointHigh, s2Point)
```

**参数**

- `s2PointLow` — 与矩形对应的低 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `s2PointHigh` — 与矩形对应的高 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `s2Point` — 目标 S2 点索引，边界矩形应增长以包括该点。 [UInt64](../../data-types/int-uint.md).

**返回值**

- `s2PointLow` — 与增长后的矩形对应的低 S2 单元 ID。 [UInt64](../../data-types/int-uint.md).
- `s2PointHigh` — 与增长后的矩形对应的高 S2 单元 ID。 [UInt64](../../data-types/float.md).

**示例**

查询：

```sql
SELECT s2RectAdd(5178914411069187297, 5177056748191934217, 5179056748191934217) AS rectAdd;
```

结果：

```text
┌─rectAdd───────────────────────────────────┐
│ (5179062030687166815,5177056748191934217) │
└───────────────────────────────────────────┘
```

**语法**

```sql
s2RectContains(s2PointLow, s2PointHi, s2Point)
```

**参数**

- `s2PointLow` — 与矩形对应的低 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `s2PointHigh` — 与矩形对应的高 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `s2Point` — 目标 S2 点索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- `1` — 如果矩形包含给定的 S2 点。
- `0` — 如果矩形不包含给定的 S2 点。

**示例**

查询：

```sql

```

结果：

```text
┌─rectContains─┐
│            0 │
└──────────────┘
```

## s2RectUnion {#s2rectunion}

返回包含该矩形和给定矩形联合的最小矩形。在 S2 系统中，矩形由一种称为 `S2LatLngRect` 的 S2Region 类型表示，表示在纬度-经度空间中的矩形。

**语法**

```sql
s2RectUnion(s2Rect1PointLow, s2Rect1PointHi, s2Rect2PointLow, s2Rect2PointHi)
```

**参数**

- `s2Rect1PointLow`, `s2Rect1PointHi` — 与第一个矩形对应的低和高 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `s2Rect2PointLow`, `s2Rect2PointHi` — 与第二个矩形对应的低和高 S2 点索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- `s2UnionRect2PointLow` — 与联合矩形对应的低 S2 单元 ID。 [UInt64](../../data-types/int-uint.md).
- `s2UnionRect2PointHi` — 与联合矩形对应的高 S2 单元 ID。 [UInt64](../../data-types/int-uint.md).

**示例**

查询：

```sql
SELECT s2RectUnion(5178914411069187297, 5177056748191934217, 5179062030687166815, 5177056748191934217) AS rectUnion;
```

结果：

```text
┌─rectUnion─────────────────────────────────┐
│ (5179062030687166815,5177056748191934217) │
└───────────────────────────────────────────┘
```

## s2RectIntersection {#s2rectintersection}

返回包含该矩形和给定矩形交集的最小矩形。在 S2 系统中，矩形由一种称为 `S2LatLngRect` 的 S2Region 类型表示，表示在纬度-经度空间中的矩形。

**语法**

```sql
s2RectIntersection(s2Rect1PointLow, s2Rect1PointHi, s2Rect2PointLow, s2Rect2PointHi)
```

**参数**

- `s2Rect1PointLow`, `s2Rect1PointHi` — 与第一个矩形对应的低和高 S2 点索引。 [UInt64](../../data-types/int-uint.md).
- `s2Rect2PointLow`, `s2Rect2PointHi` — 与第二个矩形对应的低和高 S2 点索引。 [UInt64](../../data-types/int-uint.md).

**返回值**

- `s2UnionRect2PointLow` — 与包含给定矩形交集的矩形对应的低 S2 单元 ID。 [UInt64](../../data-types/int-uint.md).
- `s2UnionRect2PointHi` — 与包含给定矩形交集的矩形对应的高 S2 单元 ID。 [UInt64](../../data-types/int-uint.md).

**示例**

查询：

```sql
SELECT s2RectIntersection(5178914411069187297, 5177056748191934217, 5179062030687166815, 5177056748191934217) AS rectIntersection;
```

结果：

```text
┌─rectIntersection──────────────────────────┐
│ (5178914411069187297,5177056748191934217) │
└───────────────────────────────────────────┘
```
