# ClickHouse SQL 语句 \{#clickhouse-sql-statements\}

用户可以通过 SQL 语句与 ClickHouse 进行交互。ClickHouse 支持常见的 SQL 语句，例如 [SELECT](select/index.md) 和 [CREATE](create/index.md)，同时也提供了诸如 [KILL](kill.md) 和 [OPTIMIZE](optimize.md) 之类的专用语句。
