# “ClickHouse”是什么意思？ \{#what-does-clickhouse-mean\}

它是“**Click**stream”（点击流）和“Data ware**House**”（数据仓库）的组合。这个名字源自 ClickHouse 在 Yandex.Metrica 中的最初用例：ClickHouse 需要保存来自整个互联网、所有用户的点击记录，而它至今仍然胜任这项工作。你可以在 [ClickHouse 历史](../../about-us/history.md) 页面了解更多相关用例。

这个由两部分组成的名字有两个要点：

- 书写 Click**H**ouse 时，唯一正确的方式是将 H 大写。
- 如果需要缩写，请使用 **CH**。由于一些历史原因，在中国使用 CK 作为缩写也很常见，主要是因为最早的一次用中文介绍 ClickHouse 的演讲采用了这种形式。

:::info    
在 ClickHouse 得名很多年之后，这种将两个本身就有意义的单词组合起来的命名方式，被一项由 [Andy Pavlo](https://www.cs.cmu.edu/~pavlo/blog/2020/03/on-naming-a-database-management-system.html)（卡内基梅隆大学数据库副教授）开展的研究视为数据库命名的最佳实践。ClickHouse 与 Postgres 共同获得了他评选的“有史以来最佳数据库名称”奖。
:::
