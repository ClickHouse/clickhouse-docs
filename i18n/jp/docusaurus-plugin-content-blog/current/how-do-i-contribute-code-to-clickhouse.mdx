## ClickHouse への貢献方法 \{#how-to-contribute-in-clickhouse\}

ClickHouse は、[GitHub 上で開発されている](https://github.com/ClickHouse/ClickHouse)オープンソースプロジェクトです。

慣例に従い、貢献に関する手順はソースコードリポジトリのルートにある [CONTRIBUTING](https://github.com/ClickHouse/ClickHouse/blob/master/CONTRIBUTING.md) ファイルに掲載されています。

ClickHouse に対して大きな変更を提案したい場合は、まずメンテナーやコミュニティと議論するために、何をしたいのかを説明した [GitHub issue を作成する](https://github.com/ClickHouse/ClickHouse/issues/new/choose) ことを検討してください。[そのような RFC issue の例はこちらです](https://github.com/ClickHouse/ClickHouse/issues?q=is%3Aissue+is%3Aopen+rfc)。

セキュリティ関連のコントリビューションを行う場合は、[セキュリティポリシー](https://github.com/ClickHouse/ClickHouse/security/policy/) も確認してください。
