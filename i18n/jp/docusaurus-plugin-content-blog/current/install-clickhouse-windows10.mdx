## Microsoft Windows に ClickHouse をインストールしてテストする方法

Windows 10 に ClickHouse をインストールした環境で、たとえばデータを挿入する際にエラーが発生する場合があります。

```
DB::Exception: std::__1::__fs::filesystem::filesystem_error: filesystem error: in rename: Permission denied ["./store/711/71144174-d098-4056-8976-6ad1204205ec/tmp_insert_all_1_1_0/"] ["./store/711/71144174-d098-4056-8976-6ad1204205ec/all_1_1_0/"]. スタックトレース:
```

Windows 10 では、WSL を WSL 2 にアップグレードする必要があります。

* PowerShell アイコンを右クリックし、「管理者として実行」を選択して PowerShell を開きます。

* WSL 2 にアップグレードするには、Microsoft の次の手順に従ってください:
  [https://learn.microsoft.com/en-us/windows/wsl/install](https://learn.microsoft.com/en-us/windows/wsl/install)

* アップグレードが完了したら、PowerShell から WSL を起動します。

```
wsl
```

* テストとしてこれらの手順に従えば、同様の出力が得られるはずです：
  これはテスト目的のため、権限の問題を避けるために root ユーザーとしてログインしています。

```
sudo -i
```

* ClickHouse のディレクトリを作成します：

```
root@marspc2:~# mkdir /clickhouse
```

* 新しいディレクトリで ClickHouse をダウンロードします：

```
root@marspc2:/# cd clickhouse

root@marspc2:/clickhouse# curl https://clickhouse.com | sh

ClickHouse バイナリのダウンロードが完了しました。以下のコマンドで実行できます:
    ./clickhouse

以下のコマンドでインストールすることもできます:
    sudo ./clickhouse install
```
