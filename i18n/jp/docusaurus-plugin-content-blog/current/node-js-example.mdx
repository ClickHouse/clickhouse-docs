## NodeJS と @clickhouse/client を使う \{#use-nodejs-with-clickhouseclient\}

これは基本的なコード例を含む `main.ts` ファイルです。

`Package.json` (`./` 直下に配置します) :

```json
{
  "name": "シンプルなClickHouseクライアントの例",
  "version": "1.0.0",
  "main": "main.js",
  "license": "MIT",
  "devDependencies": {
    "typescript": "^5.3.2"
  },
  "dependencies": {
    "@clickhouse/client": "^0.2.6"
  }
}
```

`Main.ts` (`./src` 以下に配置) :

```ts
import { ClickHouseClient, createClient } from '@clickhouse/client'; // or '@clickhouse/client-web'

interface ClickHouseResultSet<T> {
  meta: Meta[];
  data: T[];
  rows: number;
  statistics: Statistics;
}

interface Statistics {
  elapsed: number;
  rows_read: number;
  bytes_read: number;
}

interface Meta {
  name: string;
  type: string;
}

interface Count {
  c: number;
}

//`host`、`username`、`password`、`database`などのクライアント接続パラメータを
//必要に応じて置き換えてください。

const initClickHouseClient = async (): Promise<ClickHouseClient> => {
  const client = createClient({
    host: 'https://FQDN.aws.clickhouse.cloud',
    username: 'default',
    password: 'password',
    database: 'default',
    application: `pingpong`,
  });

  console.log('ClickHouse ping実行中');
  if (!(await client.ping())) {
    throw new Error('ClickHouseへのpingに失敗しました');
  }
  console.log('ClickHouse pong受信完了');
  return client;
};

const main = async () => {
  console.log('ClickHouseクライアントを初期化中');
  const client = await initClickHouseClient();

  const row = await client.query({
    query: `SELECT count() AS c FROM system.tables WHERE database='system'`,
  });

  const jsonRow: ClickHouseResultSet<Count> = await row.json();

  console.log(`${jsonRow.data[0].c}個のシステムテーブルが見つかりました`);

  await client.close();
  console.log(`👋`);
};

main();
```

パッケージをインストールするには、`./` ディレクトリで `yarn` を実行します。

```
$ yarn
yarn install v1.22.19

```

`./` にある `main.ts` のコードを次のコマンドで実行します：`

```
$ npx ts-node src/main.ts
```

```

```
