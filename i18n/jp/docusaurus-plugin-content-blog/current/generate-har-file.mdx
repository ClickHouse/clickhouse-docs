## Google Chrome から \{#from-google-chrome\}

1. <kbd>F12</kbd> キー、または <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd>（Windows/Linux） / <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>I</kbd>（Mac）を押して、デベロッパーツールを開きます。
2. 「ネットワーク」タブをクリックします。
3. ページを再読み込みし、問題を再現します。
4. デベロッパーツール内でダウンロードボタンをクリックします。

## Mozilla Firefox から \{#from-mozilla-firefox\}

1. <kbd>F12</kbd> または <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>（Windows/Linux） / <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>E</kbd>（Mac）を押して開発ツール（Developer Tools）を開きます。
2. 「ネットワーク」タブをクリックします。
3. ページを再読み込みして、問題を再現します。
4. 開発ツールのリクエスト一覧上で右クリックし、「HAR としてすべて保存」（Save All As HAR）を選択します。

## Microsoft Edge から \{#from-microsoft-edge\}

1. <kbd>F12</kbd> または <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd> (Windows/Linux) / <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>I</kbd> (Mac) を押して開発者ツールを開きます。
2. 「Network」タブをクリックします。
3. ページを再読み込みし、問題を再現します。
4. 開発者ツールの Network リスト内を右クリックし、「Save all as HAR with content」を選択します。

## Safari から \{#from-safari\}

1. 開発ツールを有効にします（まだ有効にしていない場合）：
   * Safari &gt; 設定 &gt; 詳細 を開きます。
   * 一番下の「メニューバーに“開発”メニューを表示」をオンにします。
2. 「開発」&gt;「Web インスペクタを表示」をクリックします。
3. 「ネットワーク」タブをクリックします。
4. ページを再読み込みして、問題を再現します。
5. 開発ツールから「書き出す」ボタンをクリックします。

## .HAR ファイルの送信 \{#sending-har-file\}

1. ファイル名を短く分かりやすい名前に変更します（例: login-issue.har）。
2. ファイルを圧縮します（任意ですが推奨します）。
3. サポートケースに添付するか、担当のサポート窓口宛てにメールで送信します。

:::note
HAR ファイルにはセッションクッキーやその他の機密データが含まれている場合があります。
権限を持つサポート担当者にのみ共有してください。
:::
