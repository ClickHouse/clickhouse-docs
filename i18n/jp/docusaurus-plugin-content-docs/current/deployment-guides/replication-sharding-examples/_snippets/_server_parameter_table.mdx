各サーバーについて、次のパラメータを指定します。

| Parameter | Description | Default Value |
|----------|-------------|---------------|
| `host`   | リモートサーバーのアドレスです。ドメイン名、IPv4 アドレス、または IPv6 アドレスのいずれかを使用できます。ドメイン名を指定した場合、サーバー起動時に DNS 問い合わせが行われ、その結果はサーバーが動作している間保持されます。DNS 問い合わせに失敗すると、サーバーは起動しません。DNS レコードを変更した場合は、サーバーを再起動する必要があります。 | -             |
| `port`   | messenger の通信に使用する TCP ポートです（設定内の `tcp_port`。通常は 9000 に設定されます）。`http_port` と混同しないでください。 | -             |
