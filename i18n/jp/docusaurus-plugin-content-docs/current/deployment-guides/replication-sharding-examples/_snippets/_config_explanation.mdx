* `config.d` ディレクトリには ClickHouse サーバー構成ファイル `config.xml` が含まれており、
  各 ClickHouse ノード向けのカスタム構成がここで定義されます。
  この構成は、すべての ClickHouse インストールに含まれるデフォルトの
  ClickHouse 構成ファイル `config.xml` とマージされます。
* `users.d` ディレクトリにはユーザー構成ファイル `users.xml` が含まれており、
  ユーザー用のカスタム構成がここで定義されます。この構成は、すべての
  ClickHouse インストールに含まれるデフォルトの ClickHouse `users.xml`
  構成ファイルとマージされます。

:::tip カスタム構成ディレクトリ
独自の構成を書く際には、`/etc/clickhouse-server/config.xml` および
`/etc/clickhouse-server/users.xml` のデフォルト構成を直接変更するのではなく、
`config.d` および `users.d` ディレクトリを利用することがベストプラクティスです。

次の行

```xml
<clickhouse replace="true">
```

`config.d` と `users.d` ディレクトリで定義された設定セクションが、`config.xml` と `users.xml` ファイルに定義された既定の設定セクションを上書きすることを保証します。
:::
