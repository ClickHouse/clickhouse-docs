* `config.d` ディレクトリには ClickHouse サーバーの設定ファイル `config.xml` が含まれており、
  各 ClickHouse ノードごとのカスタム設定がここで定義されています。
  この設定は、すべての ClickHouse のインストールに付属するデフォルトの設定ファイル
  `config.xml` とマージされます。
* `users.d` ディレクトリにはユーザー設定ファイル `users.xml` が含まれており、
  ユーザー用のカスタム設定がここで定義されています。
  この設定は、すべての ClickHouse のインストールに付属するデフォルトの設定ファイル
  `users.xml` とマージされます。

:::tip Custom configuration directories
独自の設定を書く際は、`/etc/clickhouse-server/config.xml` および
`etc/clickhouse-server/users.xml` のデフォルト設定を直接変更するのではなく、
`config.d` および `users.d` ディレクトリを利用することがベストプラクティスです。

次の行

```xml
<clickhouse replace="true">
```

`config.d` および `users.d` ディレクトリで定義された設定セクションが、デフォルトの `config.xml` および `users.xml` ファイルに定義されている設定セクションよりも優先されて上書きされることを保証します。
:::
