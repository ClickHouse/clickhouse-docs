* `config.d` ディレクトリには ClickHouse サーバー設定ファイル `config.xml` が含まれており、
  各 ClickHouse ノードごとのカスタム設定がここで定義されます。
  この設定は、すべての ClickHouse インストールに付属するデフォルトの ClickHouse 設定ファイル
  `config.xml` とマージされて使用されます。
* `users.d` ディレクトリにはユーザー設定ファイル `users.xml` が含まれており、
  ユーザーごとのカスタム設定がここで定義されます。この設定は、すべての ClickHouse
  インストールに付属するデフォルトの ClickHouse 設定ファイル `users.xml` とマージされて使用されます。

:::tip カスタム設定ディレクトリ
独自の設定を書く際は、`/etc/clickhouse-server/config.xml` および
`etc/clickhouse-server/users.xml` にあるデフォルト設定を直接編集するのではなく、
`config.d` と `users.d` ディレクトリを利用することがベストプラクティスです。

次の行

```xml
<clickhouse replace="true">
```

`config.d` と `users.d` ディレクトリで定義された設定セクションが、デフォルトの `config.xml` および `users.xml` ファイルで定義された設定セクションよりも優先されるようにします。
:::
