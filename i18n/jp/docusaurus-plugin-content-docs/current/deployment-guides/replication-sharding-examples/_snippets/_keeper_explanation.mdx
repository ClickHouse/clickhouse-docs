各設定ファイルには、以下のような固有の設定が含まれています（下記参照）。
使用する `server_id` は、そのクラスタ内で該当する ClickHouse Keeper ノードごとに一意であり、
`<raft_configuration>` セクションで定義されているサーバーの `<id>` と一致している必要があります。
`tcp_port` は ClickHouse Keeper の *クライアント* が使用するポートです。

```xml
<tcp_port>9181</tcp_port>
<server_id>{id}</server_id>
```

次のセクションでは、[Raft 合意アルゴリズム](https://en.wikipedia.org/wiki/Raft_\(algorithm\)) の
クォーラムに参加するサーバーを構成します。

```xml
<raft_configuration>
    <server>
        <id>1</id>
        <hostname>clickhouse-keeper-01</hostname>
        <!-- ClickHouse Keeperノード間通信用のTCPポート -->
        <!--highlight-next-line-->
        <port>9234</port>
    </server>
    <server>
        <id>2</id>
        <hostname>clickhouse-keeper-02</hostname>
        <port>9234</port>
    </server>
    <server>
        <id>3</id>
        <hostname>clickhouse-keeper-03</hostname>
        <port>9234</port>
    </server>
</raft_configuration>
```
