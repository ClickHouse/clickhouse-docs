各設定ファイルには、以下に示す固有の構成が含まれます。
使用する `server_id` は、そのクラスタ内の当該 ClickHouse Keeper ノードごとに一意であり、
`&lt;raft_configuration&gt;` セクションで定義されているサーバーの `&lt;id&gt;` と一致している必要があります。
`tcp_port` は ClickHouse Keeper の *クライアント* が使用するポートです。

```xml
<tcp_port>9181</tcp_port>
<server_id>{id}</server_id>
```

以下のセクションでは、[Raft コンセンサスアルゴリズム](https://en.wikipedia.org/wiki/Raft_\(algorithm\)) のクォーラムに参加するサーバーを構成します。

```xml
<raft_configuration>
    <server>
        <id>1</id>
        <hostname>clickhouse-keeper-01</hostname>
        <!-- ClickHouse Keeperノード間通信用のTCPポート -->
        <!--highlight-next-line-->
        <port>9234</port>
    </server>
    <server>
        <id>2</id>
        <hostname>clickhouse-keeper-02</hostname>
        <port>9234</port>
    </server>
    <server>
        <id>3</id>
        <hostname>clickhouse-keeper-03</hostname>
        <port>9234</port>
    </server>
</raft_configuration>
```
