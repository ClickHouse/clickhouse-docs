# Confluent Cloud と ClickHouse の統合

Confluent プラットフォームは、ClickHouse と統合するために 2 つのオプションを提供します。

* [Confluent Cloud 上の ClickHouse Connect Sink](./confluent-cloud.md)
* カスタムコネクタ機能を使用する [Confluent プラットフォーム上の ClickHouse Connect Sink](./custom-connector.md)
* HTTP または HTTPS 経由で API と Apache Kafka を統合する [Confluent プラットフォーム向け HTTP Sink Connector](./kafka-connect-http.md)
