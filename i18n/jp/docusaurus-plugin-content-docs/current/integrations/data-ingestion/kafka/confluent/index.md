# Confluent Cloud と ClickHouse の統合

Confluent プラットフォームは、ClickHouse と統合するために 2 つのオプションを提供します。

* [Confluent Cloud 上の ClickHouse Connect Sink](./confluent-cloud.md)
* カスタムコネクタ機能を使用する [Confluent Platform 上の ClickHouse Connect Sink](./custom-connector.md)
* Apache Kafka を HTTP または HTTPS 経由で API と統合する [Confluent Platform 用 HTTP Sink Connector](./kafka-connect-http.md)
