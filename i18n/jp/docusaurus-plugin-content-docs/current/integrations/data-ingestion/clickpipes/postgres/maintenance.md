# Postgres 用 ClickPipes のメンテナンスウィンドウ

Postgres 用 ClickPipes のメンテナンスウィンドウが次の日程で予定されています：

- **日付:** 2025年4月17日
- **時間:** 07:00 AM - 08:00 AM UTC

この時間帯に、Postgres 用の Pipe は短時間の停止が発生します。
メンテナンスウィンドウ終了後、ClickPipes は再び利用可能となり、通常どおりの運用を再開します。
