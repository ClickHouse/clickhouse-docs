# Postgres 用 ClickPipes のメンテナンスウィンドウ

Postgres 用 ClickPipes のメンテナンスウィンドウが以下の日時で予定されています:

- **日付:** 2025 年 4 月 17 日
- **時間:** 07:00–08:00 (UTC)

この時間帯に、Postgres Pipes は短時間の停止が発生します。
ClickPipes はメンテナンスウィンドウ終了後に再び利用可能となり、通常どおりの動作を再開します。
