# 連携

ClickHouse をお使いのツールやサービスと連携させて、強力なデータパイプラインと分析ワークフローを構築できます。

ClickHouse の連携機能は、サポートレベルごとに次のように分類されています：

| Tier | Description |
|------|-------------|
| Core integrations | ClickHouse によって開発またはメンテナンスされており、ClickHouse がサポートを提供し、ClickHouse の GitHub 組織で管理されています |
| Partner integrations | サードパーティのソフトウェアベンダーによって開発またはメンテナンスされ、そのベンダーによりサポートが提供されています |
| Community integrations | コミュニティメンバーによって開発またはメンテナンスされ、サポートされています。公開 GitHub リポジトリおよびコミュニティ Slack チャンネル以外に直接のサポートはありません |

<IntegrationGrid />

:::note Notice
サードパーティのロゴおよび商標はそれぞれの所有者に帰属し、利用可能な連携機能を示す目的のみに表示されています。
これらを支持・推奨することを意味するものではありません。
:::
