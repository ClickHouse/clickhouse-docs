ClickHouseの統合は、そのサポートレベルによって整理されています：

- **コア統合：** ClickHouseによって構築または維持され、ClickHouseによってサポートされ、ClickHouse GitHub組織内に存在します
- **パートナー統合：** 第三者ソフトウェアベンダーによって構築または維持され、サポートされます
- **コミュニティ統合：** コミュニティメンバーによって構築または維持され、サポートされます。パブリックなGitHubリポジトリやコミュニティのSlackチャンネルの他に、直接的なサポートはありません

各統合はさらに**プログラミング言語クライアント**、**データ取り込み**、**データ可視化**、**SQLクライアント**のカテゴリに分類されています。

:::note
下記にClickHouseの統合リストを積極的に編纂していますので、包括的ではありません。関連するClickHouse統合をリストに
[貢献](https://github.com/ClickHouse/clickhouse-docs#contributing)することをお勧めします。
:::
