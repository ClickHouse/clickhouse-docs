ClickHouse をお好みのツールやサービスと連携させて、強力なデータパイプラインと分析ワークフローを構築します。

ClickHouse のインテグレーションは、サポートレベルごとに分類されています。

| Tier                   | Description                                                                                               |
| ---------------------- | --------------------------------------------------------------------------------------------------------- |
| Core integrations      | ClickHouse が開発または保守しており、ClickHouse によるサポートが提供され、ClickHouse の GitHub 組織配下で公開されています                         |
| Partner integrations   | サードパーティのソフトウェアベンダーによって開発または保守され、ベンダーによるサポートが提供されています                                                      |
| Community integrations | コミュニティメンバーによって開発または保守され、コミュニティによってサポートされています。公開されている GitHub リポジトリおよびコミュニティの Slack チャンネル以外に、直接的なサポートはありません |

:::tip[ClickHouse 連携を構築していますか？]
ClickHouse に接続する製品を開発している場合は、まず [Integration development](/integrations/integration-development) ガイドの [インテグレーションの構築](/integrations/integration-development/building-integrations)、[テスト](/integrations/integration-development/testing-your-integration)、および [製品のドキュメント化](/integrations/integration-development/documenting-your-integration) を参照してください。提出の準備ができたら、[partner portal](https://clickhouse.com/partners) でインテグレーションを登録してください。
:::
