# インテグレーション

ClickHouse をお使いのツールやサービスと連携して、強力なデータパイプラインと分析ワークフローを構築できます。

ClickHouse のインテグレーションは、サポートレベルごとに分類されています。

| Tier | 説明 |
|------|-------------|
| Core integrations | ClickHouse が開発または保守しているインテグレーションで、ClickHouse によるサポートがあり、ClickHouse の GitHub 組織内で公開されています |
| Partner integrations | サードパーティのソフトウェアベンダーが開発または保守し、サポートしているインテグレーションです |
| Community integrations | コミュニティメンバーが開発または保守し、サポートしているインテグレーションです。公開されている GitHub リポジトリとコミュニティの Slack チャンネル以外に、直接のサポートはありません |

:::note 注意
サードパーティのロゴおよび商標はそれぞれの所有者に帰属し、利用可能なインテグレーションを示す目的でのみ表示されています。
いかなる推奨も意図するものではありません。
:::
