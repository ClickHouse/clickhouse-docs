# インテグレーション \{#integrations\}

ClickHouse をお好みのツールやサービスと連携させて、強力なデータパイプラインと分析ワークフローを構築します。

ClickHouse のインテグレーションは、サポートレベルごとに分類されています。

| Tier | Description |
|------|-------------|
| Core integrations | ClickHouse が開発または保守しており、ClickHouse によるサポートが提供され、ClickHouse の GitHub 組織配下で公開されています |
| Partner integrations | サードパーティのソフトウェアベンダーによって開発または保守され、ベンダーによるサポートが提供されています |
| Community integrations | コミュニティメンバーによって開発または保守され、コミュニティによってサポートされています。公開されている GitHub リポジトリおよびコミュニティの Slack チャンネル以外に、直接的なサポートはありません |

:::note Notice
サードパーティのロゴおよび商標は、それぞれの所有者に帰属し、利用可能なインテグレーションを示す目的でのみ表示されています。
これらを推奨または保証するものではありません。
:::
