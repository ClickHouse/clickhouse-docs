ClickHouse の統合はサポートレベルによって整理されています：

- **コア統合**： ClickHouse によって構築または維持されており、ClickHouse によってサポートされ、ClickHouse GitHub 組織に存在します
- **パートナー統合**：構築または維持され、サポートされている、サードパーティのソフトウェアベンダーによるもの
- **コミュニティ統合**：コミュニティメンバーによって構築または維持され、サポートされています。公開 GitHub リポジトリやコミュニティ Slack チャンネルを除いて、直接サポートはありません

各統合は **言語クライアント**、**データ取り込み**、**データ可視化**、および **SQL クライアント** カテゴリにさらに分類されています。

:::note
以下の ClickHouse 統合のリストを積極的に編纂していますので、網羅的ではありません。関連する ClickHouse 統合を自由に
[貢献](https://github.com/ClickHouse/clickhouse-docs#contributing)してください。
:::
