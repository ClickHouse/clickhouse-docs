:::tip SQL console
SQL クライアント接続が必要な場合は、ClickHouse Cloud サービスに付属の Web ベースの SQL コンソールを利用できます。詳細は、以下の **Connect to SQL console** を展開してください。
:::

<details>
  <summary>Connect to SQL console</summary>

  ClickHouse Cloud のサービス一覧から、対象のサービスをクリックします。
