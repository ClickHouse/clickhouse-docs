:::tip SQL console
SQL クライアントで接続したい場合は、ClickHouse Cloud サービスに紐づいた Web ベースの SQL コンソールが利用できます。詳しくは、下の **Connect to SQL console** を展開してください。
:::

<details>
  <summary>Connect to SQL console</summary>

  ClickHouse Cloud のサービス一覧から、対象のサービスをクリックします。

  <Image img={cloud_connect_to_sql_console} alt="Connect to SQL Console" size="lg" force border />

  SQL コンソールにリダイレクトされます。

  <Image img={createservice8} alt="SQL Console" size="lg" force border />
</details>
