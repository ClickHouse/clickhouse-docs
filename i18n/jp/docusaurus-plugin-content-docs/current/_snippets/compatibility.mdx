:::note 互換性
セルフホスト型の ClickHouse デプロイメントと ClickHouse Cloud サービスの間で動作に違いが見られる場合、
それは[互換性設定](/operations/settings/settings#compatibility)に関連している可能性があります。
ClickHouse Cloud では、互換性はサービス作成時に設定され、サービスがアップグレードされてもクライアントが一貫した動作を得られるように、サービスレベルでは変更されません。
互換性を変更したい場合は、[サポート](https://clickhouse.com/support/program)を通じてリクエストできます。
:::
