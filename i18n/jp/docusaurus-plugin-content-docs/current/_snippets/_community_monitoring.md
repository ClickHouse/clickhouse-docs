## コミュニティ監視ソリューション {#community-monitoring}

ClickHouseコミュニティは、主要な可観測性スタックと統合できる包括的な監視ソリューションを開発しています。[ClickHouse Monitoring](https://github.com/duyet/clickhouse-monitoring)は、事前構築されたダッシュボードを備えた完全な監視セットアップを提供します。このオープンソースプロジェクトは、確立されたベストプラクティスと実績のあるダッシュボード構成でClickHouse監視を導入したいチームに、クイックスタート手法を提供します。

:::note
他の直接データベース監視手法と同様に、このソリューションはClickHouseシステムテーブルに直接クエリを実行するため、インスタンスがアイドル状態になることを妨げ、コスト最適化に影響します。
:::
