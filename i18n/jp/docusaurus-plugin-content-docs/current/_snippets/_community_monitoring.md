## コミュニティ監視ソリューション {#community-monitoring}

ClickHouseコミュニティは、主要な可観測性スタックと統合する包括的な監視ソリューションを開発しています。[ClickHouse Monitoring](https://github.com/duyet/clickhouse-monitoring)は、事前構築されたダッシュボードを備えた完全な監視セットアップを提供します。このオープンソースプロジェクトは、確立されたベストプラクティスと実証済みのダッシュボード構成でClickHouse監視を実装したいチームに、クイックスタート方式を提供します。

:::note
他の直接データベース監視アプローチと同様に、このソリューションはClickHouseシステムテーブルに直接クエリを実行するため、インスタンスがアイドル状態になることを妨げ、コスト最適化に影響します。
:::
