# ClickHouse Cloud

ClickHouse Cloud は、人気のあるオープンソース OLAP データベース ClickHouse のオリジナルの開発者が提供するクラウドサービスです。 
[無料トライアルを開始](https://console.clickhouse.cloud/signUp)して、ClickHouse Cloud を体験できます。



## ClickHouse Cloudの利点 {#clickhouse-cloud-benefits}

ClickHouse Cloudを使用する主な利点は以下の通りです:

- **迅速な価値実現**: クラスタのサイジングやスケーリングを気にすることなく、すぐに構築を開始できます。
- **シームレスなスケーリング**: 自動スケーリングが変動するワークロードに対応するため、ピーク時の使用量を見越した過剰なプロビジョニングが不要です。
- **サーバーレス運用**: サイジング、スケーリング、セキュリティ、信頼性、アップグレードは当社が管理しますので、安心してお任せください。
- **透明性の高い料金体系**: リソース予約とスケーリング制御により、使用した分だけお支払いいただけます。
- **総所有コストの削減**: 最高の価格対性能比と低い管理オーバーヘッドを実現します。
- **広範なエコシステム**: お気に入りのデータコネクタ、可視化ツール、SQLおよび言語クライアントをそのまま利用できます。

## ClickHouse CloudはどのバージョンのClickHouseを使用していますか？ {#what-version-of-clickhouse-does-clickhouse-cloud-use}

ClickHouse Cloudは、お客様のサービスを継続的に新しいバージョンへアップグレードします。オープンソースでコアデータベースバージョンを公開した後、クラウドステージング環境で追加の検証を実施します。この検証には通常6〜8週間を要し、その後本番環境へのロールアウトが行われます。ロールアウトは、クラウドサービスプロバイダー、サービスタイプ、およびリージョンごとに段階的に実施されます。

通常のリリーススケジュールより先行して更新を受け取るための「Fast」リリースチャネルを提供しています。詳細については、[「Fastリリースチャネル」](/manage/updates#fast-release-channel-early-upgrades)を参照してください。

以前のバージョンの機能に依存している場合は、サービスの互換性設定を使用することで、場合によっては以前の動作に戻すことが可能です。
