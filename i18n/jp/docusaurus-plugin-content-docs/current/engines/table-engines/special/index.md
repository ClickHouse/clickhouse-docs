# 特殊なテーブルエンジン

テーブルエンジンには主に3つのカテゴリがあります：

- [MergeTreeエンジンファミリー](../../../engines/table-engines/mergetree-family/index.md) 主な生産用途のため。
- [Logエンジンファミリー](../../../engines/table-engines/log-family/index.md) 小規模一時データのため。
- [統合のためのテーブルエンジン](../../../engines/table-engines/integrations/index.md)。

残りのエンジンは目的がユニークであり、まだファミリーにグループ化されていないため、この「特殊な」カテゴリに配置されています。
