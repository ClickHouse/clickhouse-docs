## 説明 {#description}

`Null`フォーマットでは、出力は何もありません。 
最初は奇妙に聞こえるかもしれませんが、重要なのは、何も出力しなくても、クエリは処理されるということです。 
コマンドラインクライアントを使用する場合、データはクライアントに送信されます。 

:::tip
`Null`フォーマットは、パフォーマンステストに役立ちます。
:::
