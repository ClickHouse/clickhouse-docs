## 説明 \{#description\}

[`PrettySpace`](./PrettySpace.md) 形式とは、[ANSI エスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code) を使用しない点が異なります。
これは、この形式をブラウザで表示したり、`watch` コマンドラインユーティリティで使用したりするために必要です。
