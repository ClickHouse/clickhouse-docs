## Description {#description}

[`PrettySpace`](./PrettySpace.md)フォーマットとは異なり、[ANSIエスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code)を使用しません。
これは、ブラウザでこのフォーマットを表示する際や、`watch`コマンドラインユーティリティを使用する際に必要となります。
