## 説明 {#description}

[`PrettyCompact`](./PrettyCompact.md)フォーマットとは異なり、[ANSIエスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code)は使用されません。 
これは、ブラウザでフォーマットを表示するためや、'watch'コマンドラインユーティリティを使用するために必要です。
