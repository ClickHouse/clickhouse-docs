## Description {#description}

[`PrettyCompact`](./PrettyCompact.md)フォーマットとの違いは、[ANSIエスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code)を使用しない点です。
これは、ブラウザでの表示や、`watch`コマンドラインユーティリティの使用時に必要となります。
