## 説明 {#description}

[`PrettyCompact`](./PrettyCompact.md) 形式とは、[ANSI エスケープシーケンス](http://en.wikipedia.org/wiki/ANSI_escape_code) を使用しない点が異なります。
これは、ブラウザでこの形式を表示する場合や、`watch` コマンドラインユーティリティで使用する場合に必要です。
