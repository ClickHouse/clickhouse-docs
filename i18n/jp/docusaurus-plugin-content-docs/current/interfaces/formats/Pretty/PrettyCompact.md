## 説明 {#description}

[`Pretty`](./Pretty.md) 形式とは異なり、行間にグリッドが描画されたテーブルが表示されます。 
そのため、結果はよりコンパクトになります。

:::note
この形式は、インタラクティブモードのコマンドラインクライアントでデフォルトで使用されます。
:::
