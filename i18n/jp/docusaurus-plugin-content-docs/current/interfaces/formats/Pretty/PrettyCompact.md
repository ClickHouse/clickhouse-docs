## 説明 {#description}

[`Pretty`](./Pretty.md)形式とは異なり、行間にグリッド線が引かれたテーブルとして表示されます。
そのため、結果はよりコンパクトになります。

:::note
この形式は、コマンドラインクライアントの対話モードでデフォルトで使用されます。
:::
