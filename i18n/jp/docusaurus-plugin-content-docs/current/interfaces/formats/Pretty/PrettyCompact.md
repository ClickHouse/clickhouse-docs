## 説明 {#description}

[`Pretty`](./Pretty.md) フォーマットとは異なり、行の間にグリッドが描画された状態でテーブルが表示されます。  
このため、結果はよりコンパクトになります。

:::note
このフォーマットは、インタラクティブモードのコマンドラインクライアントでデフォルトで使用されます。
:::
