## 説明 \\{#description\\}

[`Pretty`](./Pretty.md) フォーマットとは異なり、行と行の間に罫線を表示してテーブルを描画します。  
そのため、結果はより省スペースな表示になります。

:::note
このフォーマットは、コマンドラインクライアントのインタラクティブモードで既定で使用されます。
:::
