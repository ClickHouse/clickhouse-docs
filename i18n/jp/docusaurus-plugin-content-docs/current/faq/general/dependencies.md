# ClickHouse を実行するためのサードパーティ依存関係は何ですか？

ClickHouse には実行時の依存関係はありません。完全に自己完結した単一のバイナリアプリケーションとして配布されています。このアプリケーションがクラスターのすべての機能を提供し、クエリを処理し、クラスター内のワーカーノードとして動作し、RAFT コンセンサスアルゴリズムを提供する調整システムとして機能し、さらにクライアントまたはローカルクエリエンジンとしても動作します。

このユニークなアーキテクチャの選択によって、専用のフロントエンド、バックエンド、アグリゲーションノードを持つことが多い他のシステムと一線を画しています。これにより、デプロイ、クラスター管理、およびモニタリングが容易になります。

:::info
以前は、ClickHouse は分散クラスターの調整に ZooKeeper を必要としていました。現在は不要であり、ZooKeeper の利用は引き続きサポートしていますが、もはや推奨はしていません。
:::
