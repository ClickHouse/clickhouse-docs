# ClickHouse を実行するためのサードパーティ依存関係は何ですか？

ClickHouse には実行時の依存関係はありません。完全に自己完結した 1 つのバイナリアプリケーションとして配布されています。このアプリケーションがクラスタのすべての機能を提供し、クエリを処理し、クラスタ内のワーカーノードとして動作し、RAFT コンセンサスアルゴリズムを提供する調整システムとして機能し、さらにクライアントやローカルクエリエンジンとしても動作します。

このユニークなアーキテクチャ上の選択により、専用のフロントエンド、バックエンド、あるいは集約ノードを持つことが多い他のシステムとは一線を画しており、デプロイ、クラスタ管理、および監視を容易にします。

:::info
かつては、ClickHouse で分散クラスタを調整するために ZooKeeper が必要でした。現在は不要となっており、ZooKeeper の利用自体はサポートされていますが、もはや推奨されていません。
:::
