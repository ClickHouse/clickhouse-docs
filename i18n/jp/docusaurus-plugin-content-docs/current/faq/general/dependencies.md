# ClickHouse の実行に必要なサードパーティ依存関係は何ですか？

ClickHouse にはランタイムの依存関係がありません。完全に自己完結した単一のバイナリアプリケーションとして配布されています。このアプリケーションがクラスタのすべての機能を提供し、クエリを処理し、クラスタ内のワーカーノードとして動作し、Raft コンセンサスアルゴリズムを提供する調整システムとして、さらにはクライアントまたはローカルクエリエンジンとして機能します。

この独自のアーキテクチャ上の設計判断により、専用のフロントエンド、バックエンド、あるいは集約ノードを持つことが多い他のシステムと一線を画しており、デプロイ、クラスタ管理、およびモニタリングを容易にします。

:::info
かつては、ClickHouse は分散クラスタの調整に ZooKeeper を必要としていました。現在は不要であり、ZooKeeper の利用自体はサポートしていますが、もはや推奨はしていません。
:::
