## 説明 \{#description\}

現在、分片間で移動中のパーツとその進行状況に関する情報が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — move が実行されるデータベース名。
* `table` ([String](../../sql-reference/data-types/)) — move が実行されるテーブル名。
* `task_name` ([String](../../sql-reference/data-types/)) — 移動タスクの名前。
* `task_uuid` ([UUID](../../sql-reference/data-types/)) — 移動タスクの識別子。
* `create_time` ([DateTime](../../sql-reference/data-types/)) — タスクが作成された時刻。
* `part_name` ([String](../../sql-reference/data-types/)) — 移動中のパート名。
* `part_uuid` ([UUID](../../sql-reference/data-types/)) — 移動中のパートの UUID。
* `to_shard` ([String](../../sql-reference/data-types/)) — 宛先分片の名前。
* `dst_part_name` ([String](../../sql-reference/data-types/)) — 移動後のパート名。
* `update_time` ([DateTime](../../sql-reference/data-types/)) — 最後に更新された時刻。
* `state` ([String](../../sql-reference/data-types/)) — move の現在の状態。
* `rollback` ([UInt8](../../sql-reference/data-types/)) — 操作がロールバックされたかどうかを示すフラグ。
* `num_tries` ([UInt32](../../sql-reference/data-types/)) — 操作の完了までに試行した回数。
* `last_exception` ([String](../../sql-reference/data-types/)) — 直近の例外名 (存在する場合) 。

{/*AUTOGENERATED_END*/ }
