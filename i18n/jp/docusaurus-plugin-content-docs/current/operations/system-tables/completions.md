## 説明 \{#description\}

補完トークンの一覧が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `word` ([String](../../sql-reference/data-types/)) — 補完トークン。
* `context` ([String](../../sql-reference/data-types/)) — トークンが表すエンティティの種類 (例: テーブル) 。
* `belongs` ([Nullable(String)](../../sql-reference/data-types/)) — エンティティに対応するトークン。このトークンが属するエンティティです (例: 所有元のデータベース名) 。
