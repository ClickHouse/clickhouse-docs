## 説明 \{#description\}

[ユーザー定義関数 (UDF)](/sql-reference/functions/udf.md) の読み込みステータス、エラー情報、および設定メタデータを含みます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — UDF の名前。
* `load_status` ([Enum8(&#39;Success&#39; = 0, &#39;Failed&#39; = 1)](../../sql-reference/data-types/)) — 読み込みステータス。設定可能な値:
  * **Success** — UDF は正常に読み込まれており、使用可能です
  * **Failed** — UDF の読み込みに失敗しました (詳細は &#39;loading&#95;error&#95;message&#39; フィールドを参照してください) 。
* `loading_error_message` ([String](../../sql-reference/data-types/)) — 読み込みに失敗した場合の詳細なエラーメッセージ。正常に読み込まれた場合は空です。
* `last_successful_update_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 最後に正常に更新された時刻のタイムスタンプ。一度も成功していない場合は NULL です。
* `loading_duration_ms` ([UInt64](../../sql-reference/data-types/)) — UDF の読み込みにかかった時間 (ミリ秒) 。
* `type` ([Enum8(&#39;executable&#39; = 0, &#39;executable&#95;pool&#39; = 1)](../../sql-reference/data-types/)) — UDF のタイプ: &#39;executable&#39; (単一プロセス) または &#39;executable&#95;pool&#39; (プロセスプール) 。
* `command` ([String](../../sql-reference/data-types/)) — この UDF で実行するスクリプトまたはコマンド。
* `format` ([String](../../sql-reference/data-types/)) — I/O に使用するデータフォーマット (例: &#39;TabSeparated&#39;、&#39;JSONEachRow&#39;) 。
* `return_type` ([String](../../sql-reference/data-types/)) — 関数の戻り値の型 (例: &#39;String&#39;、&#39;UInt64&#39;) 。
* `return_name` ([String](../../sql-reference/data-types/)) — 任意の戻り値識別子。設定されていない場合は空です。
* `argument_types` ([Array(String)](../../sql-reference/data-types/)) — 引数の型の配列 (例: [&#39;String&#39;, &#39;UInt64&#39;]) 。
* `argument_names` ([Array(String)](../../sql-reference/data-types/)) — 引数名の配列。名前のない引数には空文字列が入ります。
* `max_command_execution_time` ([UInt64](../../sql-reference/data-types/)) — データ block の処理に許可される最大秒数。&#39;executable&#95;pool&#39; 型でのみ使用されます。
* `command_termination_timeout` ([UInt64](../../sql-reference/data-types/)) — コマンドプロセスに SIGTERM を送信するまでの秒数。
* `command_read_timeout` ([UInt64](../../sql-reference/data-types/)) — コマンドの stdout から読み取る際のタイムアウト (ミリ秒) 。
* `command_write_timeout` ([UInt64](../../sql-reference/data-types/)) — コマンドの stdin に書き込む際のタイムアウト (ミリ秒) 。
* `pool_size` ([UInt64](../../sql-reference/data-types/)) — コマンドプロセスのインスタンス数。&#39;executable&#95;pool&#39; 型でのみ使用されます。
* `send_chunk_header` ([UInt8](../../sql-reference/data-types/)) — 各データ chunk の前に行数を送信するかどうか (ブール値) 。
* `execute_direct` ([UInt8](../../sql-reference/data-types/)) — コマンドを直接実行するか (1) 、/bin/bash 経由で実行するか (0) 。
* `lifetime` ([UInt64](../../sql-reference/data-types/)) — リロード間隔 (秒) 。0 はリロードが無効であることを示します。
* `deterministic` ([UInt8](../../sql-reference/data-types/)) — 関数が同じ引数に対して常に同じ結果を返すかどうか (ブール値) 。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

すべての UDF とその読み込みステータスを表示:

```sql
SELECT
    name,
    load_status,
    type,
    command,
    return_type,
    argument_types
FROM system.user_defined_functions
FORMAT Vertical;
```

```response
Row 1:
──────
name:           my_sum_udf
load_status:    Success
type:           executable
command:        /var/lib/clickhouse/user_scripts/sum.py
return_type:    UInt64
argument_types: ['UInt64','UInt64']
```

失敗した UDF を見つける:

```sql
SELECT
    name,
    loading_error_message
FROM system.user_defined_functions
WHERE load_status = 'Failed';
```
