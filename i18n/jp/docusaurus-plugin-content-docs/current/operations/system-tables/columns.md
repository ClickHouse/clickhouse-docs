すべてのテーブル内のカラムに関する情報を格納しています。

このテーブルを使用すると、複数のテーブルに対してまとめて、[DESCRIBE TABLE](../../sql-reference/statements/describe-table.md) クエリと同様の情報を取得できます。

[一時テーブル](../../sql-reference/statements/create/table.md#temporary-tables) のカラムは、それらが作成されたセッションでのみ `system.columns` に表示されます。これらは空の `database` フィールドで表示されます。
