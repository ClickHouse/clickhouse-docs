デタッチされた各テーブルに関する情報を保持します。

列:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — テーブルが属しているデータベース名。
* `table` ([String](../../sql-reference/data-types/)) — テーブル名。
* `uuid` ([UUID](../../sql-reference/data-types/)) — Atomic データベースにおけるテーブルの UUID。
* `metadata_path` ([String](../../sql-reference/data-types/)) — ファイルシステム内のテーブルメタデータへのパス。
* `is_permanently` ([UInt8](../../sql-reference/data-types/)) — テーブルが完全にデタッチされているかどうか。

{/*AUTOGENERATED_END*/ }

**例**

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
Row 1:
──────
database:                   base
table:                      t1
uuid:                       81b1c20a-b7c6-4116-a2ce-7583fb6b6736
metadata_path:              /var/lib/clickhouse/store/461/461cf698-fd0b-406d-8c01-5d8fd5748a91/t1.sql
is_permanently:             1
```
