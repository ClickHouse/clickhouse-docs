## 説明 \{#description\}

`system.unicode` テーブルは、Unicode 文字およびそのプロパティに関する情報を提供する仮想テーブルです (https://unicode-org.github.io/icu/userguide/strings/properties.html を参照) 。このテーブルは要求時に動的に生成されます。

ICU の Unicode コードポイントのプロパティ名は、カラム名として使用するために `snake_case` に変換されています。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `code_point` ([String](../../sql-reference/data-types/)) — U+XXXX 形式で表される Unicode コードポイント。
* `code_point_value` ([Int32](../../sql-reference/data-types/)) — Unicode コードポイントの整数値。
* `notation` ([String](../../sql-reference/data-types/)) — 文字の表記 (コードポイントの視覚的表現) 。

{/*AUTOGENERATED_END*/ }

SELECT * FROM system.unicode WHERE code_point = 'a' LIMIT 1;
