## 説明 \{#description\}

[S3Queue](../../engines/table-engines/integrations/s3queue.md) テーブルの設定に関する情報を含みます。サーバーバージョン `24.10` 以降で利用可能です。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — S3Queue エンジンを使用するテーブルのデータベース。
* `table` ([String](../../sql-reference/data-types/)) — S3Queue エンジンを使用するテーブル名。
* `name` ([String](../../sql-reference/data-types/)) — 設定名。
* `value` ([String](../../sql-reference/data-types/)) — 設定値。
* `type` ([String](../../sql-reference/data-types/)) — 設定の型 (実装固有の文字列値) 。
* `changed` ([UInt8](../../sql-reference/data-types/)) — 設定が config で明示的に定義されているか、明示的に変更されている場合は 1。
* `description` ([String](../../sql-reference/data-types/)) — 設定の説明。
* `alterable` ([UInt8](../../sql-reference/data-types/)) — 現在のユーザーが ALTER TABLE MODIFY SETTING でこの設定を変更できるかどうかを示します。0 — 変更可能、1 — 変更不可。

{/*AUTOGENERATED_END*/ }
