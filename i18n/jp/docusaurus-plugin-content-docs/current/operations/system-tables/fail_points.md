## 説明 \{#description\}

サーバーに登録されている利用可能なすべてのフェイルポイントを列挙し、それぞれの種類と現在の有効状態を示します。

フェイルポイントは、実行時に `SYSTEM ENABLE FAILPOINT` および `SYSTEM DISABLE FAILPOINT` 文を使用して有効化および無効化できます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — フェイルポイントの名前。
* `type` ([Enum8(&#39;once&#39; = 0, &#39;regular&#39; = 1, &#39;pauseable&#95;once&#39; = 2, &#39;pauseable&#39; = 3)](../../sql-reference/data-types/)) — フェイルポイントの種類。&#39;once&#39; は 1 回だけ発生した後に自動で無効化され、&#39;regular&#39; は毎回発生し、&#39;pauseable&#95;once&#39; は 1 回だけ実行をブロックし、&#39;pauseable&#39; は再開されるまで毎回実行をブロックします。
* `enabled` ([UInt8](../../sql-reference/data-types/)) — フェイルポイントが現在有効 (1) か無効 (0) かを示します。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

```sql title="Query"
SYSTEM ENABLE FAILPOINT replicated_merge_tree_insert_retry_pause;
SELECT * FROM system.fail_points WHERE enabled = 1
```
