## 説明 \{#description\}

ログエントリを含むテーブルです。このテーブルに出力されるログレベルは、`text_log.level` サーバー設定で制限できます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — クエリを実行するサーバーのホスト名。
* `event_date` ([Date](/sql-reference/data-types/date)) — エントリの日付。
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — エントリの時刻。
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — マイクロ秒精度のエントリ時刻。
* `thread_name` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — ログが記録されたスレッドの名前。
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — OS スレッド ID。
* `level` ([Enum8(&#39;Fatal&#39; = 1, &#39;Critical&#39; = 2, &#39;Error&#39; = 3, &#39;Warning&#39; = 4, &#39;Notice&#39; = 5, &#39;Information&#39; = 6, &#39;Debug&#39; = 7, &#39;Trace&#39; = 8, &#39;Test&#39; = 9)](/sql-reference/data-types/enum)) — エントリのレベル。設定可能な値: 1 または &#39;Fatal&#39;、2 または &#39;Critical&#39;、3 または &#39;Error&#39;、4 または &#39;Warning&#39;、5 または &#39;Notice&#39;、6 または &#39;Information&#39;、7 または &#39;Debug&#39;、8 または &#39;Trace&#39;。
* `query_id` ([String](/sql-reference/data-types/string)) — クエリの ID。
* `logger_name` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — ロガーの名前 (例: DDLWorker) 。
* `message` ([String](/sql-reference/data-types/string)) — メッセージ本体。
* `revision` ([UInt32](/sql-reference/data-types/int-uint)) — ClickHouse のリビジョン。
* `source_file` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — ログが記録されたソースファイル。
* `source_line` ([UInt64](/sql-reference/data-types/int-uint)) — ログが記録されたソース行。
* `message_format_string` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — メッセージのフォーマットに使用されたフォーマット文字列。
* `value1` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 1。
* `value2` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 2。
* `value3` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 3。
* `value4` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 4。
* `value5` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 5。
* `value6` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 6。
* `value7` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 7。
* `value8` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 8。
* `value9` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 9。
* `value10` ([String](/sql-reference/data-types/string)) — メッセージのフォーマットに使用された引数 10。

{/*AUTOGENERATED_END*/ }
