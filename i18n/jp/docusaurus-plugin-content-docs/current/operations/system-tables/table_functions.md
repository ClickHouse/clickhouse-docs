## 説明 \{#description\}

利用可能なすべてのテーブル関数とその説明の一覧です。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — テーブル関数の名前。
* `description` ([String](../../sql-reference/data-types/)) — テーブル関数の簡単な説明。
* `allow_readonly` ([UInt8](../../sql-reference/data-types/)) — 読み取り専用ユーザーがこの関数を使用できるかどうかを示すフラグ。
