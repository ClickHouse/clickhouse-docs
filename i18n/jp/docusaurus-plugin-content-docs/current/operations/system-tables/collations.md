## 説明 \{#description\}

文字列をアルファベット順に比較するために使用できる、利用可能なすべての照合順序の一覧が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 照合順序の名称。
* `language` ([Nullable(String)](../../sql-reference/data-types/)) — 言語。

{/*AUTOGENERATED_END*/ }
