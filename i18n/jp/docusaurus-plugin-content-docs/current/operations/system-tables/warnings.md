## 説明 \{#description\}

clickhouse-client がサーバーへの接続直後に表示する、サーバー設定に関する警告が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `message` ([String](../../sql-reference/data-types/)) — ClickHouse サーバーが出力した警告メッセージ。
* `message_format_string` ([LowCardinality(String)](../../sql-reference/data-types/)) — メッセージの整形に使用されたフォーマット文字列。

{/*AUTOGENERATED_END*/ }
