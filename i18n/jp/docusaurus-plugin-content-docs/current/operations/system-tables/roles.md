## 説明 \{#description\}

設定されている[ロール](../../guides/sre/user-management/index.md#role-management)に関する情報を保持します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — ロール名。
* `id` ([UUID](../../sql-reference/data-types/)) — ロール ID。
* `storage` ([String](../../sql-reference/data-types/)) — ロールの保存先へのパス。`access_control_path` パラメータで設定します。

{/*AUTOGENERATED_END*/ }
