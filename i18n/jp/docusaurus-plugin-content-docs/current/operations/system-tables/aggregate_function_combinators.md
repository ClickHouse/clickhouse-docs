## 説明 \{#description\}

利用可能なすべての集約関数コンビネータの一覧です。これらは集約関数に適用でき、動作を変更します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — コンビネータの名前。
* `is_internal` ([UInt8](../../sql-reference/data-types/)) — このコンビネータが内部使用専用であるかどうか。

{/*AUTOGENERATED_END*/ }
