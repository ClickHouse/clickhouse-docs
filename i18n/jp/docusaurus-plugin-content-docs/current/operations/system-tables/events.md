システム内で発生したイベント数に関する情報を含みます。たとえば、このテーブルでは、ClickHouse サーバーの起動以降に処理された `SELECT` クエリの件数を確認できます。

列:

{/*AUTOGENERATED_START*/ }

* `event` ([String](../../sql-reference/data-types/)) — イベント名。
* `value` ([UInt64](../../sql-reference/data-types/)) — 発生したイベントの数。
* `description` ([String](../../sql-reference/data-types/)) — イベントの説明。

{/*AUTOGENERATED_END*/ }
