## 説明 \{#description\}

各 Replicated データベースのレプリカに関する情報およびステータスを保持します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Replicated データベースの名前。
* `is_readonly` ([UInt8](../../sql-reference/data-types/)) — データベースのレプリカが読み取り専用モードかどうか。
* `max_log_ptr` ([Int32](../../sql-reference/data-types/)) — 一般的なアクティビティログ内のエントリ番号の最大値。
* `replica_name` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper 内のレプリカ名。
* `replica_path` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper 内のレプリカデータへのパス。
* `zookeeper_path` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper 内のデータベースデータへのパス。
* `shard_name` ([String](../../sql-reference/data-types/)) — クラスター内の分片名。
* `log_ptr` ([Int32](../../sql-reference/data-types/)) — レプリカが実行キューにコピーした一般的なアクティビティログ内の最大エントリ番号に 1 を加えた値。
* `total_replicas` ([UInt32](../../sql-reference/data-types/)) — このデータベースで認識されているレプリカの総数。
* `zookeeper_exception` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper から情報を取得する際にエラーが発生した場合の、直近の例外メッセージ。
* `is_session_expired` ([UInt8](../../sql-reference/data-types/)) — ClickHouse Keeper とのセッションの有効期限が切れているかどうか。基本的には `is_readonly` と同じです。

{/*AUTOGENERATED_END*/ }

SELECT * FROM system.database_replicas FORMAT Vertical;
