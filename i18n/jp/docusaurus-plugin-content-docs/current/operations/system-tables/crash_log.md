## 説明 \{#description\}

致命的なエラーのスタックトレース情報を保持します。テーブルはデフォルトではデータベース内に存在せず、致命的なエラーが発生したときにのみ作成されます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — クラッシュが発生したホスト名。
* `event_date` ([Date](/sql-reference/data-types/date)) — クラッシュが発生した日付。
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — クラッシュが発生した時刻。
* `timestamp_ns` ([UInt64](/sql-reference/data-types/int-uint)) — ナノ秒精度のイベントタイムスタンプ。
* `signal` ([Int32](/sql-reference/data-types/int-uint)) — シグナル番号。
* `signal_code` ([Int32](/sql-reference/data-types/int-uint)) — シグナルコード。シグナルの原因に関する追加情報を示します。
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — スレッド ID。
* `query_id` ([String](/sql-reference/data-types/string)) — Query ID。
* `query` ([String](/sql-reference/data-types/string)) — クラッシュ発生時に実行されていたクエリのテキスト。
* `trace` ([Array(UInt64)](/sql-reference/data-types/array)) — クラッシュ発生時点のスタックトレース。各要素は ClickHouse サーバープロセス内の仮想メモリアドレスです。
* `trace_full` ([Array(String)](/sql-reference/data-types/array)) — クラッシュ発生時点のスタックトレース。各要素には、ClickHouse サーバープロセス内で呼び出されたメソッドが含まれます。
* `fault_address` ([Nullable(UInt64)](/sql-reference/data-types/nullable)) — 障害の原因となったメモリアドレス。
* `fault_access_type` ([String](/sql-reference/data-types/string)) — 障害の原因となったメモリアクセスの種類 (例: &#39;read&#39;、&#39;write&#39;) 。
* `signal_description` ([String](/sql-reference/data-types/string)) — signal&#95;code に基づく、人が読んでわかる説明 (例: &#39;Address not mapped to object&#39;) 。
* `current_exception_trace_full` ([Array(String)](/sql-reference/data-types/array)) — シンボル化されたフレームを含む現在の例外スタックトレース (`std::terminate` によるクラッシュのみ) 。
* `version` ([String](/sql-reference/data-types/string)) — ClickHouse サーバーのバージョン。
* `revision` ([UInt32](/sql-reference/data-types/int-uint)) — ClickHouse サーバーのリビジョン。
* `build_id` ([String](/sql-reference/data-types/string)) — コンパイラが生成した Build ID。
* `git_hash` ([String](/sql-reference/data-types/string)) — ClickHouse ソースコードの Git コミットハッシュ。
* `architecture` ([String](/sql-reference/data-types/string)) — CPU アーキテクチャ (例: x86&#95;64、aarch64) 。

{/*AUTOGENERATED_END*/ }
