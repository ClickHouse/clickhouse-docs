# system.tokenizers \{#systemtokenizers\}

<SystemTableCloud />

利用可能なトークナイザーの一覧を表示します。
これらは関数 [tokens](../../sql-reference/functions/splitting-merging-functions.md#tokens)、[hasAllTokens](../../sql-reference/functions/string-search-functions.md#hasAllTokens)、[hasAnyTokens](../../sql-reference/functions/string-search-functions.md#hasAnyTokens)、および [text index](../../engines/table-engines/mergetree-family/textindexes.md) で使用できます。

カラム:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — トークナイザーの名前

{/*AUTOGENERATED_END*/ }

**例**

```sql
SELECT * FROM system.tokenizers;
```

```text
┌─name────────────┐
│ ngrams          │
│ splitByNonAlpha │
│ sparseGrams     │
│ tokenbf_v1      │
│ ngrambf_v1      │
│ array           │
│ splitByString   │
│ sparse_grams    │
└─────────────────┘
```
