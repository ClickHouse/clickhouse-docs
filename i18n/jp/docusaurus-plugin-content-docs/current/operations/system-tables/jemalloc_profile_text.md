## 説明 \{#description\}

シンボル化された jemalloc ヒーププロファイルを表示します。先に &#39;SYSTEM JEMALLOC FLUSH PROFILE&#39; を実行してプロファイルを生成してください。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `line` ([String](../../sql-reference/data-types/)) — シンボル化された jemalloc ヒーププロファイル内の行。

{/*AUTOGENERATED_END*/ }
