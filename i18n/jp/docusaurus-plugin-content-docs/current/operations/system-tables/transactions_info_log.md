現在のサーバーで実行されたすべてのトランザクションに関する情報が含まれます。

このテーブルは、いつでも truncate または drop しても問題ありません。

カラム:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — トランザクションが実行されたホスト名。
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/sql-reference/data-types/enum)) — トランザクションの種類。設定可能な値: Begin, Commit, Rollback, AddPart, LockPart, UnlockPart。
* `event_date` ([Date](/sql-reference/data-types/date)) — エントリの日付。
* `event_time` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — エントリの時刻。
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — スレッドの識別子。
* `query_id` ([String](/sql-reference/data-types/string)) — トランザクションのスコープ内で実行されたクエリの ID。
* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — トランザクションの識別子。
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — 識別子のハッシュ値。
* `csn` ([UInt64](/sql-reference/data-types/int-uint)) — Commit Sequence Number。
* `database` ([String](/sql-reference/data-types/string)) — トランザクションの実行対象のデータベース名。
* `table` ([String](/sql-reference/data-types/string)) — トランザクションの実行対象のテーブル名。
* `uuid` ([UUID](/sql-reference/data-types/uuid)) — トランザクションの実行対象のテーブルの UUID。
* `part` ([String](/sql-reference/data-types/string)) — トランザクションに含まれるパートの名前。

{/*AUTOGENERATED_END*/ }
