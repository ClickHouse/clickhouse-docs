# system.users \{#systemusers\}

サーバー上で設定されている[ユーザーアカウント](../../guides/sre/user-management/index.md#user-account-management)の一覧が含まれます。

カラム:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — ユーザー名。
* `id` ([UUID](../../sql-reference/data-types/)) — ユーザー ID。
* `storage` ([String](../../sql-reference/data-types/)) — ユーザーのストレージへのパス。`access_control_path` パラメータで設定されます。
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](../../sql-reference/data-types/)) — 認証タイプを示します。ユーザーを識別する方法として、パスワードなし、平文パスワード、SHA256 でハッシュ化されたパスワード、二重 SHA-1 でハッシュ化されたパスワード、bcrypt でハッシュ化されたパスワードなど、複数の方法があります。
* `auth_params` ([Array(String)](../../sql-reference/data-types/)) — `auth_type` に応じた JSON 形式の認証パラメータ。
* `valid_until` ([Array(DateTime)](../../sql-reference/data-types/)) — ユーザー認証情報の有効期限日時。
* `host_ip` ([Array(String)](../../sql-reference/data-types/)) — ClickHouse サーバーへの接続を許可されているホストの IP アドレス。
* `host_names` ([Array(String)](../../sql-reference/data-types/)) — ClickHouse サーバーへの接続を許可されているホスト名。
* `host_names_regexp` ([Array(String)](../../sql-reference/data-types/)) — ClickHouse サーバーへの接続を許可されているホスト名に対する正規表現パターン。
* `host_names_like` ([Array(String)](../../sql-reference/data-types/)) — LIKE 述語を使用して設定された、ClickHouse サーバーへの接続を許可されているホスト名。
* `default_roles_all` ([UInt8](../../sql-reference/data-types/)) — 付与されたすべてのロールがユーザーのデフォルトロールとして設定されていることを示します。
* `default_roles_list` ([Array(String)](../../sql-reference/data-types/)) — デフォルトで付与されるロールの一覧。
* `default_roles_except` ([Array(String)](../../sql-reference/data-types/)) — 列挙されたものを除き、すべての付与されたロールがデフォルトとして設定されていることを示します。
* `grantees_any` ([UInt8](../../sql-reference/data-types/)) — 任意の GRANT オプションを持つユーザーが、それを任意の相手に付与できるかどうかを示すフラグ。
* `grantees_list` ([Array(String)](../../sql-reference/data-types/)) — このユーザーがオプションを付与することを許可されているユーザーまたはロールの一覧。
* `grantees_except` ([Array(String)](../../sql-reference/data-types/)) — このユーザーがオプションを付与することを禁止されているユーザーまたはロールの一覧。
* `default_database` ([String](../../sql-reference/data-types/)) — このユーザーのデフォルトデータベース名。

{/*AUTOGENERATED_END*/ }
