## 説明 \{#description\}

jemalloc の STATISTICS を、1 行 1 カラムの形式で返します。SYSTEM JEMALLOC STATS 命令と同等です。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `stats` ([String](../../sql-reference/data-types/)) — malloc&#95;stats&#95;print による JEMalloc の STATISTICS 出力。

{/*AUTOGENERATED_END*/ }
