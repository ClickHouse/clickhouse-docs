## 説明 \{#description\}

圧縮および暗号化コーデックに関する情報を保持しています。

このテーブルを使用して、利用可能な圧縮および暗号化コーデックに関する情報を取得できます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — コーデック名。
* `method_byte` ([UInt8](../../sql-reference/data-types/)) — 圧縮ファイル内でコーデックを示すバイト値。
* `is_compression` ([UInt8](../../sql-reference/data-types/)) — このコーデックが圧縮を行う場合は True。それ以外の場合は、圧縮を補助する単なる変換であることがあります。
* `is_generic_compression` ([UInt8](../../sql-reference/data-types/)) — このコーデックが lz4 や zstd のような汎用圧縮アルゴリズムであることを示します。
* `is_encryption` ([UInt8](../../sql-reference/data-types/)) — このコーデックが暗号化を行うことを示します。
* `is_timeseries_codec` ([UInt8](../../sql-reference/data-types/)) — このコーデックが浮動小数点時系列向けのコーデックであることを示します。
* `is_experimental` ([UInt8](../../sql-reference/data-types/)) — このコーデックが実験的機能であることを示します。
* `description` ([String](../../sql-reference/data-types/)) — コーデックの概要説明。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```

```text title="Response"
Row 1:
──────
name:                   LZ4
method_byte:            130
is_compression:         1
is_generic_compression: 1
is_encryption:          0

description:            Extremely fast; good compression; balanced speed and efficiency.
```
