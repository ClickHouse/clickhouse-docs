## 説明 \{#description\}

現在のユーザーが利用可能なデータベースに関する情報を格納します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — データベース名。
* `engine` ([String](../../sql-reference/data-types/)) — データベースエンジン。
* `data_path` ([String](../../sql-reference/data-types/)) — データパス。
* `metadata_path` ([String](../../sql-reference/data-types/)) — メタデータパス。
* `uuid` ([UUID](../../sql-reference/data-types/)) — データベース UUID。
* `engine_full` ([String](../../sql-reference/data-types/)) — データベースエンジンのパラメータ。
* `comment` ([String](../../sql-reference/data-types/)) — データベースのコメント。
* `is_external` ([UInt8](../../sql-reference/data-types/)) — データベースが外部データベースかどうかを示します (例: PostgreSQL、DataLakeCatalog) 。

{/*AUTOGENERATED_END*/ }

**別名:**

* `database` — `name` のエイリアス。

この system テーブルの `name` カラムは、`SHOW DATABASES` クエリの実装に使用されます。

## 例 \{#example\}

データベースを作成する。

```sql
CREATE DATABASE test;
```

ユーザーが利用可能なすべてのデータベースを確認する。

```sql title="Query"
SELECT * FROM system.databases;
```
