## 説明 \{#description\}

S3Queue メタデータのインメモリ状態と、各ファイルで現在処理中の行を保持します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `zookeeper_path` ([String](../../sql-reference/data-types/)) — ZooKeeper 内のメタデータへのパス
* `file_path` ([String](../../sql-reference/data-types/)) — 処理対象のファイルのパス
* `file_name` ([String](../../sql-reference/data-types/)) — 処理対象のファイル名
* `rows_processed` ([UInt64](../../sql-reference/data-types/)) — 現在までに処理された行数
* `status` ([String](../../sql-reference/data-types/)) — 処理ステータス: Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — ファイルの処理が開始された時刻
* `processing_end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — ファイルの処理が終了した時刻
* `exception` ([String](../../sql-reference/data-types/)) — 処理中に発生した例外

{/*AUTOGENERATED_END*/ }
