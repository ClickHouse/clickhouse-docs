## 説明 \{#description\}

ClickHouse サーバーのビルドオプションに関する情報を保持します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — ビルドオプションの名前。
* `value` ([String](../../sql-reference/data-types/)) — ビルドオプションの値。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
