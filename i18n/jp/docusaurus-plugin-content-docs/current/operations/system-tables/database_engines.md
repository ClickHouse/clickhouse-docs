このシステムテーブルには、サーバーがサポートしているデータベースエンジンの一覧が含まれます。

このテーブルには次の列が含まれます（括弧内は列の型を示します）。

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — データベースエンジン名。

{/*AUTOGENERATED_END*/ }

例：

```sql
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text
┌─name─────┐
│ Ordinary │
│ Atomic   │
│ Lazy     │
└──────────┘
```
