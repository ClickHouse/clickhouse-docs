## 説明 \{#description\}

サーバーでサポートされているデータベースエンジンの一覧を保持します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — データベースエンジン名。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

```sql title="Query"
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text title="Response"
┌─name─────┐
│ Ordinary │
│ Atomic   │
│ Lazy     │
└──────────┘
```
