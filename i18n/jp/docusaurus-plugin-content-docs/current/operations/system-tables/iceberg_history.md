# system.iceberg&#95;history \{#systemiceberg&#95;history\}

この system テーブルには、ClickHouse 内に存在する Iceberg テーブルのスナップショット履歴が含まれます。ClickHouse に Iceberg テーブルが 1 つも存在しない場合、このテーブルは空になります。

列:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — データベース名。
* `table` ([String](../../sql-reference/data-types/)) — テーブル名。
* `made_current_at` ([Nullable(DateTime64(3))](../../sql-reference/data-types/)) — このスナップショットが現在のスナップショットになった日時。
* `snapshot_id` ([UInt64](../../sql-reference/data-types/)) — スナップショットを識別するための ID。
* `parent_id` ([UInt64](../../sql-reference/data-types/)) — 親スナップショットの ID。
* `is_current_ancestor` ([UInt8](../../sql-reference/data-types/)) — このスナップショットが現在のスナップショットの祖先であるかどうかを示すフラグ。

{/*AUTOGENERATED_END*/ }
