現在のユーザーのQUOTA使用状況：使用量と残量を示します。

列：

{/*AUTOGENERATED_START*/ }


* `quota_name` ([String](../../sql-reference/data-types/)) — QUOTA 名。
* `quota_key` ([String](../../sql-reference/data-types/)) — キー値。
* `start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — リソース消費量の計算開始時刻。
* `end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — リソース消費量の計算終了時刻。
* `duration` ([Nullable(UInt32)](../../sql-reference/data-types/)) — リソース消費量を計算する時間間隔の長さ (秒) 。
* `queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在実行されたクエリ数。
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 実行可能な全種類のクエリの最大数。
* `query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在実行された SELECT クエリ数。
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 実行可能な SELECT クエリの最大数。
* `query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在実行された INSERT クエリ数。
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 実行可能な INSERT クエリの最大数。
* `errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在エラーになったクエリ数。
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可されるエラー発生クエリの最大数。
* `result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの結果セットの総行数の現在値。
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、すべてのクエリの結果セットの総行数の最大値。
* `result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの結果セットの総バイト数の現在値。
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、すべてのクエリの結果セットの総バイト数の最大値。
* `read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの実行中に読み取られた総行数の現在値。
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、すべてのクエリの実行中に読み取れる行数の最大値。
* `read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの実行中に読み取られた総バイト数の現在値。
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、すべてのクエリの実行中に読み取れるバイト数の最大値。
* `execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 現在の期間内にクエリの実行に費やされた合計時間の現在値 (ナノ秒) 
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 指定期間内にすべてのクエリの実行に許可される合計時間の最大値 (ナノ秒) 
* `written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの実行中に書き込まれた総バイト数の現在値。
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、すべてのクエリの実行中に書き込めるバイト数の最大値。
* `failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、連続した認証失敗回数の現在値。
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、連続した認証失敗回数の最大値。
* `queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、単一の正規化クエリの最大実行回数の現在値。
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定期間内に許可される、単一の正規化クエリの最大実行回数。
