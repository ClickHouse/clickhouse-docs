## 説明 \{#description\}

設定されたユーザーディレクトリに関する情報が含まれます。ユーザーディレクトリとは、ClickHouse サーバーがユーザー提供データの読み取りを許可されているファイルシステム上のディレクトリです。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — ディレクトリ名。
* `type` ([String](../../sql-reference/data-types/)) — アクセスストレージの種類 (例: users.xml、replicated、memory など) 。
* `params` ([String](../../sql-reference/data-types/)) — アクセスストレージのパラメータを含む JSON。
* `precedence` ([UInt64](../../sql-reference/data-types/)) — このディレクトリが設定内で宣言される順序。ClickHouse がユーザーまたはロールを検索するときにも、同じ順序が使用されます。

{/*AUTOGENERATED_END*/ }
