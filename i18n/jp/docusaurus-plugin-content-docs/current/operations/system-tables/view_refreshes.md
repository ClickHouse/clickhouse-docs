# system.view&#95;refreshes

<SystemTableCloud />

[Refreshable Materialized Views](../../sql-reference/statements/create/view.md#refreshable-materialized-view) に関する情報。リフレッシュ処理の進行中かどうかに関係なく、すべてのリフレッシュ可能なマテリアライズドビューを含みます。

列:

{/*AUTOGENERATED_START*/ }

{/*AUTOGENERATED_END*/ }

**例**

```sql
SELECT
    database,
    view,
    status,
    last_refresh_result,
    last_refresh_time,
    next_refresh_time
FROM system.view_refreshes

┌─database─┬─view───────────────────────┬─status────┬─last_refresh_result─┬───last_refresh_time─┬───next_refresh_time─┐
│ default  │ hello_documentation_reader │ Scheduled │ Finished            │ 2023-12-01 01:24:00 │ 2023-12-01 01:25:00 │
└──────────┴────────────────────────────┴───────────┴─────────────────────┴─────────────────────┴─────────────────────┘
```
