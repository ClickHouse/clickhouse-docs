## 説明 \{#description\}

現在有効なすべてのロールを含むsystem テーブルであり、現在のユーザーの現在のロールおよびそのロールに付与されているロールを含みます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — ロール名。
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — ロールが `ADMIN OPTION` 権限を持つ場合は 1。
* `is_current` ([UInt8](../../sql-reference/data-types/)) — `enabled_role` が現在のユーザーの現在のロールかどうかを示すフラグ。
* `is_default` ([UInt8](../../sql-reference/data-types/)) — `enabled_role` がデフォルトのロールかどうかを示すフラグ。

{/*AUTOGENERATED_END*/ }
