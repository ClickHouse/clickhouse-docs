## 説明 \{#description\}

トランザクションとその状態の一覧です。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — トランザクションの識別子。
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — 識別子のハッシュ。
* `elapsed` ([Float64](/sql-reference/data-types/float)) — 処理中のトランザクションの経過時間。
* `is_readonly` ([UInt8](/sql-reference/data-types/int-uint)) — トランザクションで書き込み操作が一切実行されていないかどうかを示すフラグ。
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/sql-reference/data-types/enum)) — トランザクションの状態。設定可能な値: RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.

{/*AUTOGENERATED_END*/ }
