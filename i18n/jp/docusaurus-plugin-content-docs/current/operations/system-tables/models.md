## 説明 \{#description\}

LibraryBridge のメモリに読み込まれている CatBoost モデルの一覧と、それぞれの読み込み時刻が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `model_path` ([String](../../sql-reference/data-types/)) — 学習済みモデルへのパス。
* `type` ([String](../../sql-reference/data-types/)) — モデルの種類。現在は catboost のみ。
* `loading_start_time` ([DateTime](../../sql-reference/data-types/)) — モデルの読み込みが開始された時刻。
* `loading_duration` ([Float32](../../sql-reference/data-types/)) — モデルの読み込みに要した時間。

{/*AUTOGENERATED_END*/ }
