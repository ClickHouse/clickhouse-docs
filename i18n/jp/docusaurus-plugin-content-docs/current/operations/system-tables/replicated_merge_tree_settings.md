## 説明 \{#description\}

ReplicatedMergeTree エンジン固有のすべての設定について、現在値、デフォルト値、および説明の一覧を示します。これらはすべて、CREATE クエリの SETTINGS セクションで変更できます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 設定名。
* `value` ([String](../../sql-reference/data-types/)) — 設定値。
* `default` ([String](../../sql-reference/data-types/)) — 設定のデフォルト値。
* `changed` ([UInt8](../../sql-reference/data-types/)) — 設定が設定ファイルで明示的に定義されているか、明示的に変更されている場合は 1。
* `description` ([String](../../sql-reference/data-types/)) — 設定の説明。
* `min` ([Nullable(String)](../../sql-reference/data-types/)) — 制約によって設定されている場合の最小値。設定に最小値がない場合は NULL になります。
* `max` ([Nullable(String)](../../sql-reference/data-types/)) — 制約によって設定されている場合の最大値。設定に最大値がない場合は NULL になります。
* `disallowed_values` ([Array(String)](../../sql-reference/data-types/)) — 許可されていない値の一覧
* `readonly` ([UInt8](../../sql-reference/data-types/)) — 現在のユーザーがこの設定を変更できるかどうかを示します: 0 — 現在のユーザーはこの設定を変更できます、1 — 現在のユーザーはこの設定を変更できません。
* `type` ([String](../../sql-reference/data-types/)) — 設定の型 (実装固有の文字列値) 。
* `is_obsolete` ([UInt8](../../sql-reference/data-types/)) — 設定が廃止済みかどうかを示します。
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](../../sql-reference/data-types/)) —
  この機能のサポートレベルです。ClickHouse の機能は、現在の
  開発状況や使用時に期待できる内容に応じて、ティアごとに分類されています:

- PRODUCTION: この機能は安定しており、安全に使用できます。また、他の PRODUCTION 機能と組み合わせても問題はありません。
- BETA: この機能は安定しており、安全です。他の機能と組み合わせた場合の結果は不明で、正しさは保証されません。テストや報告を歓迎します。
- EXPERIMENTAL: この機能は開発中です。開発者および ClickHouse 愛好家向けです。機能は動作する場合もしない場合もあり、いつでも削除される可能性があります。
- OBSOLETE: すでにサポートされていません。すでに削除されているか、今後のリリースで削除される予定です。

{/*AUTOGENERATED_END*/ }
