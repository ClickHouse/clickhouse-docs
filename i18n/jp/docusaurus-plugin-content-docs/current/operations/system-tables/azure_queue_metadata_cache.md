## 説明 \{#description\}

AzureQueue メタデータのインメモリ状態と、ファイルごとに現在処理中の行が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `zookeeper_path` ([String](../../sql-reference/data-types/)) — メタデータへの zookeeper 内のパス
* `file_path` ([String](../../sql-reference/data-types/)) — 処理中のファイルのパス
* `file_name` ([String](../../sql-reference/data-types/)) — 処理中のファイル名
* `rows_processed` ([UInt64](../../sql-reference/data-types/)) — 現在処理中の行数
* `status` ([String](../../sql-reference/data-types/)) — 処理のステータス: `Processed`、`Processing`、`Failed`
* `processing_start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — ファイルの処理が開始された時刻
* `processing_end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — ファイルの処理が終了した時刻
* `exception` ([String](../../sql-reference/data-types/)) — 処理中に発生した例外

{/*AUTOGENERATED_END*/ }
