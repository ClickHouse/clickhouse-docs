## 説明 \{#description\}

SQLクエリで作成された、または設定ファイルから解析された、すべての 名前付きコレクション の一覧が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — コレクション名。
* `collection` ([Map(String, String)](../../sql-reference/data-types/)) — コレクションの内部データ。
* `source` ([String](../../sql-reference/data-types/)) — 名前付きコレクションのソース。
* `create_query` ([String](../../sql-reference/data-types/)) — 名前付きコレクションの CREATE クエリ。

{/*AUTOGENERATED_END*/ }
