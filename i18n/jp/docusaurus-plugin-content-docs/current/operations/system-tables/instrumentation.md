## 説明 \{#description\}

LLVM の XRay 機能を使用したインストルメンテーションポイントを含むシステムテーブルです。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `id` ([UInt32](../../sql-reference/data-types/)) — インストルメンテーションポイントの ID
* `function_id` ([Int32](../../sql-reference/data-types/)) — ELF バイナリの xray&#95;instr&#95;map セクションで関数に割り当てられた ID。
* `function_name` ([LowCardinality(String)](../../sql-reference/data-types/)) — 関数のインストルメントに使用される名前。
* `handler` ([LowCardinality(String)](../../sql-reference/data-types/)) — 関数のインストルメンテーションポイントにパッチ適用された handler。
* `entry_type` ([Enum8(&#39;Entry&#39; = 0, &#39;Exit&#39; = 1, &#39;EntryAndExit&#39; = 2)](../../sql-reference/data-types/)) — パッチのエントリ種別。
* `symbol` ([LowCardinality(String)](../../sql-reference/data-types/)) — 完全なデマングル済みシンボル名。
* `parameters` ([Array(Dynamic)](../../sql-reference/data-types/)) — handler 呼び出しのパラメータ。

{/*AUTOGENERATED_END*/ }
