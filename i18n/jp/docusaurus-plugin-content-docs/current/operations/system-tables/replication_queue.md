## 説明 \{#description\}

ClickHouse Keeper または ZooKeeper に保存されている、`ReplicatedMergeTree` ファミリーのテーブル用のレプリケーションキュー内のタスクに関する情報が含まれます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — データベース名。
* `table` ([String](../../sql-reference/data-types/)) — テーブル名。
* `replica_name` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper 内のレプリカ名。同じテーブルのレプリカにはそれぞれ異なる名前が付けられます。
* `position` ([UInt32](../../sql-reference/data-types/)) — キュー内でのタスクの位置。
* `node_name` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper 内のノード名。
* `type` ([String](../../sql-reference/data-types/)) — キュー内のタスクの種類。次のいずれかです: • GET&#95;PART — 別のレプリカからパーツを取得します。 • ATTACH&#95;PART — パーツをアタッチします (detached フォルダ内で見つかった場合は、自身のレプリカから取得することもあります) 。両者はほぼ同一のため、いくつかの最適化が加えられた GET&#95;PART と考えることができます。 • MERGE&#95;PARTS — パーツをマージします。 • DROP&#95;RANGE — 指定した番号範囲の、指定したパーティション内のパーツを削除します。 • CLEAR&#95;COLUMN — 注意: 非推奨。指定したパーティションから特定のカラムを削除します。 • CLEAR&#95;INDEX — 注意: 非推奨。指定したパーティションから特定の索引を削除します。 • REPLACE&#95;RANGE — 特定範囲のパーツを削除し、新しいパーツで置き換えます。 • MUTATE&#95;PART — パーツに 1 つ以上の mutation を適用します。 • ALTER&#95;METADATA — グローバルな /metadata および /columns パスに従って alter の変更を適用します。
* `create_time` ([DateTime](../../sql-reference/data-types/)) — タスクが実行のために登録された日時。
* `required_quorum` ([UInt32](../../sql-reference/data-types/)) — 完了確認付きでタスクの完了を待機しているレプリカ数。このカラムは GET&#95;PARTS タスクにのみ関係します。
* `source_replica` ([String](../../sql-reference/data-types/)) — ソースレプリカ名。
* `new_part_name` ([String](../../sql-reference/data-types/)) — 新しいパーツ名。
* `parts_to_merge` ([Array(String)](../../sql-reference/data-types/)) — マージまたは更新するパーツの名前。
* `is_detach` ([UInt8](../../sql-reference/data-types/)) — DETACH&#95;PARTS タスクがキューに入っているかどうかを示すフラグ。
* `is_currently_executing` ([UInt8](../../sql-reference/data-types/)) — 特定のタスクが現在実行中かどうかを示すフラグ。
* `num_tries` ([UInt32](../../sql-reference/data-types/)) — タスクの完了に失敗した試行回数。
* `last_exception` ([String](../../sql-reference/data-types/)) — 最後に発生したエラーに関するテキストメッセージ (ある場合) 。
* `last_exception_time` ([DateTime](../../sql-reference/data-types/)) — 最後にエラーが発生した日時。
* `last_attempt_time` ([DateTime](../../sql-reference/data-types/)) — タスクが最後に試行された日時。
* `num_postponed` ([UInt32](../../sql-reference/data-types/)) — アクションが延期された回数。
* `postpone_reason` ([String](../../sql-reference/data-types/)) — タスクが延期された理由。
* `last_postpone_time` ([DateTime](../../sql-reference/data-types/)) — タスクが最後に延期された日時。
* `merge_type` ([String](../../sql-reference/data-types/)) — 現在のマージの種類。mutation の場合は空です。

{/*AUTOGENERATED_END*/ }
