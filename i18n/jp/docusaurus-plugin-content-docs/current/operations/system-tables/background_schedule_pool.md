# system.background_schedule_pool \{#systembackground_schedule_pool\}

<SystemTableCloud />

## 説明 \{#description\}

バックグラウンドスケジュールプール内のタスクに関する情報を含みます。バックグラウンドスケジュールプールは、分散送信、バッファフラッシュ、メッセージブローカー処理などの定期的なタスクの実行に使用されます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `pool` ([String](../../sql-reference/data-types/)) — プール名。設定可能な値：`schedule` — 汎用スケジュール用プール、`buffer_flush` — Buffer テーブルデータをフラッシュするためのプール、`distributed` — 分散テーブル操作用プール、`message_broker` — メッセージブローカー操作用プール。
* `database` ([String](../../sql-reference/data-types/)) — データベース名。
* `table` ([String](../../sql-reference/data-types/)) — テーブル名。
* `table_uuid` ([UUID](../../sql-reference/data-types/)) — テーブル UUID。
* `query_id` ([String](../../sql-reference/data-types/)) — クエリ ID (現在実行中の場合) 。注意：これは実際のクエリではなく、`system.text_log` 内のログを照合するためにランダムに生成された ID です。
* `elapsed_ms` ([UInt64](../../sql-reference/data-types/)) — タスクの実行時間 (現在実行中の場合) 。
* `log_name` ([String](../../sql-reference/data-types/)) — タスクのログ名。
* `deactivated` ([UInt8](../../sql-reference/data-types/)) — タスクが無効化されているかどうか (無効化されたタスクはプールから削除されるため、常に false) 。
* `scheduled` ([UInt8](../../sql-reference/data-types/)) — タスクが実行対象としてスケジュールされているかどうか。
* `delayed` ([UInt8](../../sql-reference/data-types/)) — タスクが遅延付きでスケジュールされているかどうか。
* `executing` ([UInt8](../../sql-reference/data-types/)) — タスクが現在実行中かどうか。

{/*AUTOGENERATED_END*/ }
