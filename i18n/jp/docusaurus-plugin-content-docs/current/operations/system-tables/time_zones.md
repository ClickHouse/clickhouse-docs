# system.time&#95;zones \{#systemtime&#95;zones\}

ClickHouse サーバーがサポートしているタイムゾーンの一覧を保持します。このタイムゾーン一覧は、ClickHouse のバージョンによって異なる場合があります。

列:

{/*AUTOGENERATED_START*/ }

* `time_zone` ([String](../../sql-reference/data-types/)) — サポートされているタイムゾーンの一覧。

{/*AUTOGENERATED_END*/ }

**例**

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
