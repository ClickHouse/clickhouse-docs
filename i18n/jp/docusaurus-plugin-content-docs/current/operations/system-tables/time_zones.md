## 説明 \{#description\}

ClickHouse サーバーがサポートしているタイムゾーンの一覧を保持します。この一覧は ClickHouse のバージョンによって異なる場合があります。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `time_zone` ([String](../../sql-reference/data-types/)) — 対応しているタイムゾーンの一覧。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │

│ Africa/Addis_Ababa │
│ Africa/Algiers     │

│ Africa/Bissau      │
└────────────────────┘
```
