## 説明 \{#description\}

[`GraphiteMergeTree`](../../engines/table-engines/mergetree-family/graphitemergetree.md) エンジンを使用するテーブルで利用されるパラメータ [graphite&#95;rollup](../../operations/server-configuration-parameters/settings.md#graphite) に関する情報が含まれています。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `config_name` ([String](../../sql-reference/data-types/)) — graphite&#95;rollup パラメーター名。
* `rule_type` ([String](../../sql-reference/data-types/)) — ルールの種類。設定可能な値: RuleTypeAll = 0 - デフォルト。正規表現を使用し、古いスキームと互換性があります。RuleTypePlain = 1 - プレーンなメトリクス。正規表現を使用し、古いスキームと互換性があります。RuleTypeTagged = 2 - タグ付きメトリクス。正規表現を使用し、古いスキームと互換性があります。RuleTypeTagList = 3 - タグ付きメトリクス。正規表現を使用します (&#39;retention=10min ; env=(staging|prod)&#39; のような文字列から RuleTypeTagged に変換)
* `regexp` ([String](../../sql-reference/data-types/)) — メトリクス名のパターン。
* `function` ([String](../../sql-reference/data-types/)) — 集約関数の名前。
* `age` ([UInt64](../../sql-reference/data-types/)) — データの最小経過時間 (秒) 。
* `precision` ([UInt64](../../sql-reference/data-types/)) — データの経過時間を秒単位でどの程度正確に定義するか。
* `priority` ([UInt16](../../sql-reference/data-types/)) — パターンの優先度。
* `is_default` ([UInt8](../../sql-reference/data-types/)) — そのパターンがデフォルトかどうか。
* `Tables.database` ([Array(String)](../../sql-reference/data-types/)) — `config_name` パラメーターを使用するデータベーステーブル名の配列。
* `Tables.table` ([Array(String)](../../sql-reference/data-types/)) — `config_name` パラメーターを使用するテーブル名の配列。

{/*AUTOGENERATED_END*/ }
