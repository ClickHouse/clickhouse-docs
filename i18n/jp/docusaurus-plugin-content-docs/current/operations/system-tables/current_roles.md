## 説明 \{#description\}

現在のユーザーに対してアクティブなロールを含みます。`SET ROLE` によってこのテーブルの内容が変更されます。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — ロール名。
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — ロールに ADMIN OPTION 特権がある場合は 1。
* `is_default` ([UInt8](../../sql-reference/data-types/)) — ロールがデフォルトとして設定されている場合は 1。

{/*AUTOGENERATED_END*/ }
