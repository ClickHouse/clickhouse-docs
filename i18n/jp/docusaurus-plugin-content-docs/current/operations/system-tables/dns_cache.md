## 説明 \{#description\}

キャッシュされた DNS レコードに関する情報を保持します。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([String](../../sql-reference/data-types/)) — ホスト名。
* `ip_address` ([String](../../sql-reference/data-types/)) — IPアドレス。
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](../../sql-reference/data-types/)) — IPアドレスファミリー。
* `cached_at` ([DateTime](../../sql-reference/data-types/)) — レコードがキャッシュされた時点のタイムスタンプ。

{/*AUTOGENERATED_END*/ }

## 例 \{#example\}

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
