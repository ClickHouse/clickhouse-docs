# system.resources {#systemresources}

ローカルサーバー上に存在する[リソース](/operations/workload-scheduling.md#workload_entity_storage)に関する情報が含まれます。テーブルには、各リソースにつき 1 行が格納されます。

例:

```sql
SELECT *
FROM system.resources
FORMAT Vertical
```

```text
行 1:
──────
name:         io_read
read_disks:   ['s3']
write_disks:  []
create_query: CREATE RESOURCE io_read (READ DISK s3)

行 2:
──────
name:         io_write
read_disks:   []
write_disks:  ['s3']
create_query: CREATE RESOURCE io_write (WRITE DISK s3)
```

列：

{/*AUTOGENERATED_START*/ }

{/*AUTOGENERATED_END*/ }
