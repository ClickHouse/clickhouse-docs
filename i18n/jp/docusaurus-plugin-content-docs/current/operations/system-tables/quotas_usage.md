# system.quotas_usage \{#systemquotas_usage\}

<SystemTableCloud />

すべてのユーザーのクォータ使用状況。

列:

* `quota_name` ([String](../../sql-reference/data-types/)) — QUOTA 名。
* `quota_key` ([String](../../sql-reference/data-types/)) — キーの値。
* `is_current` ([UInt8](../../sql-reference/data-types/)) — 現在のユーザーのQUOTA使用状況。
* `start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — リソース消費量を計算する際の開始時刻。
* `end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — リソース消費量の計算対象となる終了時刻。
* `duration` ([Nullable(UInt32)](../../sql-reference/data-types/)) — リソース消費量の計算対象となる時間間隔の長さ (秒) 。
* `queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在までに実行されたクエリ数。
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 実行が許可される、あらゆる種類のクエリの最大数。
* `query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の実行済みSELECTクエリ数。
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 実行できる `SELECT` クエリの最大数です。
* `query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の実行済み INSERT クエリ数。
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 実行できる INSERT クエリの最大数。
* `errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在、エラーとなったクエリの数。
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定した期間内にエラーになってよいクエリの最大数。
* `result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの結果セット内の現在の総行数。
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定された期間内に許可される、すべてのクエリの結果セットに含まれる総行数の最大値。
* `result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、すべてのクエリの結果セットの合計バイト数。
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定した期間内に許可される、すべてのクエリの結果セットの合計バイト数の上限です。
* `read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内に、すべてのクエリの実行中に読み取られた行の現在の総数。
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定した期間内に許可されるすべてのクエリの実行中に読み取ることができる最大行数。
* `read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内に実行されたすべてのクエリについて、実行中に読み取られたバイト数の現在の合計。
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定した期間内に許可されたすべてのクエリの実行中に読み取れる最大バイト数。
* `execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 現在の期間内にクエリの実行に費やされた合計時間 (ナノ秒)
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 指定された期間内に、すべてのクエリの実行に許可される最大時間 (ナノ秒)
* `written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内に実行されたすべてのクエリにおいて、実行中に書き込まれたバイト数の現在の合計。
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定した期間内に許可される、すべてのクエリの実行中に書き込める最大バイト数。
* `failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、連続した認証失敗の現在の回数。
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定された期間内に許容される連続した認証失敗の最大回数。
* `queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 現在の期間内における、単一の正規化クエリの最大実行回数。
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 指定した期間内に許可される、1つの正規化クエリの最大実行回数。
