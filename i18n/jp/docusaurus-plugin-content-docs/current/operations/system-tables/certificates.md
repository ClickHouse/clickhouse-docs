## 説明 \{#description\}

利用可能な証明書とその取得元に関する情報。

## カラム \{#columns\}

{/*AUTOGENERATED_START*/ }

* `version` ([Int32](../../sql-reference/data-types/)) — 証明書のバージョン。値は v1 が 0、v2 が 1、v3 が 2 です。
* `serial_number` ([Nullable(String)](../../sql-reference/data-types/)) — 発行者がこの証明書に割り当てたシリアル番号。
* `signature_algo` ([Nullable(String)](../../sql-reference/data-types/)) — 署名アルゴリズム。発行者がこの証明書への署名に使用するアルゴリズムです。
* `issuer` ([Nullable(String)](../../sql-reference/data-types/)) — 発行者。この証明書を発行した認証局を一意に識別する識別子です。
* `not_before` ([Nullable(String)](../../sql-reference/data-types/)) — この証明書が有効となる期間の開始時刻。
* `not_after` ([Nullable(String)](../../sql-reference/data-types/)) — この証明書が有効となる期間の終了時刻。
* `subject` ([Nullable(String)](../../sql-reference/data-types/)) — サブジェクト。公開鍵の所有者を識別します。
* `pkey_algo` ([Nullable(String)](../../sql-reference/data-types/)) — 公開鍵アルゴリズム。公開鍵に使用できるアルゴリズムを定義します。
* `path` ([String](../../sql-reference/data-types/)) — この証明書を含むファイルまたはディレクトリのパス。
* `default` ([UInt8](../../sql-reference/data-types/)) — 証明書がデフォルトの証明書格納場所にあります。
* `protocol` ([String](../../sql-reference/data-types/)) — 証明書がプロトコルごとの TLS 設定に由来する場合はプロトコル名、それ以外の場合は空文字列です。

{/*AUTOGENERATED_END*/ }
