このデータセットには、2011年から2020年12月6日までのすべてのGitHubイベントが含まれており、レコード数は31億件です。ダウンロードサイズは75 GBで、`lz4`圧縮を使用したテーブルに保存する場合、最大で200 GBのディスク容量が必要になります。

データセットの詳細な説明、インサイト、ダウンロード手順、およびインタラクティブなクエリは[こちら](https://ghe.clickhouse.tech/)に掲載されています。
