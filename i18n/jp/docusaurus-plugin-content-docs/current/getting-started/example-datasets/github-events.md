このデータセットには、2011年から2020年12月6日までのGitHub上の全イベントが含まれており、31億レコードで構成されています。ダウンロードサイズは75 GBで、lz4圧縮を使用してテーブルに格納する場合、ディスク上に最大200 GBの容量が必要です。

データセットの詳細な説明、分析結果、ダウンロード手順、およびインタラクティブクエリについては[こちら](https://ghe.clickhouse.tech/)をご覧ください。
