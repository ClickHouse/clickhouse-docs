このデータセットには 2011 年から 2020 年 12 月 6 日までの GitHub 上のすべてのイベントが含まれており、レコード数は 31 億件です。ダウンロードサイズは 75 GB で、LZ4 圧縮を使用したテーブルに保存する場合、最大 200 GB のディスク容量が必要です。

データセットの詳細な説明、インサイト、ダウンロード手順、およびインタラクティブなクエリは[こちら](https://ghe.clickhouse.tech/)に掲載されています。
