このデータセットには、2011年から2020年12月6日までのGitHub上のすべてのイベントが含まれており、レコード数は31億件です。ダウンロードサイズは75 GBで、`lz4`圧縮を使用したテーブルに保存する場合、ディスク上で最大200 GBの空き容量が必要です。

データセットの詳細な説明、インサイト、ダウンロード手順、および対話的に実行できるクエリは[こちら](https://ghe.clickhouse.tech/)に掲載されています。
