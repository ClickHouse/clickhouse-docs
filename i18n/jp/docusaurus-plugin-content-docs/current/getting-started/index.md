# チュートリアルとサンプルデータセット

ClickHouseの使い方を学び、始めるための豊富なリソースを用意しています：

- ClickHouseをセットアップして実行する場合は、[クイックスタート](/get-started/quick-start)をご確認ください
- [ClickHouseチュートリアル](../tutorial.md)では、ニューヨーク市のタクシー乗車データセットを分析します

さらに、サンプルデータセットを使用することで、ClickHouseの操作方法、重要なテクニックやコツの習得、そしてClickHouseの多数の強力な関数を活用する方法を実践的に学ぶことができます。サンプルデータセットには以下が含まれます：
