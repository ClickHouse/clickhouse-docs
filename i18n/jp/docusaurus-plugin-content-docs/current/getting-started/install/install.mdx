自分で管理したい場合は、以下のオープンソース版 ClickHouse のオプションから 1 つ選択してください。
ローカルですばやくセットアップするには ClickHouse CLI を使用し、本番環境へのデプロイには
パッケージまたはバイナリから ClickHouse Server をインストールしてください。

    ローカルのバージョンやサーバーをインストールおよび管理します。
