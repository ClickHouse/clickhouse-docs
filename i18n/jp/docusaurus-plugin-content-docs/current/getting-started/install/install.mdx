<br />

自分で管理したい場合は、以下のオープンソース版 ClickHouse のいずれかを選択してください。
ローカル環境のセットアップにはクイックインストールスクリプトまたは `clickhousectl` CLI を使用し、
本番環境へのデプロイにはパッケージまたはバイナリから ClickHouse Server をインストールしてください。

<h3 className="install-group-heading">ローカル開発</h3>

<p className="install-group-description">
  自分のマシンで ClickHouse を実行して、構築、テスト、検証を行えます。
  クイックインストールを使えば単一バイナリをすばやく入手でき、clickhousectl
  を使えばローカルのバージョンやサーバーをインストールして管理できます。
</p>
