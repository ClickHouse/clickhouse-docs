# 上級ガイド

このセクションには、次の上級ガイドが含まれます：

| Guide                                                                                                                  | Description                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alternative Query Languages](../developer/alternative-query-languages)                                         | サポートされている代替クエリ方言とその使い方についてのガイドです。各方言でのクエリ例を示します。                                                                                                                                                                                                                                             |
| [Cascading Materialized Views](../developer/cascading-materialized-views)                                       | マテリアライズドビューを作成し、それらを連鎖させて複数のソーステーブルを 1 つの宛先テーブルにまとめる方法についてのガイドです。連鎖するマテリアライズドビューを使用して、ドメイン名のグループに対して月別・年別にデータを集計する例を含みます。                                                                                                 |
| [Debugging memory issues](../developer/debugging-memory-issues)                                                 | ClickHouse 内で発生するメモリ関連の問題をデバッグする方法についてのガイドです。                                                                                                                                                                                                                                                               |
| [Deduplicating Inserts on Retries](../developer/deduplicating-inserts-on-retries)                               | 失敗した挿入をリトライする場合にどのように対処するかについてのガイドです。                                                                                                                                                                                                                                                                      |
| [Deduplication strategies](../developer/deduplication)                                                          | データ重複排除を詳しく解説するガイドであり、これはデータベースから重複行を削除する手法です。OLTP システムにおける主キーに基づく重複排除との違い、ClickHouse における重複排除のアプローチ、および ClickHouse のクエリ内で重複データのシナリオをどのように扱うかを説明します。                                                                  |
| [Filling gaps in time-series data](../developer/time-series-filling-gaps)                                       | 時系列データを扱うための ClickHouse の機能について解説するガイドであり、より完全で連続的な時系列情報を得るために、データの欠損部分を補完する手法を取り上げます。                                                                                                                                                                                   |
| [Manage Data with TTL (Time-to-live)](../developer/ttl)                                                         | TTL を用いたデータ管理方法について解説するガイドです。データの有効期限の設定方法、TTL による自動削除や移動の仕組み、およびさまざまなユースケースにおけるベストプラクティスを扱います。                                                                                                                                                          |
| [Stored procedures & query parameters](../developer/stored-procedures-and-prepared-statements)                  | ClickHouse が従来型のストアドプロシージャをサポートしないことを説明し、その代替として推奨される User-Defined Functions (UDF)、パラメータ化ビュー、マテリアライズドビュー、および外部オーケストレーションについて解説するガイドです。また、プリペアドステートメントに類似した、安全なパラメータ化クエリのためのクエリパラメータについても説明します。 |
| [Understanding query execution with the Analyzer](../developer/understanding-query-execution-with-the-analyzer) | analyzer ツールを紹介し、ClickHouse のクエリ実行を分かりやすく説明するガイドです。analyzer がクエリを一連のステップに分解する方法を解説し、クエリ全体の実行プロセスを可視化・トラブルシュートして性能を最適化できるようにします。                                                                                                           |
| [Using JOINs in ClickHouse](../joining-tables)                                                                  | ClickHouse におけるテーブルの結合を分かりやすく説明するガイドです。さまざまな結合タイプ (`INNER`, `LEFT`, `RIGHT` など) を扱い、小さいテーブルを右側に配置するなど効率的な結合のベストプラクティスを解説し、複雑なデータ関係に対するクエリを最適化するために役立つ ClickHouse の内部結合アルゴリズムについての知見を提供します。                       |
