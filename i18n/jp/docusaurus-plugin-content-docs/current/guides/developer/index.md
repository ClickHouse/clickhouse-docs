# 発展的なガイド

このセクションには次の発展的なガイドが含まれます：

| Guide                                                                                                                  | Description                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alternative Query Languages](../developer/alternative-query-languages)                                         | サポートされている別のクエリ方言とその使用方法についてのガイドです。各方言でのクエリ例を示します。                                                                                                                                                                                                                                                |
| [Cascading Materialized Views](../developer/cascading-materialized-views)                                       | マテリアライズドビューの作成方法と、それらを連鎖させて複数のソーステーブルを 1 つの宛先テーブルにまとめる方法についてのガイドです。ドメイン名のグループに対して、月別・年別にデータを集計するためにカスケード型マテリアライズドビューを使用する例も含まれます。                                                                                         |
| [Debugging memory issues](../developer/debugging-memory-issues)                                                 | ClickHouse 内で発生するメモリ関連の問題をデバッグする方法についてのガイドです。                                                                                                                                                                                                                                                                    |
| [Deduplicating Inserts on Retries](../developer/deduplicating-inserts-on-retries)                               | 失敗した挿入を再試行する可能性がある状況をどのように扱うかについてのガイドです。                                                                                                                                                                                                                                                                     |
| [Deduplication strategies](../developer/deduplication)                                                          | データ重複排除について掘り下げるガイドであり、データベースから重複行を削除する手法を扱います。OLTP システムにおけるプライマリキーに基づく重複排除との違い、ClickHouse における重複排除のアプローチ、および ClickHouse のクエリ内で重複データのシナリオを処理する方法を説明します。                                                         |
| [Filling gaps in time-series data](../developer/time-series-filling-gaps)                                       | 時系列データを扱うための ClickHouse の機能についての洞察を提供するガイドであり、より完全で連続的な時系列情報の表現を得るためにデータの欠損部分を埋める手法を含みます。                                                                                                                                                                                  |
| [Manage Data with TTL (Time-to-live)](../developer/ttl)                                                         | TTL (Time-to-live) を使ってデータを管理する方法について説明するガイドです。TTL 設定によりデータの保持期間を制御し、不要になったデータを自動的に削除する仕組みや、さまざまな TTL の適用パターンとベストプラクティスを取り上げます。                                                                                                                  |
| [Stored procedures & query parameters](../developer/stored-procedures-and-prepared-statements)                  | ClickHouse は従来型のストアドプロシージャをサポートしていないことを説明し、その代替として推奨される User-Defined Functions (UDF)、パラメータ化ビュー、マテリアライズドビュー、外部オーケストレーションなどを紹介するガイドです。また、プリペアドステートメントに類似した安全なパラメータ化クエリのためのクエリパラメータについても解説します。         |
| [Understanding query execution with the Analyzer](../developer/understanding-query-execution-with-the-analyzer) | Analyzer ツールを紹介することで ClickHouse のクエリ実行を分かりやすく説明するガイドです。Analyzer がクエリを一連のステップに分解し、クエリ全体の実行プロセスを可視化・トラブルシュートして最適なパフォーマンスを実現する方法を説明します。                                                                                                 |
| [Using JOINs in ClickHouse](../joining-tables)                                                                  | ClickHouse におけるテーブルの JOIN を分かりやすく説明するガイドです。さまざまな JOIN の種類 (`INNER`、`LEFT`、`RIGHT` など) を扱い、小さいテーブルを右側に配置するなど効率的な JOIN のベストプラクティスを解説します。また、ClickHouse の内部的な JOIN アルゴリズムについての洞察を提供し、複雑なデータ関係に対するクエリ最適化に役立てます。 |
