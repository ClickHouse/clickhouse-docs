# ClickHouse SQLリファレンス

ClickHouseは、ANSI SQL標準と多くのケースで同一の宣言型クエリ言語をサポートしています。

サポートされているクエリには、GROUP BY、ORDER BY、FROM内のサブクエリ、JOIN句、IN演算子、ウィンドウ関数、スカラーサブクエリが含まれます。
