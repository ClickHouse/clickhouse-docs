値のグループの最小値を計算する集約関数。

例:

```sql
SELECT min(salary) FROM employees;
```

```sql
SELECT department, min(salary) FROM employees GROUP BY department;
```

非集約関数として 2 つの値のうち小さい方を選択する必要がある場合は、`least` を参照してください。

```sql
SELECT least(a, b) FROM table;
```
