値のグループに対して最大値を計算する集約関数。

例:

```sql
SELECT max(salary) FROM employees;
```

```sql
SELECT department, max(salary) FROM employees GROUP BY department;
```

非集約関数で 2 つの値のうち大きい方を選択したい場合は、`greatest` を使用してください。

```sql
SELECT greatest(a, b) FROM table;
```
