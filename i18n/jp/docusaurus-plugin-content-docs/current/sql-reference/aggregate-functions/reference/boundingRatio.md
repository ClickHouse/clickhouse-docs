## boundingRatio \{#boundingRatio\}

導入バージョン: v20.1

値のグループにおいて、左端の点と右端の点の間の傾きを計算します。

**構文**

```sql
boundingRatio(x, y)
```

**引数**

* `x` — X 座標の値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)
* `y` — Y 座標の値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)

**戻り値**

最左端の点と最右端の点を結ぶ直線の傾きを返します。データが空の場合は `NaN` を返します。[`Float64`](/sql-reference/data-types/float)

**例**

**サンプルデータ**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
┌─number─┬─multiply(number, 1.5)─┐
│      0 │                     0 │
│      1 │                   1.5 │
│      2 │                     3 │
│      3 │                   4.5 │
│      4 │                     6 │
│      5 │                   7.5 │
│      6 │                     9 │
│      7 │                  10.5 │
│      8 │                    12 │
│      9 │                  13.5 │
└────────┴───────────────────────┘
```

**使用例**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```

```response title=Response
┌─boundingRatio(number, multiply(number, 1.5))─┐
│                                          1.5 │
└──────────────────────────────────────────────┘
```
