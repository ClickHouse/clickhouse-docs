## groupUniqArray \{#groupUniqArray\}

導入されたバージョン: v1.1

異なる引数の値から配列を作成します。
この関数のメモリ消費量は、[`uniqExact`](/sql-reference/aggregate-functions/reference/uniqexact) 関数と同じです。

**構文**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**パラメータ**

* `max_size` — 結果配列のサイズを `max_size` 要素に制限します。`groupUniqArray(1)(x)` は `[any(x)]` と同等です。[`UInt64`](/sql-reference/data-types/int-uint)

**引数**

* `x` — 式。[`Any`](/sql-reference/data-types)

**返される値**

一意な値のみを含む配列を返します。[`Array`](/sql-reference/data-types/array)

**例**

**使用例**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**max&#95;size パラメーター指定時**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
