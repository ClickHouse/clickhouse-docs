## minMappedArrays \{#minMappedArrays\}

導入されたバージョン: v20.5

`key` 配列で指定されたキーに従って、`value` 配列から最小値を計算します。

:::note

* `key` および `value` の配列からなるタプルを渡すことは、キーの配列と値の配列を渡すことと同じです。
* 合計される各行において、`key` と `value` の要素数は同じでなければなりません。
  :::

**構文**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**引数**

* `key` — キーの配列。[`Array(T)`](/sql-reference/data-types/array)
* `value` — 値の配列。[`Array(T)`](/sql-reference/data-types/array)

**戻り値**

キーをソートした配列と、それぞれのキーに対応して計算された値の配列という 2 つの配列からなるタプルを返します。[`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**例**

**使用例**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
