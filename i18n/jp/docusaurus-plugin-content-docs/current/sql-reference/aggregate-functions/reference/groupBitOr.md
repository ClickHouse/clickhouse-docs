## groupBitOr \{#groupBitOr\}

導入バージョン: v1.1

一連の数値に対してビットごとの OR 演算を行います。

**構文**

```sql
groupBitOr(expr)
```

**別名**: `BIT_OR`

**引数**

* `expr` — `(U)Int*` 型の式。[`(U)Int*`](/sql-reference/data-types/int-uint)

**返り値**

`(U)Int*` 型の値を返します。[`(U)Int*`](/sql-reference/data-types/int-uint)

**例**

**ビット単位 OR 演算の例**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
