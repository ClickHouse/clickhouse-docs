## maxMappedArrays \{#maxMappedArrays\}

導入バージョン: v20.5

`key` 配列で指定されたキーに従って、`value` 配列から最大値を計算します。

:::note

* キー配列と値配列からなるタプルを渡すことは、キーの配列と値の配列をそれぞれ渡すことと同じです。
* 集計対象となる各行において、`key` と `value` の要素数は同じでなければなりません。
  :::

**構文**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**引数**

* `key` — キーの配列。[`Array(T)`](/sql-reference/data-types/array)
* `value` — 値の配列。[`Array(T)`](/sql-reference/data-types/array)

**返される値**

2 つの配列からなるタプルを返します。ソート順のキーと、それに対応するキーに対して計算された値です。[`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**例**

**使用例**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)────────────────┐
│ [['x', 'y', 'z'], [2, 3, 1]].        │
└──────────────────────────────────────┘
```
