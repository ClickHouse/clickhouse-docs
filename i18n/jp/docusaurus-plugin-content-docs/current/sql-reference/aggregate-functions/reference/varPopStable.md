## varPopStable {#varpopstable}

母分散を返します。[`varPop`](../reference/varPop.md) と異なり、この関数は[数値的に安定な](https://en.wikipedia.org/wiki/Numerical_stability)アルゴリズムを使用します。動作は遅くなりますが、計算誤差をより小さく抑えられます。

**構文**

```sql
varPopStable(x)
```

エイリアス: `VAR_POP_STABLE`.

**パラメータ**

* `x`: 母分散を求める値の集合。[(U)Int*](../../data-types/int-uint.md)、[Float*](../../data-types/float.md)、[Decimal*](../../data-types/decimal.md)。

**戻り値**

* `x` の母分散を返します。[Float64](../../data-types/float.md)。

**例**

クエリ:

```sql
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    varPopStable(x) AS var_pop_stable
FROM test_data;
```

結果：

```response
┌─var_pop_stable─┐
│           14.4 │
└────────────────┘
```
