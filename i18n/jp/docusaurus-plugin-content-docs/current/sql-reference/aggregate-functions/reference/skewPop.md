## skewPop \{#skewPop\}

導入バージョン: v20.1

シーケンスの[歪度](https://en.wikipedia.org/wiki/Skewness)を計算します。

**構文**

```sql
skewPop(expr)
```

**引数**

* `expr` — 数値を返す式。[`Expression`](/sql-reference/data-types/special-data-types/expression)

**返り値**

指定された分布の歪度を返します。[`Float64`](/sql-reference/data-types/float)

**使用例**

**対称分布**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**右に歪んだ分布**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
