## groupArrayIntersect \{#groupArrayIntersect\}

導入バージョン: v24.2

指定された配列の共通部分を返します（すべての指定された配列に共通して含まれる要素を返します）。

**構文**

```sql
groupArrayIntersect(x)
```

**引数**

* `x` — 引数（カラム名または式）。 [`Any`](/sql-reference/data-types)

**返される値**

すべての配列に共通して含まれる要素のみを含む配列を返します。 [`Array`](/sql-reference/data-types/array)

**例**

**使用例**

```sql title=Query
-- Create table with Memory engine
CREATE TABLE numbers (
    a Array(Int32)
) ENGINE = Memory;

-- Insert sample data
INSERT INTO numbers VALUES
    ([1,2,4]),
    ([1,5,2,8,-1,0]),
    ([1,5,7,5,8,2]);

SELECT groupArrayIntersect(a) AS intersection FROM numbers;
```

```response title=Response
┌─intersection──────┐
│ [1, 2]            │
└───────────────────┘
```
