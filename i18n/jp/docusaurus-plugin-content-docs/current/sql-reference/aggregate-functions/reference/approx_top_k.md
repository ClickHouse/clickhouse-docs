# approx_top_k \{#approx_top_k\}

指定されたカラム内で、近似的に最も頻出している値とその出現回数を含む配列を返します。結果の配列は、値そのものではなく、値の近似的な頻度の降順でソートされます。

```sql
approx_top_k(N)(column)
approx_top_k(N, reserved)(column)
```

この関数は決定的な結果を保証するものではありません。特定の状況ではエラーが発生する可能性があり、最頻値ではない値を頻出値として返す場合があります。

`N` の最大値は `65536` です。

**パラメータ**

* `N` — 返す要素数。省略可能。デフォルト値: 10。
* `reserved` — 値用に予約するセル数を指定します。uniq(column) &gt; reserved の場合、topK 関数の結果は近似的なものになります。省略可能。デフォルト値: N * 3。

**引数**

* `column` — 出現頻度を計算する対象の値。

**例**

クエリ:

```sql
SELECT approx_top_k(2)(k)
FROM VALUES('k Char, w UInt64', ('y', 1), ('y', 1), ('x', 5), ('y', 1), ('z', 10));
```

結果：

```text
┌─approx_top_k(2)(k)────┐
│ [('y',3,0),('x',1,0)] │
└───────────────────────┘
```
