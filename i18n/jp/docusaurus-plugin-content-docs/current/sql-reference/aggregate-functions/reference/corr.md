# corr {#corr}

Calculates the [Pearson correlation coefficient](https://en.wikipedia.org/wiki/Pearson_correlation_coefficient):

$$
\frac{\Sigma{(x - \bar{x})(y - \bar{y})}}{\sqrt{\Sigma{(x - \bar{x})^2} * \Sigma{(y - \bar{y})^2}}}
$$

<br />
:::note This function uses a numerically unstable algorithm. If you need
[numerical stability](https://en.wikipedia.org/wiki/Numerical_stability) in
calculations, use the [`corrStable`](../reference/corrStable.md) function. It is
slower but provides a more accurate result. :::

**Syntax**

```sql
corr(x, y)
```

**引数**

- `x` — 1 番目の変数。[(U)Int\*](../../data-types/int-uint.md)、[Float\*](../../data-types/float.md)。
- `y` — 2 番目の変数。[(U)Int\*](../../data-types/int-uint.md)、[Float\*](../../data-types/float.md)。

**戻り値**

- ピアソンの相関係数。[Float64](../../data-types/float.md)。

**例**

クエリ:

```sql
DROP TABLE IF EXISTS series;
CREATE TABLE series
(
    i UInt32,
    x_value Float64,
    y_value Float64
)
ENGINE = Memory;
INSERT INTO series(i, x_value, y_value) VALUES (1, 5.6, -4.4),(2, -9.6, 3),(3, -1.3, -4),(4, 5.3, 9.7),(5, 4.4, 0.037),(6, -8.6, -7.8),(7, 5.1, 9.3),(8, 7.9, -3.6),(9, -8.2, 0.62),(10, -3, 7.3);
```

```sql
SELECT corr(x_value, y_value)
FROM series;
```

Result:

```response
┌─corr(x_value, y_value)─┐
│     0.1730265755453256 │
└────────────────────────┘
```
