## maxIntersectionsPosition \{#maxIntersectionsPosition\}

導入バージョン: v1.1

[`maxIntersections`](/sql-reference/aggregate-functions/reference/maxintersections) 関数の出現位置を計算するための集約関数です。

**構文**

```sql
maxIntersectionsPosition(start_column, end_column)
```

**引数**

* `start_column` — 各区間の開始位置を表す数値カラム。`start_column` が `NULL` または 0 の場合、その区間はスキップされます。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float)
* `end_column` — 各区間の終了位置を表す数値カラム。`end_column` が `NULL` または 0 の場合、その区間はスキップされます。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float)

**返される値**

最大数の区間が交差する開始位置を返します。[`Any`](/sql-reference/data-types)

**例**

**最大交差位置の算出**

```sql title=Query
CREATE TABLE my_events (
    start UInt32,
    end UInt32
)
ENGINE = MergeTree
ORDER BY tuple();

INSERT INTO my_events VALUES
(1, 3),
(1, 6),
(2, 5),
(3, 7);

SELECT maxIntersectionsPosition(start, end) FROM my_events;
```

```response title=Response
┌─maxIntersectionsPosition(start, end)─┐
│                                    2 │
└──────────────────────────────────────┘
```
