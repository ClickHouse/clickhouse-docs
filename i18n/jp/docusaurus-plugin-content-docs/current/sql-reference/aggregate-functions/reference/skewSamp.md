## skewSamp \{#skewSamp\}

導入バージョン: v20.1

シーケンスの[標本歪度 (sample skewness)](https://en.wikipedia.org/wiki/Skewness) を計算します。

渡された値がある確率変数の標本を構成している場合、その歪度の不偏推定量となります。

**構文**

```sql
skewSamp(expr)
```

**引数**

* `expr` — 数値を返す式。[`Expression`](/sql-reference/data-types/special-data-types/expression)

**戻り値**

この関数は、指定された分布の歪度を返します。`n <= 1`（`n` は標本サイズ）の場合は、`nan` を返します。[`Float64`](/sql-reference/data-types/float)

**例**

**対称分布**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**右に歪んだ分布**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
