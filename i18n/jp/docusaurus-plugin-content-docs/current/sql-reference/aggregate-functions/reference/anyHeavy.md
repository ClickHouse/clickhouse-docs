## anyHeavy \{#anyHeavy\}

導入バージョン: v1.1

[heavy hitters](https://doi.org/10.1145/762471.762473) アルゴリズムを使用して、高頻度に出現する値を選択します。
クエリの各実行スレッドにおいて、ケースの過半数で出現する値がある場合、その値が返されます。
通常、結果は非決定的です。

**構文**

```sql
anyHeavy(column)
```

**引数**

* `column` — カラム名。[`String`](/sql-reference/data-types/string)

**返される値**

高頻度に出現する値を返します。結果は非決定的です。[`Any`](/sql-reference/data-types)

**例**

**使用例**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
