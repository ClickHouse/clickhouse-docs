## avg \{#avg\}

導入バージョン: v1.1

算術平均値を計算します。

**構文**

```sql
avg(x)
```

**引数**

* `x` — 入力値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)

**戻り値**

算術平均を返します。入力パラメータ `x` が空の場合は `NaN` を返します。 [`Float64`](/sql-reference/data-types/float)

**使用例**

**基本的な使用方法**

```sql title=Query
SELECT avg(x) FROM VALUES('x Int8', 0, 1, 2, 3, 4, 5);
```

```response title=Response
┌─avg(x)─┐
│    2.5 │
└────────┘
```

**空のテーブルでは NaN が返される**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;

SELECT avg(t) FROM test;
```

```response title=Response
┌─avg(x)─┐
│    nan │
└────────┘
```
