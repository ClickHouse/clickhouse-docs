## exponentialTimeDecayedMax \{#exponentialTimeDecayedMax\}

導入バージョン: v21.12

時刻 `t` と `t-1` における計算済み指数平滑移動平均のうち、大きい方の値を返します。

**構文**

```sql
exponentialTimeDecayedMax(x)(value, timeunit)
```

**パラメータ**

* `x` — 半減期。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)

**引数**

* `value` — 値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)
* `timeunit` — 時間単位。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal) または [`DateTime`](/sql-reference/data-types/datetime) または [`DateTime64`](/sql-reference/data-types/datetime64)

**返り値**

`t` と `t-1` における指数平滑化された加重移動平均の最大値を返します。[`Float64`](/sql-reference/data-types/float)

**例**

**視覚的表現付きのウィンドウ関数の使用例**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 5, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedMax(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
