## groupBitXor \{#groupBitXor\}

導入バージョン: v1.1

一連の数値に対してビット単位の XOR を適用します。

**構文**

```sql
groupBitXor(expr)
```

**エイリアス**: `BIT_XOR`

**引数**

* `expr` — `(U)Int*` 型の式。[`(U)Int*`](/sql-reference/data-types/int-uint)

**返される値**

`(U)Int*` 型の値を返します。[`(U)Int*`](/sql-reference/data-types/int-uint)

**例**

**ビット単位 XOR の例**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitXor(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01101000 = 104

┌─groupBitXor(num)─┐
│              104 │
└──────────────────┘
```
