## avgWeighted \{#avgWeighted\}

導入バージョン: v20.1

[加重算術平均](https://en.wikipedia.org/wiki/Weighted_arithmetic_mean)を計算します。

**構文**

```sql
avgWeighted(x, weight)
```

**引数**

* `x` — 値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float)
* `weight` — 値の重み。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float)

**戻り値**

すべての重みが 0 である場合、または指定された重みパラメータが空の場合は `NaN` を返し、それ以外の場合は重み付き平均を返します。[`Float64`](/sql-reference/data-types/float)

**例**

**使用例**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**整数値と浮動小数点数の重みの混在**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Float64', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**すべての重みが 0 の場合は NaN を返す**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (0, 0), (1, 0), (10, 0))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│               nan │
└───────────────────┘
```

**空のテーブルでは NaN が返されます**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;
SELECT avgWeighted(t, t) FROM test
```

```response title=Response
┌─avgWeighted(t, t)─┐
│               nan │
└───────────────────┘
```
