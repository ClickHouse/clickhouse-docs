## stddevSamp \{#stddevSamp\}

導入バージョン: v1.1

数値データ系列の標本標準偏差を返します。
結果は [`varSamp`](/sql-reference/aggregate-functions/reference/varSamp) の平方根に等しくなります。

:::note
この関数では数値的に不安定なアルゴリズムを使用します。
計算で[数値安定性](https://en.wikipedia.org/wiki/Numerical_stability)が必要な場合は、[`stddevSampStable`](/sql-reference/aggregate-functions/reference/stddevsampstable) 関数を使用してください。動作は遅くなりますが、計算誤差を小さく抑えられます。
:::

**構文**

```sql
stddevSamp(x)
```

**別名**: `STDDEV_SAMP`

**引数**

* `x` — 標本分散の平方根を求める対象の値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal*`](/sql-reference/data-types/decimal)

**戻り値**

`x` の標本分散の平方根を返します。[`Float64`](/sql-reference/data-types/float)

**例**

**標本標準偏差の計算**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSamp(population)
FROM test_data;
```

```response title=Response
┌─stddevSamp(population)─┐
│                      4 │
└────────────────────────┘
```
