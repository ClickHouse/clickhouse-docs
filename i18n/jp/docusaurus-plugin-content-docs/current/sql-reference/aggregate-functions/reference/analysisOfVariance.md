## analysisOfVariance \{#analysisOfVariance\}

導入バージョン: v22.10

一元配置分散分析（ANOVA 検定）のための統計的検定を提供します。これは、正規分布に従う複数のグループに属する観測値に対して、すべてのグループの平均が同じかどうかを判定する検定です。

:::note
グループには 0 から番号が振られ、検定を実行するには少なくとも 2 つのグループが必要です。
2 つ以上の観測値を含むグループが少なくとも 1 つ必要です。
:::

**構文**

```sql
analysisOfVariance(val, group_no)
```

**別名**: `anova`

**引数**

* `val` — 値。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)
* `group_no` — `val` が属するグループ番号。[`(U)Int*`](/sql-reference/data-types/int-uint) または [`Float*`](/sql-reference/data-types/float) または [`Decimal`](/sql-reference/data-types/decimal)

**戻り値**

F統計量と p値を要素とするタプルを返します。[`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple)

**使用例**

**基本的な使用方法**

```sql title=Query
SELECT analysisOfVariance(number, number % 2) FROM numbers(1048575);
```

```response title=Response
┌─analysisOfVariance(number, modulo(number, 2))─┐
│ (0,1)                                         │
└───────────────────────────────────────────────┘
```
