# 論理関数 {#logical-functions}

以下の関数は、任意の数値型の引数に対して論理演算を実行します。
戻り値は [`UInt8`](../data-types/int-uint.md) 型の `0` または `1`、もしくは場合によっては `NULL` になります。

引数の値がゼロの場合は `false`、ゼロ以外の値は `true` と見なされます。
