# 暗号化関数

これらの関数は、AES（Advanced Encryption Standard）アルゴリズムを使用してデータの暗号化および復号を行います。

鍵長は暗号化モードによって異なり、`-128-`、`-196-`、`-256-` の各モードに対して、それぞれ `16`、`24`、`32` バイトです。

初期化ベクトルの長さは常に 16 バイトであり、16 バイトを超える部分は無視されます。
