ClickHouse Cloud は、**プライマリデータストア**としても、**分析レイヤー**としても適したサービスです。

ClickHouse のカラムナ型アーキテクチャ、ベクトル化処理、およびクラウドネイティブな設計により、
スピードとスケールの両方を要求する分析ワークロードに対して、比類のない適合性を発揮します。
ClickHouse Cloud の代表的なユースケースは次のとおりです。

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud は、カラムナストレージアーキテクチャとベクトル化実行エンジンにより、数十億行規模のデータに対してサブセカンドの応答時間でクエリ結果を返すことで、リアルタイム分析において優れた性能を発揮します。このプラットフォームは、1 秒あたり数百万件に達する高スループットなデータ取り込みを処理できるうえ、事前集計を必要とせずに生データへ直接クエリを実行できます。マテリアライズドビューによりリアルタイム集計や事前計算された結果を提供でき、分位点や件数を求める近似関数によって、インタラクティブなダッシュボードやリアルタイムな意思決定に最適な即時インサイトを実現します。                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud は、テラバイト規模のログ、メトリクス、トレースを容易に取り込み・クエリできる、時系列データ向けに最適化された専用エンジンと関数を備えており、オブザーバビリティワークロードに非常に適しています。ClickHouse の包括的なオブザーバビリティソリューションである ClickStack を通じて、組織はログ、メトリクス、トレースという従来の 3 つのサイロを解消し、すべてのオブザーバビリティデータを単一のプラットフォームに統合することで、相関分析を可能にし、個別システムを管理する複雑さを排除できます。この統合アプローチは、アプリケーションパフォーマンス監視、インフラ監視、エンタープライズ規模でのセキュリティイベント分析に理想的であり、ClickStack がデータサイロのない完全なオブザーバビリティワークフローに必要なツールと統合機能を提供します。 |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | ClickHouse のデータウェアハウスエコシステムとの接続性により、ユーザーは数回のクリックでセットアップを完了し、データを容易に ClickHouse に取り込むことができます。履歴データ分析、データレイク、クエリのフェデレーション、およびネイティブなデータ型としての JSON を強力にサポートしているため、ユーザーは大規模なデータをコスト効率良く保存できます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud は、探索や前処理から、学習、テスト、推論に至るまで、ML バリューチェーン全体で利用できます。ClickHouse-local、ClickHouse-server、chdb などのツールはデータ探索、発見、変換に利用でき、ClickHouse 自体はフィーチャーストア、ベクターストア、あるいは MLOps 向けオブザーバビリティストアとして利用可能です。さらに、フルマネージドなリモート MCP サーバー、クエリ向けインラインテキスト補完、AI を活用したチャート設定、プロダクト内の Ask AI などの組み込みツールにより、エージェント型アナリティクスを実現します。                                                                                                                                                                                                                                                         |
