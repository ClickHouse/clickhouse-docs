ClickHouse Cloud は、**プライマリデータストア** と **分析レイヤー** の両方として利用できます。

ClickHouse のカラム型アーキテクチャ、ベクトル化処理、クラウドネイティブな設計により、
高いスピードとスケーラビリティを同時に求められる分析ワークロードに特に適しています。
一般的に、ClickHouse Cloud の主なユースケースは次のとおりです。

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud は、カラム型ストレージアーキテクチャとベクトル化実行エンジンにより、数十億行に対してサブ秒レイテンシのクエリ応答を実現し、リアルタイム分析に優れています。数百万イベント/秒クラスの高スループットなデータ取り込みを処理しつつ、事前集計を必要とせずに生データへ直接クエリを実行できます。Materialized Views によりリアルタイム集計や事前計算された結果を提供でき、分位点や件数向けの近似関数と組み合わせることで、インタラクティブなダッシュボードやリアルタイムな意思決定に最適な即時インサイトを提供します。                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud はオブザーバビリティ系のワークロードに適しており、ログ、メトリクス、トレースといったタイムシリーズデータをテラバイト規模で容易に取り込み・クエリできる、専用エンジンと関数を備えています。ClickHouse の包括的なオブザーバビリティソリューションである ClickStack を通じて、ログ、メトリクス、トレースという従来の 3 つのサイロを単一プラットフォーム上で統合し、オブザーバビリティデータの相関分析を可能にしつつ、個別システムを運用する複雑さを排除できます。この統合アプローチにより、エンタープライズ規模でのアプリケーションパフォーマンス監視、インフラ監視、セキュリティイベント分析に最適であり、ClickStack がデータサイロのない完全なオブザーバビリティワークフローに必要なツールと連携機能を提供します。 |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | ClickHouse のデータウェアハウス向けエコシステムとの高い接続性により、ユーザーは少ないクリック数でセットアップを完了し、データを容易に ClickHouse に取り込むことができます。履歴データ分析、データレイク、クエリフェデレーション、ネイティブデータ型としての JSON を優れた形でサポートしており、大規模でもコスト効率よくデータを保存できます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud は、探索や前処理から、トレーニング、テスト、推論に至るまで、ML バリューチェーン全体で利用できます。ClickHouse-local、ClickHouse-server、chDB などのツールは、データ探索、発見、変換に利用でき、ClickHouse 自体は Feature Store、ベクトルストア、MLOps 向けオブザーバビリティストアとして利用可能です。さらに、フルマネージドなリモート MCP サーバー、クエリ向けインラインテキスト補完、AI によるチャート設定、製品内の Ask AI などの組み込みツールにより、エージェント型アナリティクスも実現できます。                                                                                                                                                                                                                                                         |
