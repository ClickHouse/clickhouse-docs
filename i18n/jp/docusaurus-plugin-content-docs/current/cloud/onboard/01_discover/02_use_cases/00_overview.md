ClickHouse Cloud は、**プライマリデータストア**としても**分析レイヤー**としても利用できます。

ClickHouse のカラムナ型アーキテクチャ、ベクトル化処理、クラウドネイティブな設計により、
高速性とスケーラビリティの両方が求められる分析ワークロードに特に適しています。
大まかに言うと、ClickHouse Cloud の代表的なユースケースは次のとおりです。

| Use case                                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Real-Time analytics](/cloud/get-started/cloud/use-cases/real-time-analytics)            | ClickHouse Cloud は、カラムナ型ストレージアーキテクチャとベクトル化実行エンジンにより、数十億行に対するサブセカンドのクエリ応答を実現し、リアルタイム分析において高い性能を発揮します。毎秒数百万イベントという高スループットなデータ取り込みを処理しつつ、事前集約を行わずに生データへ直接クエリを実行できます。`Materialized Views` によってリアルタイム集約や事前計算済みの結果を提供し、分位点やカウント向けの近似関数により、インタラクティブなダッシュボードやリアルタイムな意思決定に最適な即時インサイトをもたらします。                                                                                                                                                                                                                              |
| [Observability](/cloud/get-started/cloud/use-cases/observability)                        | ClickHouse Cloud はオブザーバビリティワークロードに非常に適しており、ログ・メトリクス・トレースといったタイムシリーズデータをテラバイト規模で容易に取り込み・クエリできるよう最適化された専用エンジンと関数を備えています。ClickHouse の包括的なオブザーバビリティソリューションである ClickStack により、ログ・メトリクス・トレースという従来の 3 つのサイロを単一プラットフォーム上で統合し、相関分析を可能にするとともに、個別システムを管理する複雑さを解消します。この統合アプローチにより、エンタープライズ規模でのアプリケーション性能監視、インフラ監視、セキュリティイベント分析に最適な基盤を提供し、ClickStack がデータサイロのない完全なオブザーバビリティワークフローを実現するためのツールや連携機能を提供します。 |
| [Data warehousing](/cloud/get-started/cloud/use-cases/data_lake_and_warehouse)           | ClickHouse のデータウェアハウスエコシステムとの接続性により、ユーザーは数回のクリックでセットアップを完了し、簡単にデータを ClickHouse に取り込むことができます。履歴データ分析、データレイク、クエリフェデレーション、ネイティブな `JSON` データ型に対する優れたサポートにより、ユーザーはデータをコスト効率よく大規模に保存できます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Machine Learning and Artificial Intelligence](/cloud/get-started/cloud/use-cases/AI_ML) | ClickHouse Cloud は、探索や前処理から、学習、テスト、推論に至るまで、ML バリューチェーン全体で利用できます。`Clickhouse-local`、`Clickhouse-server`、`chdb` といったツールは、データの探索、検証、変換に利用でき、ClickHouse は `feature store`、`vector store`、あるいは MLOps 向けオブザーバビリティストアとして活用できます。さらに、完全マネージドなリモート MCP サーバー、クエリ向けインラインテキスト補完、AI を活用したチャート設定、製品内の Ask AI などの組み込みツールにより、エージェント型アナリティクスも実現できます。                                                                                                                                                                                                                                                         |
