## ClickHouse Cloud とは何ですか？ {#what-is-clickhouse-cloud}

ClickHouse Cloud は、世界最速かつ最も人気のあるオープンソースのカラムナ型オンライン分析処理データベースである ClickHouse の開発元によって提供される、フルマネージドのクラウドサービスです。

Cloud では、インフラストラクチャ、メンテナンス、スケーリング、運用がすべて管理されるため、組織や顧客への価値提供という本質的な業務に、より迅速に集中して取り組むことができます。



## ClickHouse Cloud の利点 {#benefits-of-clickhouse-cloud}

ClickHouse Cloud には、オープンソース版と比較していくつかの大きな利点があります。

- **価値創出までの速さ**: クラスターのサイジングやスケーリングを気にせず、すぐに構築を開始できます。
- **シームレスなスケーリング**: 自動スケーリングによりワークロードの変動に応じて調整されるため、ピーク時を見越した過剰なプロビジョニングが不要です。
- **サーバーレス運用**: サイジング、スケーリング、セキュリティ、信頼性、アップグレードはすべて当社側で対応するため、運用負荷を大幅に軽減できます。
- **透明性の高い料金体系**: 実際に使用した分だけ支払えばよく、リソース予約やスケーリング制御も利用できます。
- **総保有コスト (TCO)**: 優れた価格/性能比を実現し、運用管理コストも低く抑えられます。
- **幅広いエコシステム**: お使いのデータコネクタ、可視化ツール、SQL および各種言語クライアントをそのまま利用できます。
