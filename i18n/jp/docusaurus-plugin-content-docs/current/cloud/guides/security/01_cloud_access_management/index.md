# クラウドでのアクセス管理 \{#cloud-access-management\}

このセクションでは、ClickHouse Cloud におけるアクセス管理に関する詳細なガイドを提供します。

| ページ                                                   | 説明                                           |
|--------------------------------------------------------|-------------------------------------------------------|
| [Manage my account](/cloud/security/manage-my-account) | パスワード、多要素認証（MFA）、アカウント復旧など、自身のユーザーアカウントの管理方法について説明します |

| [Manage database users](/cloud/security/manage-database-users) | データベースユーザーの管理方法を説明する管理者向けガイドです |
| [SAML SSO setup](/cloud/security/saml-setup) | SAML 連携の設定およびトラブルシューティングに関する管理者向けガイドです |
| [Common access management queries](/cloud/security/common-access-management-queries) | データベース権限の設定および検証手順を示す詳細な例を紹介します |
