:::note
MySQL および MongoDB の CDC ClickPipes については、General Availability (GA) になるまでは無料でご利用いただけます。GA リリース前にお客様へ事前に通知いたしますので、その時点で ClickPipes の利用状況を確認し、最適化してください。
:::

[ClickPipes](/integrations/clickpipes) の課金は、**コンピュート使用量** と **取り込みデータ量** に基づきます。各カテゴリの料金モデルの詳細については、専用の課金ページをご覧ください。
