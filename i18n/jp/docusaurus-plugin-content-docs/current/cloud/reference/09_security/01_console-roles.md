## 組織ロール {#organization-roles}

組織ロールの割り当て手順については、[クラウドユーザーの管理](/cloud/security/manage-cloud-users)を参照してください。

ClickHouseには、ユーザー管理のための4つの組織レベルロールがあります。管理者ロールのみがデフォルトでサービスへのアクセス権を持ちます。その他のすべてのロールは、サービスと連携するためにサービスレベルロールと組み合わせる必要があります。

| ロール      | 説明                                                                                                                                                                                                                 |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Admin     | 組織のすべての管理操作を実行し、すべての設定を制御します。このロールはデフォルトで組織の最初のユーザーに割り当てられ、すべてのサービスに対するサービス管理者権限を自動的に持ちます。 |
| Developer | 組織への閲覧アクセスと、同等以下の権限を持つAPIキーを生成する機能。                                                                                                                        |
| Billing   | 使用状況と請求書の閲覧、および支払い方法の管理。                                                                                                                                                                        |
| Member    | サインインのみ可能で、個人プロファイル設定を管理する機能を持ちます。デフォルトでSAML SSOユーザーに割り当てられます。                                                                                                                   |


## サービスロール {#service-roles}

サービスロールの割り当て手順については、[クラウドユーザーの管理](/cloud/security/manage-cloud-users)を参照してください。

サービス権限は、管理者ロール以外のユーザーに対して、管理者が明示的に付与する必要があります。Service Adminには、SQLコンソールの管理者アクセス権が事前設定されていますが、権限を削減または削除するように変更できます。

| ロール              | 説明                 |
| ----------------- | --------------------------- |
| Service read only | サービスと設定を表示します。 |
| Service admin     | サービス設定を管理します。    |


## SQLコンソールロール {#sql-console-roles}

SQLコンソールロールの割り当て手順については、[SQLコンソールロール割り当ての管理](/cloud/guides/sql-console/manage-sql-console-role-assignments)を参照してください。

| ロール                  | 説明                                                                                    |
| --------------------- | ---------------------------------------------------------------------------------------------- |
| SQL console read only | サービス内のデータベースに対する読み取り専用アクセス権限。                                              |
| SQL console admin     | サービス内のデータベースに対する管理者アクセス権限。デフォルトデータベースロールと同等。 |
