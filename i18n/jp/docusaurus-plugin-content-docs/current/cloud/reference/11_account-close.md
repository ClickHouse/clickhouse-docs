## アカウントの閉鎖と削除 {#account-close--deletion}

私たちの目標は、お客様のプロジェクトの成功を支援することです。このサイトで回答されていない質問がある場合や、特定のユースケースの評価についてサポートが必要な場合は、[support@clickhouse.com](mailto:support@clickhouse.com)までお問い合わせください。

アカウントの閉鎖が必要となる状況があることを理解しています。このガイドでは、その手順についてご案内します。


## アカウントのクローズと削除の違い {#close-vs-delete}

クローズされたアカウントには再度ログインして、使用状況、請求情報、アカウントレベルのアクティビティログを確認できます。これにより、ユースケースの記録から年末の税務目的での請求書のダウンロードまで、さまざまな目的に役立つ詳細情報に簡単にアクセスできます。
また、製品アップデートの受信も継続されるため、お待ちいただいていた機能が利用可能になった際にお知らせを受け取ることができます。さらに、クローズされたアカウントはいつでも再開して新しいサービスを開始できます。

個人データの削除を要求される場合は、これが不可逆的なプロセスであることをご認識ください。アカウントと関連情報は利用できなくなります。製品アップデートを受信できなくなり、アカウントを再開することもできません。なお、ニュースレターの購読には影響しません。

ニュースレター購読者は、アカウントをクローズしたり情報を削除したりすることなく、ニュースレターメールの下部にある購読解除リンクを使用していつでも購読を解除できます。


## アカウント閉鎖の準備 {#preparing-for-closure}

アカウント閉鎖を申請する前に、以下の手順でアカウントを準備してください。

1. 保持が必要なサービスのデータをすべてエクスポートしてください。
2. サービスを停止して削除してください。これにより、アカウントへの追加料金の発生を防ぐことができます。
3. 閉鎖を申請する管理者以外のすべてのユーザーを削除してください。これにより、処理完了までの間に新しいサービスが作成されないようにすることができます。
4. コントロールパネルの「Usage」タブと「Billing」タブを確認して、すべての料金が支払い済みであることを確認してください。未払い残高があるアカウントは閉鎖できません。


## アカウント閉鎖のリクエスト {#request-account-closure}

アカウントの閉鎖および削除のリクエストには認証が必要です。リクエストを迅速に処理するため、以下の手順に従ってください。

1. clickhouse.cloudアカウントにサインインします。
2. 上記の「閉鎖の準備」セクションに記載されている残りの手順をすべて完了します。
3. ヘルプボタン(画面右上の疑問符アイコン)をクリックします。
4. 「Support」の下にある「Create case」をクリックします。
5. 「Create new case」画面で、以下の内容を入力します:

```text
Priority: Severity 3
Subject: Please close my ClickHouse account
Description: キャンセル理由について簡単にお知らせいただけますと幸いです。
```

5. 「Create new case」をクリックします
6. アカウントを閉鎖し、完了後に確認メールをお送りします。


## 個人データの削除リクエスト {#request-personal-data-deletion}

ClickHouseからの個人データ削除をリクエストできるのは、アカウント管理者のみです。アカウント管理者でない場合は、ClickHouseアカウント管理者に連絡して、アカウントからの削除をリクエストしてください。

データ削除をリクエストするには、上記の「アカウント閉鎖のリクエスト」の手順に従ってください。ケース情報を入力する際は、件名を「ClickHouseアカウントを閉鎖し、個人データを削除してください」に変更してください。

個人データ削除リクエストは30日以内に完了いたします。
