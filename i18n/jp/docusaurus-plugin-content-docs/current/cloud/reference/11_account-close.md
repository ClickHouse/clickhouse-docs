## アカウントの解約と削除 \{#account-close--deletion\}

私たちの目標は、お客様のプロジェクトの成功を支援することです。このサイトで解決できない疑問がある場合や、個別のユースケースの評価についてサポートが必要な場合は、[support@clickhouse.com](mailto:support@clickhouse.com) までご連絡ください。

やむを得ずアカウントの解約が必要となる状況があることも理解しています。本ガイドでは、その手順について説明します。

## アカウントのクローズと削除の違い \{#close-vs-delete\}
お客様は、クローズしたアカウントにも再度ログインして、利用状況、請求情報、およびアカウントレベルのアクティビティログを確認できます。これにより、ユースケースの記録から、年末に税務申告用として請求書をダウンロードすることまで、さまざまな目的に役立つ詳細情報へ簡単にアクセスできます。
また、待ち望んでいた機能が利用可能になったかどうかが分かるように、今後も製品アップデートを受け取り続けることができます。さらに、クローズしたアカウントは、いつでも再開して新しいサービスを開始できます。

個人データの削除を希望するお客様は、これが元に戻せない処理であることにご注意ください。アカウントおよび関連情報は今後利用できなくなります。製品アップデートは受信できず、アカウントを再開することもできません。これはニュースレター購読には影響しません。

ニュースレターの購読者は、アカウントをクローズしたり情報を削除したりすることなく、ニュースレターメールの末尾にある購読解除リンクを使用して、いつでも購読を解除できます。

## アカウントの閉鎖準備 \{#preparing-for-closure\}

アカウントの閉鎖を申請する前に、次の手順を行ってアカウントを準備してください。
1. 保持する必要があるデータを、利用中のサービスからエクスポートします。
2. サービスを停止して削除します。これにより、アカウントに追加の料金が発生しないようにできます。
3. 閉鎖を申請する管理者以外のすべてのユーザーを削除します。これにより、処理が完了するまでに新しいサービスが作成されることを防止できます。
4. コントロールパネルの「Usage」と「Billing」タブを確認し、すべての料金が支払済みであることを確かめてください。未払い残高のあるアカウントは閉鎖できません。

## アカウントの閉鎖をリクエストする \{#request-account-closure\}

閉鎖および削除のリクエストについては、認証が必要です。リクエストを迅速に処理できるよう、以下の手順に従ってください。

1. clickhouse.cloud アカウントにサインインします。
2. 上記の *Preparing for Closure* セクションの残りの手順をすべて完了します。
3. ヘルプボタン（画面右上のクエスチョンマーク）をクリックします。
4. 「Support」で「Create case」をクリックします。
5. 「Create new case」画面で、次の情報を入力します。

```text
Priority: Severity 3
Subject: ClickHouseアカウントの閉鎖をお願いします
Description: 解約される理由について簡単にお知らせいただけますと幸いです。
```

5. 「新しいケースを作成」をクリックします
6. アカウントを閉鎖し、処理が完了しましたら確認メールをお送りします。

## ご自身の個人データの削除を依頼する \{#request-personal-data-deletion\}
ClickHouse に個人データの削除を依頼できるのは、アカウント管理者のみです。アカウント管理者でない場合は、
アカウントからの削除を依頼するために、ClickHouse のアカウント管理者に連絡してください。

データ削除を依頼するには、上記の「アカウント閉鎖の依頼」の手順に従ってください。ケース情報を入力する際、件名を
「ClickHouse アカウントの閉鎖と個人データの削除をお願いします。」に変更してください。

個人データ削除のご依頼は 30 日以内に処理を完了します。
