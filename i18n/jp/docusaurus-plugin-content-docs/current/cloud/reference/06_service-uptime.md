## 稼働時間アラート {#uptime-alerts}

ユーザーは[ステータスページ](https://status.clickhouse.com/)でリージョン別の稼働時間を確認し、サービス障害に関するアラートを受信登録できるようになりました。


## SLA {#sla}

一定額以上のコミット契約に対して、サービスレベル契約(SLA)も提供しています。SLAポリシーの詳細については、[sales@clickhouse.com](mailto:sales@clickhouse.com)までお問い合わせください。
