## 稼働時間アラート {#uptime-alerts}

ユーザーは、[ステータスページ](https://status.clickhouse.com/)でリージョンごとの稼働時間を確認し、サービス障害時のアラートを受信するよう登録できます。



## SLA {#sla}

また、特定のコミットメント契約については、Service Level Agreement（SLA）も提供しています。SLA ポリシーの詳細については、[sales@clickhouse.com](mailto:sales@clickhouse.com) までお問い合わせください。
