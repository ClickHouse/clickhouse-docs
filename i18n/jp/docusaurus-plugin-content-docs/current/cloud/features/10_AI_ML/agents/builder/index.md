エージェントビルダーでは、エージェントを作成して設定します。

:::tip ClickHouse Agents を初めてお使いですか？
まずは[クイックスタート](/cloud/features/ai-ml/agents/quickstart)から始めてください。以下のページでは、ビルダーの各機能を詳しく説明しています。
:::

このセクションでは、エージェントビルダーの各機能を取り上げており、以下のページで構成されています。

| ページ                                                                       | 説明                             |
| ------------------------------------------------------------------------- | ------------------------------ |
| [コードインタープリター](/cloud/features/ai-ml/agents/builder/code-interpreter)      | サンドボックス環境でコードを実行します。           |
| [ウェブ検索](/cloud/features/ai-ml/agents/builder/web-search)                  | 公開ウェブを検索します。                   |
| [Image generation](/cloud/features/ai-ml/agents/builder/image-generation) | テキストから画像を生成します。                |
| [ビジョン](/cloud/features/ai-ml/agents/builder/vision)                       | 画像入力を受け付けます。                   |
| [MCPサーバー](/cloud/features/ai-ml/agents/builder/mcp-servers)               | サードパーティ製のMCPサーバーをエージェントに接続します。 |
| [スキル](/cloud/features/ai-ml/agents/builder/skills)                        | 再利用可能な命令パック。                   |
| [サブエージェント](/cloud/features/ai-ml/agents/builder/subagents)                | 子エージェントに作業を委任します。              |
