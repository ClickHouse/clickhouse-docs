このドキュメントでは、別途、第三者のモデル開発者規約の適用対象となる ClickHouse Cloud のモデルを一覧で示します。
現在使用されているすべてのモデルは AWS Bedrock でホストされており、該当するモデル開発者自身がホストしているものではありません。
これらのリンクは、お客様の便宜のために提供されています。適用されるモデル規約および使用ポリシーを遵守する責任は、お客様にあります。
