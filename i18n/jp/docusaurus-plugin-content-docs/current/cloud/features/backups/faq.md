### バックアップストレージの請求額が上がっているのはなぜですか？ \{#why-bill-going-up\}

ClickHouse Cloud では、信頼性の高い復旧のために、フルバックアップと増分バックアップを組み合わせて、データのバックアップを最大 8 世代保持しています。これまでは、請求書に反映されていたのはこのデータの一部にすぎませんでした。2026 年 7 月 1 日以降は、請求額が実際に保持されているストレージ容量と一致するよう、これを修正します。

:::note
バックアップストレージの 1 GB あたりの価格は変更されていません。この修正により、請求書に記載されるバックアップ世代数が、実際にサービスで保持されている数と一致するようになります。
:::

### これは値上げですか？ \{#is-this-a-price-increase\}

いいえ。1GBあたりの価格に変更はありません。請求書に記載されるバックアップ世代数を修正しています。

### 過去の過少請求分を後から請求されますか？ \{#past-undercharges\}

いいえ。過去の請求期間にさかのぼって請求したり、不足分を回収したりすることはありません。この修正は、2026年7月1日以降の請求にのみ適用されます。

### 保持されているすべてのバックアップ世代はどこで確認できますか？ \{#see-backup-versions\}

保持されているすべてのバックアップ世代は、4月23日から ClickHouse Cloud console で確認できるようになります。また、特定のバージョンが現在課金対象になっているか、あるいは保持されているもののまだ課金されていないかも確認できます。更新された[料金計算ツール](https://clickhouse.com/pricing)にも、これらのコストが反映されています。

### 質問がある場合は、どこに問い合わせればよいですか？ \{#contact\}

[ClickHouse Support](https://clickhouse.com/support/program) にお問い合わせください。お客様固有の状況については、担当のアカウントチームにご相談いただくこともできます。

### これはバックアップの信頼性や可用性に影響しますか？ \{#reliability\}

いいえ。バックアップの対象範囲、保持期間、復旧機能に変更はありません。修正されるのはバックアップの請求方法のみで、保存方法や管理方法は変わりません。

### 詳細はどこで確認できますか？ \{#more-details\}

4月23日に、詳細を記載した通知メールをお送りします。実際のバックアップ費用をより正確に見積もるには、[料金計算ツール](https://clickhouse.com/pricing)をご利用ください。

### 請求に関する変更はいつ適用されますか？ \{#when-changes-go-live\}

請求に関する変更は、最初の通知から60日後の2026年7月1日に適用されます。
