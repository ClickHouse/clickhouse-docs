&#39;Launch ClickStack&#39; を選択して、ClickStack UI (HyperDX) にアクセスします。自動的に認証されてリダイレクトされます。各 OTel ソース（logs、traces、metrics）ごとにデータソースが事前に作成されています。

<Image img={clickstack_managed_ui} size="md" alt="ClickStack UI" border />

以上で準備完了です。🎉

ClickStack をさっそく探索してみてください。logs や traces を検索し、logs・traces・metrics がリアルタイムでどのように相関付けられるかを確認し、ダッシュボードを構築し、サービスマップを探索し、イベントの変化やパターンを発見し、アラートを設定して問題を先回りで検知できるようにしましょう。
