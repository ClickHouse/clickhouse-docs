このセクションでは、ClickStack の利用開始に役立つ、さまざまなサンプルデータセットと例を紹介します。これらの例では、ローカル開発から本番環境まで、ClickStack でオブザーバビリティデータを扱うさまざまな方法を示します。

| Dataset | Description |
|---------|-------------|
| [Sample Data](sample-data.md) | デモ環境から取得したログ、トレース、メトリクスを含むサンプルデータセットを読み込む |
| [Local Data](local-data.md) | ローカルシステムのメトリクスとログを収集し、分析のために ClickStack に送信する |
| [Remote Demo Data](remote-demo-data.md) | リモートのデモクラスタに接続し、特定のインシデントを調査する |
