## ClickStack デモデイズ \{#clickstack-demo-days\}

これらのフォルダには、社内の開発者向け定例である デモデイズ の録画が含まれています。ここでは、現在 ClickStack で構築中のものや検証している内容を共有しています。

ここでご覧いただく内容の一部は、今後製品に反映されます。一部は、早い段階でフィードバックを得るために共有しているものです。また、一部は実験的な取り組みであり、そのままリリースされない可能性もあります。

気になるものがあれば、リンクされている PR を確認し、ぜひご意見をお寄せください。コミュニティが現在の動きを身近に把握し、ClickStack の今後の方向性をともに形作れる場にしたいと考えています。

質問やフィードバックがある場合は、ClickHouse Slack チャンネル [#o11y-clickstack](https://clickhouse.com/slack) または [公開リポジトリ](https://github.com/hyperdxio/hyperdx/tree/main) からご連絡ください。
