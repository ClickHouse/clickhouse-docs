時系列ユースケースガイドへようこそ。このガイドでは、時系列データ向けに ClickHouse をセットアップし、活用する方法を学びます。

時系列データは、現代の分析のあらゆる場面に登場します。システムメトリクスやアプリケーションログから、ビジネスイベントやセンサーの観測値に至るまで、時間の経過とともに収集されるデータポイントは、システムやプロセスにおけるトレンド、パターン、異常を理解するのに役立ちます。

ClickHouse は時系列データの処理を得意としており、保存と分析の両方に対して強力な機能を提供します。シンプルなモニタリングダッシュボードの構築から、ペタバイト級のセンサーデータをリアルタイムに処理する場合まで、ClickHouse は必要なツールとパフォーマンスを提供します。

このガイドでは、ClickHouse で時系列データを扱ううえで知っておくべき内容を、基本概念から高度な最適化手法まで順を追って解説します。次の内容を学ぶことができます:

* [ユースケースに適した日付・時刻データ型を選択する](/use-cases/time-series/date-time-data-types)
* [一般的な時系列処理および集計を実行する](/use-cases/time-series/basic-operations)
* [時間ベースデータ向けの特化した分析関数を適用する](/use-cases/time-series/analysis-functions)
* [時間データのストレージ効率を最適化する](/use-cases/time-series/storage-efficiency)
* [時系列ワークロードのクエリパフォーマンスをチューニングする](/use-cases/time-series/query-performance)

時系列分析が初めての方も、既存の実装を最適化したい方も、このガイドは ClickHouse の時系列機能を最大限に活用するうえでの助けとなるでしょう。
