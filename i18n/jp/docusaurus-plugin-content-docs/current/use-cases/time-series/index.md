タイムシリーズ・ユースケースガイドへようこそ。このガイドでは、タイムシリーズデータ向けに ClickHouse をセットアップし、活用する方法を学びます。

タイムシリーズデータは、現代の分析のあらゆる場面に存在します。システムメトリクスやアプリケーションログから、ビジネスイベントやセンサー計測値まで、時間とともに収集されるデータポイントは、システムや業務プロセスにおけるトレンド、パターン、異常を理解するのに役立ちます。

ClickHouse はタイムシリーズデータの処理を得意としており、ストレージと分析の両方に強力な機能を提供します。シンプルな監視ダッシュボードを構築する場合でも、ペタバイト規模のセンサーデータをリアルタイムに処理する場合でも、ClickHouse は必要なツールとパフォーマンスを提供します。

このガイドでは、ClickHouse でタイムシリーズデータを扱ううえで知っておくべき内容を、基本概念から高度な最適化手法まで順を追って解説します。次のようなことを学びます。

* [ユースケースに適した日付・時刻データ型を選択する](/use-cases/time-series/date-time-data-types)
* [一般的なタイムシリーズ処理と集約を実行する](/use-cases/time-series/basic-operations)
* [時間ベースのデータ向けの専用分析関数を適用する](/use-cases/time-series/analysis-functions)
* [時系列データのストレージ効率を最適化する](/use-cases/time-series/storage-efficiency)
* [タイムシリーズワークロード向けにクエリパフォーマンスをチューニングする](/use-cases/time-series/query-performance)

タイムシリーズ分析に初めて取り組む場合でも、既存の実装の最適化に取り組んでいる場合でも、このガイドは ClickHouse のタイムシリーズ機能を最大限に活用するうえで役立ちます。
