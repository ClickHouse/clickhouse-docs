時系列ユースケースガイドへようこそ。このガイドでは、時系列データ向けに ClickHouse をセットアップして利用する方法を説明します。

時系列データは、現代のデータ分析のあらゆる場面に存在します。システムメトリクスやアプリケーションログから、ビジネスイベントやセンサー計測値まで、時間の経過とともに収集されるデータポイントは、システムや業務プロセスにおけるトレンド、パターン、異常を理解するのに役立ちます。

ClickHouse は時系列データの処理に優れており、保存と分析の両方に強力な機能を提供します。シンプルなモニタリングダッシュボードの構築から、ペタバイト規模のセンサーデータをリアルタイムで処理する場合まで、ClickHouse は必要なツールとパフォーマンスを提供します。

このガイドでは、ClickHouse で時系列データを扱ううえで知っておくべき内容を、基本概念から高度な最適化手法まで順を追って解説します。次の内容について学びます。

* [ユースケースに適した日付・時刻データ型を選択する](/use-cases/time-series/date-time-data-types)
* [一般的な時系列処理と集約を実行する](/use-cases/time-series/basic-operations)
* [時間ベースのデータ向けの専用分析関数を適用する](/use-cases/time-series/analysis-functions)
* [時系列データのストレージ効率を最適化する](/use-cases/time-series/storage-efficiency)
* [時系列ワークロード向けにクエリのパフォーマンスをチューニングする](/use-cases/time-series/query-performance)

時系列分析が初めての方でも、既存の実装を最適化したい方でも、このガイドは ClickHouse の時系列機能を最大限に活用するための助けとなります。
