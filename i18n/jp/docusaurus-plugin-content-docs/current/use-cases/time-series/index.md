時系列ユースケースガイドへようこそ。このガイドでは、時系列データ向けに ClickHouse をセットアップし、活用する方法を学びます。

時系列データは、現代の分析のあらゆる場面に存在します。システムメトリクスやアプリケーションログから、ビジネスイベントやセンサー計測値に至るまで、時間の経過とともに収集されたデータポイントは、システムやプロセスにおけるトレンド、パターン、異常を理解するのに役立ちます。

ClickHouse は時系列データの処理を得意としており、保存と分析の双方において強力な機能を提供します。シンプルなモニタリングダッシュボードの構築から、ペタバイト級のセンサーデータをリアルタイムで処理するケースまで、ClickHouse は必要なツールとパフォーマンスを提供します。

このガイドでは、ClickHouse で時系列データを扱ううえで押さえておくべきポイントを、基本概念から高度な最適化手法まで順を追って説明します。次の内容を学ぶことができます。

* [ユースケースに適した日付・時刻データ型を選択する](/use-cases/time-series/date-time-data-types)
* [一般的な時系列処理と集約を実行する](/use-cases/time-series/basic-operations)
* [時間ベースデータ向けの専用分析関数を適用する](/use-cases/time-series/analysis-functions)
* [時系列データの保存効率を最適化する](/use-cases/time-series/storage-efficiency)
* [時系列ワークロード向けにクエリパフォーマンスをチューニングする](/use-cases/time-series/query-performance)

時系列分析をこれから始める方も、既存の実装をさらに最適化したい方も、このガイドを通じて ClickHouse の時系列機能を最大限に引き出すことができます。
