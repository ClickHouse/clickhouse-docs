[Model Context Protocol](https://www.anthropic.com/news/model-context-protocol) (MCP) は、Anthropic によって開発された標準仕様であり、AI アシスタントが外部システムとシームレスに連携できるようにします。
このプロトコルにより、AI アシスタントはデータソース、API、データベースなどに対して、安全かつ標準化された方法で接続できるようになります。

MCP は AI モデルとさまざまなサービスとの間に共通インターフェースを提供し、統合ごとにカスタムツールを実装する必要性をなくします。
AI システム向けに特化して設計された汎用 API 標準と考えることができます。

MCP の主な利点は、AI ライブラリがプロトコル対応を一度実装するだけでよい点です。
その後は、すべての MCP 対応サービスにすぐにアクセスできるようになり、AI ライブラリを保守する開発者の大幅な工数削減につながります。


## MCPのアーキテクチャ {#mcp-architecture}

MCPはクライアント・サーバーアーキテクチャを採用しています：

- クライアント（Claude Desktop、Cursor、VS Codeなど）がMCPサーバーへの接続を確立します。クライアントの一覧は[awesome-mcp-clients](https://github.com/punkpeye/awesome-mcp-clients?tab=readme-ov-file#windsurf) GitHubリポジトリで確認できます。
- サーバーは標準化されたインターフェースを通じてツールと機能を公開します。サーバーの一覧は[awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) GitHubリポジトリで確認できます。
- AIモデルは必要に応じてこれらのツールを使用し、外部データや機能にアクセスできます

## ClickHouseにはMCPサーバーがありますか？ {#clickhouse-mcp-server}

はい、あります！
[ClickHouse MCP Server](https://github.com/ClickHouse/mcp-clickhouse)は以下のツールを提供します：

- `run_select_query` - ClickHouseクラスタ上でSQLクエリを実行します。
- `list_databases` - ClickHouseクラスタ上のすべてのデータベースを一覧表示します。
- `list_tables` - データベース内のすべてのテーブルを一覧表示します。
