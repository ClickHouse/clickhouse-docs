## Question \{#question\}

여러 SQL 문과 여러 줄에 대해 ClickHouse Client를 사용할 때 `send_logs_level` 출력을 파일로 저장하려면 어떻게 해야 합니까?

## 답변 \{#answer\}

* SQL 문이 포함된 SQL 파일을 생성합니다. 예를 들어 `send_logs_level_example.sql`과 같이 생성합니다.

```
SET send_logs_level = 'trace';
SELECT * FROM db1.table1;
```

* 화면과 파일 모두에 출력하는 다음 명령을 실행합니다:

```
cat send_logs_level_example.sql | ./clickhouse client -n -m --host abc123.us-west-2.aws.clickhouse.cloud --secure --port 9440 --password ABC123 --send_logs_level=trace 2>&1 | tee send_log_results.txt
```
