## Question \{#question\}

커넥터를 사용해 Power BI에서 ClickHouse에 연결을 시도하면 인증 오류가 발생합니다.

이 오류는 보통 다음과 같은 형태로 나타납니다:

```
We encountered an error while trying to connect.
Details: "ODBC: ERROR [HY000] HTTP status code: 403
Received error:
Code: 516. DB::Exception: default: Authentication failed: password is incorrect, or there is no user with such name.
If you have installed ClickHouse and forgot password you can reset it in the configuration file.
The password for default user is typically located at /etc/clickhouse-server/users.d/default-password.xml and deleting this file will reset the password.
See also /etc/clickhouse-server/users.ml on the server where
ClickHouse is installed.
```

<Image img={powerbi_error} size="md" alt="Power BI ODBC 인증 오류 대화 상자" border />

## 답변 \{#answer\}

ClickHouse ODBC Driver를 버전 [1.4.1](https://github.com/ClickHouse/clickhouse-odbc/releases/tag/1.4.1.20250523)로 업데이트하십시오.

업데이트가 불가능한 경우, 비밀번호에 물결표(~)가 포함되어 있는지 확인하십시오. 이 문자는 [1.4.1](https://github.com/ClickHouse/clickhouse-odbc/releases/tag/1.4.1.20250523) 이전 버전에서 문제를 일으킬 수 있습니다.

해당 연결에만 사용하는 전용 사용자를 생성하고 비밀번호를 수동으로 설정할 것을 권장합니다. ClickHouse Cloud를 사용하고 있고 `default` 사용자와 동등한 관리자 수준의 접근 권한이 필요한 경우, 새 사용자를 생성한 후 해당 사용자에게 `default_role`을 할당하십시오.

자세한 내용은 다음을 참조하십시오:\
https://clickhouse.com/docs/operations/access-rights#user-account-management\
https://clickhouse.com/docs/cloud/security/cloud-access-management#database-roles
