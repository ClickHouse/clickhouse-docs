## 질문 \{#question\}

쿼리 실행 시간 제한을 어떻게 설정합니까?

## 답변 \{#answer\}

[`max_execution_time`](/docs/operations/settings/settings#max_execution_time) 설정(초 단위로 지정)을 사용할 수 있습니다.

```sql
clickhouse-cloud :) SELECT 1 SETTINGS max_execution_time=0.0001

SELECT 1
SETTINGS max_execution_time = 0.0001
```
