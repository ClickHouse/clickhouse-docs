## 수행해야 할 작업 \{#what-to-do\}

Oracle ODBC 드라이버를 통해 ClickHouse 외부 딕셔너리의 소스로 Oracle을 사용하는 경우, `/etc/default/clickhouse`에서 `NLS_LANG` 환경 변수에 올바른 값을 설정해야 합니다.

자세한 내용은 [Oracle NLS&#95;LANG FAQ](https://www.oracle.com/technetwork/products/globalization/nls-lang-099431.html)를 참조하십시오.

**예제**

```sql
NLS_LANG=RUSSIAN_RUSSIA.UTF8
```
