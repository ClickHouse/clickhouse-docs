## 개요 \{#overview\}

이 가이드는 [ClickPipes reverse private endpoint](/integrations/clickpipes/aws-privatelink#msk-multi-vpc)와 함께 사용할 **MSK 다중 VPC(MSK multi-VPC)** 환경을 구성하는 방법을 안내합니다.

## 요구 사항 \{#requirements\}

MSK 클러스터 VPC는 ClickPipes 리전 중 하나에 있어야 합니다. 지원되는 리전 목록은 [ClickPipes 리전](/integrations/clickpipes/aws-privatelink#aws-privatelink-regions)에서 확인할 수 있습니다.

## 다중 VPC 연결 활성화 \{#enabling-multi-vpc-connectivity\}

1. MSK 클러스터로 이동합니다.
   * Amazon MSK 콘솔의 왼쪽 탐색 창에서 &quot;Clusters&quot;를 선택합니다.
   * 다중 VPC 연결을 구성할 MSK 클러스터를 선택합니다.
2. MSK 다중 VPC 연결을 활성화합니다.
   * **Connectivity** 탭에서 **Multi-VPC connectivity** 섹션을 찾습니다.
   * **Edit**을 클릭합니다.
   * **Turn-on MSK multi-VPC connectivity** 옵션을 활성화합니다.
   * 안내에 따라 설정을 완료합니다.
3. 클러스터 정책에 ClickPipes 계정 주체를 추가합니다.
   * **Configuration** 탭으로 이동합니다.
   * **Cluster policy** 섹션에서 **Edit**을 클릭합니다.
   * **IAM policy**에 `arn:aws:iam::072088201116:root`를 포함합니다. 예:
     ```json
     {
         "Version": "2012-10-17",
         "Statement": [
             {
                 "Effect": "Allow",
                 "Principal": {
                     "AWS": [
                         "arn:aws:iam::072088201116:root"
                     ]
                 },
                 "Action": [
                     "kafka-cluster:Connect",
                     "kafka-cluster:DescribeCluster",
                     "kafka-cluster:ListClusters"
                 ]
             }
         ]
     }
     ```

:::tip
클러스터 IAM 정책을 통해 ClickPipes가 MSK 클러스터로의 연결을 시작할 수 있습니다.
MSK 클러스터에 IAM 인증을 구성하려면 [IAM 인증 문서](/integrations/clickpipes/kafka/best-practices#iam)를 참조하십시오.
:::
