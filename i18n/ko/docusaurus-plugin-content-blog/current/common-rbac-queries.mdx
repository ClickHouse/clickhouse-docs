## 소개 \{#introduction\}

다음은 특정 권한을 사용자에게 부여할 때 사용할 수 있는 일반적인 쿼리입니다.

## 현재 사용자와 동일한 권한을 다른 사용자에게 부여하려면 어떻게 하면 됩니까? \{#how-do-i-grant-the-same-permissions-as-the-current-user-to-another-user\}

```sql
GRANT CURRENT GRANTS ON *.* TO another_user;
```

## 현재 사용자의 권한을 기준으로 특정 사용자에게 특정 권한을 부여하려면 어떻게 해야 합니까? \{#how-do-i-grant-a-specific-permission-to-a-user-based-on-the-grants-of-the-current-user\}

아래 예시에서 `another_user`는 현재 사용자가 가진 모든 데이터베이스와 테이블에 대해 `SELECT` 명령을 실행할 수 있습니다.

```sql
GRANT CURRENT GRANTS(SELECT ON *.*) TO another_user;
```

## 현재 사용자에게 부여된 권한을 기준으로 특정 데이터베이스에 대해 특정 사용자에게 특정 권한을 부여하려면 어떻게 해야 합니까? \{#how-do-i-grant-a-specific-permission-to-a-user-for-a-specific-database-based-on-the-grants-of-the-current-user\}

아래 예시에서 `another_user`는 `my_database`의 모든 테이블에 대해 `INSERT` 명령을 수행할 수 있게 됩니다.

```sql
GRANT INSERT ON my_database.* TO another_user;
```

## 기본 사용자를 기준으로 특정 사용자에게 모든 권한을 부여하려면 어떻게 해야 합니까? \{#how-do-i-give-access-to-all-grants-for-a-specific-user-based-on-the-default-user\}

```sql
GRANT default_role TO another_user;
```
