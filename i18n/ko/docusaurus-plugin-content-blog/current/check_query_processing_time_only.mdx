## Question \{#question\}

쿼리가 많은 행을 반환하지만, 쿼리 처리 시간에만 관심이 있습니다. 쿼리 결과 출력은 생략하고 쿼리 처리 시간만 확인하려면 어떻게 해야 합니까?

## 답변 \{#answer\}

쿼리에 `FORMAT Null`을 추가하여 출력 형식을 `Null`로 설정합니다. 이렇게 하면 데이터가 클라이언트로 전송되지 않게 됩니다.

예를 들어:

```sql
SELECT
    customer_id,
    count() AS total,
    any(review_headline)
FROM amazon_reviews
GROUP BY customer_id
ORDER BY total DESC
FORMAT Null
```

응답에는 처리된 행 수와 경과 시간이 포함되지만, 반환되는 행은 0개입니다:

```response
0 rows in set. Elapsed: 25.288 sec. Processed 222.04 million rows, 13.50 GB (8.78 million rows/s., 533.77 MB/s.)
```
