## ClickHouse Cloud 서비스에 액세스할 수 없습니다 \{#i-am-unable-to-access-a-clickhouse-cloud-service\}

다음과 같은 오류 메시지가 표시되는 경우 IP Access List에서 해당 접근을 차단하고 있을 수 있습니다.

```response
curl: (35) error:02FFF036:system library:func(4095):Connection reset by peer
```

또는

```response
curl: (35) LibreSSL SSL_connect: SSL_ERROR_SYSCALL in connection to HOSTNAME.clickhouse.cloud:8443
```

또는

```response
Code: 210. DB::NetException: SSL connection unexpectedly closed (e46453teek.us-east-2.aws.clickhouse-staging.com:9440). (NETWORK_ERROR)
```

[IP Access List](/cloud/security/setting-ip-filters)를 확인하십시오. 허용된 목록에 포함되지 않은 위치에서 연결을 시도하면 연결이 실패합니다.
