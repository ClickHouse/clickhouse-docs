## 질문 \{#question\}

ClickHouse Cloud에서 청구 담당자(Billing Contact)를 어떻게 변경할 수 있습니까?

## 답변 \{#answer\}

Admin으로 결제 담당자(Billing Contact)를 변경하려면 다음 단계를 따르십시오.

1. 새 사용자를 Admin 권한으로 Cloud Organization에 초대합니다.
2. 해당 사용자가 초대를 수락한 후 ClickHouse Cloud Console의 결제 페이지(Admin → Billing)로 이동하여 「Billing contacts」 섹션을 찾습니다.
3. Edit 버튼을 클릭하여 새 Admin 사용자를 결제 담당자(Billing Contact)로 지정합니다.
