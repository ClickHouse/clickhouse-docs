## Google Chrome에서 \{#from-google-chrome\}

1. Windows/Linux에서는 <kbd>F12</kbd> 또는 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd>를, Mac에서는 <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>I</kbd>를 눌러 개발자 도구(Developer Tools)를 엽니다.
2. &quot;Network&quot; 탭을 클릭합니다.
3. 페이지를 새로 고치고 문제를 재현합니다.
4. 개발자 도구에서 다운로드 버튼을 클릭합니다.

## Mozilla Firefox에서 \{#from-mozilla-firefox\}

1. <kbd>F12</kbd> 또는 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd> (Windows/Linux) / <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>E</kbd> (Mac)을 눌러 개발자 도구(Developer Tools)를 엽니다.
2. 「Network」 탭을 클릭합니다.
3. 페이지를 새로 고침하고 문제를 재현합니다.
4. 개발자 도구에서 요청 목록을 마우스 오른쪽 버튼으로 클릭한 다음 「Save All As HAR」를 선택합니다.

## Microsoft Edge에서 \{#from-microsoft-edge\}

1. <kbd>F12</kbd> 또는 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd> (Windows/Linux) / <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>I</kbd> (Mac)을 눌러 개발자 도구(Developer Tools)를 엽니다.
2. 「Network」 탭을 클릭합니다.
3. 페이지를 새로고침한 다음 문제를 다시 재현합니다.
4. 개발자 도구의 네트워크 목록에서 마우스 오른쪽 버튼을 클릭하고 「Save all as HAR with content」를 선택합니다.

## Safari에서 \{#from-safari\}

1. 개발자 도구를 활성화합니다(이미 활성화되어 있지 않다면):
   * Safari &gt; Settings &gt; Advanced로 이동합니다.
   * 맨 아래에서 &quot;Show Develop menu in menu bar&quot;를 선택합니다.
2. Develop &gt; Show Web Inspector를 클릭합니다.
3. &quot;Network&quot; 탭을 클릭합니다.
4. 페이지를 새로고침한 후 문제를 다시 재현합니다.
5. Developer Tools에서 &quot;Export&quot; 버튼을 클릭합니다.

## .HAR 파일 보내기 \{#sending-har-file\}

1. 파일 이름을 짧고 내용을 잘 나타내도록 변경합니다(예: login-issue.har).
2. 파일을 압축합니다(선택 사항이지만 권장됩니다).
3. 지원 케이스에 첨부하거나 지정된 지원 담당자에게 이메일로 전송합니다.

:::note
HAR 파일에는 세션 쿠키나 기타 민감한 데이터가 포함될 수 있습니다.
승인된 지원 인력에게만 공유하십시오.
:::
