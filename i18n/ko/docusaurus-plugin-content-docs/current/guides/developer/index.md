# 고급 가이드 \{#advanced-guides\}

이 섹션에는 다음과 같은 고급 가이드가 포함되어 있습니다.

| Guide                                                                                                                  | Description                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alternative Query Languages](../developer/alternative-query-languages)                                         | 지원되는 대체 SQL 방언과 사용 방법을 다루는 가이드입니다. 각 방언별 쿼리 예제를 제공합니다.                                                                                                                                                                                                                                                   |
| [Cascading Materialized Views](../developer/cascading-materialized-views)                                       | materialized view를 생성하고 이를 연쇄적으로(cascade) 구성하여 여러 개의 소스 테이블을 하나의 대상 테이블로 결합하는 방법을 설명하는 가이드입니다. 도메인 이름 그룹에 대해 월별 및 연도별로 데이터를 집계하기 위해 연쇄 materialized view를 사용하는 예제가 포함되어 있습니다.                                                              |
| [Debugging memory issues](../developer/debugging-memory-issues)                                                 | ClickHouse에서 메모리 관련 문제를 디버깅하는 방법을 설명하는 가이드입니다.                                                                                                                                                                                                                                                                   |
| [Deduplicating Inserts on Retries](../developer/deduplicating-inserts-on-retries)                               | 실패한 INSERT를 재시도해야 할 수 있는 상황을 처리하는 방법을 설명하는 가이드입니다.                                                                                                                                                                                                                                                          |
| [Deduplication strategies](../developer/deduplication)                                                          | 데이터 중복 제거(deduplication) 기법, 즉 데이터베이스에서 중복된 행을 제거하는 기술을 심도 있게 다루는 가이드입니다. OLTP 시스템의 기본 키 기반 중복 제거와의 차이점, ClickHouse의 중복 제거 방식, 그리고 ClickHouse 쿼리 내에서 중복 데이터 시나리오를 처리하는 방법을 설명합니다.                                                        |
| [Filling gaps in time-series data](../developer/time-series-filling-gaps)                                       | 시계열 데이터 처리에 대한 ClickHouse의 기능을 설명하는 가이드입니다. 보다 완전하고 연속적인 시계열 표현을 만들기 위해 데이터의 공백을 메우는 다양한 기법을 다룹니다.                                                                                                                                                                         |
| [Manage Data with TTL (Time-to-live)](../developer/ttl)                                                         | 시계열 데이터의 공백을 메우기 위해 `WITH FILL` 절을 사용하는 방법을 다루는 가이드입니다. 0 값으로 공백을 채우는 방법, 공백 채우기를 시작할 시작 지점을 지정하는 방법, 특정 종료 지점까지 공백을 채우는 방법, 누적 계산을 위한 값을 보간하는 방법을 설명합니다.                                                                            |
| [Stored procedures &amp; query parameters](../developer/stored-procedures-and-prepared-statements)                  | ClickHouse는 전통적인 저장 프로시저를 지원하지 않는다는 점을 설명하고, User-Defined Functions (UDF), 매개변수화된 뷰, materialized view, 외부 오케스트레이션 등 권장되는 대안을 제시하는 가이드입니다. 또한 prepared statement와 유사한 안전한 매개변수화 쿼리를 위한 쿼리 파라미터 활용 방법을 다룹니다.                       |
| [Understanding query execution with the Analyzer](../developer/understanding-query-execution-with-the-analyzer) | Analyzer 도구를 소개하여 ClickHouse 쿼리 실행 방식을 이해하기 쉽게 설명하는 가이드입니다. Analyzer가 쿼리를 여러 단계로 분해하여 전체 실행 과정을 시각화하고 문제를 해결하며 최적의 성능을 달성할 수 있도록 하는 방법을 설명합니다.                                                                                                  |
| [Using JOINs in ClickHouse](../joining-tables)                                                                  | ClickHouse에서 테이블 조인을 보다 쉽게 이해하도록 돕는 가이드입니다. 다양한 조인 유형(`INNER`, `LEFT`, `RIGHT` 등)을 다루고, 작은 테이블을 오른쪽에 두는 것과 같은 효율적인 조인을 위한 모범 사례를 설명합니다. 또한 복잡한 데이터 관계에서 쿼리를 최적화할 수 있도록 ClickHouse 내부 조인 알고리즘에 대한 인사이트를 제공합니다.     |
