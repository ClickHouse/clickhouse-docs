시계열 사용 사례 가이드에 오신 것을 환영합니다. 이 가이드에서는 시계열 데이터를 위해 ClickHouse를 설정하고 활용하는 방법을 알아봅니다.

시계열 데이터는 현대 분석 환경 곳곳에 존재합니다. 시스템 메트릭과 애플리케이션 로그부터 비즈니스 이벤트와 센서 측정값까지, 시간 경과에 따라 수집된 데이터 포인트는 시스템과 프로세스의 추세, 패턴, 이상 징후를 이해하는 데 도움을 줍니다.

ClickHouse는 시계열 데이터를 처리하는 데 뛰어나며, 저장과 분석 모두를 위한 강력한 기능을 제공합니다. 단순한 모니터링 대시보드를 구축하든, 실시간으로 페타바이트 규모의 센서 데이터를 처리하든, ClickHouse는 필요한 도구와 성능을 제공합니다.

이 가이드는 ClickHouse에서 시계열 데이터를 다루는 데 알아야 할 내용을 기본 개념부터 고급 최적화 기법까지 차례로 안내합니다. 다음과 같은 내용을 학습할 수 있습니다.

* [사용 사례에 맞는 날짜 및 시간 데이터 타입 선택하기](/use-cases/time-series/date-time-data-types)
* [일반적인 시계열 연산 및 집계 수행하기](/use-cases/time-series/basic-operations)
* [시간 기반 데이터에 특화된 분석 함수 적용하기](/use-cases/time-series/analysis-functions)
* [시계열(시간) 데이터 저장 효율 최적화하기](/use-cases/time-series/storage-efficiency)
* [시계열 워크로드에 대한 쿼리 성능 튜닝하기](/use-cases/time-series/query-performance)

시계열 분석을 처음 접했거나 기존 구현을 최적화하려는 경우에도, 이 가이드는 ClickHouse의 시계열 기능을 최대한 활용하는 데 도움이 됩니다.
