# Machine Learning 및 GenAI \{#machine-learning-and-genai\}

ClickHouse는 Machine Learning 워크로드를 처리하는 실시간 데이터베이스로 최적입니다.
ClickHouse를 사용하면 분석 데이터에 GenAI를 활용하는 것이 그 어느 때보다 쉬워집니다.
이 섹션에서는 ClickHouse가 Machine Learning 및 GenAI에 어떻게 사용되는지에 대한
가이드를 확인할 수 있습니다.

| Section                                                                        | Description                                                                                                                                   |
| ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| [AI chat](/use-cases/AI_ML/AIChat)                                             | 이 가이드는 ClickHouse Cloud Console에서 AI Chat 기능을 활성화하고 사용하는 방법을 설명합니다.                                                |
| [Semantic Layer](/use-cases/AI_ML/AIChat/semantic-layer)                      | AGENTS.md를 사용해 Ask AI 채팅 에이전트를 사용자 정의하여 비즈니스별 규칙, 계산, 데이터 구조 지침을 제공하는 방법을 알아봅니다.    |
| [MCP](/use-cases/AI/MCP)                                                       | ClickHouse와 함께 Model Context Protocol(MCP)을 사용하도록 설정하는 데 도움이 되는 가이드 모음입니다.                                                    |
| [AI-powered SQL generation](/use-cases/AI/ai-powered-sql-generation)           | 이 기능을 사용하면 데이터 요구 사항을 자연어로 기술할 수 있으며, 시스템이 이를 해당 SQL 문으로 변환합니다. |
