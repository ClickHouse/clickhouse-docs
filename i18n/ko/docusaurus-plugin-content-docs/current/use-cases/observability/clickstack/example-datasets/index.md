이 섹션에서는 ClickStack을 시작하는 데 도움이 되는 다양한 샘플 데이터 세트와 예제를 제공합니다. 이 예제들은 로컬 개발부터 프로덕션 시나리오에 이르기까지 ClickStack에서 관측 가능성 데이터 작업에 대한 다양한 방법을 보여줍니다.

| 데이터 세트 | 설명 |
|-------------|------|
| [샘플 데이터](sample-data.md) | 우리의 데모 환경에서 로그, 추적 및 메트릭을 포함하는 샘플 데이터 세트를 로드합니다. |
| [로컬 데이터](local-data.md) | 로컬 시스템 메트릭 및 로그를 수집하여 ClickStack으로 전송하여 분석합니다. |
| [원격 데모 데이터](remote-demo-data.md) | 우리의 원격 데모 클러스터에 연결하여 문제를 탐색합니다. |
