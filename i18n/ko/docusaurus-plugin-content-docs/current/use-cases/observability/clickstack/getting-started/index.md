ClickStack은 운영 방식과 요구 사항에 따라 여러 방식으로 배포할 수 있습니다:

- [Managed ClickStack](/use-cases/observability/clickstack/getting-started/managed) - **가장 쉽고 간편하게** 시작하는 방법으로, **Managed ClickStack**을 **ClickHouse Cloud**에 배포합니다.
- [오픈 소스 ClickStack](/use-cases/observability/clickstack/getting-started/oss) - 완전한 오픈 소스 아키텍처를 직접 운영하려는 경우에 적합합니다.
