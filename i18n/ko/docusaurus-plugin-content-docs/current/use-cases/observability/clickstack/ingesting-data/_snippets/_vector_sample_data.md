샘플 데이터 세트로 실험해 보고 싶다면, 다음 nginx 예제를 다운로드하십시오.

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

:::note
이 데이터는 파싱을 더 쉽게 하기 위해 로그를 JSON 형식으로 출력하도록 구성된 Nginx 인스턴스에서 수집되었습니다. 이러한 로그를 위한 Nginx 구성은 [&quot;ClickStack으로 Nginx 로그 모니터링&quot;](/use-cases/observability/clickstack/integrations/nginx#configure-nginx)을 참조하십시오.
:::
