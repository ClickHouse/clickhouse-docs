# 데이터베이스 엔진 \{#database-engines\}

데이터베이스 엔진은 테이블을 사용할 수 있게 해 줍니다. 기본적으로 ClickHouse는 구성 가능한 [테이블 엔진](../../engines/table-engines/index.md)과 [SQL 방언](../../sql-reference/syntax.md)을 제공하는 [Atomic](../../engines/database-engines/atomic.md) 데이터베이스 엔진을 사용합니다.

사용 가능한 데이터베이스 엔진의 전체 목록은 아래와 같습니다. 자세한 내용은 각 링크를 참고하십시오:

{/* 이 페이지의 목차 테이블은 다음 스크립트에 의해 자동으로 생성됩니다.
  https://github.com/ClickHouse/clickhouse-docs/blob/main/scripts/autogenerate-table-of-contents.sh
  YAML 프런트 매터 필드(slug, description, title)에서 생성됩니다.

  오류를 발견한 경우 해당 페이지의 YML 프런트 매터를 수정하십시오.
  */ }

| Page                                                                        | Description                                                                                                              |
| --------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| [Atomic](/engines/database-engines/atomic)                                  | `Atomic` 엔진은 차단되지 않는 `DROP TABLE` 및 `RENAME TABLE` 쿼리와 원자적인 `EXCHANGE TABLES` 쿼리를 지원합니다. `Atomic` 데이터베이스 엔진이 기본으로 사용됩니다. |
| [Shared](/engines/database-engines/shared)                                  | ClickHouse Cloud에서 사용할 수 있는 `Shared` 데이터베이스 엔진을 설명하는 페이지입니다.                                                             |
| [Replicated](/engines/database-engines/replicated)                          | 이 엔진은 Atomic 엔진을 기반으로 합니다. ZooKeeper에 기록되는 DDL 로그를 통해 메타데이터 복제를 지원하며, 지정된 데이터베이스의 모든 레플리카에서 실행됩니다.                       |
| [PostgreSQL](/engines/database-engines/postgresql)                          | 원격 PostgreSQL 서버의 데이터베이스에 연결할 수 있습니다.                                                                                    |
| [MySQL](/engines/database-engines/mysql)                                    | 원격 MySQL 서버의 데이터베이스에 연결하여 `INSERT` 및 `SELECT` 쿼리를 수행하고 ClickHouse와 MySQL 간에 데이터를 교환할 수 있습니다.                             |
| [SQLite](/engines/database-engines/sqlite)                                  | SQLite 데이터베이스에 연결하여 `INSERT` 및 `SELECT` 쿼리를 수행하고 ClickHouse와 SQLite 간에 데이터를 교환할 수 있습니다.                                  |
| [Backup](/engines/database-engines/backup)                                  | 백업에 있는 테이블/데이터베이스를 읽기 전용 모드로 즉시 attach할 수 있습니다.                                                                          |
| [MaterializedPostgreSQL](/engines/database-engines/materialized-postgresql) | PostgreSQL 데이터베이스의 테이블로 구성된 ClickHouse 데이터베이스를 생성합니다.                                                                    |
| [DataLakeCatalog](/engines/database-engines/datalakecatalog)                | DataLakeCatalog 데이터베이스 엔진은 ClickHouse를 외부 데이터 카탈로그에 연결하고 오픈 테이블 형식 데이터를 쿼리할 수 있도록 지원합니다.                                 |
