# 특수 테이블 엔진

테이블 엔진에는 세 가지 주요 범주가 있습니다:

- [MergeTree 엔진 패밀리](../../../engines/table-engines/mergetree-family/index.md)는 주요 생산 사용을 위한 것입니다.
- [Log 엔진 패밀리](../../../engines/table-engines/log-family/index.md)는 작고 임시 데이터를 위한 것입니다.
- [통합을 위한 테이블 엔진](../../../engines/table-engines/integrations/index.md)입니다.

나머지 엔진은 그 목적이 독특하며 아직 패밀리로 그룹화되지 않았기 때문에 이 "특수" 범주에 배치됩니다.
