# ClickHouse Cloud 지원 서비스 \{#clickhouse-cloud-support-services\}

ClickHouse는 ClickHouse Cloud 사용자와 고객을 위해 지원 서비스를 제공합니다. ClickHouse 제품을 대표하는 지원 서비스 팀을 지향하며, 이는 타의 추종을 불허하는 성능, 사용 편의성, 그리고 예외적으로 빠르고 높은 품질의 결과를 의미합니다. 자세한 내용은 [ClickHouse Support Program](https://clickhouse.com/support/program/) 페이지를 방문하십시오.

[Cloud 콘솔에 로그인](https://console.clickhouse.cloud/support)한 후 메뉴에서 **Help -&gt; Support**를 선택하면 새로운 지원 케이스를 생성하고, 제출한 케이스의 상태를 확인할 수 있습니다.

또한 [Status 페이지](https://status.clickhouse.com)를 구독하여 플랫폼에 영향을 주는 인시던트에 대한 알림을 신속하게 받을 수 있습니다.

:::note
구독 고객에게만 지원 인시던트에 대한 서비스 수준 계약(SLA)이 적용됩니다. 현재 ClickHouse Cloud 사용자가 아니어도 질문에 답변하기 위해 노력하지만, 대신 다음과 같은 커뮤니티 리소스를 이용하시기를 권장합니다:

* [ClickHouse 커뮤니티 Slack 채널](https://clickhouse.com/slack)
* [기타 커뮤니티 옵션](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
  :::
