:::note
MySQL 및 MongoDB CDC ClickPipes는 General Availability(GA) 단계에 도달하기 전까지는 무료로 사용할 수 있습니다. 고객에게는 GA 출시 전에 사전 안내가 제공되며, 이때 ClickPipes 사용량을 검토하고 최적화할 수 있습니다.
:::

[ClickPipes](/integrations/clickpipes) 청구는 **컴퓨트 사용량**과 **수집된 데이터**를 기준으로 이루어집니다. 각 항목별 요금 모델에 대한 자세한 내용은 전용 청구 페이지를 참고하십시오.
