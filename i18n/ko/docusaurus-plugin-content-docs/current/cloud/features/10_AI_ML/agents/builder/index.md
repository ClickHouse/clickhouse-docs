Agent Builder는 에이전트를 생성하고 구성하는 곳입니다.

:::tip ClickHouse Agents가 처음이신가요?
[빠른 시작](/cloud/features/ai-ml/agents/quickstart)부터 시작하세요. 아래 페이지에서는 빌더의 각 구성 요소를 자세히 설명합니다.
:::

이 섹션에서는 Agent Builder의 각 구성 요소를 다루며, 다음 페이지로 이루어져 있습니다:

| 페이지                                                               | 설명                      |
| ----------------------------------------------------------------- | ----------------------- |
| [코드 인터프리터](/cloud/features/ai-ml/agents/builder/code-interpreter) | 샌드박스 환경에서 코드를 실행합니다.    |
| [웹 검색](/cloud/features/ai-ml/agents/builder/web-search)           | 공개 웹을 검색합니다.            |
| [이미지 생성](/cloud/features/ai-ml/agents/builder/image-generation)   | 텍스트로 이미지를 생성합니다.        |
| [비전](/cloud/features/ai-ml/agents/builder/vision)                 | 이미지 입력을 받을 수 있습니다.      |
| [MCP 서버](/cloud/features/ai-ml/agents/builder/mcp-servers)        | 에이전트에 타사 MCP 서버를 연결합니다. |
| [스킬](/cloud/features/ai-ml/agents/builder/skills)                 | 재사용 가능한 지침 모음입니다.       |
| [서브에이전트](/cloud/features/ai-ml/agents/builder/subagents)          | 작업을 하위 에이전트에 위임합니다.     |
