Agent Marketplace는 조직 내 다른 사용자가 공유한 에이전트를 찾아보거나 검색하고 실행할 수 있는 곳입니다. 또한 관리자가 조직 전체에 게시한 에이전트도 포함됩니다.

## 둘러보기 \{#browse\}

에이전트는 카테고리별로 분류되어 있습니다. 예를 들어 *Analytics*, *Operations*, *Engineering*이 있습니다. 마켓플레이스 상단의 탭을 사용해 카테고리 간에 전환하세요. 각 에이전트 카드에는 이름, 설명, 아바타와 대화를 바로 시작할 수 있는 빠른 실행 항목이 표시됩니다.

검색창을 사용해 모든 카테고리에서 이름, 설명 또는 키워드로 에이전트를 찾으세요.

## 에이전트 열기 \{#open-an-agent\}

에이전트 카드를 클릭하면 지침, 사용 중인 모델, 연결된 도구, 공유한 사용자 등 자세한 정보를 모두 확인할 수 있습니다. 상세 보기에서 다음 작업을 수행할 수 있습니다:

* **대화 시작** — 에이전트와 새 채팅을 엽니다.
* **사본 저장** — 편집 권한이 있으면 에이전트를 내 목록으로 복제해 수정할 수 있습니다.

## 직접 게시 \{#publish-your-own\}

마켓플레이스에서 에이전트를 찾을 수 있게 하려면 [공유 및 액세스](/cloud/features/ai-ml/agents/sharing-and-access) 패널에서 적절한 범위로 공유하십시오. 표시 수준은 다음과 같습니다.

* **비공개** — 본인만 볼 수 있습니다. 마켓플레이스에는 표시되지 않습니다.
* **사용자 또는 그룹과 공유** — 해당 사용자 또는 그룹에 표시됩니다. 해당 사용자의 마켓플레이스에 표시됩니다.
* **조직 전체** — 조직의 모든 사용자가 볼 수 있습니다. 관리자의 마켓플레이스 권한이 적용됩니다.

게시하기 전에 에이전트를 분류하고 설명을 명확하게 작성하십시오. 이 필드들은 검색과 카테고리별 탐색 환경 모두에 영향을 줍니다.

## 관리자 설정 \{#admin-controls\}

조직 관리자는 다음 작업을 수행할 수 있습니다:

* 마켓플레이스에 표시할 카테고리를 선별합니다.
* 개별 에이전트를 조직 전체 보기로 승격합니다.
* 역할별로 마켓플레이스 액세스를 허용하거나 제한합니다.

권한 모델은 [공유 및 액세스](/cloud/features/ai-ml/agents/sharing-and-access)를 참조하십시오.
