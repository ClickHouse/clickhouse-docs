ClickHouse Agents는 ClickHouse Cloud에서 제공되는 관리형 에이전트 기능입니다. 자연어로 분석 작업을 설명하면, 에이전트가 필요한 단계를 계획하고 Agent Builder에서 추가한 도구를 호출한 다음 ClickHouse 서비스에서 작업을 실행합니다.

이 기능은 베타 단계입니다. 동작 방식과 노출되는 기능 범위는 일반 제공 전에 변경될 수 있습니다.

## 수행할 수 있는 작업 \{#what-you-can-do\}

* 코드 없이 맞춤형 에이전트를 만들 수 있습니다 — 지침을 설정하고, 모델을 선택하고, 도구를 연결합니다.
* ClickHouse 서비스에서 대화를 실행할 수 있습니다. 에이전트는 필요에 따라 도구를 호출합니다.
* 에이전트를 팀원과 공유하거나 마켓플레이스에 게시할 수 있습니다.

## 이 섹션의 내용 \{#in-this-section\}

* [빠른 시작](/cloud/features/ai-ml/agents/quickstart) — 첫 번째 에이전트를 만들고 쿼리를 실행합니다.
* [채팅](/cloud/features/ai-ml/agents/chat) — 대화, 북마크, 포크, 다중 대화 및 공유.
* [Agent Builder](/cloud/features/ai-ml/agents/builder) — 에이전트를 생성하고 구성합니다. 모델 매개변수를 설정하고 도구, MCP 서버, skills 및 하위 에이전트를 추가합니다.
* [프롬프트](/cloud/features/ai-ml/agents/prompts) — 저장된 프롬프트 라이브러리.
* [메모리](/cloud/features/ai-ml/agents/memory) — 여러 대화에 걸쳐 유지되는 컨텍스트.
* [마켓플레이스](/cloud/features/ai-ml/agents/marketplace) — 조직 내에서 에이전트를 공유하고 찾아볼 수 있습니다.
* [공유 및 액세스](/cloud/features/ai-ml/agents/sharing-and-access) — 에이전트의 권한 모델.
