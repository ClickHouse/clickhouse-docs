ClickHouse Cloud가 처음이라 어디서부터 시작해야 할지 모르겠습니까? 이 문서 섹션에서는
빠르게 시작하는 데 필요한 모든 내용을 안내합니다. ClickHouse Cloud를 살펴보는 동안
프로세스의 각 단계를 따라갈 수 있도록 이 시작하기 섹션을 세 개의 하위 섹션으로
구성했습니다.

  * ClickHouse Cloud가 무엇인지, 그리고 오픈소스 버전과 어떻게 다른지 [알아보세요](/cloud/overview)
  * ClickHouse Cloud의 주요 사용 사례를 [살펴보세요](/cloud/get-started/cloud/use-cases/overview)

  ## ClickHouse Cloud 설정하기 \{#get-set-up-with-clickhouse-cloud\}

  이제 ClickHouse Cloud가 무엇인지 알게 되었으므로, 데이터를 ClickHouse Cloud로 가져오는 프로세스를
  안내하고, 사용할 수 있는 주요 기능을 보여드리며, 알아두면 좋은 일반적인 모범 사례를
  소개합니다.

  주제는 다음과 같습니다:

  * 다양한 플랫폼에서의 [마이그레이션 가이드](/integrations/migration/overview)
