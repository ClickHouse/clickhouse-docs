# 프라이빗 네트워킹 설정 \{#private-networking-setup\}

ClickHouse BYOC는 보안을 강화하고 서비스에 대한 직접 연결을 지원하기 위해 다양한 프라이빗 네트워킹 옵션을 제공합니다. 이 가이드에서는 자체 AWS 또는 GCP 계정에서 실행되는 ClickHouse Cloud 배포를 내부 애플리케이션이나 분석 도구와 같은 다른 네트워크 또는 서비스에 안전하게 연결하기 위한 권장 방법을 안내합니다. VPC 피어링, AWS PrivateLink, GCP Private Service Connect와 같은 옵션을 다루며, 각 옵션의 주요 단계와 고려 사항을 설명합니다.

ClickHouse BYOC 배포에 프라이빗 네트워크 연결이 필요한 경우, 각 가이드의 단계를 따르거나 더 복잡한 시나리오에 대한 지원이 필요하면 ClickHouse Support에 문의하십시오.
