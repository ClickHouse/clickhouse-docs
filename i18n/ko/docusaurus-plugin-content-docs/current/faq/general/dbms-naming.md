# "ClickHouse"는 무슨 뜻을 의미합니까? \{#what-does-clickhouse-mean\}

"**Click**stream"과 "Data ware**House**"를 조합해 만든 이름입니다. Yandex.Metrica의 최초 사용 사례에서 비롯되었으며, 전 세계 인터넷 사용자들의 모든 클릭 기록을 보관하는 역할을 위해 ClickHouse가 설계되었고, 지금도 그 역할을 수행하고 있습니다. 이 사용 사례에 대해 더 알고 싶다면 [ClickHouse history](../../about-us/history.md) 페이지를 참고하십시오.

이 두 부분으로 이루어진 이름에는 두 가지 의미상의 특징이 있습니다:

- Click**H**ouse를 쓸 때는 H를 대문자로 쓰는 것이 올바른 표기입니다.
- 약어가 필요하다면 **CH**를 사용하십시오. 역사적인 이유로, 중국에서는 CK라는 약어도 널리 쓰이는데, 이는 ClickHouse에 대해 중국어로 진행된 초기 발표 중 하나에서 이 표기를 사용했기 때문입니다.

:::info    
ClickHouse라는 이름이 붙은 지 수년 후, 이렇게 서로 독립적으로 의미를 가지는 두 단어를 결합하는 방식이 데이터베이스 이름을 짓는 최선의 방법으로 강조되었습니다. 이는 Carnegie Mellon University의 데이터베이스 부교수인 Andy Pavlo가 작성한 [연구](https://www.cs.cmu.edu/~pavlo/blog/2020/03/on-naming-a-database-management-system.html)에서 제시된 내용입니다. ClickHouse는 Postgres와 함께 그의 「역대 최고의 데이터베이스 이름」 상을 공동 수상했습니다.
:::
