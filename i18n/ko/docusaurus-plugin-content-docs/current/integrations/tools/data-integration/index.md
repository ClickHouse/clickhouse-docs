# 데이터 통합 \{#data-integrations\}

| Page      | Description                                                                                                                     |
|-----------|---------------------------------------------------------------------------------------------------------------------------------|
| [Easypanel](/integrations/easypanel) | Easypanel을 사용하면 자체 서버에 ClickHouse를 배포할 수 있습니다.                                                                    |
| [Retool](/integrations/retool)    | 풍부한 사용자 인터페이스를 갖춘 웹 및 모바일 앱을 빠르게 구축하고 복잡한 작업을 자동화하며 AI를 통합할 수 있으며, 이 모든 기능은 사용자의 데이터로 구동됩니다. |
| [Splunk](/integrations/audit-splunk)     | ClickHouse Cloud 감사 로그를 Splunk에 저장할 수 있습니다.                                                                                  |
