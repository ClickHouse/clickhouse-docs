# 데이터 통합

| 페이지      | 설명                                                                                                                         |
|-----------|----------------------------------------------------------------------------------------------------------------------------|
| [Easypanel](/integrations/easypanel) | Easypanel은 자체 서버에서 ClickHouse를 배포할 수 있도록 해줍니다.                                                               |
| [Retool](/integrations/retool)    | 풍부한 사용자 인터페이스를 갖춘 웹 및 모바일 앱을 신속하게 구축하고, 복잡한 작업을 자동화하며, AI를 통합합니다. 모든 것은 귀하의 데이터로 구동됩니다. |
| [Splunk](/integrations/audit-splunk)     | ClickHouse Cloud 감사 로그를 Splunk에 저장합니다.                                                                             |
