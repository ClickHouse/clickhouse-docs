# Postgres용 ClickPipes 점검 시간대 \{#maintenance-windows-for-clickpipes-for-postgres\}

Postgres ClickPipes에 대한 예정된 점검 시간대는 다음과 같습니다:

* **날짜:** 2025년 4월 17일
* **시간:** 오전 07:00 - 오전 08:00 (UTC)

이 시간 동안 Postgres ClickPipes는 짧은 중단이 발생합니다.
점검 시간대 이후 ClickPipes는 다시 사용 가능해지며 정상적으로 운영을 재개합니다.
