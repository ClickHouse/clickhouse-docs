# Fivetran 및 ClickHouse Cloud

<ClickHouseSupportedBadge/>

## 개요 {#overview}

[Fivetran](https://www.fivetran.com)은 클라우드 데이터 플랫폼에서 데이터 이동을 자동화하는 플랫폼으로, 데이터를 클라우드 데이터 플랫폼으로 이동시키거나 그 사이에서 이동시킵니다.

[ClickHouse Cloud](https://clickhouse.com/cloud)는 [Fivetran 목적지](https://fivetran.com/docs/destinations/clickhouse)로 지원되어, 사용자가 다양한 소스에서 ClickHouse로 데이터를 로드할 수 있게 합니다.

:::note
[ClickHouse Cloud 목적지](https://fivetran.com/docs/destinations/clickhouse)는 현재 비공식 미리보기 상태입니다. 문제 발생 시 ClickHouse 지원팀에 문의하시기 바랍니다.
:::
