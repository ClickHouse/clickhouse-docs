# 통합 \{#integrations\}

ClickHouse를 선호하는 도구 및 서비스와 연동하여 강력한 데이터 파이프라인과 분석 워크플로우를 구축합니다.

ClickHouse 통합은 지원 수준에 따라 다음과 같이 구분됩니다:

| 계층 | 설명 |
|------|-------------|
| 핵심 통합(Core integrations) | ClickHouse에서 개발하거나 유지 관리하며, ClickHouse에서 지원하고 ClickHouse GitHub 조직에 포함되어 있습니다 |
| 파트너 통합(Partner integrations) | 서드파티 소프트웨어 공급업체에서 개발하거나 유지 관리하고 지원합니다 |
| 커뮤니티 통합(Community integrations) | 커뮤니티 구성원이 개발하거나 유지 관리하고 지원합니다. 공개 GitHub 리포지토리와 커뮤니티 Slack 채널에서 제공되는 것 외에는 별도의 직접 지원은 제공되지 않습니다 |

<IntegrationGrid />

:::note 안내
서드파티 로고와 상표는 각 소유권자에게 귀속되며, 사용 가능한 통합을 표시하기 위한 목적으로만 표시됩니다.
이는 어떠한 보증이나 추천을 의미하지 않습니다.
:::
