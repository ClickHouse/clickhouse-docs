## 설명 {#description}

이 형식에서 ClickHouse는 각 행을 구분된 줄 바꿈 JSON 객체로 출력합니다.

## 사용 예 {#example-usage}

데이터를 삽입합니다:

```sql
INSERT INTO football FROM INFILE 'football.json' FORMAT JSONEachRow;
```

### 데이터 읽기 {#reading-data}

`JSONEachRow` 형식을 사용하여 데이터를 읽습니다:

```sql
SELECT *
FROM football
FORMAT JSONEachRow
```
