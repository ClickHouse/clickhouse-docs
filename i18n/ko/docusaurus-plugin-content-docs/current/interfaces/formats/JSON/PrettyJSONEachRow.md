## 설명 \{#description\}

[JSONEachRow](./JSONEachRow.md)와의 차이점은 JSON이 줄 바꿈 구분자와 4칸 공백 들여쓰기를 사용하는 Pretty 형식으로 포맷팅된다는 점뿐입니다.

### 데이터 읽기 \{#reading-data\}

`PrettyJSONEachRow` 형식을 사용하여 데이터를 읽습니다:

```sql
SELECT *
FROM football
FORMAT PrettyJSONEachRow
```
