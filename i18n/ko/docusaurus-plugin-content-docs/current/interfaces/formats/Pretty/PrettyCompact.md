## 설명 {#description}

[`Pretty`](./Pretty.md) 형식과 다르게, 이 형식은 행 사이에 그리드가 그려진 테이블로 표시됩니다. 
이로 인해 결과가 더 컴팩트해집니다.

:::note
이 형식은 대화형 모드의 명령줄 클라이언트에서 기본적으로 사용됩니다.
:::
