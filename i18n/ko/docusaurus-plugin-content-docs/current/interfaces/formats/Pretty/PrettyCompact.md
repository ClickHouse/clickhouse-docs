## 설명 \{#description\}

테이블의 행 사이에 격자가 그려진 형태로 표시된다는 점에서 [`Pretty`](./Pretty.md) 형식과 다릅니다. 
그래서 결과가 더 간결하게 표시됩니다.

:::note
이 형식은 대화형 모드의 명령줄 클라이언트에서 기본 형식으로 사용됩니다.
:::
