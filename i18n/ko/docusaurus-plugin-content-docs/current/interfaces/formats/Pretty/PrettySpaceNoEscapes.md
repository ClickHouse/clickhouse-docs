## 설명 {#description}

[`PrettySpace`](./PrettySpace.md) 형식과 다르며 [ANSI-escape 시퀀스](http://en.wikipedia.org/wiki/ANSI_escape_code)가 사용되지 않습니다. 
이는 브라우저에서 이 형식을 표시하고 'watch' 명령줄 유틸리티를 사용하는 데 필요합니다.
