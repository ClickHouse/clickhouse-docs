## 설명 \{#description\}

[ANSI-escape sequences](http://en.wikipedia.org/wiki/ANSI_escape_code)를 사용하지 않는다는 점에서 [`PrettySpace`](./PrettySpace.md) 형식과 다릅니다. 
이는 브라우저에서 이 형식을 표시하거나 `watch` 명령줄 유틸리티에서 사용하기 위해 필요합니다.
