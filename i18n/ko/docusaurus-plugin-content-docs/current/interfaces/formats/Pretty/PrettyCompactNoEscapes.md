## 설명 \{#description\}

[`PrettyCompact`](./PrettyCompact.md) 형식과 다른 점은 [ANSI 이스케이프 시퀀스](http://en.wikipedia.org/wiki/ANSI_escape_code)를 사용하지 않는다는 것입니다.  
이는 이 형식을 브라우저에 표시하고 `watch` 명령줄 유틸리티에서 사용하기 위해 필요합니다.
