## Description {#description}

[`PrettyCompact`](./PrettyCompact.md) 형식과 다르며 [ANSI-escape sequences](http://en.wikipedia.org/wiki/ANSI_escape_code)가 사용되지 않습니다. 
이것은 브라우저에서 형식을 표시하고, 'watch' 커맨드라인 유틸리티를 사용하는 데 필요합니다.
