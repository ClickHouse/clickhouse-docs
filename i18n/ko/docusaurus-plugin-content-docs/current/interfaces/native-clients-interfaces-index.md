# 네이티브 클라이언트 및 인터페이스 \{#native-clients-interfaces\}

ClickHouse는 ClickHouse에 연결하기 위한 여러 네이티브 클라이언트와 인터페이스를 제공합니다.

자세한 내용은 아래 페이지를 참조하십시오:

| Section                                                      | Summary                                                                                  |
|--------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [Command-Line Client](/interfaces/cli)               | 명령줄 옵션과 설정 파일을 지원하는 네이티브 명령줄 클라이언트입니다.                       |
| [Drivers &amp; Interfaces](/interfaces/overview)         | 여러 네트워크 인터페이스, 라이브러리 및 시각적 인터페이스를 제공합니다.                    |
| [SQL Console](/integrations/sql-clients/sql-console) | ClickHouse Cloud의 데이터를 빠르고 쉽게 조회하고 조작할 수 있는 방법입니다.               |
