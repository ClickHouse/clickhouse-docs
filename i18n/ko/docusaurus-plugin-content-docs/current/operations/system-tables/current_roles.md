## 설명 \{#description\}

현재 사용자의 활성 역할을 포함합니다. `SET ROLE`은 이 테이블의 내용을 변경합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `role_name` ([String](../../sql-reference/data-types/)) — 역할 이름입니다.
* `with_admin_option` ([UInt8](../../sql-reference/data-types/)) — 역할에 ADMIN OPTION privilege가 있으면 1입니다.
* `is_default` ([UInt8](../../sql-reference/data-types/)) — 역할이 기본으로 설정되어 있으면 1입니다.

{/*AUTOGENERATED_END*/ }
