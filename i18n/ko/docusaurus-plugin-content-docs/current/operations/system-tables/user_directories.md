## 설명 \{#description\}

구성된 사용자 디렉터리, 즉 ClickHouse 서버가 사용자 제공 데이터를 읽을 수 있도록 허용된 파일 시스템상의 디렉터리에 대한 정보를 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 디렉터리 이름입니다.
* `type` ([String](../../sql-reference/data-types/)) — 액세스 저장소의 유형입니다. 예: users.xml, 복제된 또는 memory 등입니다.
* `params` ([String](../../sql-reference/data-types/)) — 액세스 저장소의 매개변수가 포함된 JSON입니다.
* `precedence` ([UInt64](../../sql-reference/data-types/)) — 이 디렉터리가 config에 선언된 순서입니다. ClickHouse가 사용자 또는 역할을 찾을 때도 동일한 순서가 사용됩니다.

{/*AUTOGENERATED_END*/ }
