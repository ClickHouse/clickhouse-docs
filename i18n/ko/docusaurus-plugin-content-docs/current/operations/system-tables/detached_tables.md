각 분리(detached)된 테이블에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 테이블이 속한 데이터베이스의 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블 이름입니다.
* `uuid` ([UUID](../../sql-reference/data-types/)) — 테이블 UUID입니다(Atomic 데이터베이스).
* `metadata_path` ([String](../../sql-reference/data-types/)) — 파일 시스템에서 테이블 메타데이터가 저장된 경로입니다.
* `is_permanently` ([UInt8](../../sql-reference/data-types/)) — 테이블이 영구적으로 분리(detach)되었는지 여부입니다.

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
Row 1:
──────
database:                   base
table:                      t1
uuid:                       81b1c20a-b7c6-4116-a2ce-7583fb6b6736
metadata_path:              /var/lib/clickhouse/store/461/461cf698-fd0b-406d-8c01-5d8fd5748a91/t1.sql
is_permanently:             1
```
