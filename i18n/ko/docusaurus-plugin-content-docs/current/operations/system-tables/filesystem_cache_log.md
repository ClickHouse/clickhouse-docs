원격 파일 시스템의 객체와 관련해 파일 시스템 캐시에서 발생한 모든 이벤트의 이력을 포함합니다.

이 테이블은 언제든지 안전하게 truncate 또는 drop할 수 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 호스트명
* `event_date` ([Date](/sql-reference/data-types/date)) — 이벤트 날짜
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 이벤트 시간
* `query_id` ([String](/sql-reference/data-types/string)) — 쿼리 ID
* `source_file_path` ([String](/sql-reference/data-types/string)) — 파일 시스템의 파일 세그먼트 경로
* `file_segment_range` ([Tuple(UInt64, UInt64)](/sql-reference/data-types/tuple)) — 파일 세그먼트 범위
* `total_requested_range` ([Tuple(UInt64, UInt64)](/sql-reference/data-types/tuple)) — 전체 읽기 범위
* `key` ([String](/sql-reference/data-types/string)) — 파일 세그먼트 키
* `offset` ([UInt64](/sql-reference/data-types/int-uint)) — 파일 세그먼트 오프셋
* `size` ([UInt64](/sql-reference/data-types/int-uint)) — 읽기 크기
* `read_type` ([String](/sql-reference/data-types/string)) — 읽기 유형: READ&#95;FROM&#95;CACHE, READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE, READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/sql-reference/data-types/int-uint)) — 캐시에서 읽기를 시도했는지 여부
* `read_buffer_id` ([String](/sql-reference/data-types/string)) — 내부 구현의 읽기 버퍼 ID
* `user_id` ([String](/sql-reference/data-types/string)) — 파일 세그먼트를 생성한 사용자의 ID
