## 설명 \{#description\}

사용 가능한 인증서와 해당 출처에 대한 정보를 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `version` ([Int32](../../sql-reference/data-types/)) — 인증서 버전입니다. 값은 v1은 0, v2는 1, v3는 2입니다.
* `serial_number` ([Nullable(String)](../../sql-reference/data-types/)) — 발급자가 인증서에 부여한 일련번호입니다.
* `signature_algo` ([Nullable(String)](../../sql-reference/data-types/)) — 서명 알고리즘으로, 발급자가 이 인증서에 서명할 때 사용하는 알고리즘입니다.
* `issuer` ([Nullable(String)](../../sql-reference/data-types/)) — 발급자로, 이 인증서를 발급한 인증 기관(CA)의 고유 식별자입니다.
* `not_before` ([Nullable(String)](../../sql-reference/data-types/)) — 이 인증서의 유효 기간이 시작되는 시점입니다.
* `not_after` ([Nullable(String)](../../sql-reference/data-types/)) — 이 인증서의 유효 기간이 종료되는 시점입니다.
* `subject` ([Nullable(String)](../../sql-reference/data-types/)) — 주체로, 공개 키의 소유자를 식별합니다.
* `pkey_algo` ([Nullable(String)](../../sql-reference/data-types/)) — 공개 키 알고리즘으로, 해당 공개 키와 함께 사용할 수 있는 알고리즘을 정의합니다.
* `path` ([String](../../sql-reference/data-types/)) — 이 인증서를 포함하는 파일 또는 디렉터리의 경로입니다.
* `default` ([UInt8](../../sql-reference/data-types/)) — 인증서가 기본 인증서 위치에 있음을 나타냅니다.
* `protocol` ([String](../../sql-reference/data-types/)) — 인증서가 프로토콜별 TLS 구성에서 가져온 경우 프로토콜 이름이며, 그렇지 않으면 비어 있습니다.
