[*GraphiteMergeTree](../../engines/table-engines/mergetree-family/graphitemergetree.md) 엔진을 사용하는 테이블에서 사용되는 [graphite&#95;rollup](../../operations/server-configuration-parameters/settings.md#graphite) 매개변수에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `config_name` ([String](../../sql-reference/data-types/)) — graphite&#95;rollup 매개변수 이름입니다.
* `rule_type` ([String](../../sql-reference/data-types/)) — 규칙 유형입니다. 가능한 값은 다음과 같습니다: RuleTypeAll = 0 - 기본값, 정규식 사용, 기존 스키마와 호환됨; RuleTypePlain = 1 - 일반 메트릭, 정규식 사용, 기존 스키마와 호환됨; RuleTypeTagged = 2 - 태그된 메트릭, 정규식 사용, 기존 스키마와 호환됨; RuleTypeTagList = 3 - 태그된 메트릭, 정규식 사용(문자열 &#39;retention=10min ; env=(staging|prod)&#39;에서 RuleTypeTagged로 변환됨).
* `regexp` ([String](../../sql-reference/data-types/)) — 메트릭 이름에 대한 패턴입니다.
* `function` ([String](../../sql-reference/data-types/)) — 집계 FUNCTION의 이름입니다.
* `age` ([UInt64](../../sql-reference/data-types/)) — 데이터의 최소 경과 시간(초)입니다.
* `precision` ([UInt64](../../sql-reference/data-types/)) — 데이터 경과 시간을 초 단위로 어느 정도 정밀도로 지정하는지입니다.
* `priority` ([UInt16](../../sql-reference/data-types/)) — 패턴 우선순위입니다.
* `is_default` ([UInt8](../../sql-reference/data-types/)) — 패턴이 기본값인지 여부입니다.
* `Tables.database` ([Array(String)](../../sql-reference/data-types/)) — `config_name` 매개변수를 사용하는 데이터베이스 테이블 이름의 배열입니다.
* `Tables.table` ([Array(String)](../../sql-reference/data-types/)) — `config_name` 매개변수를 사용하는 테이블 이름의 배열입니다.
