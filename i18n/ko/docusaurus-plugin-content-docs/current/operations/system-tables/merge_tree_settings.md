# system.merge_tree_settings \{#systemmerge_tree_settings\}

`MergeTree` 테이블 설정에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 설정 이름입니다.
* `value` ([String](../../sql-reference/data-types/)) — 설정값입니다.
* `default` ([String](../../sql-reference/data-types/)) — 설정 기본값입니다.
* `changed` ([UInt8](../../sql-reference/data-types/)) — 설정이 구성(config)에서 명시적으로 정의되었거나 명시적으로 변경된 경우 1입니다.
* `description` ([String](../../sql-reference/data-types/)) — 설정 설명입니다.
* `min` ([Nullable(String)](../../sql-reference/data-types/)) — 제약 조건(constraints)을 통해 설정된 경우 설정의 최소값입니다. 설정에 최소값이 없으면 NULL을 포함합니다.
* `max` ([Nullable(String)](../../sql-reference/data-types/)) — 제약 조건(constraints)을 통해 설정된 경우 설정의 최대값입니다. 설정에 최대값이 없으면 NULL을 포함합니다.
* `disallowed_values` ([Array(String)](../../sql-reference/data-types/)) — 허용되지 않는 값 목록입니다.
* `readonly` ([UInt8](../../sql-reference/data-types/)) — 현재 사용자가 설정을 변경할 수 있는지 여부를 나타냅니다: 0 — 현재 사용자가 설정을 변경할 수 있음, 1 — 현재 사용자가 설정을 변경할 수 없음.
* `type` ([String](../../sql-reference/data-types/)) — 설정 타입(구현별 문자열 값)입니다.
* `is_obsolete` ([UInt8](../../sql-reference/data-types/)) — 설정이 더 이상 사용되지 않는지 여부를 나타냅니다.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](../../sql-reference/data-types/)) —
  이 기능에 대한 지원 수준입니다. ClickHouse 기능은 계층(tier)으로 구성되며, 각 계층은 기능의 현재 개발 상태와 사용 시 기대할 수 있는 특성에 따라 구분됩니다:

- PRODUCTION: 기능이 안정적이며 사용해도 안전하고, 다른 PRODUCTION 기능과 상호 작용하는 데 문제가 없습니다.
- BETA: 기능이 안정적이고 안전합니다. 다른 기능과 함께 사용했을 때의 결과는 알려져 있지 않으며, 정확성이 보장되지 않습니다. 테스트 및 보고를 환영합니다.
- EXPERIMENTAL: 기능이 개발 중입니다. 개발자와 ClickHouse 열성 사용자를 위한 것입니다. 기능이 동작할 수도, 동작하지 않을 수도 있으며 언제든지 제거될 수 있습니다.
- OBSOLETE: 더 이상 지원되지 않습니다. 이미 제거되었거나 향후 릴리스에서 제거될 예정입니다.

{/*AUTOGENERATED_END*/ }
