# system.users \{#systemusers\}

서버에 설정된 [사용자 계정](../../guides/sre/user-management/index.md#user-account-management) 목록입니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 사용자 이름.
* `id` ([UUID](../../sql-reference/data-types/)) — 사용자 ID.
* `storage` ([String](../../sql-reference/data-types/)) — 사용자 스토리지 경로. `access_control_path` 파라미터에서 설정합니다.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](../../sql-reference/data-types/)) — 인증 유형을 나타냅니다. 사용자 식별에는 여러 방식이 있으며, 비밀번호 없음, 평문 비밀번호, SHA256으로 인코딩된 비밀번호, 이중 SHA-1로 인코딩된 비밀번호, bcrypt로 인코딩된 비밀번호 등을 사용할 수 있습니다.
* `auth_params` ([Array(String)](../../sql-reference/data-types/)) — `auth_type`에 따라 JSON 형식으로 표현되는 인증 파라미터.
* `valid_until` ([Array(DateTime)](../../sql-reference/data-types/)) — 사용자 자격 증명의 만료 일시.
* `host_ip` ([Array(String)](../../sql-reference/data-types/)) — ClickHouse 서버에 접속이 허용된 호스트의 IP 주소.
* `host_names` ([Array(String)](../../sql-reference/data-types/)) — ClickHouse 서버에 접속이 허용된 호스트 이름.
* `host_names_regexp` ([Array(String)](../../sql-reference/data-types/)) — ClickHouse 서버에 접속이 허용된 호스트 이름에 대한 정규 표현식.
* `host_names_like` ([Array(String)](../../sql-reference/data-types/)) — `LIKE` 술어를 사용해 설정된, ClickHouse 서버에 접속이 허용된 호스트 이름.
* `default_roles_all` ([UInt8](../../sql-reference/data-types/)) — 부여된 모든 역할이 사용자에 대해 기본값으로 설정되어 있음을 나타냅니다.
* `default_roles_list` ([Array(String)](../../sql-reference/data-types/)) — 기본값으로 부여되는 역할 목록.
* `default_roles_except` ([Array(String)](../../sql-reference/data-types/)) — 나열된 역할을 제외하고 기본값으로 설정된 모든 부여된 역할.
* `grantees_any` ([UInt8](../../sql-reference/data-types/)) — GRANT 옵션을 가진 사용자가 이를 임의의 대상에게 부여할 수 있는지를 나타내는 플래그입니다.
* `grantees_list` ([Array(String)](../../sql-reference/data-types/)) — 이 사용자가 GRANT 옵션을 부여할 수 있는 사용자 또는 역할 목록.
* `grantees_except` ([Array(String)](../../sql-reference/data-types/)) — 이 사용자에게 GRANT 옵션 부여가 금지된 사용자 또는 역할 목록.
* `default_database` ([String](../../sql-reference/data-types/)) — 이 사용자의 기본 데이터베이스 이름.
