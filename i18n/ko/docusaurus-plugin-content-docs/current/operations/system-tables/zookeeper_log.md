## 설명 \{#description\}

이 테이블에는 ZooKeeper 서버에 대한 요청의 파라미터와 그에 대한 응답의 파라미터 정보가 저장됩니다.

요청 시에는 요청 파라미터에 해당하는 컬럼만 채워지며, 나머지 컬럼은 기본값(`0` 또는 `NULL`)으로 채워집니다. 응답이 도착하면 응답에서 가져온 데이터가 나머지 컬럼에 추가됩니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 쿼리를 실행하는 서버의 호스트명입니다.
* `type` ([Enum8(&#39;Request&#39; = 1, &#39;Response&#39; = 2, &#39;Finalize&#39; = 3)](/sql-reference/data-types/enum)) — ZooKeeper client의 이벤트 유형입니다. 다음 값 중 하나를 가질 수 있습니다: Request — 요청이 전송됨, Response — 응답을 수신함, Finalize — 연결이 끊겨 응답을 수신하지 못함.
* `event_date` ([Date](/sql-reference/data-types/date)) — 이벤트가 발생한 날짜입니다.
* `event_time` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 이벤트가 발생한 날짜와 시간입니다.
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — 이 요청을 실행한 스레드의 ID입니다.
* `query_id` ([String](/sql-reference/data-types/string)) — 이 요청이 실행된 쿼리의 ID입니다.
* `address` ([IPv6](/sql-reference/data-types/ipv6)) — 요청에 사용된 ZooKeeper 서버의 IP 주소입니다.
* `port` ([UInt16](/sql-reference/data-types/int-uint)) — 요청에 사용된 ZooKeeper 서버의 포트입니다.
* `session_id` ([Int64](/sql-reference/data-types/int-uint)) — ZooKeeper 서버가 각 연결에 대해 설정하는 세션 ID입니다.
* `duration_microseconds` ([UInt64](/sql-reference/data-types/int-uint)) — ZooKeeper가 요청을 처리하는 데 걸린 시간입니다.
* `xid` ([Int64](/sql-reference/data-types/int-uint)) — 세션 내 요청의 ID입니다. 일반적으로 순차적인 요청 번호입니다. 요청 행과 짝을 이루는 응답/finalize 행에서 동일한 값을 가집니다.
* `has_watch` ([UInt8](/sql-reference/data-types/int-uint)) — watch가 설정되었는지 여부를 나타냅니다.
* `op_num` ([Enum16(&#39;Close&#39; = -11, &#39;Error&#39; = -1, &#39;Watch&#39; = 0, &#39;Create&#39; = 1, &#39;Remove&#39; = 2, &#39;Exists&#39; = 3, &#39;Get&#39; = 4, &#39;Set&#39; = 5, &#39;GetACL&#39; = 6, &#39;SetACL&#39; = 7, &#39;SimpleList&#39; = 8, &#39;Sync&#39; = 9, &#39;Heartbeat&#39; = 11, &#39;List&#39; = 12, &#39;Check&#39; = 13, &#39;Multi&#39; = 14, &#39;Create2&#39; = 15, &#39;Reconfig&#39; = 16, &#39;CheckWatch&#39; = 17, &#39;RemoveWatch&#39; = 18, &#39;MultiRead&#39; = 22, &#39;Auth&#39; = 100, &#39;SetWatch&#39; = 101, &#39;SetWatch2&#39; = 105, &#39;AddWatch&#39; = 106, &#39;FilteredList&#39; = 500, &#39;CheckNotExists&#39; = 501, &#39;CreateIfNotExists&#39; = 502, &#39;RemoveRecursive&#39; = 503, &#39;CheckStat&#39; = 504, &#39;TryRemove&#39; = 505, &#39;FilteredListWithStatsAndData&#39; = 506, &#39;ListRecursive&#39; = 507, &#39;SessionID&#39; = 997)](/sql-reference/data-types/enum)) — 요청 또는 응답 유형입니다.
* `path` ([String](/sql-reference/data-types/string)) — 요청에서 지정한 ZooKeeper 노드의 경로 또는 요청에 경로 지정이 필요하지 않은 경우 빈 문자열입니다.
* `data` ([String](/sql-reference/data-types/string)) — ZooKeeper 노드에 기록되는 데이터입니다(SET 및 CREATE 요청의 경우 요청에서 기록하려는 내용, GET 요청에 대한 응답의 경우 읽은 내용). 또는 빈 문자열입니다.
* `is_ephemeral` ([UInt8](/sql-reference/data-types/int-uint)) — ZooKeeper 노드가 임시 노드(ephemeral node)로 생성되는지 여부입니다.
* `is_sequential` ([UInt8](/sql-reference/data-types/int-uint)) — ZooKeeper 노드가 sequential 모드로 생성되는지 여부입니다.
* `version` ([널 허용(Int32)](/sql-reference/data-types/nullable)) — 요청 실행 시 예상하는 ZooKeeper 노드의 버전입니다. CHECK, SET, REMOVE 요청에서 지원됩니다(요청이 버전 검사를 수행하지 않으면 -1이며, 버전 검사를 지원하지 않는 다른 요청에서는 NULL입니다).
* `requests_size` ([UInt32](/sql-reference/data-types/int-uint)) — 다중 요청에 포함된 요청의 수입니다(다중 요청은 여러 개의 연속된 일반 요청으로 구성되며, 이를 원자적으로 실행하는 특수 요청입니다). 다중 요청에 포함된 모든 요청은 동일한 xid를 가집니다.
* `request_idx` ([UInt32](/sql-reference/data-types/int-uint)) — 다중 요청에 포함된 요청 번호입니다(다중 요청의 경우 0이고, 이후에는 1부터 순서대로 증가합니다).
* `zxid` ([Int64](/sql-reference/data-types/int-uint)) — ZooKeeper 트랜잭션 ID입니다. 성공적으로 처리된 요청에 대해 ZooKeeper 서버가 발급하는 일련번호입니다(요청이 실행되지 않았거나, 오류를 반환했거나, 클라이언트가 요청의 실행 여부를 알 수 없는 경우 0).
* `error` ([Nullable(Enum8(&#39;ZNOWATCHER&#39; = -121, &#39;ZNOTREADONLY&#39; = -119, &#39;ZSESSIONMOVED&#39; = -118, &#39;ZNOTHING&#39; = -117, &#39;ZCLOSING&#39; = -116, &#39;ZAUTHFAILED&#39; = -115, &#39;ZINVALIDACL&#39; = -114, &#39;ZINVALIDCALLBACK&#39; = -113, &#39;ZSESSIONEXPIRED&#39; = -112, &#39;ZNOTEMPTY&#39; = -111, &#39;ZNODEEXISTS&#39; = -110, &#39;ZNOCHILDRENFOREPHEMERALS&#39; = -108, &#39;ZBADVERSION&#39; = -103, &#39;ZNOAUTH&#39; = -102, &#39;ZNONODE&#39; = -101, &#39;ZAPIERROR&#39; = -100, &#39;ZOUTOFMEMORY&#39; = -10, &#39;ZINVALIDSTATE&#39; = -9, &#39;ZBADARGUMENTS&#39; = -8, &#39;ZOPERATIONTIMEOUT&#39; = -7, &#39;ZUNIMPLEMENTED&#39; = -6, &#39;ZMARSHALLINGERROR&#39; = -5, &#39;ZCONNECTIONLOSS&#39; = -4, &#39;ZDATAINCONSISTENCY&#39; = -3, &#39;ZRUNTIMEINCONSISTENCY&#39; = -2, &#39;ZSYSTEMERROR&#39; = -1, &#39;ZOK&#39; = 0))](/sql-reference/data-types/nullable)) — 오류 코드입니다. 다양한 값을 가질 수 있으며, 여기에 그중 일부를 표시합니다: ZOK — 요청이 성공적으로 실행되었습니다, ZCONNECTIONLOSS — 연결이 끊어졌습니다, ZOPERATIONTIMEOUT — 요청 실행 타임아웃이 초과되었습니다, ZSESSIONEXPIRED — 세션이 만료되었습니다, NULL — 요청이 완료되었습니다.
* `watch_type` ([널 허용(Enum8(&#39;NOTWATCHING&#39; = -2, &#39;SESSION&#39; = -1, &#39;CREATED&#39; = 1, &#39;DELETED&#39; = 2, &#39;CHANGED&#39; = 3, &#39;CHILD&#39; = 4))](/sql-reference/data-types/nullable)) — watch 이벤트의 유형입니다(`op&#95;num = Watch`인 응답의 경우). 그 외 응답에서는 `NULL`입니다.
* `watch_state` ([Nullable(Enum16(&#39;AUTH&#95;FAILED&#39; = -113, &#39;EXPIRED&#95;SESSION&#39; = -112, &#39;CONNECTING&#39; = 1, &#39;ASSOCIATING&#39; = 2, &#39;CONNECTED&#39; = 3, &#39;READONLY&#39; = 5, &#39;NOTCONNECTED&#39; = 999))](/sql-reference/data-types/nullable)) — watch 이벤트의 상태입니다(`op&#95;num = Watch`인 응답의 경우). 그 외 응답에서는 `NULL`입니다.
* `path_created` ([String](/sql-reference/data-types/string)) — 생성된 ZooKeeper 노드의 경로입니다(CREATE 요청에 대한 응답). 노드가 sequential로 생성된 경우 이 경로는 path와 다를 수 있습니다.
* `stat_czxid` ([Int64](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드가 생성되게 한 변경의 zxid입니다.
* `stat_mzxid` ([Int64](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드를 마지막으로 수정한 변경 작업의 zxid입니다.
* `stat_pzxid` ([Int64](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드의 자식 노드를 마지막으로 수정한 변경 작업의 트랜잭션 ID입니다.
* `stat_version` ([Int32](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드 데이터의 변경 횟수입니다.
* `stat_cversion` ([Int32](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드의 자식 노드에 발생한 변경 횟수입니다.
* `stat_dataLength` ([Int32](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드에 있는 데이터 필드의 길이입니다.
* `stat_numChildren` ([Int32](/sql-reference/data-types/int-uint)) — 이 ZooKeeper 노드의 하위 노드 수입니다.
* `children` ([Array(String)](/sql-reference/data-types/array)) — 하위 ZooKeeper 노드 목록입니다(LIST 요청에 대한 응답).

{/*AUTOGENERATED_END*/ }
