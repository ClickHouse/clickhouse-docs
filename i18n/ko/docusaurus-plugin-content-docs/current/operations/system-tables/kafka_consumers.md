Kafka 컨슈머에 대한 정보를 포함합니다.
[Kafka 테이블 엔진](../../engines/table-engines/integrations/kafka) (ClickHouse 네이티브 통합)에 적용됩니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — Kafka Engine 테이블이 속한 데이터베이스.
* `table` ([String](../../sql-reference/data-types/)) — Kafka Engine 테이블 이름.
* `consumer_id` ([String](../../sql-reference/data-types/)) — Kafka consumer 식별자. 하나의 테이블에 여러 consumer가 있을 수 있습니다. `kafka_num_consumers` 파라미터로 지정합니다.
* `assignments.topic` ([Array(String)](../../sql-reference/data-types/)) — Kafka 토픽.
* `assignments.partition_id` ([Array(Int32)](../../sql-reference/data-types/)) — Kafka 파티션 ID. 하나의 파티션에는 오직 하나의 consumer만 할당될 수 있습니다.
* `assignments.current_offset` ([Array(Int64)](../../sql-reference/data-types/)) — 현재 오프셋.
* `assignments.intent_size` ([Array(Nullable(Int64))](../../sql-reference/data-types/)) — 새 StorageKafka 엔진에서 푸시되었으나 아직 커밋되지 않은 메시지 수.
* `exceptions.time` ([Array(DateTime)](../../sql-reference/data-types/)) — 가장 최근 예외 10개의 생성 시각 타임스탬프.
* `exceptions.text` ([Array(String)](../../sql-reference/data-types/)) — 가장 최근 예외 10개의 텍스트.
* `last_poll_time` ([DateTime](../../sql-reference/data-types/)) — 가장 최근 poll의 타임스탬프.
* `num_messages_read` ([UInt64](../../sql-reference/data-types/)) — consumer가 읽은 메시지 수.
* `last_commit_time` ([DateTime](../../sql-reference/data-types/)) — 가장 최근 poll의 타임스탬프.
* `num_commits` ([UInt64](../../sql-reference/data-types/)) — 해당 consumer의 전체 커밋 횟수.
* `last_rebalance_time` ([DateTime](../../sql-reference/data-types/)) — 가장 최근 Kafka 리밸런스의 타임스탬프.
* `num_rebalance_revocations` ([UInt64](../../sql-reference/data-types/)) — consumer에서 파티션 할당이 취소된 횟수.
* `num_rebalance_assignments` ([UInt64](../../sql-reference/data-types/)) — consumer가 Kafka 클러스터에 할당된 횟수.
* `is_currently_used` ([UInt8](../../sql-reference/data-types/)) — 현재 consumer가 사용 중인지 나타내는 플래그.
* `last_used` ([DateTime64(6)](../../sql-reference/data-types/)) — 이 consumer가 마지막으로 사용된 시각.
* `rdkafka_stat` ([String](../../sql-reference/data-types/)) — 라이브러리 내부 통계. 비활성화하려면 statistics&#95;interval&#95;ms를 0으로 설정하며, 기본값은 3000(3초에 한 번)입니다.
* `dependencies` ([Array(Array(String))](../../sql-reference/data-types/)) — 전이적(transitive) 데이터베이스 종속성.
* `missing_dependencies` ([Array(Array(String))](../../sql-reference/data-types/)) — 누락된 전이적(transitive) 데이터베이스 종속성.

{/*AUTOGENERATED_END*/ }
