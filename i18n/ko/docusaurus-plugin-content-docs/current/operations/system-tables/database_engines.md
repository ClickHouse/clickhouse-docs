## 설명 \{#description\}

서버에서 지원하는 데이터베이스 엔진 목록을 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 데이터베이스 엔진의 이름입니다.

{/*AUTOGENERATED_END*/ }

## 예시 \{#example\}

```sql title="Query"
SELECT *
FROM system.database_engines
WHERE name IN ('Atomic', 'Lazy', 'Ordinary')
```

```text title="Response"
┌─name─────┐
│ Ordinary │
│ Atomic   │
│ Lazy     │
└──────────┘
```
