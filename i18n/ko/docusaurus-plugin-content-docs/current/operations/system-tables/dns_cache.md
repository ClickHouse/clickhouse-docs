캐시된 DNS 레코드에 대한 정보가 포함되어 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `hostname` ([String](../../sql-reference/data-types/)) — 호스트명.
* `ip_address` ([String](../../sql-reference/data-types/)) — IP 주소.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](../../sql-reference/data-types/)) — IP 주소 패밀리.
* `cached_at` ([DateTime](../../sql-reference/data-types/)) — 레코드가 캐시된 시각.

{/*자동 생성 구간 종료*/ }

**예제**

쿼리:

```sql
SELECT * FROM system.dns_cache;
```

결과:

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
