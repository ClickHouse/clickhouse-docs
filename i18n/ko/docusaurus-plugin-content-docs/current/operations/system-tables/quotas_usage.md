# system.quotas_usage \{#systemquotas_usage\}

<SystemTableCloud />

모든 사용자에 대한 쿼터 사용량입니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `quota_name` ([String](../../sql-reference/data-types/)) — Quota 이름.
* `quota_key` ([String](../../sql-reference/data-types/)) — 키 값.
* `is_current` ([UInt8](../../sql-reference/data-types/)) — 현재 사용자에 대한 Quota 사용량.
* `start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 리소스 사용량 계산 시작 시각.
* `end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 리소스 사용량 계산 종료 시각.
* `duration` ([Nullable(UInt32)](../../sql-reference/data-types/)) — 리소스 사용량 계산을 위한 시간 구간의 길이(초 단위).
* `queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재까지 실행된 쿼리 수.
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용된 모든 유형의 쿼리 최대 개수.
* `query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재까지 실행된 SELECT 쿼리 수.
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용된 SELECT 쿼리의 최대 개수.
* `query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재까지 실행된 INSERT 쿼리 수.
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용된 INSERT 쿼리의 최대 개수.
* `errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 오류가 발생한 쿼리의 현재 개수.
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 오류가 발생하는 것이 허용되는 쿼리의 최대 개수.
* `result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 결과 집합에 포함된 행의 총 개수.
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 모든 쿼리 결과 집합에 포함될 수 있도록 허용된 행의 최대 총 개수.
* `result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 결과 집합에 포함된 총 바이트 수.
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 모든 쿼리 결과 집합에 포함될 수 있도록 허용된 최대 총 바이트 수.
* `read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리를 실행하는 동안 읽은 행의 총 개수.
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 모든 쿼리를 실행하는 동안 읽을 수 있도록 허용된 행의 최대 개수.
* `read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리를 실행하는 동안 읽은 총 바이트 수.
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 모든 쿼리를 실행하는 동안 읽을 수 있도록 허용된 최대 바이트 수.
* `execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 현재 기간 동안 쿼리 실행에 소요된 총 시간(나노초 단위).
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 지정된 기간 동안 모든 쿼리 실행에 허용되는 최대 시간(나노초 단위).
* `written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리를 실행하는 동안 기록된 총 바이트 수.
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 모든 쿼리를 실행하는 동안 기록할 수 있도록 허용된 최대 바이트 수.
* `failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 연속으로 실패한 인증의 개수.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 동안 연속으로 실패할 수 있도록 허용된 인증의 최대 개수.
