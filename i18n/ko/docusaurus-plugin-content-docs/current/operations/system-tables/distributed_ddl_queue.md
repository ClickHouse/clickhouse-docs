클러스터에서 실행된 [분산 DDL 쿼리(ON CLUSTER 절)](../../sql-reference/distributed-ddl.md)에 대한 정보를 담고 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `entry` ([String](../../sql-reference/data-types/)) — 쿼리 ID입니다.
* `entry_version` ([Nullable(UInt8)](../../sql-reference/data-types/)) — 엔트리 버전입니다.
* `initiator_host` ([Nullable(String)](../../sql-reference/data-types/)) — DDL 작업을 시작한 호스트입니다.
* `initiator_port` ([Nullable(UInt16)](../../sql-reference/data-types/)) — 시작한 측에서 사용한 포트입니다.
* `cluster` ([String](../../sql-reference/data-types/)) — 클러스터 이름입니다. 결정되지 않은 경우 비어 있습니다.
* `query` ([String](../../sql-reference/data-types/)) — 실행된 쿼리입니다.
* `settings` ([Map(String, String)](../../sql-reference/data-types/)) — DDL 작업에서 사용된 설정입니다.
* `query_create_time` ([DateTime](../../sql-reference/data-types/)) — 쿼리가 생성된 시각입니다.
* `host` ([Nullable(String)](../../sql-reference/data-types/)) — 호스트 이름입니다.
* `port` ([Nullable(UInt16)](../../sql-reference/data-types/)) — 호스트 포트 번호입니다.
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](../../sql-reference/data-types/)) — 쿼리 상태입니다.
* `exception_code` ([Nullable(UInt16)](../../sql-reference/data-types/)) — 예외 코드입니다.
* `exception_text` ([Nullable(String)](../../sql-reference/data-types/)) — 예외 메시지입니다.
* `query_finish_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 쿼리가 종료된 시각입니다.
* `query_duration_ms` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 쿼리 실행 시간(밀리초 단위)입니다.

{/*AUTOGENERATED_END*/ }
