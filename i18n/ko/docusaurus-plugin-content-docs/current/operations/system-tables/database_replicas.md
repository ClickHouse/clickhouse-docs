각 복제된 데이터베이스의 레플리카에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 복제된 데이터베이스(Replicated database)의 이름입니다.
* `is_readonly` ([UInt8](../../sql-reference/data-types/)) — 데이터베이스 레플리카가 읽기 전용 모드인지 여부를 나타냅니다.
* `max_log_ptr` ([Int32](../../sql-reference/data-types/)) — 일반 작업 로그에서의 최대 엔트리 번호입니다.
* `replica_name` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper에서 레플리카의 이름입니다.
* `replica_path` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper에서 레플리카 데이터의 경로입니다.
* `zookeeper_path` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper에서 데이터베이스 데이터의 경로입니다.
* `shard_name` ([String](../../sql-reference/data-types/)) — 클러스터에서 세그먼트(shard)의 이름입니다.
* `log_ptr` ([Int32](../../sql-reference/data-types/)) — 레플리카가 실행 큐로 복사한 일반 작업 로그의 최대 엔트리 번호에 1을 더한 값입니다.
* `total_replicas` ([UInt32](../../sql-reference/data-types/)) — 이 데이터베이스에 대해 알려진 전체 레플리카 수입니다.
* `zookeeper_exception` ([String](../../sql-reference/data-types/)) — ClickHouse Keeper에서 정보를 가져오는 중 오류가 발생했을 때 기록된 마지막 예외 메시지입니다.
* `is_session_expired` ([UInt8](../../sql-reference/data-types/)) — ClickHouse Keeper와의 세션이 만료되었는지를 나타냅니다. 기본적으로 `is_readonly`와 동일합니다.

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.database_replicas FORMAT Vertical;
```

```text
Row 1:
──────

zookeeper_path:      /test/db_2
shard_name:          shard1
log_ptr:             2
total_replicas:      1
zookeeper_exception: 
is_session_expired:  0
```
