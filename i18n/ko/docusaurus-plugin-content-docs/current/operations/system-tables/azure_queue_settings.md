[AzureQueue](../../engines/table-engines/integrations/azure-queue.md) 테이블의 설정 정보를 포함합니다.
`24.10` 서버 버전부터 사용 가능합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — S3Queue Engine이 있는 테이블의 데이터베이스입니다.
* `table` ([String](../../sql-reference/data-types/)) — S3Queue Engine이 있는 테이블의 이름입니다.
* `name` ([String](../../sql-reference/data-types/)) — 설정 이름입니다.
* `value` ([String](../../sql-reference/data-types/)) — 설정 값입니다.
* `type` ([String](../../sql-reference/data-types/)) — 설정 유형입니다(구현별 문자열 값).
* `changed` ([UInt8](../../sql-reference/data-types/)) — 설정이 구성 파일에서 명시적으로 정의되었거나 명시적으로 변경되었으면 1입니다.
* `description` ([String](../../sql-reference/data-types/)) — 설정 설명입니다.
* `alterable` ([UInt8](../../sql-reference/data-types/)) — 현재 사용자가 ALTER TABLE MODIFY SETTING을 통해 설정을 변경할 수 있는지를 나타냅니다: 0 — 현재 사용자가 설정을 변경할 수 있음, 1 — 현재 사용자가 설정을 변경할 수 없음.

{/*AUTOGENERATED_END*/ }
