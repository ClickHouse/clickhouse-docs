지원되는 [데이터 타입](../../sql-reference/data-types/index.md)에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 데이터 타입 이름입니다.
* `case_insensitive` ([UInt8](../../sql-reference/data-types/)) — 쿼리에서 데이터 타입 이름을 대소문자를 구분하지 않고 사용할 수 있는지 여부를 나타내는 속성입니다. 예를 들어, `Date`와 `date`는 모두 유효합니다.
* `alias_to` ([String](../../sql-reference/data-types/)) — `name`이 별칭인 데이터 타입 이름입니다.

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.data_type_families WHERE alias_to = 'String'
```

```text
┌─name───────┬─case_insensitive─┬─alias_to─┐
│ LONGBLOB   │                1 │ String   │
│ LONGTEXT   │                1 │ String   │
│ TINYTEXT   │                1 │ String   │
│ TEXT       │                1 │ String   │
│ VARCHAR    │                1 │ String   │
│ MEDIUMBLOB │                1 │ String   │
│ BLOB       │                1 │ String   │
│ TINYBLOB   │                1 │ String   │
│ CHAR       │                1 │ String   │
│ MEDIUMTEXT │                1 │ String   │
└────────────┴──────────────────┴──────────┘
```
