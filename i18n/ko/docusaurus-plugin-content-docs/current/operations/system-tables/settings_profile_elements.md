## 설명 \{#description\}

SETTINGS PROFILE의 내용을 설명합니다:

* 제약 조건.
* 해당 설정이 적용되는 역할과 사용자.
* 상위 SETTINGS PROFILE.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `profile_name` ([Nullable(String)](../../sql-reference/data-types/)) — SETTINGS PROFILE 이름.
* `user_name` ([Nullable(String)](../../sql-reference/data-types/)) — 사용자 이름.
* `role_name` ([Nullable(String)](../../sql-reference/data-types/)) — 역할 이름.
* `index` ([UInt64](../../sql-reference/data-types/)) — SETTINGS PROFILE 요소의 일련 번호.
* `setting_name` ([Nullable(String)](../../sql-reference/data-types/)) — 설정 이름.
* `value` ([Nullable(String)](../../sql-reference/data-types/)) — 설정값.
* `min` ([Nullable(String)](../../sql-reference/data-types/)) — 설정의 최솟값. 설정되지 않은 경우 NULL.
* `max` ([Nullable(String)](../../sql-reference/data-types/)) — 설정의 최댓값. 설정되지 않은 경우 NULL.
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](../../sql-reference/data-types/)) — 설정을 변경할 수 있는지 여부를 나타내는 속성.
* `inherit_profile` ([Nullable(String)](../../sql-reference/data-types/)) — 이 SETTINGS PROFILE의 상위 프로필. 설정되지 않은 경우 NULL입니다. SETTINGS PROFILE은 상위 프로필로부터 모든 설정값과 제약 조건(min, max, readonly)을 상속합니다.
