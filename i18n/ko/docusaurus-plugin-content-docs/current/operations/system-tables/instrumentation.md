## 설명 \{#description\}

LLVM의 XRay 기능을 사용한 계측 지점을 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `id` ([UInt32](../../sql-reference/data-types/)) — 계측 지점의 ID
* `function_id` ([Int32](../../sql-reference/data-types/)) — elf-binary의 xray&#95;instr&#95;map 섹션에서 함수에 할당된 ID입니다.
* `function_name` ([LowCardinality(String)](../../sql-reference/data-types/)) — 함수 계측에 사용되는 이름입니다.
* `handler` ([LowCardinality(String)](../../sql-reference/data-types/)) — 함수의 계측 지점에 패치된 handler입니다.
* `entry_type` ([Enum8(&#39;Entry&#39; = 0, &#39;Exit&#39; = 1, &#39;EntryAndExit&#39; = 2)](../../sql-reference/data-types/)) — 패치의 엔트리 유형입니다.
* `symbol` ([LowCardinality(String)](../../sql-reference/data-types/)) — 전체 디맹글링 심볼 이름입니다.
* `parameters` ([Array(Dynamic)](../../sql-reference/data-types/)) — handler 호출에 사용되는 매개변수입니다.

{/*AUTOGENERATED_END*/ }
