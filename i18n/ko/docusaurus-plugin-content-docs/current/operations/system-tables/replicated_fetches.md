# system.replicated_fetches \{#systemreplicated_fetches\}

<SystemTableCloud />

현재 실행 중인 백그라운드 가져오기 작업에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 데이터베이스 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블 이름입니다.
* `elapsed` ([Float64](../../sql-reference/data-types/)) — 현재 실행 중인 백그라운드 페치가 표시되기 시작한 이후 경과 시간(초)입니다.
* `progress` ([Float64](../../sql-reference/data-types/)) — 0에서 1 사이의 완료 작업 비율입니다.
* `result_part_name` ([String](../../sql-reference/data-types/)) — 현재 실행 중인 백그라운드 페치의 결과로 생성될 파트의 이름입니다.
* `result_part_path` ([String](../../sql-reference/data-types/)) — 현재 실행 중인 백그라운드 페치의 결과로 생성될 파트의 절대 경로입니다.
* `partition_id` ([String](../../sql-reference/data-types/)) — 파티션 ID입니다.
* `total_size_bytes_compressed` ([UInt64](../../sql-reference/data-types/)) — 결과 파트에 포함된 압축 데이터의 총 크기(바이트)입니다.
* `bytes_read_compressed` ([UInt64](../../sql-reference/data-types/)) — 결과 파트에서 읽은 압축 바이트 수입니다.
* `source_replica_path` ([String](../../sql-reference/data-types/)) — 소스 레플리카의 절대 경로입니다.
* `source_replica_hostname` ([String](../../sql-reference/data-types/)) — 소스 레플리카의 호스트 이름입니다.
* `source_replica_port` ([UInt16](../../sql-reference/data-types/)) — 소스 레플리카의 포트 번호입니다.
* `interserver_scheme` ([String](../../sql-reference/data-types/)) — interserver 스킴 이름입니다.
* `URI` ([String](../../sql-reference/data-types/)) — URI(Uniform Resource Identifier)입니다.
* `to_detached` ([UInt8](../../sql-reference/data-types/)) — 현재 실행 중인 백그라운드 페치가 `TO DETACHED` 표현식을 사용하여 수행되는지 나타내는 플래그입니다.
* `thread_id` ([UInt64](../../sql-reference/data-types/)) — 스레드 식별자입니다.

{/*AUTOGENERATED_END*/ }
