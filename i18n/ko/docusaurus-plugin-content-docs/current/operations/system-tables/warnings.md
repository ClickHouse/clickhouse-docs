## 설명 \{#description\}

서버에 연결한 직후 `clickhouse-client`에 표시되는 서버 구성 관련 경고를 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `message` ([String](../../sql-reference/data-types/)) — ClickHouse 서버에서 발생한 경고 메시지입니다.
* `message_format_string` ([LowCardinality(String)](../../sql-reference/data-types/)) — 메시지 포맷에 사용된 포맷 문자열입니다.

{/*AUTOGENERATED_END*/ }
