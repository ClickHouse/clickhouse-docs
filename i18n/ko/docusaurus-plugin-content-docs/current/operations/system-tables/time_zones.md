# system.time_zones \{#systemtime_zones\}

ClickHouse 서버에서 지원하는 시간대 목록입니다. 이 시간대 목록은 사용 중인 ClickHouse 버전에 따라 달라질 수 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `time_zone` ([String](../../sql-reference/data-types/)) — 지원되는 시간대 목록입니다.

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
