## 설명 \{#description\}

SQL 쿼리를 통해 생성되었거나 설정 파일에서 파싱된 모든 이름이 지정된 컬렉션 목록을 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 컬렉션 이름입니다.
* `collection` ([Map(String, String)](../../sql-reference/data-types/)) — 컬렉션의 내부 정보입니다.
* `source` ([String](../../sql-reference/data-types/)) — 명명된 컬렉션의 소스입니다.
* `create_query` ([String](../../sql-reference/data-types/)) — 명명된 컬렉션을 생성하는 쿼리입니다.

{/*AUTOGENERATED_END*/ }
