현재 서버에서 실행된 모든 트랜잭션에 대한 정보를 포함합니다.

이 테이블은 언제든지 안전하게 truncate하거나 drop할 수 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 트랜잭션이 실행된 호스트명입니다.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/sql-reference/data-types/enum)) — 트랜잭션 유형입니다. 가능한 값: Begin, Commit, Rollback, AddPart, LockPart, UnlockPart.
* `event_date` ([Date](/sql-reference/data-types/date)) — 항목의 날짜입니다.
* `event_time` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 항목의 시간입니다
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — 스레드 식별자입니다.
* `query_id` ([String](/sql-reference/data-types/string)) — 트랜잭션 범위 내에서 실행된 쿼리의 ID입니다.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — 트랜잭션 식별자입니다.
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — 식별자의 해시입니다.
* `csn` ([UInt64](/sql-reference/data-types/int-uint)) — Commit Sequence Number입니다
* `database` ([String](/sql-reference/data-types/string)) — 트랜잭션이 실행된 데이터베이스 이름입니다.
* `table` ([String](/sql-reference/data-types/string)) — 트랜잭션이 실행된 테이블 이름입니다.
* `uuid` ([UUID](/sql-reference/data-types/uuid)) — 트랜잭션이 실행된 테이블의 UUID입니다.
* `part` ([String](/sql-reference/data-types/string)) — 트랜잭션에 참여한 part의 이름입니다.
