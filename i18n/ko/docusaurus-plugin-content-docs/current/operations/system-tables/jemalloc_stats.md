## 설명 \{#description\}

jemalloc 통계를 하나의 컬럼만 있는 단일 행으로 반환합니다. SYSTEM JEMALLOC STATS 명령어와 동일합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `stats` ([String](../../sql-reference/data-types/)) — malloc&#95;stats&#95;print의 JEMalloc 통계 출력 결과입니다.

{/*AUTOGENERATED_END*/ }
