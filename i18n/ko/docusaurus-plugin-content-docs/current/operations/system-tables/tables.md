## 설명 \{#description\}

서버가 인식하고 있는 각 테이블의 메타데이터를 포함합니다.

[Detached](../../sql-reference/statements/detach.md) 테이블은 `system.tables`에 표시되지 않습니다.

[Temporary tables](../../sql-reference/statements/create/table.md#temporary-tables)는 생성된 해당 세션에서만 `system.tables`에 표시됩니다. 이러한 테이블은 `database` 필드가 비어 있으며 `is_temporary` 플래그가 켜진 상태로 표시됩니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 테이블이 속한 데이터베이스 이름입니다.
* `name` ([String](../../sql-reference/data-types/)) — 테이블 이름.
* `uuid` ([UUID](../../sql-reference/data-types/)) — 테이블 UUID(Atomic 데이터베이스).
* `engine` ([String](../../sql-reference/data-types/)) — 테이블 엔진 이름(매개변수 없음).
* `is_temporary` ([UInt8](../../sql-reference/data-types/)) — 테이블이 임시인지 나타내는 플래그입니다.
* `data_paths` ([Array(String)](../../sql-reference/data-types/)) — 파일 시스템에 있는 테이블 데이터의 경로입니다.
* `metadata_path` ([String](../../sql-reference/data-types/)) — 파일 시스템에 있는 테이블 메타데이터의 경로입니다.
* `metadata_modification_time` ([DateTime](../../sql-reference/data-types/)) — 테이블 메타데이터의 마지막 수정 시각입니다.
* `metadata_version` ([Int32](../../sql-reference/data-types/)) — ReplicatedMergeTree 테이블의 메타데이터 버전입니다. ReplicatedMergeTree 테이블이 아니면 0입니다.
* `dependencies_database` ([Array(String)](../../sql-reference/data-types/)) — 데이터베이스 의존성.
* `dependencies_table` ([Array(String)](../../sql-reference/data-types/)) — 테이블 종속성(현재 테이블의 materialized view)입니다.
* `create_table_query` ([String](../../sql-reference/data-types/)) — 테이블을 생성할 때 사용된 쿼리입니다.
* `engine_full` ([String](../../sql-reference/data-types/)) — 테이블 엔진 매개변수입니다.
* `as_select` ([String](../../sql-reference/data-types/)) — 뷰의 SELECT 쿼리입니다.
* `parameterized_view_parameters` ([Array(Tuple(name String, type String))](../../sql-reference/data-types/)) — 매개변수화된 뷰에 사용되는 매개변수입니다.
* `partition_key` ([String](../../sql-reference/data-types/)) — 테이블에 지정된 파티션 키 표현식입니다.
* `sorting_key` ([String](../../sql-reference/data-types/)) — 테이블에 지정된 sorting key 표현식입니다.
* `primary_key` ([String](../../sql-reference/data-types/)) — 테이블에 지정된 기본 키 표현식입니다.
* `sampling_key` ([String](../../sql-reference/data-types/)) — 테이블에 지정된 샘플링 키 표현식입니다.
* `storage_policy` ([String](../../sql-reference/data-types/)) — 스토리지 정책입니다. MergeTree 및 Distributed 엔진을 사용하는 테이블에 적용됩니다.
* `total_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 테이블의 정확한 총 행 수를 빠르게 확인할 수 있으면 해당 값을, 그렇지 않으면 NULL을 나타냅니다(기본 Buffer 테이블 포함).
* `total_bytes` ([널 허용(UInt64)](../../sql-reference/data-types/)) — 스토리지에서 테이블의 정확한 바이트 수를 빠르게 확인할 수 있는 경우 해당 총 바이트 수를 나타내고, 그렇지 않으면 NULL입니다(기반 스토리지는 포함하지 않음). 테이블이 데이터를 디스크에 저장하는 경우 디스크에서 사용 중인 공간(즉, 압축된 크기)을 반환합니다. 테이블이 데이터를 메모리에 저장하는 경우 메모리에서 사용 중인 바이트 수의 근삿값을 반환합니다.
* `total_bytes_uncompressed` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 압축되지 않은 바이트의 총수입니다. 스토리지의 테이블에 대해 part 체크섬으로 정확한 바이트 수를 빠르게 확인할 수 있으면 해당 값을 반환하고, 그렇지 않으면 NULL을 반환합니다(기반 스토리지가 있는 경우에도 이를 고려하지 않습니다).
* `parts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 이 테이블에 있는 전체 파트 수입니다.
* `active_parts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 이 테이블에서 활성 상태인 파트 수입니다.
* `total_marks` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 이 테이블의 모든 파트에 포함된 마크의 총 개수입니다.
* `active_on_fly_data_mutations` ([UInt64](../../sql-reference/data-types/)) — 즉시 적용 가능한 활성 데이터 뮤테이션(UPDATE 및 DELETE)의 총 개수입니다.
* `active_on_fly_alter_mutations` ([UInt64](../../sql-reference/data-types/)) — 즉시 적용할 수 있는 활성 alter mutation(MODIFY COLUMN)의 총개수입니다.
* `active_on_fly_metadata_mutations` ([UInt64](../../sql-reference/data-types/)) — 즉시 적용 가능한 활성 메타데이터 뮤테이션(RENAME)의 총개수입니다.
* `columns_descriptions_cache_size` ([UInt64](../../sql-reference/data-types/)) — *MergeTree 테이블의 컬럼 설명 캐시의 크기
* `lifetime_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 서버 시작 이후 삽입된 총 행 수(Buffer 테이블에만 해당).
* `lifetime_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 서버가 시작된 이후 삽입된 총 바이트 수(Buffer 테이블에만 해당).
* `comment` ([String](../../sql-reference/data-types/)) — 테이블 설명입니다.
* `has_own_data` ([UInt8](../../sql-reference/data-types/)) — 테이블 자체가 일부 데이터를 디스크에 저장하는지, 아니면 다른 소스에만 액세스하는지를 나타내는 플래그입니다.
* `loading_dependencies_database` ([Array(String)](../../sql-reference/data-types/)) — 데이터베이스 로드 의존성(현재 객체보다 먼저 로드되어야 하는 객체 목록)입니다.
* `loading_dependencies_table` ([Array(String)](../../sql-reference/data-types/)) — 테이블 로딩 의존 관계(현재 객체보다 먼저 로드되어야 하는 객체 목록).
* `loading_dependent_database` ([Array(String)](../../sql-reference/data-types/)) — 종속적으로 로드되는 데이터베이스입니다.
* `loading_dependent_table` ([Array(String)](../../sql-reference/data-types/)) — 로딩이 의존하는 테이블입니다.
* `definer` ([String](../../sql-reference/data-types/)) — 테이블에 사용되는 SQL 보안 definer의 이름입니다.

{/*AUTOGENERATED_END*/ }
