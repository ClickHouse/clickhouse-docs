## 설명 \{#description\}

모든 QUOTA의 모든 구간에 대한 최대값 정보를 포함합니다. 하나의 QUOTA에 임의 개수(0개 포함)의 행이 대응될 수 있습니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `quota_name` ([String](../../sql-reference/data-types/)) — QUOTA 이름입니다.
* `duration` ([UInt32](../../sql-reference/data-types/)) — 리소스 사용량을 계산하기 위한 시간 인터벌의 길이이며, 초 단위입니다.
* `is_randomized_interval` ([UInt8](../../sql-reference/data-types/)) — 불리언 값입니다. 인터벌이 무작위화되었는지 여부를 나타냅니다. 무작위화되지 않은 경우 인터벌은 항상 같은 시각에 시작됩니다. 예를 들어 1분 인터벌은 항상 분 단위의 정확한 시각에 시작됩니다(즉, 11:20:00에는 시작할 수 있지만 11:20:01에는 시작하지 않습니다). 하루 인터벌은 항상 UTC 자정에 시작됩니다. 인터벌이 무작위화된 경우 첫 번째 인터벌은 임의의 시각에 시작되며, 이후 인터벌은 순차적으로 시작됩니다. 값: 0 — 인터벌이 무작위화되지 않음, 1 — 인터벌이 무작위화됨.
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용되는 모든 유형의 쿼리 최대 개수입니다.
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용되는 SELECT 쿼리의 최대 개수입니다.
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용되는 INSERT 쿼리의 최대 개수입니다.
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 오류 발생 쿼리의 최대 개수입니다.
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 모든 쿼리 결과 집합의 총 행 수 최댓값입니다.
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 모든 쿼리 결과 집합의 총 바이트 수 최댓값입니다.
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 모든 쿼리 실행 중 읽을 수 있는 최대 행 수입니다.
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 모든 쿼리 실행 중 읽을 수 있는 최대 바이트 수입니다.
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 실행에 허용되는 최대 시간(나노초)입니다.
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 모든 쿼리 실행 중 기록할 수 있는 최대 바이트 수입니다.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 허용되는 연속 인증 실패의 최대 횟수입니다.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 정규화된 단일 쿼리 해시당 허용되는 최대 실행 횟수입니다.
