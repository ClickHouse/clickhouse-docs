[서버 설정](../../engines/table-engines/mergetree-family/mergetree.md#table_engine-mergetree-multiple-volumes_configure)에 정의된 디스크 정보가 포함됩니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 서버 설정에 지정된 디스크 이름입니다.
* `path` ([String](../../sql-reference/data-types/)) — 파일 시스템에서 마운트 지점의 경로입니다.
* `free_space` ([UInt64](../../sql-reference/data-types/)) — 디스크의 사용 가능한 공간(바이트 단위)입니다.
* `total_space` ([UInt64](../../sql-reference/data-types/)) — 디스크 용량(바이트 단위)입니다.
* `unreserved_space` ([UInt64](../../sql-reference/data-types/)) — 예약에 의해 점유되지 않은 사용 가능한 공간입니다(현재 실행 중인 머지, INSERT 및 기타 디스크 쓰기 작업을 위해 예약된 공간을 `free_space`에서 뺀 값).
* `keep_free_space` ([UInt64](../../sql-reference/data-types/)) — 디스크에 항상 남겨 두어야 하는 여유 공간(바이트 단위)입니다. 디스크 설정의 `keep_free_space_bytes` 매개변수로 정의됩니다.
* `type` ([String](../../sql-reference/data-types/)) — 이 디스크가 데이터를 어디에 저장하는지 나타내는 디스크 유형입니다. RAM, 로컬 드라이브 또는 원격 스토리지일 수 있습니다.
* `object_storage_type` ([String](../../sql-reference/data-types/)) — 디스크 유형이 `object_storage`인 경우 객체 스토리지의 유형입니다.
* `metadata_type` ([String](../../sql-reference/data-types/)) — 디스크 유형이 `object_storage`인 경우 메타데이터 스토리지의 유형입니다.
* `is_encrypted` ([UInt8](../../sql-reference/data-types/)) — 이 디스크가 실제 데이터를 암호화하는지 여부를 나타내는 플래그입니다.
* `is_read_only` ([UInt8](../../sql-reference/data-types/)) — 이 디스크에서 읽기 작업만 수행할 수 있는지 여부를 나타내는 플래그입니다.
* `is_write_once` ([UInt8](../../sql-reference/data-types/)) — 디스크가 한 번만 쓰기(write-once)인지 여부를 나타내는 플래그입니다. 이 디스크로의 BACKUP은 지원하지만, 이 디스크에 있는 MergeTree 테이블에 대한 INSERT는 지원하지 않습니다.
* `is_remote` ([UInt8](../../sql-reference/data-types/)) — 이 디스크에 대한 작업이 네트워크와의 상호 작용을 수반하는지 여부를 나타내는 플래그입니다.
* `is_broken` ([UInt8](../../sql-reference/data-types/)) — 디스크가 손상(broken)되었는지 여부를 나타내는 플래그입니다. 손상된 디스크는 공간이 0으로 표시되며 사용할 수 없습니다.
* `cache_path` ([String](../../sql-reference/data-types/)) — 디스크가 캐싱을 지원하는 경우 로컬 드라이브에 있는 캐시 디렉터리의 경로입니다.

{/*AUTOGENERATED_END*/ }
