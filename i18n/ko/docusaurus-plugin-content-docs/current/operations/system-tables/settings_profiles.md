# system.settings_profiles \{#systemsettings_profiles\}

구성된 설정 프로파일의 속성 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 설정 프로파일 이름.
* `id` ([UUID](../../sql-reference/data-types/)) — 설정 프로파일 ID.
* `storage` ([String](../../sql-reference/data-types/)) — 설정 프로파일 저장소 경로. `access_control_path` 매개변수에서 구성합니다.
* `num_elements` ([UInt64](../../sql-reference/data-types/)) — 이 프로파일의 요소 개수(`system.settings_profile_elements` 테이블 기준).
* `apply_to_all` ([UInt8](../../sql-reference/data-types/)) — SETTINGS PROFILE이 모든 역할 및/또는 사용자에 설정되어 있음을 나타냅니다.
* `apply_to_list` ([Array(String)](../../sql-reference/data-types/)) — SETTINGS PROFILE이 적용되는 역할 및/또는 사용자 목록입니다.
* `apply_to_except` ([Array(String)](../../sql-reference/data-types/)) — SETTINGS PROFILE이 나열된 역할 및/또는 사용자를 제외한 모든 역할 및/또는 사용자에게 적용됩니다.

{/*AUTOGENERATED_END*/ }
