## 설명 \{#description\}

LibraryBridge 메모리에 로드된 CatBoost 모델 목록과 각 모델의 로드 시각을 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `model_path` ([String](../../sql-reference/data-types/)) — 학습된 모델의 경로입니다.
* `type` ([String](../../sql-reference/data-types/)) — 모델 유형입니다. 현재는 catboost만 지원합니다.
* `loading_start_time` ([DateTime](../../sql-reference/data-types/)) — 모델 로드가 시작된 시각입니다.
* `loading_duration` ([Float32](../../sql-reference/data-types/)) — 모델을 로드하는 데 걸린 시간입니다.

{/*AUTOGENERATED_END*/ }
