세그먼트로 전송하기 위해 대기열에 들어 있는 로컬 파일에 대한 정보를 포함합니다. 이러한 로컬 파일에는 비동기 모드로 분산 테이블(Distributed table)에 새 데이터를 삽입할 때 생성되는 새로운 파트가 포함됩니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 데이터베이스 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블 이름입니다.
* `data_path` ([String](../../sql-reference/data-types/)) — 로컬 파일이 있는 폴더의 경로입니다.
* `is_blocked` ([UInt8](../../sql-reference/data-types/)) — 로컬 파일을 서버로 전송하는 작업이 차단되었는지를 나타내는 플래그입니다.
* `error_count` ([UInt64](../../sql-reference/data-types/)) — 오류 개수입니다.
* `data_files` ([UInt64](../../sql-reference/data-types/)) — 폴더에 있는 로컬 파일 개수입니다.
* `data_compressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 로컬 파일에 있는 압축된 데이터의 크기(바이트 단위)입니다.
* `broken_data_files` ([UInt64](../../sql-reference/data-types/)) — 오류로 인해 손상된 것으로 표시된 파일 개수입니다.
* `broken_data_compressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 손상된 파일에 있는 압축된 데이터의 크기(바이트 단위)입니다.
* `last_exception` ([String](../../sql-reference/data-types/)) — 마지막으로 발생한 오류에 대한 텍스트 메시지(있는 경우)입니다.
* `last_exception_time` ([DateTime](../../sql-reference/data-types/)) — 마지막 예외가 발생한 시각입니다.

{/*AUTOGENERATED_END*/ }
