# system.table_engines \{#systemtable_engines\}

서버에서 지원하는 테이블 엔진에 대한 설명과 각 엔진의 기능 지원 정보가 포함됩니다.

이 테이블에는 다음 컬럼이 포함됩니다(컬럼 타입은 괄호 안에 표시됩니다).

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 테이블 엔진의 이름입니다.
* `supports_settings` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 SETTINGS 절을 지원하는지 여부를 나타내는 플래그입니다.
* `supports_skipping_indices` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 스키핑 인덱스를 지원하는지 여부를 나타내는 플래그입니다.
* `supports_projections` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 프로젝션을 지원하는지 여부를 나타내는 플래그입니다.
* `supports_sort_order` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 PARTITION&#95;BY, PRIMARY&#95;KEY, ORDER&#95;BY 및 SAMPLE&#95;BY 절을 지원하는지 여부를 나타내는 플래그입니다.
* `supports_ttl` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 TTL을 지원하는지 여부를 나타내는 플래그입니다.
* `supports_replication` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 데이터 복제를 지원하는지 여부를 나타내는 플래그입니다.
* `supports_deduplication` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 데이터 중복 제거를 지원하는지 여부를 나타내는 플래그입니다.
* `supports_parallel_insert` ([UInt8](../../sql-reference/data-types/)) — 테이블 엔진이 병렬 insert를 지원하는지 여부를 나타내는 플래그입니다(max&#95;insert&#95;threads 설정 참조).

{/*AUTOGENERATED_END*/ }

예시:

```sql
SELECT *
FROM system.table_engines
WHERE name IN ('Kafka', 'MergeTree', 'ReplicatedCollapsingMergeTree')
```
