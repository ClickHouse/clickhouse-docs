## 설명 \{#description\}

사용 가능한 모든 집계 함수 결합자의 목록으로, 집계 함수에 적용하여 동작 방식을 변경할 수 있습니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — combinator의 이름입니다.
* `is_internal` ([UInt8](../../sql-reference/data-types/)) — 이 combinator가 내부 용도로만 사용되는지 여부를 나타냅니다.

{/*AUTOGENERATED_END*/ }
