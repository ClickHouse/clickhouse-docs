## 설명 \{#description\}

ReplicatedMergeTree 엔진별 모든 설정과 각 설정의 현재 값, 기본값 및 설명 목록을 포함합니다. 이들 설정은 CREATE 쿼리의 SETTINGS 섹션에서 변경할 수 있습니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 설정 이름입니다.
* `value` ([String](../../sql-reference/data-types/)) — 설정 값입니다.
* `default` ([String](../../sql-reference/data-types/)) — 설정 기본값입니다.
* `changed` ([UInt8](../../sql-reference/data-types/)) — 설정이 config에 명시적으로 정의되었거나 명시적으로 변경된 경우 1입니다.
* `description` ([String](../../sql-reference/data-types/)) — 설정 설명입니다.
* `min` ([Nullable(String)](../../sql-reference/data-types/)) — 제약 조건을 통해 최소값이 설정된 경우 해당 설정의 최소값입니다. 설정에 최소값이 없으면 NULL을 포함합니다.
* `max` ([Nullable(String)](../../sql-reference/data-types/)) — 제약 조건을 통해 최대값이 설정된 경우 해당 설정의 최대값입니다. 설정에 최대값이 없으면 NULL을 포함합니다.
* `disallowed_values` ([Array(String)](../../sql-reference/data-types/)) — 허용되지 않는 값 목록입니다.
* `readonly` ([UInt8](../../sql-reference/data-types/)) — 현재 사용자가 이 설정을 변경할 수 있는지 여부를 나타냅니다: 0 — 현재 사용자가 이 설정을 변경할 수 있습니다, 1 — 현재 사용자가 이 설정을 변경할 수 없습니다.
* `type` ([String](../../sql-reference/data-types/)) — 설정 유형입니다(구현에 따라 달라지는 문자열 값).
* `is_obsolete` ([UInt8](../../sql-reference/data-types/)) — 설정이 폐기되었는지 여부를 나타냅니다.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](../../sql-reference/data-types/)) —
  이 기능의 지원 수준입니다. ClickHouse 기능은 현재 개발 상태와 사용 시 기대할 수 있는 수준에 따라 여러 티어로 구분됩니다:

- PRODUCTION: 이 기능은 안정적이고 안전하게 사용할 수 있으며, 다른 PRODUCTION 기능과 함께 사용해도 상호 작용 문제가 없습니다.
- BETA: 이 기능은 안정적이고 안전합니다. 다만 다른 기능과 함께 사용할 때의 결과는 알려져 있지 않으며, 정확성은 보장되지 않습니다. 테스트와 보고를 환영합니다.
- EXPERIMENTAL: 이 기능은 개발 중입니다. 개발자와 ClickHouse 애호가를 위한 기능입니다. 정상적으로 동작할 수도 있고 그렇지 않을 수도 있으며, 언제든 제거될 수 있습니다.
- OBSOLETE: 더 이상 지원되지 않습니다. 이미 제거되었거나 향후 릴리스에서 제거될 예정입니다.
