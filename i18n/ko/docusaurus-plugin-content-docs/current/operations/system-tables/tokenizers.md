# system.tokenizers \{#systemtokenizers\}

<SystemTableCloud />

사용 가능한 모든 토크나이저를 표시합니다.
이는 [tokens](../../sql-reference/functions/splitting-merging-functions.md#tokens), [hasAllTokens](../../sql-reference/functions/string-search-functions.md#hasAllTokens), [hasAnyTokens](../../sql-reference/functions/string-search-functions.md#hasAnyTokens) 함수와 [텍스트 인덱스](../../engines/table-engines/mergetree-family/textindexes.md)에서 사용할 수 있습니다.

열:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 토크나이저 이름

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.tokenizers;
```

```text
┌─name────────────┐
│ ngrams          │
│ splitByNonAlpha │
│ sparseGrams     │
│ tokenbf_v1      │
│ ngrambf_v1      │
│ array           │
│ splitByString   │
│ sparse_grams    │
└─────────────────┘
```
