## 설명 \{#description\}

AzureQueue 메타데이터의 인메모리 상태와 파일별로 현재 처리 중인 행 정보를 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `zookeeper_path` ([String](../../sql-reference/data-types/)) — 메타데이터가 저장된 zookeeper 경로
* `file_path` ([String](../../sql-reference/data-types/)) — 처리 중인 파일의 경로
* `file_name` ([String](../../sql-reference/data-types/)) — 처리 중인 파일의 이름
* `rows_processed` ([UInt64](../../sql-reference/data-types/)) — 현재까지 처리된 행 수
* `status` ([String](../../sql-reference/data-types/)) — 처리 상태: Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 파일 처리가 시작된 시각
* `processing_end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 파일 처리가 종료된 시각
* `exception` ([String](../../sql-reference/data-types/)) — 처리 중 발생한 예외

{/*AUTOGENERATED_END*/ }
