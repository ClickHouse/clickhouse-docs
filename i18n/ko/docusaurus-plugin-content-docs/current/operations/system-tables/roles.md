# system.roles \{#systemroles\}

구성된 [roles](../../guides/sre/user-management/index.md#role-management)에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 역할 이름.
* `id` ([UUID](../../sql-reference/data-types/)) — 역할 ID.
* `storage` ([String](../../sql-reference/data-types/)) — 역할 스토리지가 저장되는 경로입니다. `access_control_path` 매개변수로 설정합니다.
