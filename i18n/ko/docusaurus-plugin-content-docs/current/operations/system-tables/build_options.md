ClickHouse 서버의 빌드 옵션 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 빌드 옵션의 이름입니다.
* `value` ([String](../../sql-reference/data-types/)) — 빌드 옵션의 값입니다.

{/*AUTOGENERATED_END*/ }

**예시**

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
