## 설명 \{#description\}

로그 항목을 포함합니다. 이 테이블로 기록되는 로그 레벨은 `text_log.level` 서버 설정으로 제한할 수 있습니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 쿼리를 실행한 서버의 호스트명입니다.
* `event_date` ([Date](/sql-reference/data-types/date)) — 항목의 날짜입니다.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 항목의 시간입니다.
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 마이크로초 정밀도를 포함한 항목 시간입니다.
* `thread_name` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 로그가 기록된 스레드의 이름입니다.
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — OS 스레드 ID입니다.
* `level` ([Enum8(&#39;Fatal&#39; = 1, &#39;Critical&#39; = 2, &#39;Error&#39; = 3, &#39;Warning&#39; = 4, &#39;Notice&#39; = 5, &#39;Information&#39; = 6, &#39;Debug&#39; = 7, &#39;Trace&#39; = 8, &#39;Test&#39; = 9)](/sql-reference/data-types/enum)) — 항목 수준입니다. 가능한 값은 1 또는 &#39;Fatal&#39;, 2 또는 &#39;Critical&#39;, 3 또는 &#39;Error&#39;, 4 또는 &#39;Warning&#39;, 5 또는 &#39;Notice&#39;, 6 또는 &#39;Information&#39;, 7 또는 &#39;Debug&#39;, 8 또는 &#39;Trace&#39;입니다.
* `query_id` ([String](/sql-reference/data-types/string)) — 쿼리 ID입니다.
* `logger_name` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 로거 이름입니다(예: DDLWorker).
* `message` ([String](/sql-reference/data-types/string)) — 메시지 자체입니다.
* `revision` ([UInt32](/sql-reference/data-types/int-uint)) — ClickHouse 리비전입니다.
* `source_file` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 로그가 기록된 소스 파일입니다.
* `source_line` ([UInt64](/sql-reference/data-types/int-uint)) — 로그가 기록된 소스 줄입니다.
* `message_format_string` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 메시지 형식을 지정하는 데 사용된 형식 문자열입니다.
* `value1` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 1입니다.
* `value2` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 2입니다.
* `value3` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 3입니다.
* `value4` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 4입니다.
* `value5` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 5입니다.
* `value6` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 6입니다.
* `value7` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 7입니다.
* `value8` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 8입니다.
* `value9` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 9입니다.
* `value10` ([String](/sql-reference/data-types/string)) — 메시지 형식을 지정하는 데 사용된 인수 10입니다.
