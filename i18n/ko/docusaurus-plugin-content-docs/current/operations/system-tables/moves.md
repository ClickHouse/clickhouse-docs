# system.moves \{#systemmoves\}

이 테이블에는 [MergeTree](/engines/table-engines/mergetree-family/mergetree.md) 테이블에서 진행 중인 [데이터 파트 이동](/sql-reference/statements/alter/partition#move-partitionpart)에 대한 정보가 포함됩니다. 각 데이터 파트 이동은 테이블의 하나의 행으로 표현됩니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 데이터베이스 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 이동 중인 데이터 파트가 포함된 테이블 이름입니다.
* `elapsed` ([Float64](../../sql-reference/data-types/)) — 데이터 파트 이동이 시작된 후 경과한 시간(초)입니다.
* `target_disk_name` ([String](../../sql-reference/data-types/)) — 데이터 파트가 이동되는 디스크 이름입니다.
* `target_disk_path` ([String](../../sql-reference/data-types/)) — 파일 시스템에서 해당 디스크 마운트 지점의 경로입니다.
* `part_name` ([String](../../sql-reference/data-types/)) — 이동 중인 데이터 파트 이름입니다.
* `part_size` ([UInt64](../../sql-reference/data-types/)) — 데이터 파트 크기입니다.
* `thread_id` ([UInt64](../../sql-reference/data-types/)) — 이동 작업을 수행하는 스레드의 식별자입니다.

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.moves
```
