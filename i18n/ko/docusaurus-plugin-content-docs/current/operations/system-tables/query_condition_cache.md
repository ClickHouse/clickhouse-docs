# system.query_condition_cache \{#systemquery_condition_cache\}

<SystemTableCloud />

[query condition cache](../query-condition-cache.md)에 저장된 내용을 보여줍니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `key_hash` ([UInt128](../../sql-reference/data-types/)) — (table&#95;uuid, part&#95;name, condition&#95;hash)의 해시값.
* `entry_size` ([UInt64](../../sql-reference/data-types/)) — 엔트리의 크기(바이트 단위).
* `matching_marks` ([String](../../sql-reference/data-types/)) — 일치하는 마크.

{/*AUTOGENERATED_END*/ }

```sql
SELECT * FROM system.query_condition_cache FORMAT Vertical;
```
