# system.projections \{#systemprojections\}

모든 테이블에 존재하는 프로젝션 정보가 포함됩니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 데이터베이스 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블 이름입니다.
* `name` ([String](../../sql-reference/data-types/)) — 프로젝션 이름입니다.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](../../sql-reference/data-types/)) — 프로젝션 유형입니다.
* `sorting_key` ([Array(String)](../../sql-reference/data-types/)) — 프로젝션 정렬 키입니다.
* `query` ([String](../../sql-reference/data-types/)) — 프로젝션 쿼리입니다.
* `settings` ([Map(String, String)](../../sql-reference/data-types/)) — 프로젝션 설정입니다.

{/*AUTOGENERATED_END*/ }

**예제**

```sql
SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       landing
name:        improved_sorting_key
type:        Normal
sorting_key: ['user_id','date']
query:       SELECT * ORDER BY user_id, date
settings:     {}

Row 2:
──────
database:    default
table:       landing
name:        agg_no_key
type:        Aggregate
sorting_key: []
query:       SELECT count()
settings:     {}
```
