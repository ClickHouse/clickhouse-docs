## 설명 \{#description\}

치명적 오류에 대한 스택 트레이스 정보를 포함합니다. 이 테이블은 데이터베이스에 기본적으로 존재하지 않으며, 치명적 오류가 발생할 때에만 생성됩니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 크래시가 발생한 호스트명입니다.
* `event_date` ([Date](/sql-reference/data-types/date)) — 크래시가 발생한 날짜입니다.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 크래시가 발생한 시간입니다.
* `timestamp_ns` ([UInt64](/sql-reference/data-types/int-uint)) — 나노초 단위까지 포함한 이벤트 타임스탬프입니다.
* `signal` ([Int32](/sql-reference/data-types/int-uint)) — 시그널 번호입니다.
* `signal_code` ([Int32](/sql-reference/data-types/int-uint)) — 시그널 코드입니다. 시그널 원인에 대한 추가 정보를 제공합니다.
* `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) — 스레드 ID입니다.
* `query_id` ([String](/sql-reference/data-types/string)) — Query ID입니다.
* `query` ([String](/sql-reference/data-types/string)) — 크래시가 발생했을 때 실행 중이던 쿼리 텍스트입니다.
* `trace` ([Array(UInt64)](/sql-reference/data-types/array)) — 크래시 시점의 스택 트레이스입니다. 각 요소는 ClickHouse 서버 프로세스 내부의 가상 메모리 주소입니다.
* `trace_full` ([Array(String)](/sql-reference/data-types/array)) — 크래시 시점의 스택 트레이스입니다. 각 요소에는 ClickHouse 서버 프로세스 내부에서 호출된 메서드가 포함됩니다.
* `fault_address` ([Nullable(UInt64)](/sql-reference/data-types/nullable)) — 오류를 일으킨 메모리 주소입니다.
* `fault_access_type` ([String](/sql-reference/data-types/string)) — 오류를 일으킨 메모리 접근 유형입니다(예: &#39;read&#39;, &#39;write&#39;).
* `signal_description` ([String](/sql-reference/data-types/string)) — signal&#95;code를 기반으로 한 사람이 이해할 수 있는 설명입니다(예: &#39;Address not mapped to object&#39;).
* `current_exception_trace_full` ([Array(String)](/sql-reference/data-types/array)) — 심볼화된 프레임이 포함된 현재 예외 스택 트레이스입니다(`std::terminate` 크래시에만 해당).
* `version` ([String](/sql-reference/data-types/string)) — ClickHouse 서버 버전입니다.
* `revision` ([UInt32](/sql-reference/data-types/int-uint)) — ClickHouse 서버 리비전입니다.
* `build_id` ([String](/sql-reference/data-types/string)) — 컴파일러가 생성한 BuildID입니다.
* `git_hash` ([String](/sql-reference/data-types/string)) — ClickHouse 소스 코드의 Git 커밋 해시입니다.
* `architecture` ([String](/sql-reference/data-types/string)) — CPU 아키텍처입니다(예: x86&#95;64, aarch64).

{/*AUTOGENERATED_END*/ }
