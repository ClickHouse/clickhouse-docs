## 설명 \{#description\}

성공 및 실패한 모든 로그인 및 로그아웃 이벤트에 대한 정보를 포함합니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 쿼리를 실행하는 서버의 호스트명입니다.
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/sql-reference/data-types/enum)) — 로그인/로그아웃 결과입니다. 가능한 값: LoginFailure — 로그인 오류. LoginSuccess — 로그인 성공. Logout — 시스템 로그아웃.
* `auth_id` ([UUID](/sql-reference/data-types/uuid)) — 인증 ID입니다. 사용자가 로그인할 때마다 자동으로 생성되는 UUID입니다.
* `session_id` ([String](/sql-reference/data-types/string)) — 클라이언트가 HTTP 인터페이스를 통해 전달한 세션 ID입니다.
* `event_date` ([Date](/sql-reference/data-types/date)) — 로그인/로그아웃 날짜입니다.
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 로그인/로그아웃 시간입니다.
* `event_time_microseconds` ([DateTime64(6)](/sql-reference/data-types/datetime64)) — 로그인/로그아웃 시작 시각을 마이크로초 정밀도로 나타낸 값입니다.
* `user` ([Nullable(String)](/sql-reference/data-types/nullable)) — 사용자 이름입니다.
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/sql-reference/data-types/nullable)) — 인증 유형입니다.
* `profiles` ([Array(LowCardinality(String))](/sql-reference/data-types/array)) — 모든 역할 및/또는 사용자에 설정된 프로필 목록입니다.
* `roles` ([Array(LowCardinality(String))](/sql-reference/data-types/array)) — 프로필이 적용된 역할 목록입니다.
* `settings` ([Array(Tuple(LowCardinality(String), String))](/sql-reference/data-types/array)) — 클라이언트가 로그인하거나 로그아웃할 때 변경된 설정입니다.
* `client_address` ([IPv6](/sql-reference/data-types/ipv6)) — 로그인/로그아웃에 사용된 IP 주소입니다.
* `client_port` ([UInt16](/sql-reference/data-types/int-uint)) — 로그인/로그아웃에 사용된 클라이언트 포트입니다.
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/sql-reference/data-types/enum)) — 로그인이 시작된 인터페이스입니다.
* `client_hostname` ([String](/sql-reference/data-types/string)) — `clickhouse-client` 또는 다른 TCP 클라이언트가 실행되는 클라이언트 머신의 호스트명입니다.
* `client_name` ([String](/sql-reference/data-types/string)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 이름입니다.
* `client_revision` ([UInt32](/sql-reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 리비전입니다.
* `client_version_major` ([UInt32](/sql-reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 메이저 버전입니다.
* `client_version_minor` ([UInt32](/sql-reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 마이너 버전입니다.
* `client_version_patch` ([UInt32](/sql-reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트 버전의 패치 버전입니다.
* `failure_reason` ([String](/sql-reference/data-types/string)) — 로그인/로그아웃 실패 원인이 포함된 예외 메시지입니다.

{/*AUTOGENERATED_END*/ }
