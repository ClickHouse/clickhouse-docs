S3Queue 엔진에서 처리한 파일 정보가 포함된 로그 항목을 담고 있습니다.

이 테이블은 언제든지 안전하게 truncate하거나 drop할 수 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `hostname` ([LowCardinality(String)](/sql-reference/data-types/lowcardinality)) — 호스트명
* `event_date` ([Date](/sql-reference/data-types/date)) — 이 로그 행이 기록된 이벤트 날짜
* `event_time` ([DateTime](/sql-reference/data-types/datetime)) — 이 로그 행이 기록된 이벤트 시간
* `database` ([String](/sql-reference/data-types/string)) — 큐 테이블(`S3Queue` 또는 `AzureQueue`)이 있는 데이터베이스 이름
* `table` ([String](/sql-reference/data-types/string)) — 큐 테이블(`S3Queue` 또는 `AzureQueue`)의 이름
* `uuid` ([String](/sql-reference/data-types/string)) — 큐 테이블(`S3Queue` 또는 `AzureQueue`)의 UUID
* `file_name` ([String](/sql-reference/data-types/string)) — 처리 대상 파일의 이름
* `rows_processed` ([UInt64](/sql-reference/data-types/int-uint)) — 처리된 행 수
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/sql-reference/data-types/enum)) — 처리 대상 파일의 상태
* `processing_start_time` ([Nullable(DateTime)](/sql-reference/data-types/nullable)) — 파일 처리가 시작된 시간
* `processing_end_time` ([Nullable(DateTime)](/sql-reference/data-types/nullable)) — 파일 처리가 종료된 시간
* `exception` ([String](/sql-reference/data-types/string)) — 예외가 발생한 경우의 메시지
* `commit_id` ([UInt64](/sql-reference/data-types/int-uint)) — 이 파일이 커밋된 트랜잭션의 ID
* `commit_time` ([DateTime](/sql-reference/data-types/datetime)) — 파일이 Keeper에 커밋된 시간(실패 또는 처리 완료 상태)
* `transaction_start_time` ([DateTime](/sql-reference/data-types/datetime)) — 전체 처리 트랜잭션이 시작된 시간
* `get_object_time_ms` ([UInt64](/sql-reference/data-types/int-uint)) — 객체 스토리지에서 객체를 찾는 데 걸린 시간

{/*AUTOGENERATED_END*/ }
