# system.s3_queue_settings \{#systems3_queue_settings\}

[S3Queue](../../engines/table-engines/integrations/s3queue.md) 테이블의 설정 정보가 포함됩니다. 서버 버전 `24.10`부터 사용 가능합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — S3Queue Engine을 사용하는 테이블이 속한 데이터베이스.
* `table` ([String](../../sql-reference/data-types/)) — S3Queue Engine을 사용하는 테이블 이름.
* `name` ([String](../../sql-reference/data-types/)) — 설정 이름.
* `value` ([String](../../sql-reference/data-types/)) — 설정 값.
* `type` ([String](../../sql-reference/data-types/)) — 설정 유형(구현에 따라 달라지는 문자열 값).
* `changed` ([UInt8](../../sql-reference/data-types/)) — 설정이 설정 파일에서 명시적으로 정의되었거나 명시적으로 변경된 경우 1.
* `description` ([String](../../sql-reference/data-types/)) — 설정 설명.
* `alterable` ([UInt8](../../sql-reference/data-types/)) — 현재 사용자가 ALTER TABLE MODIFY SETTING을 통해 설정을 변경할 수 있는지 여부를 나타냅니다: 0 — 현재 사용자는 설정을 변경할 수 있음, 1 — 현재 사용자는 설정을 변경할 수 없음.

{/*AUTOGENERATED_END*/ }
