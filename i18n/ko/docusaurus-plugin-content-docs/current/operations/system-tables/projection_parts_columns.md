# system.projection_parts_columns \{#systemprojection_parts_columns\}

이 테이블은 MergeTree 계열 테이블의 프로젝션 파트에 있는 컬럼 정보를 제공합니다.



## 컬럼 \{#columns\}



{/*AUTOGENERATED_START*/ }

* `partition` ([String](../../sql-reference/data-types/)) — 파티션의 이름입니다.
* `name` ([String](../../sql-reference/data-types/)) — 데이터 파트의 이름입니다.
* `part_type` ([String](../../sql-reference/data-types/)) — 데이터 파트의 저장 포맷입니다.
* `parent_name` ([String](../../sql-reference/data-types/)) — 원본(부모) 데이터 파트의 이름입니다.
* `parent_uuid` ([UUID](../../sql-reference/data-types/)) — 원본(부모) 데이터 파트의 UUID입니다.
* `parent_part_type` ([String](../../sql-reference/data-types/)) — 원본(상위) 데이터 파트의 저장 형식입니다.
* `active` ([UInt8](../../sql-reference/data-types/)) — 데이터 파트가 활성 상태인지 여부를 나타내는 플래그입니다
* `marks` ([UInt64](../../sql-reference/data-types/)) — 마크의 수입니다.
* `rows` ([UInt64](../../sql-reference/data-types/)) — 행 수입니다.
* `bytes_on_disk` ([UInt64](../../sql-reference/data-types/)) — 모든 데이터 파트 파일 크기를 바이트 단위로 합산한 값입니다.
* `data_compressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 데이터 파트의 압축된 데이터 총 크기입니다. 모든 보조 파일(예: 마크 파일)은 포함되지 않습니다.
* `data_uncompressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 데이터 파트에서 압축되지 않은 데이터의 전체 크기입니다. 예를 들어 마크 파일과 같은 모든 보조 파일은 포함되지 않습니다.
* `marks_bytes` ([UInt64](../../sql-reference/data-types/)) — 마크가 저장된 파일의 크기입니다.
* `parent_marks` ([UInt64](../../sql-reference/data-types/)) — 소스(상위) 파트에 포함된 마크 개수입니다.
* `parent_rows` ([UInt64](../../sql-reference/data-types/)) — 소스(상위) 파트의 행 수입니다.
* `parent_bytes_on_disk` ([UInt64](../../sql-reference/data-types/)) — 모든 원본(부모) 데이터 파트 파일의 총 크기(바이트 단위)입니다.
* `parent_data_compressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 원본(부모) 데이터 파트의 압축된 데이터 전체 크기입니다.
* `parent_data_uncompressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 소스(상위) 데이터 파트의 압축되지 않은 데이터 총 크기입니다.
* `parent_marks_bytes` ([UInt64](../../sql-reference/data-types/)) — 소스(상위) 데이터 파트의 마크 파일 크기입니다.
* `modification_time` ([DateTime](../../sql-reference/data-types/)) — 데이터 파트가 저장된 디렉터리가 마지막으로 수정된 시간입니다. 일반적으로 데이터 파트가 생성된 시각에 해당합니다.
* `remove_time` ([DateTime](../../sql-reference/data-types/)) — 데이터 파트가 비활성화된 시각입니다.
* `refcount` ([UInt32](../../sql-reference/data-types/)) — 데이터 파트가 사용되는 위치 수입니다. 값이 2보다 크면 해당 데이터 파트가 쿼리나 머지 작업에서 사용 중임을 나타냅니다.
* `min_date` ([Date](../../sql-reference/data-types/)) — 파티션 키에 포함되어 있는 경우 Date 컬럼의 최소값입니다.
* `max_date` ([Date](../../sql-reference/data-types/)) — 파티션 키에 Date 컬럼이 포함되어 있는 경우 해당 컬럼의 최대 값입니다.
* `min_time` ([DateTime](../../sql-reference/data-types/)) — 파티션 키에 포함된 경우 DateTime 컬럼의 최솟값입니다.
* `max_time` ([DateTime](../../sql-reference/data-types/)) — 파티션 키에 포함되어 있다면 DateTime 컬럼의 최대값입니다.
* `partition_id` ([String](../../sql-reference/data-types/)) — 파티션의 ID입니다.
* `min_block_number` ([Int64](../../sql-reference/data-types/)) — 머지 후 현재 파트를 구성하는 데이터 파트들 가운데 가장 작은 번호입니다.
* `max_block_number` ([Int64](../../sql-reference/data-types/)) — 머지 후 현재 파트를 구성하는 데이터 파트들의 번호 중 최댓값입니다.
* `level` ([UInt32](../../sql-reference/data-types/)) — 머지 트리의 깊이입니다. 0이면 현재 파트는 다른 파트들을 머지한 것이 아니라 INSERT로 생성되었음을 의미합니다.
* `data_version` ([UInt64](../../sql-reference/data-types/)) — 데이터 파트에 어떤 뮤테이션을 적용해야 하는지 결정하는 데 사용되는 숫자입니다(버전이 `data_version`보다 높은 뮤테이션).
* `primary_key_bytes_in_memory` ([UInt64](../../sql-reference/data-types/)) — 기본 키 값이 차지하는 메모리 크기(바이트 단위)입니다.
* `primary_key_bytes_in_memory_allocated` ([UInt64](../../sql-reference/data-types/)) — 기본 키 값에 대해 예약된 메모리의 크기(바이트 단위)입니다.
* `database` ([String](../../sql-reference/data-types/)) — 데이터베이스의 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블의 이름입니다.
* `engine` ([String](../../sql-reference/data-types/)) — 매개변수를 포함하지 않는 테이블 엔진의 이름입니다.
* `disk_name` ([String](../../sql-reference/data-types/)) — 데이터 파트를 저장하는 디스크의 이름입니다.
* `path` ([String](../../sql-reference/data-types/)) — 데이터 파트 파일들이 있는 폴더의 절대 경로입니다.
* `column` ([String](../../sql-reference/data-types/)) — 컬럼 이름입니다.
* `type` ([String](../../sql-reference/data-types/)) — 컬럼 타입입니다.
* `column_position` ([UInt64](../../sql-reference/data-types/)) — 테이블에서 컬럼의 순서를 나타내며 1부터 시작합니다.
* `default_kind` ([String](../../sql-reference/data-types/)) — 기본값에 대한 표현식 유형(DEFAULT, MATERIALIZED, ALIAS)으로, 정의되지 않은 경우 빈 문자열입니다.
* `default_expression` ([String](../../sql-reference/data-types/)) — 기본값을 위한 표현식이며, 정의되지 않은 경우 빈 문자열입니다.
* `column_bytes_on_disk` ([UInt64](../../sql-reference/data-types/)) — 컬럼의 총 크기(바이트 단위)입니다.
* `column_data_compressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 해당 컬럼의 압축된 데이터 총 크기(바이트 단위)입니다.
* `column_data_uncompressed_bytes` ([UInt64](../../sql-reference/data-types/)) — 컬럼의 압축 해제된 데이터 전체 크기(바이트 단위)입니다.
* `column_marks_bytes` ([UInt64](../../sql-reference/data-types/)) — 마크 컬럼의 크기(바이트 단위)입니다.
* `column_modification_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 컬럼이 마지막으로 수정된 시각입니다.
