DROP TABLE이 실행되었지만 아직 데이터 정리 작업이 수행되지 않은 테이블에 대한 정보를 포함합니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `index` ([UInt32](../../sql-reference/data-types/)) — marked&#95;dropped&#95;tables 큐에서의 인덱스입니다.
* `database` ([String](../../sql-reference/data-types/)) — 데이터베이스 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블 이름입니다.
* `uuid` ([UUID](../../sql-reference/data-types/)) — 테이블 UUID입니다.
* `engine` ([String](../../sql-reference/data-types/)) — 테이블 엔진 이름입니다.
* `metadata_dropped_path` ([String](../../sql-reference/data-types/)) — metadata&#95;dropped 디렉터리 내 테이블 메타데이터 파일의 경로입니다.
* `table_dropped_time` ([DateTime](../../sql-reference/data-types/)) — 테이블 데이터 삭제를 다음에 시도하도록 예약된 시각입니다. 일반적으로는 테이블이 삭제된 시각에 `database_atomic_delay_before_drop_table_sec` 값을 더한 시각입니다.

{/*AUTOGENERATED_END*/ }

**예제**

다음 예제는 `dropped_tables` 정보 조회 방법을 보여줍니다.

```sql
SELECT *
FROM system.dropped_tables\G
```

```text
Row 1:
──────
index:                 0
database:              default
table:                 test
uuid:                  03141bb2-e97a-4d7c-a172-95cc066bb3bd
engine:                MergeTree
metadata_dropped_path: /data/ClickHouse/build/programs/data/metadata_dropped/default.test.03141bb2-e97a-4d7c-a172-95cc066bb3bd.sql
table_dropped_time:    2023-03-16 23:43:31
```
