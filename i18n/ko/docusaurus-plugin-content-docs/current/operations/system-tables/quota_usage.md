현재 사용자의 QUOTA 사용량: 사용된 양과 남은 양을 나타냅니다.


컬럼:

{/*AUTOGENERATED_START*/ }

* `quota_name` ([String](../../sql-reference/data-types/)) — 쿼터 이름.
* `quota_key` ([String](../../sql-reference/data-types/)) — 키 값.
* `start_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 리소스 사용량 계산을 시작하는 시각.
* `end_time` ([Nullable(DateTime)](../../sql-reference/data-types/)) — 리소스 사용량 계산을 종료하는 시각.
* `duration` ([Nullable(UInt32)](../../sql-reference/data-types/)) — 리소스 사용량을 계산하는 시간 구간의 길이(초 단위).
* `queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재까지 실행된 쿼리 수.
* `max_queries` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 모든 유형을 포함하여 실행이 허용되는 최대 쿼리 수.
* `query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재까지 실행된 SELECT 쿼리 수.
* `max_query_selects` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용되는 SELECT 쿼리의 최대 개수.
* `query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재까지 실행된 INSERT 쿼리 수.
* `max_query_inserts` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 실행이 허용되는 INSERT 쿼리의 최대 개수.
* `errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 오류로 종료된 쿼리의 현재 개수.
* `max_errors` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 오류로 종료되는 것이 허용되는 쿼리의 최대 개수.
* `result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 결과 집합에 포함된 행의 총 개수.
* `max_result_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 결과 집합에 포함되는 것이 허용되는 행의 최대 총 개수.
* `result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 결과 집합에 포함된 바이트의 총 개수.
* `max_result_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 결과 집합에 포함되는 것이 허용되는 바이트의 최대 총 개수.
* `read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 실행 시 읽은 행의 총 개수.
* `max_read_rows` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 실행 시 읽을 수 있도록 허용된 행의 최대 개수.
* `read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 실행 시 읽은 바이트의 총 개수.
* `max_read_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 실행 시 읽을 수 있도록 허용된 바이트의 최대 개수.
* `execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 현재 기간 동안 쿼리 실행에 소요된 시간의 현재 총합(나노초 단위).
* `max_execution_time` ([Nullable(Float64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 실행에 허용되는 시간의 최대값(나노초 단위).
* `written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 모든 쿼리 실행 시 기록한 바이트의 총 개수.
* `max_written_bytes` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 모든 쿼리 실행 시 기록할 수 있도록 허용된 바이트의 최대 개수.
* `failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 현재 기간 동안 연속으로 실패한 인증 시도의 현재 개수.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](../../sql-reference/data-types/)) — 지정된 기간 내에 연속으로 실패하는 것이 허용되는 인증 시도의 최대 개수.
