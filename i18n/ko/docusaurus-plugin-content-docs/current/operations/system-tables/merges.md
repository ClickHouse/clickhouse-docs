# system.merges \{#systemmerges\}

<SystemTableCloud />

MergeTree 패밀리의 테이블에서 현재 진행 중인 머지 작업과 파트 뮤테이션에 대한 정보가 포함되어 있습니다.

컬럼:

{/*AUTOGENERATED_START*/ }

* `database` ([String](../../sql-reference/data-types/)) — 테이블이 속한 데이터베이스 이름입니다.
* `table` ([String](../../sql-reference/data-types/)) — 테이블 이름입니다.
* `elapsed` ([Float64](../../sql-reference/data-types/)) — 머지 작업이 시작된 후 경과한 시간(초)입니다.
* `progress` ([Float64](../../sql-reference/data-types/)) — 완료된 작업의 비율로, 0에서 1까지의 값입니다.
* `num_parts` ([UInt64](../../sql-reference/data-types/)) — 머지 대상이 되는 파트의 개수입니다.
* `source_part_names` ([Array(String)](../../sql-reference/data-types/)) — 소스 파트 이름 목록입니다.
* `result_part_name` ([String](../../sql-reference/data-types/)) — 머지 결과로 생성될 파트의 이름입니다.
* `source_part_paths` ([Array(String)](../../sql-reference/data-types/)) — 각 소스 파트의 경로 목록입니다.
* `result_part_path` ([String](../../sql-reference/data-types/)) — 머지 결과로 생성될 파트의 경로입니다.
* `partition_id` ([String](../../sql-reference/data-types/)) — 머지가 수행 중인 파티션의 식별자입니다.
* `partition` ([String](../../sql-reference/data-types/)) — 파티션 이름입니다.
* `is_mutation` ([UInt8](../../sql-reference/data-types/)) — 이 프로세스가 파트 뮤테이션이면 1입니다.
* `total_size_bytes_compressed` ([UInt64](../../sql-reference/data-types/)) — 머지된 청크에 포함된 압축된 데이터의 총 크기(바이트)입니다.
* `total_size_bytes_uncompressed` ([UInt64](../../sql-reference/data-types/)) — 머지된 청크에 포함된 압축되지 않은 데이터의 총 크기(바이트)입니다.
* `total_size_marks` ([UInt64](../../sql-reference/data-types/)) — 머지된 파트에 있는 마크의 총 개수입니다.
* `bytes_read_uncompressed` ([UInt64](../../sql-reference/data-types/)) — 압축되지 않은 기준으로 읽은 데이터의 바이트 수입니다.
* `rows_read` ([UInt64](../../sql-reference/data-types/)) — 읽은 행의 개수입니다.
* `bytes_written_uncompressed` ([UInt64](../../sql-reference/data-types/)) — 압축되지 않은 기준으로 기록한 데이터의 바이트 수입니다.
* `rows_written` ([UInt64](../../sql-reference/data-types/)) — 기록한 행의 개수입니다.
* `columns_written` ([UInt64](../../sql-reference/data-types/)) — 기록한 컬럼의 개수입니다(수직 병합 알고리즘의 경우).
* `memory_usage` ([UInt64](../../sql-reference/data-types/)) — 머지 프로세스가 사용하는 메모리 양입니다.
* `thread_id` ([UInt64](../../sql-reference/data-types/)) — 머지 프로세스의 스레드 ID입니다.
* `merge_type` ([String](../../sql-reference/data-types/)) — 현재 머지 유형입니다. 뮤테이션인 경우 비어 있습니다.
* `merge_algorithm` ([String](../../sql-reference/data-types/)) — 현재 머지에 사용되는 알고리즘입니다. 뮤테이션인 경우 비어 있습니다.

{/*AUTOGENERATED_END*/ }
