## 설명 \{#description\}

트랜잭션 목록과 각 트랜잭션의 상태를 나타냅니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `tid` ([Tuple(UInt64, UInt64, UUID)](/sql-reference/data-types/tuple)) — 트랜잭션의 식별자입니다.
* `tid_hash` ([UInt64](/sql-reference/data-types/int-uint)) — 식별자의 해시값입니다.
* `elapsed` ([Float64](/sql-reference/data-types/float)) — 처리 중인 트랜잭션의 경과 시간입니다.
* `is_readonly` ([UInt8](/sql-reference/data-types/int-uint)) — 트랜잭션이 쓰기 작업을 수행하지 않았는지 여부를 나타내는 플래그입니다.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/sql-reference/data-types/enum)) — 트랜잭션의 상태입니다. 가능한 값: RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.

{/*AUTOGENERATED_END*/ }
