압축 및 암호화 코덱에 대한 정보를 포함합니다.

이 테이블을 사용하여 사용 가능한 압축 및 암호화 코덱에 대한 정보를 조회할 수 있습니다.

`system.codecs` 테이블에는 다음과 같은 컬럼이 포함되어 있습니다(컬럼 타입은 대괄호 안에 표시됩니다):

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 코덱 이름입니다.
* `method_byte` ([UInt8](../../sql-reference/data-types/)) — 압축 파일에서 코덱을 나타내는 바이트 값입니다.
* `is_compression` ([UInt8](../../sql-reference/data-types/)) — 이 코덱이 데이터를 압축하면 true입니다. 그렇지 않으면 압축을 돕는 변환일 수 있습니다.
* `is_generic_compression` ([UInt8](../../sql-reference/data-types/)) — 코덱이 lz4, zstd와 같은 범용 압축 알고리즘인지 여부입니다.
* `is_encryption` ([UInt8](../../sql-reference/data-types/)) — 데이터 암호화를 수행하는 코덱인지 여부입니다.
* `is_timeseries_codec` ([UInt8](../../sql-reference/data-types/)) — 부동 소수점 시계열 데이터를 위한 코덱인지 여부입니다.
* `is_experimental` ([UInt8](../../sql-reference/data-types/)) — 코덱이 실험적인지 여부입니다.
* `description` ([String](../../sql-reference/data-types/)) — 코덱에 대한 상위 수준 설명입니다.

{/*AUTOGENERATED_END*/ }

**예제**

쿼리:

```sql
SELECT * FROM system.codecs WHERE name='LZ4'
```

결과:

```text
Row 1:
──────
name:                   LZ4
method_byte:            130
is_compression:         1
is_generic_compression: 1
is_encryption:          0
is_timeseries_codec:    0
is_experimental:        0
description:            Extremely fast; good compression; balanced speed and efficiency.
```
