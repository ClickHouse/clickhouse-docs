## 설명 \{#description\}

사용 가능한 모든 테이블 함수와 각 함수에 대한 설명 목록입니다.

## 컬럼 \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) — 테이블 함수의 이름입니다.
* `description` ([String](../../sql-reference/data-types/)) — 테이블 함수에 대한 간략한 설명입니다.
* `allow_readonly` ([UInt8](../../sql-reference/data-types/)) — 읽기 전용 사용자가 이 함수를 사용할 수 있는지 여부를 나타내는 플래그입니다.
