## м„ӨлӘ… \{#description\}

`config.xml`м—җ м§Җм •лҗң м„ңлІ„ м „м—ӯ м„Өм •м—җ лҢҖн•ң м •ліҙлҘј нҸ¬н•Ён•©лӢҲлӢӨ.
нҳ„мһ¬ мқҙ н…Ңмқҙлё”мқҖ `config.xml`мқҳ мөңмғҒмң„ мҲҳмӨҖм—җ мһҲлҠ” м„Өм •л§Ң н‘ңмӢңн•ҳл©°, мӨ‘мІ©лҗң м„Өм •(мҳҲ: [logger](../../operations/server-configuration-parameters/settings.md#logger))мқҖ м§Җмӣҗн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

## м»¬лҹј \{#columns\}

{/*AUTOGENERATED_START*/ }

* `name` ([String](../../sql-reference/data-types/)) вҖ” м„ңлІ„ м„Өм • мқҙлҰ„мһ…лӢҲлӢӨ.
* `value` ([String](../../sql-reference/data-types/)) вҖ” м„ңлІ„ м„Өм • к°’мһ…лӢҲлӢӨ.
* `default` ([String](../../sql-reference/data-types/)) вҖ” м„ңлІ„ м„Өм • кё°ліёк°’мһ…лӢҲлӢӨ.
* `changed` ([UInt8](../../sql-reference/data-types/)) вҖ” м„Өм •мқҙ config.xmlм—җ лӘ…мӢңлҗҳм—ҲлҠ”м§Җ м—¬л¶ҖлҘј лӮҳнғҖлғ…лӢҲлӢӨ.
* `description` ([String](../../sql-reference/data-types/)) вҖ” м„ңлІ„ м„Өм •м—җ лҢҖн•ң к°„лӢЁн•ң м„ӨлӘ…мһ…лӢҲлӢӨ.
* `type` ([String](../../sql-reference/data-types/)) вҖ” м„ңлІ„ м„Өм • к°’мқҳ мң нҳ•мһ…лӢҲлӢӨ.
* `changeable_without_restart` ([Enum8(&#39;No&#39; = 0, &#39;IncreaseOnly&#39; = 1, &#39;DecreaseOnly&#39; = 2, &#39;Yes&#39; = 3)](../../sql-reference/data-types/)) вҖ” м„Өм •мқ„ лҹ°нғҖмһ„м—җ ліҖкІҪн•  мҲҳ мһҲлҠ”м§Җ м—¬л¶ҖлҘј лӮҳнғҖлғ…лӢҲлӢӨ.
* `is_obsolete` ([UInt8](../../sql-reference/data-types/)) вҖ” м„Өм •мқҙ нҸҗкё°лҗҳм—ҲлҠ”м§Җ м—¬л¶ҖлҘј лӮҳнғҖлғ…лӢҲлӢӨ.

## мҳҲмӢң \{#example\}

лӢӨмқҢ мҳҲмӢңлҠ” мқҙлҰ„м—җ `thread_pool`мқ„ нҸ¬н•Ён•ҳлҠ” м„ңлІ„ м„Өм • м •ліҙлҘј к°Җм ёмҳӨлҠ” л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ.

```sql
SELECT *
FROM system.server_settings
WHERE name LIKE '%thread_pool%'
```
