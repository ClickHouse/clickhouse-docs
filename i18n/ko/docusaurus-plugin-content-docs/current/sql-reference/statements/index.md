# ClickHouse SQL 문 \{#clickhouse-sql-statements\}

사용자는 SQL 문으로 ClickHouse와 상호 작용합니다. ClickHouse는 [SELECT](select/index.md), [CREATE](create/index.md)와 같은 일반적인 SQL 문을 지원할 뿐만 아니라, [KILL](kill.md), [OPTIMIZE](optimize.md)와 같은 특수한 SQL 문도 제공합니다.
