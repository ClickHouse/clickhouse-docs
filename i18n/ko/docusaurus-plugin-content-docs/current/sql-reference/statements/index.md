# ClickHouse SQL 문

사용자는 SQL 문을 사용하여 ClickHouse와 상호 작용합니다. ClickHouse는 [SELECT](select/index.md) 및 [CREATE](create/index.md)와 같은 일반 SQL 문을 지원하지만, [KILL](kill.md) 및 [OPTIMIZE](optimize.md)와 같은 전문화된 문도 제공합니다.
