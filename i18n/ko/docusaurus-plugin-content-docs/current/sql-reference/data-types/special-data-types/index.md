# 특수 데이터 타입 \{#special-data-types\}

특수 데이터 타입 값은 직렬화하여 테이블에 저장하거나 쿼리 결과로 출력할 수는 없지만, 쿼리 실행 중에는 중간 결과로 사용할 수 있습니다.
