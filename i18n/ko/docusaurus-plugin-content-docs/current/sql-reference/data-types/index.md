# ClickHouse의 데이터 유형

이 섹션에서는 ClickHouse에서 지원하는 데이터 유형에 대해 설명합니다. 예를 들어 [정수](int-uint.md), [부동 소수점](float.md) 및 [문자열](string.md)이 있습니다.

시스템 테이블 [system.data_type_families](/operations/system-tables/data_type_families)는 사용 가능한 모든 데이터 유형의 개요를 제공합니다. 
그뿐만 아니라 데이터 유형이 다른 데이터 유형의 별칭인지 여부와 이름이 대소문자를 구분하는지(예: `bool`과 `BOOL`)도 보여줍니다.
