## groupArrayIntersect \{#groupArrayIntersect\}

도입 버전: v24.2

주어진 배열들의 교집합을 반환합니다(모든 배열에 공통으로 포함된 모든 항목을 반환합니다).

**구문**

```sql
groupArrayIntersect(x)
```

**인수**

* `x` — 인수(컬럼 이름 또는 표현식). [`Any`](/sql-reference/data-types)

**반환 값**

모든 배열에 공통으로 포함된 요소를 포함하는 배열을 반환합니다. [`Array`](/sql-reference/data-types/array)

**예시**

**사용 예시**

```sql title=Query
-- Create table with Memory engine
CREATE TABLE numbers (
    a Array(Int32)
) ENGINE = Memory;

-- Insert sample data
INSERT INTO numbers VALUES
    ([1,2,4]),
    ([1,5,2,8,-1,0]),
    ([1,5,7,5,8,2]);

SELECT groupArrayIntersect(a) AS intersection FROM numbers;
```

```response title=Response
┌─intersection──────┐
│ [1, 2]            │
└───────────────────┘
```
