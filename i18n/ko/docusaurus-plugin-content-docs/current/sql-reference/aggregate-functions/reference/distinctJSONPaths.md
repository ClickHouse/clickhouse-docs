## distinctJSONPaths \{#distinctJSONPaths\}

도입된 버전: v24.9

[JSON](https://clickhouse.com/docs/sql-reference/data-types/newjson) 컬럼에 저장된 고유 경로 목록을 계산합니다.

**구문**

```sql
distinctJSONPaths(json)
```

**인수**

* `json` — JSON 컬럼. [`JSON`](/sql-reference/data-types/newjson)

**반환값**

정렬된 경로 목록을 반환합니다. [`Array(String)`](/sql-reference/data-types/array)

**예제**

**중첩된 JSON의 기본 사용 예**

```sql title=Query
DROP TABLE IF EXISTS test_json;
CREATE TABLE test_json(json JSON) ENGINE = Memory;
INSERT INTO test_json VALUES ('{"a" : 42, "b" : "Hello"}'), ('{"b" : [1, 2, 3], "c" : {"d" : {"e" : "2020-01-01"}}}'), ('{"a" : 43, "c" : {"d" : {"f" : [{"g" : 42}]}}}');

SELECT distinctJSONPaths(json) FROM test_json;
```

```response title=Response
┌─distinctJSONPaths(json)───┐
│ ['a','b','c.d.e','c.d.f'] │
└───────────────────────────┘
```

**선언한 JSON 경로 사용 시**

```sql title=Query
DROP TABLE IF EXISTS test_json;
CREATE TABLE test_json(json JSON) ENGINE = Memory;
INSERT INTO test_json VALUES ('{"a" : 42, "b" : "Hello"}'), ('{"b" : [1, 2, 3], "c" : {"d" : {"e" : "2020-01-01"}}}'), ('{"a" : 43, "c" : {"d" : {"f" : [{"g" : 42}]}}}')

SELECT distinctJSONPaths(json) FROM test_json;
```

```response title=Response
┌─distinctJSONPaths(json)─┐
│ ['a','b','c']           │
└─────────────────────────┘
```
