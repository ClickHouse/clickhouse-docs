## timeSeriesGroupArray \{#timeSeriesGroupArray\}

도입된 버전: v25.9

시계열 데이터를 타임스탬프 기준으로 오름차순 정렬합니다.

:::note
이 함수는 실험적 기능이므로 `allow_experimental_ts_to_grid_aggregate_function=true` 로 설정하여 활성화합니다.
:::

**문법**

```sql
timeSeriesGroupArray(timestamp, value)
```

**인자**

* `timestamp` — 샘플의 타임스탬프입니다. [`DateTime`](/sql-reference/data-types/datetime) 또는 [`UInt32`](/sql-reference/data-types/int-uint) 또는 [`UInt64`](/sql-reference/data-types/int-uint)
* `value` — 타임스탬프에 해당하는 시계열의 값입니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**반환 값**

타임스탬프 기준으로 오름차순 정렬된 튜플 `(timestamp, value)`의 배열을 반환합니다. 동일한 타임스탬프에 여러 개의 값이 있는 경우, 함수는 이들 중 최댓값을 선택합니다. [`Array(Tuple(T1, T2))`](/sql-reference/data-types/array)

**예제**

**개별 값을 사용하는 기본 예제**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamp, value)
FROM
(
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesGroupArray(timestamp, value)───────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]  │
└──────────────────────────────────────────────────────┘
```

**동일한 크기의 배열로 타임스탬프와 값의 여러 샘플을 전달하기**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamps, values);
```

```response title=Response
┌─timeSeriesGroupArray(timestamps, values)──────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]   │
└───────────────────────────────────────────────────────┘
```
