## skewPop \{#skewPop\}

도입 버전: v20.1

시퀀스의 [왜도](https://en.wikipedia.org/wiki/Skewness)를 계산합니다.

**구문**

```sql
skewPop(expr)
```

**인수**

* `expr` — 숫자를 반환하는 식입니다. [`Expression`](/sql-reference/data-types/special-data-types/expression)

**반환 값**

주어진 분포에 대한 왜도(skewness)를 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예제**

**대칭 분포**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**오른쪽으로 치우친 분포**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
