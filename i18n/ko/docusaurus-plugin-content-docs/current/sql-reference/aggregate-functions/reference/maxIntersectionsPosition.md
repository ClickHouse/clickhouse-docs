## maxIntersectionsPosition \{#maxIntersectionsPosition\}

도입: v1.1

[`maxIntersections`](/sql-reference/aggregate-functions/reference/maxintersections) FUNCTION에서 발생하는 지점의 위치를 계산하는 집계 FUNCTION입니다.

**구문**

```sql
maxIntersectionsPosition(start_column, end_column)
```

**인수**

* `start_column` — 각 구간의 시작을 나타내는 수치형 컬럼입니다. `start_column`이 `NULL` 이거나 0이면 해당 구간은 건너뜁니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float)
* `end_column` — 각 구간의 끝을 나타내는 수치형 컬럼입니다. `end_column`이 `NULL` 이거나 0이면 해당 구간은 건너뜁니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float)

**반환 값**

서로 교차하는 구간의 개수가 최대가 되는 시작 위치들을 반환합니다. [`Any`](/sql-reference/data-types)

**예시**

**최대 교차 구간 시작 위치 찾기**

```sql title=Query
CREATE TABLE my_events (
    start UInt32,
    end UInt32
)
ENGINE = MergeTree
ORDER BY tuple();

INSERT INTO my_events VALUES
(1, 3),
(1, 6),
(2, 5),
(3, 7);

SELECT maxIntersectionsPosition(start, end) FROM my_events;
```

```response title=Response
┌─maxIntersectionsPosition(start, end)─┐
│                                    2 │
└──────────────────────────────────────┘
```
