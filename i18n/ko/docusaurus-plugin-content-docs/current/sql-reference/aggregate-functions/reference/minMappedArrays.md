## minMappedArrays \{#minMappedArrays\}

도입 버전: v20.5

`key` 배열에 지정된 키를 기준으로 `value` 배열에서 최소값을 계산합니다.

:::note

* 키 배열과 값 배열로 이루어진 튜플을 전달하는 것은 키 배열과 값 배열을 각각 전달하는 것과 동일합니다.
* 집계되는 각 행에 대해 `key`와 `value`의 원소 개수는 동일해야 합니다.
  :::

**구문**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**인수**

* `key` — 키의 배열. [`Array(T)`](/sql-reference/data-types/array)
* `value` — 값의 배열. [`Array(T)`](/sql-reference/data-types/array)

**반환 값**

정렬된 순서의 키 배열과 각 키에 대해 계산된 값 배열, 이렇게 두 개의 배열로 구성된 튜플을 반환합니다. [`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**예제**

**사용 예**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
