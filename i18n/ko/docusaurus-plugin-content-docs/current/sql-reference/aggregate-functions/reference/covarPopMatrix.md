## covarPopMatrix \{#covarPopMatrix\}

도입 버전: v23.2

N개의 변수에 대한 모집단 공분산 행렬을 반환합니다.

**구문**

```sql
covarPopMatrix(x1[, x2, ...])
```

**인수(Arguments)**

* `x1[, x2, ...]` — 임의 개수의 매개변수입니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**반환 값(Returned value)**

모집단 공분산 행렬을 반환합니다. [`Array(Array(Float64))`](/sql-reference/data-types/array)

**예시(Examples)**

**기본 모집단 공분산 행렬 계산**

```sql title=Query
DROP TABLE IF EXISTS test;
CREATE TABLE test
(
    a UInt32,
    b Float64,
    c Float64,
    d Float64
)
ENGINE = Memory;
INSERT INTO test(a, b, c, d) VALUES (1, 5.6, -4.4, 2.6), (2, -9.6, 3, 3.3), (3, -1.3, -4, 1.2), (4, 5.3, 9.7, 2.3), (5, 4.4, 0.037, 1.222), (6, -8.6, -7.8, 2.1233), (7, 5.1, 9.3, 8.1222), (8, 7.9, -3.6, 9.837), (9, -8.2, 0.62, 8.43555), (10, -3, 7.3, 6.762);

SELECT arrayMap(x -> round(x, 3), arrayJoin(covarPopMatrix(a, b, c, d))) AS covarPopMatrix
FROM test
```

