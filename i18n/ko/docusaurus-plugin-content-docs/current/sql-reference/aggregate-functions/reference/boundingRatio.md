## boundingRatio \{#boundingRatio\}

도입 버전: v20.1

값 그룹에서 가장 왼쪽 점과 가장 오른쪽 점을 이은 직선의 기울기를 계산합니다.

**구문**

```sql
boundingRatio(x, y)
```

**인수**

* `x` — X 좌표 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)
* `y` — Y 좌표 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**반환 값**

가장 왼쪽 점과 가장 오른쪽 점을 잇는 직선의 기울기를 반환하며, 데이터가 비어 있으면 `NaN`을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**샘플 데이터**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
┌─number─┬─multiply(number, 1.5)─┐
│      0 │                     0 │
│      1 │                   1.5 │
│      2 │                     3 │
│      3 │                   4.5 │
│      4 │                     6 │
│      5 │                   7.5 │
│      6 │                     9 │
│      7 │                  10.5 │
│      8 │                    12 │
│      9 │                  13.5 │
└────────┴───────────────────────┘
```

**사용 예**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```

```response title=Response
┌─boundingRatio(number, multiply(number, 1.5))─┐
│                                          1.5 │
└──────────────────────────────────────────────┘
```
