## stddevPopStable \{#stddevPopStable\}

도입된 버전: v1.1

결과는 [varPop](../../../sql-reference/aggregate-functions/reference/varPop.md)의 제곱근과 동일합니다. [stddevPop](../reference/stddevPop.md)과 달리, 이 FUNCTION은 수치적으로 안정적인 알고리즘을 사용합니다. 실행 속도는 더 느리지만 계산 오차는 더 작습니다.

**구문**

```sql
stddevPopStable(x)
```

**인수**

* `x` — 표준 편차를 계산할 값들의 모집단입니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal*`](/sql-reference/data-types/decimal)

**반환 값**

`x`의 분산의 제곱근을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예제**

**기본 예제**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population Float64,
)
ENGINE = Log;

INSERT INTO test_data SELECT randUniform(5.5, 10) FROM numbers(1000000);

SELECT
    stddevPopStable(population) AS stddev
FROM test_data;
```

```response title=Response
┌─────────────stddev─┐
│ 1.2999977786592576 │
└────────────────────┘
```
