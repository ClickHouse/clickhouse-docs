## groupBitOr \{#groupBitOr\}

도입 버전: v1.1

일련의 숫자에 대해 비트 단위 OR 연산을 수행합니다.

**구문**

```sql
groupBitOr(expr)
```

**별칭**: `BIT_OR`

**인수**

* `expr` — `(U)Int*` 형식의 표현식입니다. [`(U)Int*`](/sql-reference/data-types/int-uint)

**반환 값**

`(U)Int*` 형식의 값을 반환합니다. [`(U)Int*`](/sql-reference/data-types/int-uint)

**예시**

**비트 OR 예제**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
