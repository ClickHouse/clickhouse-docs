## entropy \{#entropy\}

도입 버전: v20.1

값 컬럼의 [Shannon 엔트로피](https://en.wikipedia.org/wiki/Entropy_\(information_theory\))를 계산합니다.

**구문**

```sql
entropy(val)
```

**인수**

* `val` — 임의 타입의 값을 담은 컬럼. [`Any`](/sql-reference/data-types)

**반환 값**

Shannon 엔트로피를 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**기본 엔트로피 계산**

```sql title=Query
CREATE TABLE entropy (`vals` UInt32,`strings` String)
ENGINE = Memory;

INSERT INTO entropy VALUES (1, 'A'), (1, 'A'), (1,'A'), (1,'A'), (2,'B'), (2,'B'), (2,'C'), (2,'D');

SELECT entropy(vals), entropy(strings) FROM entropy
```

```response title=Response
┌─entropy(vals)─┬─entropy(strings)─┐
│             1 │             1.75 │
└───────────────┴──────────────────┘
```
