## exponentialTimeDecayedMax \{#exponentialTimeDecayedMax\}

도입 버전: v21.12

시간 인덱스 `t`에서의 지수 평활 이동 평균값과 `t-1`에서의 값 중 최대값을 반환합니다.

**구문**

```sql
exponentialTimeDecayedMax(x)(value, timeunit)
```

**매개변수**

* `x` — 반감기. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**인수**

* `value` — 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)
* `timeunit` — 시간 단위. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal) 또는 [`DateTime`](/sql-reference/data-types/datetime) 또는 [`DateTime64`](/sql-reference/data-types/datetime64)

**반환값**

`t`와 `t-1`에서의 지수 평활 가중 이동 평균의 최댓값을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**시각적 표현을 포함한 윈도 함수 사용 예**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 5, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedMax(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
