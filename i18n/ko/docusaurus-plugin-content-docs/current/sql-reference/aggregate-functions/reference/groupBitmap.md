## groupBitmap \{#groupBitmap\}

도입된 버전: v20.1

부호 없는 정수 컬럼에서 비트맵(비트 배열)을 생성한 다음, 해당 비트맵에서 고유 값의 개수(카디널리티)를 반환합니다.
`-State` 조합자 접미사를 추가하면 개수를 반환하는 대신 실제 [비트맵 객체](/sql-reference/functions/bitmap-functions)를 반환합니다.

**구문**

```sql
groupBitmap(expr)
groupBitmapState(expr)
```

**인수**

* `expr` — `UInt*` 타입을 결과로 내는 표현식입니다. [`UInt*`](/sql-reference/data-types/int-uint)

**반환 값**

`-State`를 사용할 때는 비트맵 객체를, 그 외에는 `UInt64` 타입의 개수를 반환합니다. [`UInt64`](/sql-reference/data-types/int-uint)

**예시**

**사용 예시**

```sql title=Query
CREATE TABLE t (UserID UInt32) ENGINE = Memory;
INSERT INTO t VALUES (1), (1), (2), (3);

SELECT groupBitmap(UserID) AS num FROM t;
```

```response title=Response
┌─num─┐
│   3 │
└─────┘
```
