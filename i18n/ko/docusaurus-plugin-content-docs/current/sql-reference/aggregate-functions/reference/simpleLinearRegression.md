## simpleLinearRegression \{#simpleLinearRegression\}

도입 버전: v20.1

단순(일변량) 선형 회귀를 수행합니다.

**구문**

```sql
simpleLinearRegression(x, y)
```

**인수**

* `x` — 설명 변수 값이 들어 있는 컬럼. [`Float64`](/sql-reference/data-types/float)
* `y` — 종속 변수 값이 들어 있는 컬럼. [`Float64`](/sql-reference/data-types/float)

**반환 값**

결과 회귀 직선 `y = k*x + b`의 상수 `(k, b)`를 반환합니다. [`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple)

**예시**

**완전한 선형 적합**

```sql title=Query
SELECT arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [0, 1, 2, 3]);
```

```response title=Response
┌─arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [0, 1, 2, 3])─┐
│ (1,0)                                                             │
└───────────────────────────────────────────────────────────────────┘
```

**절편을 포함한 선형 회귀**

```sql title=Query
SELECT arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [3, 4, 5, 6]);
```

```response title=Response
┌─arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [3, 4, 5, 6])─┐
│ (1,3)                                                             │
└───────────────────────────────────────────────────────────────────┘
```
