## avg \{#avg\}

도입된 버전: v1.1

산술 평균을 계산합니다.

**구문**

```sql
avg(x)
```

**인수**

* `x` — 입력 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**반환 값**

산술 평균을 반환하며, 입력 매개변수 `x`가 비어 있는 경우 `NaN`을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**기본 사용법**

```sql title=Query
SELECT avg(x) FROM VALUES('x Int8', 0, 1, 2, 3, 4, 5);
```

```response title=Response
┌─avg(x)─┐
│    2.5 │
└────────┘
```

**빈 테이블에서는 NaN이 반환됩니다**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;

SELECT avg(t) FROM test;
```

```response title=Response
┌─avg(x)─┐
│    nan │
└────────┘
```
