## aggThrow \{#aggThrow\}

도입 버전: v20.1

이 함수는 예외 안전성을 테스트하는 용도로 사용할 수 있습니다. 생성 시 지정된 확률로 예외를 던집니다.

**구문**

```sql
aggThrow(throw_prob)
```

**인수**

* `throw_prob` — 생성 시 예외를 던질 확률. [`Float64`](/sql-reference/data-types/float)

**반환 값**

다음과 같은 예외를 반환합니다: `Code: 503. DB::Exception: Aggregate function aggThrow has thrown exception successfully`.

**예시**

**사용 예시**

```sql title=Query
SELECT number % 2 AS even, aggThrow(number) FROM numbers(10) GROUP BY even;
```

```response title=Response
Received exception:
Code: 503. DB::Exception: Aggregate function aggThrow has thrown exception successfully: While executing AggregatingTransform. (AGGREGATE_FUNCTION_THROW)
```
