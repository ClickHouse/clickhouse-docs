## skewSamp \{#skewSamp\}

도입된 버전: v20.1

시퀀스의 [표본 왜도](https://en.wikipedia.org/wiki/Skewness)를 계산합니다.

전달된 값들이 어떤 확률 변수의 표본을 이룰 때, 그 확률 변수 왜도의 불편 추정값을 제공합니다.

**구문**

```sql
skewSamp(expr)
```

**인수**

* `expr` — 숫자를 반환하는 표현식. [`Expression`](/sql-reference/data-types/special-data-types/expression)

**반환 값**

주어진 분포의 왜도(skewness)를 계산해 반환합니다. `n <= 1`인 경우(`n`은 표본 크기), 함수는 `nan`을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**대칭 분포**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**오른쪽으로 치우친 분포**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
