## exponentialTimeDecayedAvg \{#exponentialTimeDecayedAvg\}

도입 버전: v21.12

시점 `t`에서 시계열 값에 대한 지수 평활 가중 이동 평균을 반환합니다.

**구문**

```sql
exponentialTimeDecayedAvg(x)(v, t)
```

**매개변수**

* `x` — 반감기 기간. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**인자**

* `v` — 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)
* `t` — 시간. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal) 또는 [`DateTime`](/sql-reference/data-types/datetime) 또는 [`DateTime64`](/sql-reference/data-types/datetime64)

**반환 값**

시점 `t`에서의 지수 평활 가중 이동 평균을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**시각적 표현과 함께하는 윈도우 함수 사용**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 5, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedAvg(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    )
```
