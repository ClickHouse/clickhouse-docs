## analysisOfVariance \{#analysisOfVariance\}

도입 버전: v22.10

일원 분산 분석(ANOVA 검정)을 위한 통계적 검정을 제공합니다. 정규 분포를 따르는 여러 집단에 대해 모든 집단의 평균이 동일한지 여부를 검정합니다.

:::note
집단 번호는 0부터 시작하며, 검정을 수행하려면 최소 2개 이상의 집단이 필요합니다.
관측값이 2개 이상인 집단이 최소 한 개는 있어야 합니다.
:::

**구문**

```sql
analysisOfVariance(val, group_no)
```

**별칭**: `anova`

**인수**

* `val` — 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)
* `group_no` — `val`이 속하는 그룹 번호. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**반환 값**

F-통계량과 p-값으로 구성된 튜플을 반환합니다. [`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple)

**예제**

**기본 사용 예제**

```sql title=Query
SELECT analysisOfVariance(number, number % 2) FROM numbers(1048575);
```

```response title=Response
┌─analysisOfVariance(number, modulo(number, 2))─┐
│ (0,1)                                         │
└───────────────────────────────────────────────┘
```
