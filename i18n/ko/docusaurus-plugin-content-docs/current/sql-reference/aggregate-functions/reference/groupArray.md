## groupArray \{#groupArray\}

도입된 버전: v1.1

인수 값으로 이루어진 배열을 생성합니다.
값은 임의의(비결정적인) 순서로 배열에 추가될 수 있습니다.

두 번째 형태(`max_size` 매개변수 사용)는 결과 배열의 크기를 `max_size` 개 요소로 제한합니다. 예를 들어 `groupArray(1)(x)` 는 `[any(x)]` 와 동일합니다.

일부 경우에는 여전히 실행 순서에 의존할 수 있습니다. 이는 서브쿼리 결과가 충분히 작을 때, `ORDER BY` 를 사용하는 서브쿼리에서 `SELECT` 가 수행되는 경우에 해당합니다.

`groupArray` 함수는 결과에서 `NULL` 값을 제거합니다.

**구문**

```sql
groupArray(x)
groupArray(max_size)(x)
```

**별칭**: `array_agg`

**매개변수**

* `max_size` — 선택 사항입니다. 결과 배열의 크기를 `max_size`개의 요소로 제한합니다. [`UInt64`](/sql-reference/data-types/int-uint)

**인수**

* `x` — 배열로 수집할 값입니다. [`Any`](/sql-reference/data-types)

**반환 값**

인수 값으로 구성된 배열을 반환합니다. [`Array`](/sql-reference/data-types/array)

**예제**

**기본 사용법**

```sql title=Query
SELECT id, groupArray(10)(name) FROM default.ck GROUP BY id;
```

```response title=Response
┌─id─┬─groupArray(10)(name)─┐
│  1 │ ['zhangsan','lisi']  │
│  2 │ ['wangwu']           │
└────┴──────────────────────┘
```
