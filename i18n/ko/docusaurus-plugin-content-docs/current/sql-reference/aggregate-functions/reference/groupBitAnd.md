## groupBitAnd \{#groupBitAnd\}

도입 버전: v1.1

일련의 숫자에 비트 단위 AND 연산을 적용합니다.

**구문**

```sql
groupBitAnd(expr)
```

**별칭(Aliases)**: `BIT_AND`

**인수(Arguments)**

* `expr` — `(U)Int*` 타입의 표현식입니다. [`(U)Int*`](/sql-reference/data-types/int-uint)

**반환값(Returned value)**

`(U)Int*` 타입의 값을 반환합니다. [`(U)Int*`](/sql-reference/data-types/int-uint)

**예시(Examples)**

**비트 단위 AND 예시(Bitwise AND example)**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
