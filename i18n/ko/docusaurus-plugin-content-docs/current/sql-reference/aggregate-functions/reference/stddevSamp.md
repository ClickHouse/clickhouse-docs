## stddevSamp \{#stddevSamp\}

도입 버전: v1.1

숫자 데이터 시퀀스의 표본 표준 편차를 반환합니다.
결과는 [`varSamp`](/sql-reference/aggregate-functions/reference/varSamp)의 제곱근과 같습니다.

:::note
이 함수는 수치적으로 불안정한 알고리즘을 사용합니다.
계산에서 [수치적 안정성](https://en.wikipedia.org/wiki/Numerical_stability)이 필요하면 [`stddevSampStable`](/sql-reference/aggregate-functions/reference/stddevsampstable) 함수를 사용하십시오. 동작 속도는 더 느리지만 계산 오차가 더 작습니다.
:::

**구문**

```sql
stddevSamp(x)
```

**별칭(Aliases)**: `STDDEV_SAMP`

**인수(Arguments)**

* `x` — 표본 분산의 제곱근을 구할 값입니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal*`](/sql-reference/data-types/decimal)

**반환 값(Returned value)**

`x`에 대한 표본 분산의 제곱근을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시(Examples)**

**표본 표준편차 계산(Computing sample standard deviation)**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSamp(population)
FROM test_data;
```

```response title=Response
┌─stddevSamp(population)─┐
│                      4 │
└────────────────────────┘
```
