## groupUniqArray \{#groupUniqArray\}

도입 버전: v1.1

서로 다른 인수 값으로부터 배열을 생성합니다.
이 함수의 메모리 사용량은 [`uniqExact`](/sql-reference/aggregate-functions/reference/uniqexact) 함수와 동일합니다.

**구문**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**매개변수**

* `max_size` — 결과 배열의 요소 수를 `max_size`로 제한합니다. `groupUniqArray(1)(x)`는 `[any(x)]`와 동일합니다. [`UInt64`](/sql-reference/data-types/int-uint)

**인수**

* `x` — 표현식입니다. [`Any`](/sql-reference/data-types)

**반환 값**

고유한 값으로 이루어진 배열을 반환합니다. [`Array`](/sql-reference/data-types/array)

**예제**

**사용 예제**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**max&#95;size 매개변수를 사용하는 경우**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
