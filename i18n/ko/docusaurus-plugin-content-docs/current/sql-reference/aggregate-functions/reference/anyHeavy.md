## anyHeavy \{#anyHeavy\}

도입된 버전: v1.1

[heavy hitters](https://doi.org/10.1145/762471.762473) 알고리즘을 사용하여 자주 등장하는 값을 선택합니다.
쿼리의 각 실행 스레드에서 절반이 넘는 경우에 나타나는 값이 있으면 해당 값을 반환합니다.
일반적으로 결과는 비결정적입니다.

**구문**

```sql
anyHeavy(column)
```

**인수**

* `column` — 컬럼 이름입니다. [`String`](/sql-reference/data-types/string)

**반환 값**

자주 나타나는 값을 반환합니다. 결과는 비결정적입니다. [`Any`](/sql-reference/data-types)

**예시**

**사용 예시**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
