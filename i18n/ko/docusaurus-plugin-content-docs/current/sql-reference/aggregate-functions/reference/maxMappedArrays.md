## maxMappedArrays \{#maxMappedArrays\}

도입 버전: v20.5

`key` 배열에 지정된 키에 따라 `value` 배열에서 최댓값을 계산합니다.

:::note

* 키 배열과 값 배열로 이루어진 튜플을 전달하는 것은 키 배열과 값 배열을 각각 전달하는 것과 동일합니다.
* 집계되는 각 행에 대해 `key`와 `value`의 요소 개수는 동일해야 합니다.
  :::

**구문**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**인수**

* `key` — 키 배열. [`Array(T)`](/sql-reference/data-types/array)
* `value` — 값 배열. [`Array(T)`](/sql-reference/data-types/array)

**반환 값**

정렬된 순서의 키 배열과 해당 키에 대해 계산된 값 배열로 구성된 튜플을 반환합니다. [`Tuple(Array(T), Array(T))`](/sql-reference/data-types/tuple)

**예시**

**사용 예시**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)────────────────┐
│ [['x', 'y', 'z'], [2, 3, 1]].        │
└──────────────────────────────────────┘
```
