## avgWeighted \{#avgWeighted\}

도입된 버전: v20.1

[가중 산술 평균](https://en.wikipedia.org/wiki/Weighted_arithmetic_mean)을 계산합니다.

**구문**

```sql
avgWeighted(x, weight)
```

**인수**

* `x` — 값. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float)
* `weight` — 값의 가중치. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float)

**반환 값**

모든 가중치가 0이거나 제공된 가중치 매개변수가 비어 있으면 `NaN`을 반환하고, 그렇지 않으면 가중평균을 반환합니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**사용 예시**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**정수형과 실수형 가중치 혼합**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Float64', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**모든 가중치가 0이면 NaN이 반환됩니다**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (0, 0), (1, 0), (10, 0))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│               nan │
└───────────────────┘
```

**비어 있는 테이블에서는 NaN이 반환됩니다**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;
SELECT avgWeighted(t, t) FROM test
```

```response title=Response
┌─avgWeighted(t, t)─┐
│               nan │
└───────────────────┘
```
