## studentTTest \{#studentTTest\}

도입 버전: v21.1

두 모집단에서 나온 표본에 Student&#39;s t-검정을 적용합니다.

두 표본의 값은 `sample_data` 컬럼에 있습니다. `sample_index`가 0이면 해당 행의 값은 첫 번째 모집단의 표본에 속합니다. 그렇지 않다면 두 번째 모집단의 표본에 속합니다.
영가설은 두 모집단의 평균이 동일하다는 것입니다. 분산이 동일한 정규 분포를 가정합니다.

**함께 보기**

* [Student&#39;s t-검정](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [welchTTest 함수](/sql-reference/aggregate-functions/reference/welchttest)

**구문**

```sql
studentTTest([confidence_level])(sample_data, sample_index)
```

**매개변수**

* `confidence_level` — 옵션입니다. 신뢰 구간을 계산할 때 사용하는 신뢰 수준입니다. [`Float`](/sql-reference/data-types/float)

**인수**

* `sample_data` — 표본 데이터입니다. [`Integer`](/sql-reference/data-types/int-uint) 또는 [`Float`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)
* `sample_index` — 표본 인덱스입니다. [`Integer`](/sql-reference/data-types/int-uint)

**반환 값**

선택적 `confidence_level`이 지정된 경우, 두 개 또는 네 개의 요소로 구성된 튜플을 반환합니다. 반환되는 값은 계산된 t-통계량, 계산된 p-값, [계산된 신뢰 구간 하한], [계산된 신뢰 구간 상한]입니다. [`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple) 또는 [`Tuple(Float64, Float64, Float64, Float64)`](/sql-reference/data-types/tuple)

**예시**

**기본 사용법**

```sql title=Query
SELECT studentTTest(sample_data, sample_index) FROM student_ttest;
```

```response title=Response
│ (-0.21739130434783777,0.8385421208415731) │
└───────────────────────────────────────────┘
```
