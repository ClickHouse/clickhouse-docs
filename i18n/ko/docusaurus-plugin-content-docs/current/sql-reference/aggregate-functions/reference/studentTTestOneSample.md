## studentTTestOneSample \{#studentTTestOneSample\}

도입 버전: v25.10

표본 평균이 알려진 모평균과 다른지 판단하기 위해 단일 표본 Student의 t-검정을 적용합니다.

데이터가 정규 분포를 따른다고 가정합니다. 영가설은 표본 평균이 모평균과 같다는 것입니다.

선택적 인수 `confidence_level`을 사용하면 신뢰 구간을 계산할 수 있습니다.

**참고:**

* 최소 2개의 관측값이 필요하며, 그렇지 않으면 결과는 `(nan, nan)`이 됩니다(요청된 경우 구간 역시 `nan`이 됩니다).
* 입력 값이 상수이거나 상수에 가까운 경우, 표준 오차가 0(또는 사실상 0)이 되어 `nan`을 반환합니다.

**함께 보기**

* [Student의 t-검정](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [studentTTest 함수](/sql-reference/aggregate-functions/reference/studentttest)

**구문**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

**매개변수**

* `confidence_level` — 선택입니다. 신뢰 구간의 신뢰 수준입니다. (0, 1) 범위의 Float입니다. [`Float*`](/sql-reference/data-types/float)

**인자**

* `sample_data` — 표본 데이터입니다. [`Integer`](/sql-reference/data-types/int-uint) 또는 [`Float`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)
* `population_mean` — 검정 대상이 되는 알려진 모평균(보통 상수)입니다. [`(U)Int*`](/sql-reference/data-types/int-uint) 또는 [`Float*`](/sql-reference/data-types/float) 또는 [`Decimal`](/sql-reference/data-types/decimal)

**반환 값**

두 개 또는 네 개의 요소로 이루어진 튜플을 반환합니다(`confidence_level`을 지정한 경우 네 개). 반환 값은 계산된 t-통계량, 계산된 p-값(양측 검정), [계산된 신뢰 구간 하한], [계산된 신뢰 구간 상한]입니다. 신뢰 구간은 주어진 신뢰 수준에서의 표본 평균에 대한 값입니다. [`Tuple(Float64, Float64)`](/sql-reference/data-types/tuple) 또는 [`Tuple(Float64, Float64, Float64, Float64)`](/sql-reference/data-types/tuple)

**예시**

**신뢰 구간 없이**

```sql title=Query
SELECT studentTTestOneSample()(value, 20.0) FROM t;
```
