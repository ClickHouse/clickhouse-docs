## rankCorr \{#rankCorr\}

도입 버전: v20.9

순위 상관 계수를 계산합니다.

x와 y의 순위에 대한 순위 상관 계수를 반환합니다. 상관 계수 값의 범위는 -1부터 +1까지입니다. 인수가 2개 미만으로 전달되면 예외가 발생합니다. 값이 +1에 가까울수록 두 확률 변수 간의 선형 관계가 강하며, 한 확률 변수가 증가하면 다른 확률 변수도 증가합니다. 값이 -1에 가까울수록 두 확률 변수 간의 선형 관계가 강하며, 한 확률 변수가 증가하면 다른 확률 변수는 감소합니다. 값이 0에 가깝거나 0과 같으면 두 확률 변수 사이에 관계가 없음을 의미합니다.

**함께 보기**

* [Spearman의 순위 상관 계수](https://en.wikipedia.org/wiki/Spearman%27s_rank_correlation_coefficient)

**구문**

```sql
rankCorr(x, y)
```

**인수**

* `x` — 임의의 값. [`Float*`](/sql-reference/data-types/float)
* `y` — 임의의 값. [`Float*`](/sql-reference/data-types/float)

**반환 값**

x와 y의 순위 간 순위 상관계수를 반환합니다. 값의 범위는 -1에서 +1 사이입니다. [`Float64`](/sql-reference/data-types/float)

**예시**

**완전한 상관**

```sql title=Query
SELECT rankCorr(number, number) FROM numbers(100);
```

```response title=Response
┌─rankCorr(number, number)─┐
│                        1 │
└──────────────────────────┘
```

**비선형 관계**

```sql title=Query
SELECT roundBankers(rankCorr(exp(number), sin(number)), 3) FROM numbers(100);
```

```response title=Response
┌─roundBankers(rankCorr(exp(number), sin(number)), 3)─┐
│                                              -0.037 │
└─────────────────────────────────────────────────────┘
```
