이 가이드는 Helm 차트를 사용하여 ClickHouse Operator를 설치하는 방법을 설명합니다.

## 사전 요구 사항 \{#prerequisites\}

- Kubernetes 클러스터 v1.33.0 이상
- Helm v3.0 이상
- 클러스터와 통신하도록 설정된 kubectl

## Helm 설치 \{#install-helm\}

Helm이 설치되어 있지 않은 경우:

```bash
curl https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3 | bash
```

설치를 확인합니다:

```bash
helm version
```


## Operator 설치 \{#install-the-operator\}

:::note
Helm 차트는 기본적으로 webhook이 활성화된 ClickHouse Operator를 배포하며, cert-manager가 설치되어 있어야 합니다.
:::

### OCI helm 저장소에서 설치 \{#from-oci-helm-repository\}

최신 릴리스를 설치하십시오

```bash
    helm install clickhouse-operator oci://ghcr.io/clickhouse/clickhouse-operator-helm \
       --create-namespace \
       -n clickhouse-operator-system
```

특정 Operator 버전 설치

```bash
    helm install clickhouse-operator oci://ghcr.io/clickhouse/clickhouse-operator-helm \
       --create-namespace \
       -n clickhouse-operator-system \
       --set-json="manager.container.tag=<operator version>
```


### 로컬 차트에서 설치 \{#from-local-chart\}

리포지토리를 클론한 후 로컬 차트에서 설치합니다.

```bash
git clone https://github.com/ClickHouse/clickhouse-operator.git
cd clickhouse-operator
helm install clickhouse-operator ./dist/chart
```


### 구성 옵션 \{#configuration-options\}

고급 구성 옵션에 대해서는 Helm 차트에 포함된 [values.yaml](https://github.com/ClickHouse/clickhouse-operator/blob/main/dist/chart/values.yaml) 파일을 참조하십시오.
