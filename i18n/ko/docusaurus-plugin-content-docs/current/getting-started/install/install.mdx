직접 관리하려면 아래의 오픈소스 ClickHouse 옵션 중 하나를 선택하세요.
로컬 환경은 빠른 설치 스크립트 또는 `clickhousectl` CLI로 설정할 수 있으며, 프로덕션 배포용으로는
패키지 또는 바이너리에서 ClickHouse 서버를 설치할 수 있습니다.

<h3 className="install-group-heading">로컬 개발</h3>

<p className="install-group-description">
  자체 머신에서 ClickHouse를 실행하여 개발, 테스트, 실험을 진행하세요.
  빠른 설치를 사용하면 단일 바이너리를 신속하게 설치할 수 있으며, clickhousectl을 사용하면
  로컬 버전과 서버를 설치하고 관리할 수 있습니다.
</p>
