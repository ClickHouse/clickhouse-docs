이 데이터셋에는 2011년부터 2020년 12월 6일까지 GitHub에서 발생한 모든 이벤트가 포함되어 있으며, 규모는 31억 개 레코드입니다. 다운로드 용량은 75GB이며, lz4 압축을 사용하는 테이블에 저장하는 경우 디스크에 최대 200GB의 공간이 필요합니다.

전체 데이터셋에 대한 설명, 인사이트, 다운로드 방법 및 대화형 쿼리는 이 [저장소](https://github.com/ClickHouse/github-explorer)에 게시되어 있습니다.
