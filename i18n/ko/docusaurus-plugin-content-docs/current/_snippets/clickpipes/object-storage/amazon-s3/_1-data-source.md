**2.** **Amazon S3** 타일을 클릭합니다. 이 타일을 사용하여 ClickPipes UI에 표시되지 않은 다른 S3 호환 서비스에도 연결할 수 있습니다.

<Image img={cp_step1} alt="Select imports" size="lg" border />

:::tip
객체 스토리지 서비스 제공 업체마다 URL 형식과 API 구현이 다르기 때문에, 모든 S3 호환 서비스가 기본적으로 지원되는 것은 아닙니다. [지원되는 데이터 소스](/integrations/clickpipes/object-storage/s3/overview#supported-data-sources)에 나와 있지 않은 서비스를 사용 중 문제가 발생하는 경우, [당사 팀에 문의](https://clickhouse.com/company/contact?loc=clickpipes)해 주십시오.
:::
