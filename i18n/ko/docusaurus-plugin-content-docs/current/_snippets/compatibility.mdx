:::note 호환성
셀프 호스팅 ClickHouse 배포 환경과 ClickHouse Cloud 서비스 간에 동작에 차이가 느껴진다면,
[compatibility setting](/operations/settings/settings#compatibility)과 관련이 있을 수 있습니다.
Cloud에서는 서비스 생성 시 compatibility가 설정되며, 서비스가 업그레이드되더라도 클라이언트 입장에서 동작이 일관되도록 서비스 단위에서는 변경되지 않습니다.
compatibility를 변경하려는 경우 [support](https://clickhouse.com/support/program)를 통해 요청할 수 있습니다.
:::
